Find $a$ and $b$ such that $\displaystyle \lim_{x \to 1} \frac{x^2 + ax + b}{x - 1} = 3$.

For the limit to exist, the numerator must be $0$ at $x = 1$:

$1 + a + b = 0$, so $b = -1 - a$.

Then $x^2 + ax + b = x^2 + ax - 1 - a$. Since $x = 1$ is a root, factor: $x^2 + ax - 1 - a = (x - 1)(x + 1 + a)$.

$\dfrac{(x - 1)(x + 1 + a)}{x - 1} = x + 1 + a$ for $x \neq 1$.

$\lim\limits_{x \to 1} (x + 1 + a) = 2 + a = 3$, so $a = 1$.

Then $b = -1 - 1 = -2$.

Answer: $a = 1$, $b = -2$.
