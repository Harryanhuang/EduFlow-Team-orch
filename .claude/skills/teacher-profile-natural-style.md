# 老师资料写作风格指南

## 什么时候用

- 撰写新老师的 `class_style`（教学风格）
- 撰写或润色老师的「教学特色」「个人简介」
- 从飞书表单 / 图片信息整理成 Obsidian Markdown 时，需要生成教学风格段落
- 审核已有老师资料，判断 class_style 是否符合标准

## 风格原则

**核心定位：专业正式，适合对外介绍（面向家长和学生）。既不要 AI 腔，也不要太口语化。**

目标读感：像一位资深老师在正式场合介绍自己的教学方法和特点——有专业度，但不生硬；有人味，但不随意。

### 三条红线

| 红线 | 表现 | 处理 |
|------|------|------|
| **太 AI** | 具象化、内驱力、批判性思维培养、赋能、底层逻辑、痛点、精准把握 | 全部替换 |
| **太口语** | "话不多""讲得很细""觉得物理没那么可怕""学生都挺喜欢他" | 改为正式表达 |
| **不真实** | 写"课堂气氛活跃"但老师实际沉稳细致；写"善于互动"但老师以板书见长 | 必须真实反映老师特点，不能编造 |

### 三档对比（同一意思的三种写法）

| 含义 | 专业正式 ✅ | 太口语 ❌ | 太 AI 化 ❌ |
|------|-----------|---------|-----------|
| 板书好 | 板书规范工整，阅读性强 | 板书写得很好看 | 板书呈现结构化视觉输出 |
| 讲课清楚 | 讲解条理清晰 | 讲得很细，学生都听得懂 | 知识传递链路清晰高效 |
| 有耐心 | 授课细致耐心 | 话不多，就是慢慢教 | 以耐心陪伴为教学基石 |
| 帮学生理解 | 帮助学生建立系统的学科思维 | 觉得物理没那么可怕了 | 赋能学生建构认知框架 |
| 学生有进步 | 帮助学生高效备考 | 成绩提了不少 | 实现学业表现的显著提升 |
| 了解学生 | 从学生学情出发 | 跟学生聊聊就知道问题在哪 | 基于学情诊断的精准干预 |

### 禁用词清单

以下词汇 **不要出现在 class_style 或教学特色中**：

| 禁用词 | 替代表达 |
|--------|---------|
| 具象化 | 用例子讲清楚、让抽象变具体 |
| 内驱力 | 学习兴趣、主动性 |
| 批判性思维培养 | 引导学生独立思考 |
| 从被动接受转向主动探究 | 鼓励学生多问多想 |
| 精准把握 | 熟悉、了解清楚 |
| 灵活运用 | 会用、会结合 |
| 知识体系构建 | 梳理知识点、理清框架 |
| 系统化知识体系 | 知识框架、知识逻辑 |
| 差异化教学策略 | 因材施教、根据学生情况调整 |
| 情境化教学 | 结合生活实际、用例子讲 |
| 赋能 | 帮助、提升 |
| 闭环 | — |
| 沉浸式 | — |
| 深耕 | 做了多年、经验丰富 |
| 多维度 | 多方面、多个角度 |
| 底层逻辑 | 基本规律、核心方法 |
| 痛点 | 薄弱环节、困难点 |
| 抓手 | — |
| 颗粒度 | — |
| 课堂气氛活跃 | （仅在老师确实活泼时用） |
| 学生都挺喜欢 | 学生反馈良好 |
| 讲得很细 | 讲解细致、条理清晰 |
| 话不多 | 授课风格沉稳 |

### 字数范围

| 标准 | 字数 | 说明 |
|------|------|------|
| **目标范围** | **210–220 字** | 钟老师Winters 标准：专业正式、细节丰富、面向家长学生 |
| 可接受范围 | 170–250 字 | 低于 210 字需要补充具体细节；超过 220 字需要精简冗余 |

> 210–220 字是因为这个长度能完整呈现一位老师的教学风格、方法和特点，同时保持紧凑专业。

### 独特性要求

每位老师的 class_style 必须**独特，不能互换**。

- 不要写成通用模板套在所有老师身上
- 要体现这位老师的真实特点（如钟老师沉稳细致、板书好；Renee 亲切、善于倾听、喜欢用生活例子）
- 写完自问：把名字遮住，能不能看出这是哪一位老师？如果不能，需要重写

## 写作模板

按这个结构写 class_style：

```
[开头：风格定位一句话] + [方法：板书/课堂/知识梳理，2-3 句] + [特色：针对不同类型学生的做法，1-2 句] + [成果：备考/竞赛/实验等，1 句]
```

### 结构拆解

1. **第一句（风格定位）**：「教学风格___」或「擅长___」——定调
2. **第二句（方法—课堂）**：具体教学方法（板书、节奏、条理）
3. **第三句（方法—知识）**：如何梳理重难点、串联知识
4. **第四句（方法—个性化）**：针对某类学生的特别做法
5. **第五句（方法—具体场景）**：某个学科或教学场景的具体做法
6. **第六句（成果）**：备考、提分、竞赛等实际成果

## 参照示例

### 标杆示例：钟老师Winters（219 字，专业正式标准）

> 教学风格沉稳细致，授课节奏稳，讲解条理清晰。板书规范工整，阅读性强，学生反映课堂笔记容易整理和复习。善于梳理学科重难点，将零散的知识点串联成完整的知识框架，帮助学生建立系统的化学学科思维。针对竞赛学生，注重错题本的系统管理，每周进行知识盲区复盘，从错题中归纳总结规律。有机化学教学中注重引导学生先预测产物再验证结果。授课细致耐心，会手把手指导学生完成实验报告，从格式规范到数据分析逐一审核，考前系统梳理历年真题及常考点，帮助学生高效备考。

**为什么好**：
- 开头定调准确（"沉稳细致"），后面每一句都呼应这个特点
- 有具体方法（"板书规范工整""错题本系统管理""先预测产物再验证"）
- 有真实场景（"有机化学教学""实验报告""历年真题"）
- 没有禁用词，没有过度口语化
- 这段文字放在任何其他老师身上都不成立——独特性强

### 对比示例：太口语化 ❌

> 老师人很好，话不多但是讲得很细。学生都觉得物理没那么可怕了，成绩提了不少。

**问题**：太随意，不像对外介绍材料，像朋友闲聊。

### 对比示例：太 AI 化 ❌

> 以赋能学生建构认知框架为教学理念，通过差异化教学策略和情境化教学设计，实现学生学业表现的显著提升。

**问题**：全是 buzzword，读起来像教研论文摘要，没有人味。

## 使用前必读

写任何老师的 class_style 前：

1. 先读本文件，记住三条红线和禁用词清单
2. 先读该老师的 personal_intro 和基本信息，提取 3–5 个具体细节
3. 确认这位老师的真实特点（沉稳/活泼/耐心/严谨等），不要编造
4. 按上面六步结构起草
5. 写完后逐词扫描：有没有禁用词？有没有太口语的表达？有没有不真实的描述？
6. 数一下字数，不在 210–220 字范围内的适当增删
7. 做"遮名测试"：把老师名字遮住，读起来能认出这是哪位老师吗？如果不能，说明太通用，需要增加独特性
