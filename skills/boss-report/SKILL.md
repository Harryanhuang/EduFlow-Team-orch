# Boss Report — AI Employee Work Summary Skill

Source: https://github.com/Lord1Egypt/awesome-skill-forge/blob/main/community/clawhub/b/boss-report/SKILL.md

This local copy is installed as a reference for report structure only. `Luke_recorder` should use `boss-manager-recorder` for daily recording and use this skill when a daily, weekly, or monthly summary is explicitly requested.

## Useful Report Shape

```markdown
# Work Report — {Date}

## Overview
Brief summary of what changed.

## Completed
- Finished work items and evidence.

## In Progress
- Active work and current owner.

## Issues
- Blockers, repeated mistakes, and unresolved risks.

## Learnings
- Reusable habits, manager corrections, and colleague skill patterns.

## Next
- Next likely follow-up, without inventing commitments.
```

Keep reports concise. Prefer evidence and distilled lessons over activity narration.
