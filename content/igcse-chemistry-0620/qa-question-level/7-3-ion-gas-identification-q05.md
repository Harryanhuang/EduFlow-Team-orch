# Question: Question Q-7.3-05
**Difficulty**: Standard
**Topic**: 7.3
**Question**: How would you distinguish between iron(II) and iron(III) ions using aqueous sodium hydroxide?
**Answer**: Add aqueous NaOH. Iron(II) ions form a green precipitate of Fe(OH)₂. Iron(III) ions form a brown/orange-brown precipitate of Fe(OH)₃. Neither precipitate dissolves in excess NaOH.
**Explanation**: Green vs brown. A common error is confusing which colour corresponds to which ion.
