# Question: Question Q-11.3-05
**Difficulty**: Standard
**Topic**: 11.3
**Question**: What is formed when ethanol reacts with ethanoic acid? Name the type of reaction.
**Answer**: Ethyl ethanoate (an ester) and water. This is an esterification (condensation) reaction: CH₃COOH + C₂H₅OH → CH₃COOC₂H₅ + H₂O (catalysed by concentrated sulfuric acid).
**Explanation**: Acid + alcohol → ester + water. A common error is saying a salt is formed.
