# QQL: 2.2 — Q-2.2-08 (C)

**Question ID**: Q-2.2-08
**Difficulty**: Challenge
**Topic**: 2.2 Periodic Table configuration
**Question**: Explain why the chemical properties of elements in the same group are similar. Use Group I (alkali metals) as an example.
**Answer**: Elements in the same group have the same number of electrons in their outer shell. Chemical reactions involve the loss, gain, or sharing of these outer electrons. Since all Group I elements have 1 outer electron that is easily lost, they all react in similar ways (e.g. vigorously with water to form +1 ions).
**Explanation**: The key is linking outer electron count to chemical behaviour. A common error is saying "they have similar atomic structure" without specifying the outer electrons.
