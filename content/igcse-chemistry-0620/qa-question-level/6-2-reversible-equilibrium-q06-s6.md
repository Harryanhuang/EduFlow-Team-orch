# QQL: 6.2 — Q-6.2-06 (S)

**Question ID**: Q-6.2-06
**Difficulty**: Standard
**Topic**: 6.2 Reversible reactions, equilibrium
**Question**: In the reaction Fe³⁺(aq) + SCN⁻(aq) ⇌ FeSCN²⁺(aq), the product FeSCN²⁺ is blood-red. Describe how you would use this equilibrium to demonstrate Le Chatelier's principle experimentally.
**Answer**: Set up: mix dilute FeCl3 and KSCN solutions to establish equilibrium (pale red colour).
Test 1: Add more Fe³⁺ — colour deepens (equilibrium shifts right).
Test 2: Add more SCN⁻ — colour deepens (shifts right).
Test 3: Add Ag⁺ (precipitates SCN⁻) — colour fades (shifts left).
Each change demonstrates the system responding to counteract the disturbance.
**Explanation**: Students must describe a practical experiment with predictions. A common error is not identifying what would be observed.
