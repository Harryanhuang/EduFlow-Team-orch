# Question: Question Q-3.1-03
**Difficulty**: Standard
**Topic**: 3.1
**Question**: Balance the equation: Fe + O₂ → Fe₂O₃
**Answer**: 4Fe + 3O₂ → 2Fe₂O₃
**Explanation**: Iron: 4 on each side. Oxygen: 6 on each side (3×2 = 6, 2×3 = 6). A common error is 2Fe + 3O₂ → Fe₂O₃ (oxygen unbalanced).
