# Question: Question Q-12.1-03
**Difficulty**: Standard
**Topic**: 12.1
**Question**: What is the purpose of using a locating agent in chromatography?
**Answer**: A locating agent makes colourless substances visible on a chromatogram, allowing the positions of the spots to be identified and Rf values to be calculated.
**Explanation**: Makes invisible spots visible. A common error is saying it "reacts with the solvent."
