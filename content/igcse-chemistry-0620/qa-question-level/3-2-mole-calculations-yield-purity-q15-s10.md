# Question: Question Q-3.2-15
**Difficulty**: Foundation
**Topic**: 3.2
**Question**: State Avogadro's Law.
**Answer**: Equal volumes of gases at the same temperature and pressure contain equal numbers of molecules (or moles).
**Explanation**: Same V, T, P → same number of molecules. A common error is saying equal masses.
