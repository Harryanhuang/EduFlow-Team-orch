# Question: Question Q-3.1-06
**Difficulty**: Challenge
**Topic**: 3.1
**Question**: What is one mole?
**Answer**: One mole is the amount of substance that contains the same number of particles (atoms, molecules, or ions) as there are atoms in exactly 12 g of carbon-12. This number is 6.02 × 10²³ (Avogadro's constant).
**Explanation**: Fixed number of particles. A common error is saying "one mole = one gram."
