# Question: Question Q-7.2-01
**Difficulty**: Standard
**Topic**: 7.2
**Question**: Which method of salt preparation would you use for sodium chloride? Explain why.
**Answer**: Titration (acid + soluble alkali). Both sodium hydroxide and hydrochloric acid are soluble, so titration is needed to find the exact endpoint. After titration, the solution is evaporated and crystallised.
**Explanation**: Soluble base + soluble acid → titration. A common error is using excess base method (that's for insoluble bases).
