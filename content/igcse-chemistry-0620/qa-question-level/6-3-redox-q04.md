# Question: Question Q-6.3-04
**Difficulty**: Standard
**Topic**: 6.3
**Question**: What is an oxidising agent? What is a reducing agent?
**Answer**: An oxidising agent is a substance that causes oxidation of another substance — it itself is reduced (gains electrons / loses oxygen). A reducing agent causes reduction of another substance — it itself is oxidised (loses electrons / gains oxygen).
**Explanation**: Oxidising agent = gets reduced. Reducing agent = gets oxidised. A common error is thinking the oxidising agent is oxidised.
