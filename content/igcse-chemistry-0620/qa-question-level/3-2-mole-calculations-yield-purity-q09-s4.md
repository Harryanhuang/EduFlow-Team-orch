# Question: Question Q-3.2-09
**Difficulty**: Standard
**Topic**: 3.2
**Question**: 25.0 cm³ of 0.05 mol/dm³ H₂SO₄ neutralises 20.0 cm³ of KOH. Calculate the concentration of KOH.
**Answer**: 0.125 mol/dm³
**Explanation**: Moles of H₂SO₄ = 0.05 × 0.025 = 0.00125 mol. H₂SO₄ + 2KOH → K₂SO₄ + 2H₂O, so 1:2 ratio. Moles of KOH = 2 × 0.00125 = 0.0025 mol. Concentration = 0.0025/0.020 = 0.125 mol/dm³. A common error is using 1:1 ratio.
