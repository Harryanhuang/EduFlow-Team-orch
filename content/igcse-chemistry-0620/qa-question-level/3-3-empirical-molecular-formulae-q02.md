# Question: Question Q-3.3-02
**Difficulty**: Standard
**Topic**: 3.3
**Question**: The empirical formula of a compound is CH₂O and its Mᵣ is 60. Calculate its molecular formula. (Aᵣ: C = 12, H = 1, O = 16)
**Answer**: C₂H₄O₂
**Explanation**: Empirical formula mass = 12 + 2 + 16 = 30. n = 60/30 = 2. Molecular formula = (CH₂O) × 2 = C₂H₄O₂. A common error is using the empirical formula as the molecular formula.
