# Question: Question Q-9.1-06
**Difficulty**: Standard
**Topic**: 9.1
**Question**: Describe the uses of brass, bronze, and stainless steel, relating each to its properties.
**Answer**: Brass (Cu + Zn): malleable, attractive → musical instruments, decorative items. Bronze (Cu + Sn): hard, corrosion-resistant → statues, bearings, ship propellers. Stainless steel (Fe + Cr + Ni): strong, corrosion-resistant → cutlery, surgical instruments, kitchen sinks.
**Explanation**: Properties → uses for each. A common error is swapping brass and bronze compositions.
