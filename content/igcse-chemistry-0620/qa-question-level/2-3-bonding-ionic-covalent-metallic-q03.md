# Question: Question Q-2.3-03
**Difficulty**: Foundation
**Topic**: 2.3
**Question**: What is a covalent bond?
**Answer**: A covalent bond is a shared pair of electrons between two non-metal atoms. Each atom contributes one electron to the shared pair.
**Explanation**: Sharing between non-metals. A common error is saying covalent bonds involve ions.
