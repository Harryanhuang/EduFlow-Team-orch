# Question: Question Q-3.1-02
**Difficulty**: Foundation
**Topic**: 3.1
**Question**: Calculate the percentage by mass of nitrogen in ammonium nitrate (NH₄NO₃). (Aᵣ: N = 14, H = 1, O = 16)
**Answer**: 35%
**Explanation**: Mᵣ = 14 + (4×1) + 14 + (3×16) = 80. Total N mass = 2 × 14 = 28. %N = (28/80) × 100 = 35%. A common error is using only one nitrogen atom.
