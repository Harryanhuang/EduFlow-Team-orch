# Question: Question Q-1.2-06
**Difficulty**: Standard
**Topic**: 1.2
**Question**: What is the Rf value in chromatography? Calculate it for a substance that moves 4 cm from the baseline when the solvent front travels 8 cm.
**Answer**: Rf = distance moved by substance / distance moved by solvent front. Rf = 4/8 = 0.5. Rf values are always between 0 and 1.
**Explanation**: Dimensionless ratio. A common error is swapping numerator and denominator.
