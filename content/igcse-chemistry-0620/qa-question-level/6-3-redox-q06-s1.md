# QQL: 6.3 — Q-6.3-06 (S)

**Question ID**: Q-6.3-06
**Difficulty**: Standard
**Topic**: 6.3 Redox
**Question**: Identify which substance is oxidised and which is reduced in each reaction: (a) 2Fe2O3 + 3C → 4Fe + 3CO2 (b) CuO + H2 → Cu + H2O. Explain your answers in terms of oxygen transfer.
**Answer**: (a) Carbon is oxidised (gains oxygen → CO2). Iron(III) oxide is reduced (loses oxygen → Fe).
(b) Hydrogen is oxidised (gains oxygen → H2O). Copper(II) oxide is reduced (loses oxygen → Cu).
Oxidation is gain of oxygen; reduction is loss of oxygen.
**Explanation**: Students must identify both processes in each reaction. A common error is only identifying oxidation or only reduction.
