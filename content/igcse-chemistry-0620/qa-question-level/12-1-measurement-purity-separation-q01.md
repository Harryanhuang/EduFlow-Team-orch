# Question: Question Q-12.1-01
**Difficulty**: Standard
**Topic**: 12.1
**Question**: How can you determine whether a substance is pure using chromatography?
**Answer**: A pure substance produces a single spot on a chromatogram. If there are multiple spots, the substance is a mixture.
**Explanation**: One spot = pure. A common error is saying "more spots = purer."
