# Question: Question Q-4.2-09
**Difficulty**: Standard
**Topic**: 4.2
**Question**: Describe how electroplating works. Give an example.
**Answer**: The object to be plated is the cathode. The plating metal is the anode. The electrolyte contains ions of the plating metal. At the cathode: metal ions gain electrons and deposit as a solid layer. Example: silver plating — silver anode, object as cathode, silver nitrate solution as electrolyte. Ag⁺ + e⁻ → Ag at cathode.
**Explanation**: Object = cathode, plating metal = anode. A common error is reversing the electrodes.
