# Question: Question Q-6.4-04
**Difficulty**: Standard
**Topic**: 6.4
**Question**: Explain why enzymes are specific to particular reactions.
**Answer**: Each enzyme has a uniquely shaped active site that only fits one specific substrate (lock and key model). Only the correct substrate can bind to the active site and react. This means each enzyme catalyses only one type of reaction.
**Explanation**: Active site shape matches specific substrate. A common error is saying enzymes "choose" which reaction to catalyse.
**Tags**: enzyme, specificity, lock-and-key
