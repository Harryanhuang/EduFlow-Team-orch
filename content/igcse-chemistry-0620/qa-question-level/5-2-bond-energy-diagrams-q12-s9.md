# Question: Question Q-5.2-12
**Difficulty**: Standard
**Topic**: 5.2
**Question**: Use the data to calculate the C-Cl bond energy in CH₄ + Cl₂ → CH₃Cl + HCl, given ΔH = -101 kJ/mol. C-H = 413, Cl-Cl = 242, C-Cl = ?, H-Cl = 431 (kJ/mol).
**Answer**: 327 kJ/mol
**Explanation**: Bonds broken: 4×C-H + Cl-Cl = 1652 + 242 = 1894. Bonds made: 3×C-H + C-Cl + H-Cl = 1239 + C-Cl + 431 = 1670 + C-Cl. ΔH = 1894 - (1670 + C-Cl) = -101. C-Cl = 1894 - 1670 + 101 = 325 kJ/mol. Actually: 1894 - 1670 = 224; 224 - C-Cl = -101; C-Cl = 224 + 101 = 325 kJ/mol.
