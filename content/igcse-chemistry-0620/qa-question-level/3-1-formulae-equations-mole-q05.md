# Question: Question Q-3.1-05
**Difficulty**: Standard
**Topic**: 3.1
**Question**: Calculate the number of moles in 4.4 g of carbon dioxide (CO₂). (Aᵣ: C = 12, O = 16)
**Answer**: 0.1 mol
**Explanation**: Mᵣ of CO₂ = 12 + (2×16) = 44. Moles = mass / Mᵣ = 4.4 / 44 = 0.1 mol. A common error is using Mᵣ = 28.
