# QQL: 7.2 — Q-7.2-07 (S)

**Question ID**: Q-7.2-07
**Difficulty**: Standard
**Topic**: 7.2 Salt preparation
**Question**: Explain why potassium sulfate can be prepared by titration of potassium hydroxide with sulfuric acid, but copper(II) sulfate cannot. Describe the titration method for potassium sulfate.
**Answer**: Potassium hydroxide is a soluble base (alkali), so it can be titrated with acid using an indicator. Copper(II) oxide is insoluble, so titration is not possible.
Titration method: 1. Add measured volume of KOH to a flask with indicator. 2. Add H2SO4 from burette until colour changes. 3. Repeat without indicator. 4. Evaporate solution to crystallise potassium sulfate.
**Explanation**: Students must link solubility of the base to the choice of method. A common error is suggesting titration with an insoluble base.
