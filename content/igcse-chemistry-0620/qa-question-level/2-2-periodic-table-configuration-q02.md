# Question: Question Q-2.2-02
**Difficulty**: Standard
**Topic**: 2.2
**Question**: How many electrons are in the outer shell of an element in Group 7 (the halogens)?
**Answer**: 7 electrons.
**Explanation**: Group number = number of valence electrons (for Groups 1-7). A common error is saying 1 electron (confusing with Group 1).
