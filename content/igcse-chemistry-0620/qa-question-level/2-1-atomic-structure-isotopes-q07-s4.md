# Question: Question Q-2.1-07
**Difficulty**: Standard
**Topic**: 2.1
**Question**: What are isotopes? Explain why isotopes of the same element have the same chemical properties but different physical properties.
**Answer**: Isotopes are atoms of the same element with the same number of protons but different numbers of neutrons. They have the same chemical properties because chemical behaviour is determined by the number of electrons (which equals protons). They have different physical properties (e.g., mass, density) because they have different numbers of neutrons.
**Explanation**: Same protons/electrons = same chemistry; different neutrons = different mass. A common error is saying isotopes have different numbers of electrons.
