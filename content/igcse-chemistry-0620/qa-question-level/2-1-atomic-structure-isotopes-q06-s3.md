# Question: Question Q-2.1-06
**Difficulty**: Standard
**Topic**: 2.1
**Question**: An atom has 17 protons, 18 neutrons, and 17 electrons. State its atomic number, mass number, and electronic configuration.
**Answer**: Atomic number = 17 (number of protons). Mass number = 17 + 18 = 35. Electronic configuration = 2,8,7 (or 2.8.7). This is chlorine-35.
**Explanation**: Atomic number = protons; Mass number = protons + neutrons; Config = shells from innermost. A common error is reversing atomic and mass number.
