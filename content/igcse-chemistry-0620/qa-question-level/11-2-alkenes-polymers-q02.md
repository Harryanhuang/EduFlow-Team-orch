# Question: Question Q-11.2-02
**Difficulty**: Foundation
**Topic**: 11.2
**Question**: How can you distinguish between an alkane and an alkene using bromine water?
**Answer**: Add bromine water to each. Alkenes decolourise bromine water (orange-brown → colourless) because they undergo an addition reaction with bromine. Alkanes do not react, so the colour stays orange-brown.
**Explanation**: Bromine water test for unsaturation. A common error is saying alkanes decolourise bromine.
