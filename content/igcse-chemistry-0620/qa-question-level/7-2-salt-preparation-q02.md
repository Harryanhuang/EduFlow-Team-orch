# Question: Question Q-7.2-02
**Difficulty**: Foundation
**Topic**: 7.2
**Question**: Which method of salt preparation would you use for copper(II) sulfate? Explain why.
**Answer**: Excess base method (acid + insoluble base). Copper(II) oxide is insoluble in water, so it is added in excess to ensure all the acid is neutralised. The excess is filtered off.
**Explanation**: Insoluble base → excess method. A common error is using titration.
