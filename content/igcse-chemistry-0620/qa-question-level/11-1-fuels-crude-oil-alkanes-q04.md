# Question: Question Q-11.1-04
**Difficulty**: Standard
**Topic**: 11.1
**Question**: Write the balanced equation for the complete combustion of propane (C₃H₈).
**Answer**: C₃H₈ + 5O₂ → 3CO₂ + 4H₂O
**Explanation**: Carbon: 3 on each side. Hydrogen: 8 on each side. Oxygen: 10 on each side (5×2 = 3×2 + 4). A common error is using 4O₂ instead of 5O₂.
