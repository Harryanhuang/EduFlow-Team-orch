# QQL: 2.2 — Q-2.2-07 (C)

**Question ID**: Q-2.2-07
**Difficulty**: Challenge
**Topic**: 2.2 Periodic Table configuration
**Question**: An element X has the electronic configuration 2,8,5. (a) Which group and period is it in? Explain your reasoning. (b) Is it a metal or non-metal? (c) What type of bonding would you expect in the compound it forms with chlorine?
**Answer**: (a) Group V (5 valence electrons), Period 3 (3 electron shells). The number of valence electrons = group number; the number of shells = period number.
(b) Non-metal — elements on the right side of the Periodic Table with 5 valence electrons are non-metals (this is phosphorus).
(c) Covalent bonding — both X and chlorine are non-metals, so they share electrons.
**Explanation**: Students must link electronic configuration to position. Common errors: confusing group number with total electrons, or saying ionic because "it has electrons to lose".
