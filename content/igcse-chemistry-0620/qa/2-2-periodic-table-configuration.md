# QA: 2.2 — The Periodic Table, electronic configuration

## Topic 名称
The Periodic Table, electronic configuration（元素周期表，电子排布）

## Topic 定义
理解元素周期表的结构（周期、族），电子排布与元素位置的关系，金属与非金属的区分。

## 关键知识点
- 周期表按原子序数排列
- 周期（横行）数 = 电子层数
- 族（纵列）数 = 最外层电子数（主族元素）
- 同一族元素有相似的化学性质（最外层电子数相同）
- 同一周期从左到右：金属性减弱，非金属性增强
- 金属在左边和中间，非金属在右上方
- 电子排布与周期表位置的互推
- 0 族/第 18 族（稀有气体）：最外层电子满
- 元素分类：金属、非金属、半金属（准金属）

## 前置知识
- 2.1 原子结构、电子排布

## 常见错误
- 周期数和族数搞反
- 认为周期表按原子量排列（现代周期表按原子序数）
- 混淆最外层电子数和总电子数
- 认为同一周期性质相同（同一族才相似）
- 电子排布与位置互推时计算错误

## 可出题方向
- 给定电子排布推断元素在周期表中的位置
- 给定位置写出电子排布
- 解释同族元素性质相似的原因
- 判断元素是金属还是非金属
- 周期表中位置与性质的关系
