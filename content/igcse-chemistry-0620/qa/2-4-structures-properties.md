# QA: 2.4 — Structures and properties

## Topic 名称
Structures and properties（结构与性质）

## Topic 定义
理解物质的微观结构（离子晶体、简单分子、巨型共价、金属晶体）如何决定其宏观物理性质。

## 关键知识点
- 离子晶体：离子排列成规则晶格，强静电作用 → 高熔沸点、硬脆、固态不导电/液态或溶液导电
- 简单分子：分子间弱作用力（范德华力）→ 低熔沸点、通常不导电
- 巨型共价结构：原子间共价键网络 → 极高熔沸点、通常不导电（石墨除外）
- 金刚石：每个 C 原子与 4 个 C 形成共价键 → 极硬
- 石墨：每个 C 原子与 3 个 C 成键，有自由电子 → 导电、润滑性
- C60 富勒烯：球形分子，分子间范德华力 → 较低熔点
- 金属晶体：阳离子 + 自由电子 → 导电、延展、光泽
- 合金：不同大小原子混合 → 滑移受阻 → 更硬

## 常见错误
- 混淆分子内作用力和分子间作用力
- 认为熔化破坏了共价键（简单分子熔化只破坏分子间作用力）
- 认为石墨的导电是因为有离子
- 合金比纯金属软（相反，合金更硬）
- 认为 C60 和金刚石一样硬

## 可出题方向
- 根据结构解释物理性质
- 比较不同结构的熔沸点
- 解释为什么某物质导电或不导电
- 合金的性质和应用
- 新材料（如纳米材料、石墨烯）的性质推断
