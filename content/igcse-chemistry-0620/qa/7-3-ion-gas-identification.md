# QA: 7.3 — Identification of ions and gases

## Topic 名称
Identification of ions and gases（离子和气体的鉴定）

## Topic 定义
掌握常见阳离子、阴离子和气体的化学鉴定方法和实验现象。

## 关键知识点
- 阳离子鉴定（NaOH 溶液）：
  - Cu²⁺ → 蓝色沉淀（不溶于过量 NaOH）
  - Fe²⁺ → 绿色沉淀 → 空气中变棕
  - Fe³⁺ → 红棕沉淀
  - Al³⁺ → 白色沉淀（溶于过量 NaOH → 无色溶液）
  - Zn²⁺ → 白色沉淀（溶于过量 NaOH → 无色溶液）
  - Ca²⁺ → 白色沉淀（不溶于过量 NaOH）
  - NH₄⁺ → 加热产生 NH₃（使湿润红色石蕊变蓝）
- 阴离子鉴定：
  - CO₃²⁻ → 加酸产生 CO₂（使石灰水变浑浊）
  - Cl⁻ → AgNO₃ + HNO₃ → 白色沉淀
  - Br⁻ → AgNO₃ + HNO₃ → 乳黄色沉淀
  - I⁻ → AgNO₃ + HNO₃ → 黄色沉淀
  - SO₄²⁻ → BaCl₂/Ba(NO₃)₂ + HCl → 白色沉淀
  - NO₃⁻ → Al + NaOH + 加热 → NH₃
- 气体鉴定：
  - CO₂ → 石灰水变浑浊
  - H₂ → 点燃产生爆鸣声
  - O₂ → 使带火星木条复燃
  - NH₃ → 使湿润红色石蕊变蓝
  - Cl₂ → 使湿润蓝色石蕊先变红后漂白

## 常见错误
- 混淆不同离子的沉淀颜色
- 忘记加稀硝酸排除干扰离子（鉴定卤素离子时）
- 石灰水变浑浊的原理不清楚（生成 CaCO₃ 沉淀）
- Cl₂ 的漂白性理解错误（是与水反应生成的 HClO 起作用）
- Zn²⁺ 和 Al³⁺ 的区别（需用焰色反应或其他方法）

## 可出题方向

- 设计鉴定方案
- 写出鉴定反应的离子方程式
- 区分相似现象
- 综合推断题（未知样品分析）
