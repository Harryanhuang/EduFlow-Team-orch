# Items: 12.6 — Destructive testing of organic compounds

### Question Q-12.1-15
**Difficulty**: Standard
**Question**: Describe a simple chemical test to distinguish between ethanol and ethanoic acid.
**Answer**: Add sodium carbonate. Ethanoic acid fizzes/effervesces as it produces CO₂ gas (acid + carbonate → salt + water + CO₂). Ethanol shows no reaction. Alternatively: use limewater — CO₂ turns limewater cloudy.
**Explanation**: Acid + carbonate → CO₂ (fizz). A common error is using an indicator only (both are acidic to different degrees).
**Tags**: ethanol, ethanoic-acid, distinguishing-test, carbonate

### Question Q-12.1-16
**Difficulty**: Challenge
**Question**: Describe how you could distinguish between an alkene and an alkane using bromine water.
**Answer**: Add bromine water (orange/brown solution) to each. The alkene decolorises the bromine water (turns from orange to colourless) because bromine adds across the C=C double bond. The alkane does not react — the bromine water stays orange.
**Explanation**: C=C decolorises bromine. A common error is saying both react.
**Tags**: alkene, alkane, bromine-water, distinguishing-test

### Question Q-12.1-17
**Difficulty**: Foundation
**Question**: A white solid produces a gas that turns limewater cloudy when dilute acid is added. The original solid also gives a yellow flame. Identify the compound.
**Answer**: Sodium carbonate (Na₂CO₃). Limewater turns cloudy = CO₂ produced = carbonate present. Yellow flame = sodium ion. Together: sodium carbonate.
**Explanation**: Carbonate + sodium → Na₂CO₃. A common error is identifying only the carbonate or only the sodium.
**Tags**: sodium-carbonate, identification, limewater, flame-test
