# Items: 7.3 — Identification of ions and gases

### Question Q-7.3-01
**Difficulty**: Foundation
**Question**: Describe the test for hydrogen gas.
**Answer**: Hold a lighted splint near the mouth of a test tube containing the gas. Hydrogen produces a characteristic "pop" or "squeaky pop" sound as it ignites with oxygen in air: 2H₂ + O₂ → 2H₂O.
**Explanation**: Lighted splint → pop. A common error is saying it "burns with a blue flame" (that's a description of combustion, not the test).
**Tags**: hydrogen, gas-test, pop, identification

### Question Q-7.3-02
**Difficulty**: Foundation
**Question**: Describe the test for oxygen gas.
**Answer**: Insert a glowing splint into the test tube. Oxygen relights the glowing splint because it supports combustion.
**Explanation**: Glowing splint → relights. A common error is confusing with hydrogen (lighted splint).
**Tags**: oxygen, gas-test, glowing-splint, identification

### Question Q-7.3-03
**Difficulty**: Challenge
**Question**: A gas turns limewater milky, and the cloudiness disappears when excess gas is bubbled through for a long time. Identify the gas and explain both observations using equations.
**Answer**: The gas is carbon dioxide. It first reacts with calcium hydroxide to form insoluble calcium carbonate, which makes limewater milky: Ca(OH)₂(aq) + CO₂(g) → CaCO₃(s) + H₂O(l). In excess CO₂, calcium carbonate forms soluble calcium hydrogencarbonate, so the milkiness disappears: CaCO₃(s) + CO₂(g) + H₂O(l) → Ca(HCO₃)₂(aq).
**Explanation**: Students must connect the observation to both precipitation and redissolving in excess CO₂. Common errors are stopping at "limewater turns milky" or identifying hydrogen.
**Tags**: carbon-dioxide, limewater, equation, gas-test

### Question Q-7.3-04
**Difficulty**: Standard
**Question**: A solution contains copper(II) ions. Describe the test using aqueous sodium hydroxide and state the expected result.
**Answer**: Add aqueous NaOH dropwise. Copper(II) ions form a blue precipitate of copper(II) hydroxide: Cu²⁺ + 2OH⁻ → Cu(OH)₂(s). The precipitate does not dissolve in excess NaOH.
**Explanation**: Blue ppt, insoluble in excess. A common error is saying the precipitate is green.
**Tags**: Cu2+, NaOH-test, blue-precipitate, cation

### Question Q-7.3-05
**Difficulty**: Standard
**Question**: How would you distinguish between iron(II) and iron(III) ions using aqueous sodium hydroxide?
**Answer**: Add aqueous NaOH. Iron(II) ions form a green precipitate of Fe(OH)₂. Iron(III) ions form a brown/orange-brown precipitate of Fe(OH)₃. Neither precipitate dissolves in excess NaOH.
**Explanation**: Green vs brown. A common error is confusing which colour corresponds to which ion.
**Tags**: Fe2+, Fe3+, NaOH-test, cation, distinguishing
