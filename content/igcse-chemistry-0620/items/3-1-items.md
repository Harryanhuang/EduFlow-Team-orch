# Items: 3.1 — Chemical formulae, equations, the mole concept

### Question Q-3.1-01
**Difficulty**: Standard
**Question**: Calculate the percentage by mass of nitrogen in ammonium nitrate (NH₄NO₃). (Aᵣ: N = 14, H = 1, O = 16)
**Answer**: 35%
**Explanation**: Mᵣ = 14 + (4×1) + 14 + (3×16) = 80. Total N mass = 2 × 14 = 28. %N = (28/80) × 100 = 35%. A common error is using only one nitrogen atom.
**Tags**: percentage-mass, nitrogen, ammonium-nitrate, calculation

### Question Q-3.1-02
**Difficulty**: Foundation
**Question**: Balance the equation: Fe + O₂ → Fe₂O₃
**Answer**: 4Fe + 3O₂ → 2Fe₂O₃
**Explanation**: Iron: 4 on each side. Oxygen: 6 on each side (3×2 = 6, 2×3 = 6). A common error is 2Fe + 3O₂ → Fe₂O₃ (oxygen unbalanced).
**Tags**: balancing-equations, iron-oxide

### Question Q-3.1-03
**Difficulty**: Standard
**Question**: Calculate the number of moles in 4.4 g of carbon dioxide (CO₂). (Aᵣ: C = 12, O = 16)
**Answer**: 0.1 mol
**Explanation**: Mᵣ of CO₂ = 12 + (2×16) = 44. Moles = mass / Mᵣ = 4.4 / 44 = 0.1 mol. A common error is using Mᵣ = 28.
**Tags**: moles, mass, CO2, calculation

### Question Q-3.1-04
**Difficulty**: Foundation
**Question**: What is one mole?
**Answer**: One mole is the amount of substance that contains the same number of particles (atoms, molecules, or ions) as there are atoms in exactly 12 g of carbon-12. This number is 6.02 × 10²³ (Avogadro's constant).
**Explanation**: Fixed number of particles. A common error is saying "one mole = one gram."
**Tags**: mole, definition, Avogadro

### Question Q-3.1-05
**Difficulty**: Standard
**Question**: Calculate the mass of 0.25 mol of calcium carbonate (CaCO₃). (Aᵣ: Ca = 40, C = 12, O = 16)
**Answer**: 25 g
**Explanation**: Mᵣ of CaCO₃ = 40 + 12 + (3×16) = 100. Mass = moles × Mᵣ = 0.25 × 100 = 25 g. A common error is forgetting the ×3 for oxygen.
**Tags**: mass, moles, CaCO3, calculation
