# Items: 1.3 — Elements, compounds, and mixtures

### Question Q-2.1-01
**Difficulty**: Foundation
**Question**: What is the difference between an element and a compound?
**Answer**: An element is a substance made of only one type of atom (e.g., oxygen, iron). A compound is a substance made of two or more different types of atoms chemically bonded together (e.g., water, carbon dioxide).
**Explanation**: One type vs. multiple types of atoms. A common error is saying a compound is a mixture.
**Tags**: element, compound, definition

### Question Q-2.1-02
**Difficulty**: Foundation
**Question**: How can you separate a mixture of iron filings and sulfur powder?
**Answer**: Using a magnet. Iron is magnetic and is attracted to the magnet, while sulfur is not. This is a physical separation — no chemical reaction has occurred.
**Explanation**: Magnetic separation. A common error is saying "dissolve in water" (neither dissolves in water).
**Tags**: mixture, separation, magnetic, iron

### Question Q-2.1-03
**Difficulty**: Standard
**Question**: Explain why the properties of a compound are different from the properties of the elements it is made from.
**Answer**: When elements chemically combine to form a compound, their atoms rearrange into a new structure with different bonding. The compound has its own unique properties. For example, sodium (reactive metal) + chlorine (poisonous gas) → sodium chloride (stable, edible salt).
**Explanation**: New bonds = new structure = new properties. A common error is saying compounds are "just a mix."
**Tags**: compound, properties, chemical-bonding

### Question Q-2.1-04
**Difficulty**: Foundation
**Question**: Classify each as an element, compound, or mixture: air, water, gold, brass.
**Answer**: Air = mixture; water = compound; gold = element; brass = mixture (alloy).
**Explanation**: Element = one atom type; compound = chemically bonded; mixture = physically combined. A common error is classifying brass as a compound.
**Tags**: classification, element, compound, mixture
