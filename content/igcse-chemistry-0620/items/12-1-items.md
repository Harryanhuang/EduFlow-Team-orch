# Items: 12.1 — Measurement, purity, separation techniques

### Question Q-12.1-01
**Difficulty**: Foundation
**Question**: How can you determine whether a substance is pure using chromatography?
**Answer**: A pure substance produces a single spot on a chromatogram. If there are multiple spots, the substance is a mixture.
**Explanation**: One spot = pure. A common error is saying "more spots = purer."
**Tags**: purity, chromatography, single-spot

### Question Q-12.1-02
**Difficulty**: Standard
**Question**: A student wants to separate a mixture of ethanol and water. What technique should be used and why?
**Answer**: Fractional distillation. Ethanol (b.p. 78°C) and water (b.p. 100°C) have different boiling points. Fractional distillation separates liquids based on their different boiling points, with the lower-boiling component distilling off first.
**Explanation**: Different boiling points → fractional distillation. A common error is saying simple distillation (for separating a liquid from a solid).
**Tags**: fractional-distillation, ethanol, water, separation

### Question Q-12.1-03
**Difficulty**: Standard
**Question**: What is the purpose of using a locating agent in chromatography?
**Answer**: A locating agent makes colourless substances visible on a chromatogram, allowing the positions of the spots to be identified and Rf values to be calculated.
**Explanation**: Makes invisible spots visible. A common error is saying it "reacts with the solvent."
**Tags**: chromatography, locating-agent, visualisation

### Question Q-12.1-04
**Difficulty**: Standard
**Question**: Explain how you would separate a mixture of sand and salt (sodium chloride).
**Answer**: (1) Add water and stir — salt dissolves, sand does not. (2) Filter — sand stays on the filter paper (residue), salt solution passes through (filtrate). (3) Evaporate the filtrate to dryness to obtain solid salt.
**Explanation**: Dissolve → filter → evaporate. A common error is trying to filter without adding water.
**Tags**: separation, sand, salt, filtration, evaporation
