# Items: 3.3 — Empirical and molecular formulae

### Question Q-3.3-01
**Difficulty**: Standard
**Question**: A compound contains 80% carbon and 20% hydrogen by mass. Calculate its empirical formula. (Aᵣ: C = 12, H = 1)
**Answer**: CH₃
**Explanation**: Assume 100 g sample: C = 80 g, H = 20 g. Moles C = 80/12 = 6.67, moles H = 20/1 = 20. Divide by smallest: C = 6.67/6.67 = 1, H = 20/6.67 = 3. Ratio C:H = 1:3. Empirical formula: CH₃. A common error is using the percentage directly as the number of atoms.
**Tags**: empirical-formula, percentage-composition, calculation

### Question Q-3.3-02
**Difficulty**: Standard
**Question**: The empirical formula of a compound is CH₂O and its Mᵣ is 60. Calculate its molecular formula. (Aᵣ: C = 12, H = 1, O = 16)
**Answer**: C₂H₄O₂
**Explanation**: Empirical formula mass = 12 + 2 + 16 = 30. n = 60/30 = 2. Molecular formula = (CH₂O) × 2 = C₂H₄O₂. A common error is using the empirical formula as the molecular formula.
**Tags**: molecular-formula, empirical-formula, Mr, calculation

### Question Q-3.3-03
**Difficulty**: Standard
**Question**: In an experiment, 2.4 g of magnesium reacts with 1.6 g of oxygen. Determine the empirical formula of the compound formed. (Aᵣ: Mg = 24, O = 16)
**Answer**: MgO
**Explanation**: Moles Mg = 2.4/24 = 0.1, moles O = 1.6/16 = 0.1. Ratio Mg:O = 0.1:0.1 = 1:1. Empirical formula: MgO. A common error is using the mass ratio directly (2.4:1.6).
**Tags**: empirical-formula, reacting-mass, magnesium-oxide, calculation

### Question Q-3.3-04
**Difficulty**: Standard
**Question**: A hydrocarbon produces 4.4 g of CO₂ and 1.8 g of H₂O on complete combustion. Determine its empirical formula. (Aᵣ: C = 12, H = 1, O = 16)
**Answer**: CH₂
**Explanation**: Mass of C in CO₂ = (12/44) × 4.4 = 1.2 g → moles C = 1.2/12 = 0.1. Mass of H in H₂O = (2/18) × 1.8 = 0.2 g → moles H = 0.2/1 = 0.2. Ratio C:H = 0.1:0.2 = 1:2. Empirical formula: CH₂. A common error is forgetting to convert CO₂ mass to carbon mass.
**Tags**: empirical-formula, combustion, hydrocarbon, calculation
