# Items: 9.4 — Bond energy calculations

### Question Q-5.2-10
**Difficulty**: Standard
**Question**: Calculate the enthalpy change for the reaction: CH₄ + 2O₂ → CO₂ + 2H₂O. Bond energies: C-H = 413, O=O = 498, C=O = 805, O-H = 464 (all kJ/mol).
**Answer**: -818 kJ/mol
**Explanation**: Bonds broken: 4×C-H = 1652, 2×O=O = 996. Total in = 2648. Bonds made: 2×C=O = 1610, 4×O-H = 1856. Total out = 3466. ΔH = 2648 - 3466 = -818 kJ/mol (exothermic). A common error is getting the signs wrong.
**Tags**: bond-energy, combustion, methane, enthalpy, calculation

### Question Q-5.2-11
**Difficulty**: Standard
**Question**: A reaction has bond energies: bonds broken = 700 kJ/mol, bonds made = 500 kJ/mol. Is it exothermic or endothermic? Calculate ΔH.
**Answer**: Endothermic. ΔH = 700 - 500 = +200 kJ/mol.
**Explanation**: More energy in than out = endothermic (positive ΔH). A common error is saying exothermic because "700 is bigger."
**Tags**: bond-energy, endothermic, enthalpy, calculation

### Question Q-5.2-12
**Difficulty**: Standard
**Question**: Use the data to calculate the C-Cl bond energy in CH₄ + Cl₂ → CH₃Cl + HCl, given ΔH = -101 kJ/mol. C-H = 413, Cl-Cl = 242, C-Cl = ?, H-Cl = 431 (kJ/mol).
**Answer**: 327 kJ/mol
**Explanation**: Bonds broken: 4×C-H + Cl-Cl = 1652 + 242 = 1894. Bonds made: 3×C-H + C-Cl + H-Cl = 1239 + C-Cl + 431 = 1670 + C-Cl. ΔH = 1894 - (1670 + C-Cl) = -101. C-Cl = 1894 - 1670 + 101 = 325 kJ/mol. Actually: 1894 - 1670 = 224; 224 - C-Cl = -101; C-Cl = 224 + 101 = 325 kJ/mol.
**Explanation**: Rearrange the ΔH equation. A common error is not counting the C-H bonds correctly (4 broken, 3 made).
**Tags**: bond-energy, C-Cl, rearrangement, calculation
