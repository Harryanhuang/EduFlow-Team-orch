# Items: 2.2 — Periodic table, electronic configuration

### Question Q-2.2-01
**Difficulty**: Foundation
**Question**: How are elements arranged in the Periodic Table?
**Answer**: Elements are arranged in order of increasing atomic number (number of protons). Elements in the same group have the same number of valence (outer shell) electrons.
**Explanation**: Atomic number, not mass. A common error is saying "increasing mass."
**Tags**: periodic-table, arrangement, atomic-number

### Question Q-2.2-02
**Difficulty**: Standard
**Question**: An element has the electronic configuration 2,8,5. Which group and period is it in?
**Answer**: Group 5 (5 valence electrons), Period 3 (3 electron shells). The element is phosphorus (atomic number 15).
**Explanation**: Valence electrons → group; number of shells → period. A common error is saying Group 15.
**Tags**: electronic-configuration, group, period, phosphorus

### Question Q-2.2-03
**Difficulty**: Foundation
**Question**: Why are noble gases (Group 8/0) unreactive?
**Answer**: Noble gases have a full outer shell of electrons (2 for helium, 8 for the others). This stable electron configuration means they do not need to gain, lose, or share electrons.
**Explanation**: Full outer shell = stable. A common error is saying they "have no electrons."
**Tags**: noble-gases, reactivity, full-shell
