# Items: 6.3 — Redox: oxidation, reduction, oxidising/reducing agents

### Question Q-6.3-01
**Difficulty**: Foundation
**Question**: Define oxidation and reduction in terms of oxygen transfer.
**Answer**: Oxidation is the gain of oxygen. Reduction is the loss of oxygen. For example, in CuO + H₂ → Cu + H₂O: CuO is reduced (loses oxygen to become Cu), and H₂ is oxidised (gains oxygen to become H₂O).
**Explanation**: OIL RIG (Oxidation Is Loss of oxygen; Reduction Is Gain of oxygen) — but only for oxygen transfer. A common error is reversing them.
**Tags**: oxidation, reduction, oxygen, definition

### Question Q-6.3-02
**Difficulty**: Standard
**Question**: Define oxidation and reduction in terms of electron transfer.
**Answer**: Oxidation is the loss of electrons (OIL — Oxidation Is Loss). Reduction is the gain of electrons (RIG — Reduction Is Gain). For example: Zn → Zn²⁺ + 2e⁻ is oxidation (Zn loses electrons). Cu²⁺ + 2e⁻ → Cu is reduction (Cu²⁺ gains electrons).
**Explanation**: OIL RIG for electrons. A common error is confusing which is which.
**Tags**: oxidation, reduction, electron, OIL-RIG

### Question Q-6.3-03
**Difficulty**: Standard
**Question**: In the reaction Fe₂O₃ + 3CO → 2Fe + 3CO₂, identify the substance oxidised and the substance reduced.
**Answer**: Fe₂O₃ is reduced (loses oxygen to become Fe). CO is oxidised (gains oxygen to become CO₂). Iron goes from +3 in Fe₂O₃ to 0 in Fe (reduction). Carbon goes from +2 in CO to +4 in CO₂ (oxidation).
**Explanation**: Fe₂O₃ → Fe (reduction); CO → CO₂ (oxidation). A common error is saying Fe is oxidised.
**Tags**: redox, blast-furnace, iron, CO

### Question Q-6.3-04
**Difficulty**: Standard
**Question**: What is an oxidising agent? What is a reducing agent?
**Answer**: An oxidising agent is a substance that causes oxidation of another substance — it itself is reduced (gains electrons / loses oxygen). A reducing agent causes reduction of another substance — it itself is oxidised (loses electrons / gains oxygen).
**Explanation**: Oxidising agent = gets reduced. Reducing agent = gets oxidised. A common error is thinking the oxidising agent is oxidised.
**Tags**: oxidising-agent, reducing-agent, definition

### Question Q-6.3-05
**Difficulty**: Challenge
**Question**: Write the half-equations for the reaction: Zn + CuSO₄ → ZnSO₄ + Cu. Identify the oxidising and reducing agents.
**Answer**: Oxidation: Zn → Zn²⁺ + 2e⁻ (Zn loses electrons, so Zn is the reducing agent). Reduction: Cu²⁺ + 2e⁻ → Cu (Cu²⁺ gains electrons, so Cu²⁺ is the oxidising agent).
**Explanation**: Zn is oxidised → reducing agent. Cu²⁺ is reduced → oxidising agent. A common error is naming CuSO₄ as the oxidising agent (it's the Cu²⁺ ion that's reduced).
**Tags**: half-equations, Zn, Cu, oxidising-agent, reducing-agent
