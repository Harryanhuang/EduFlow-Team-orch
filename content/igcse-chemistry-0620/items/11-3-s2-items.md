# Items: 11.5 — Organic chemistry: functional groups and naming

### Question Q-11.3-07
**Difficulty**: Foundation
**Question**: What is a functional group?
**Answer**: A functional group is a specific atom or group of atoms in a molecule that determines the characteristic chemical reactions of that molecule.
**Explanation**: Determines chemical behaviour. A common error is saying it determines physical properties only.
**Tags**: functional-group, definition, organic

### Question Q-11.3-08
**Difficulty**: Standard
**Question**: Name and draw the structure of the alcohol with 3 carbon atoms.
**Answer**: Propanol (C₃H₇OH). Structure: CH₃-CH₂-CH₂-OH (propan-1-ol) or CH₃-CH(OH)-CH₃ (propan-2-ol). Both are isomers.
**Explanation**: -OH on a 3-carbon chain. A common error is putting the -OH on the first carbon only.
**Tags**: propanol, alcohol, isomers, structure

### Question Q-11.3-09
**Difficulty**: Foundation
**Question**: Name the homologous series with the functional group C=C.
**Answer**: Alkenes.
**Explanation**: C=C double bond. A common error is saying alkynes (C≡C).
**Tags**: alkenes, C=C, functional-group, homologous-series

### Question Q-11.3-10
**Difficulty**: Standard
**Question**: What is a homologous series? Give two characteristics shared by all members.
**Answer**: A homologous series is a family of organic compounds with the same functional group. Characteristics: (1) same general formula, (2) similar chemical properties, (3) a gradual change in physical properties with increasing chain length.
**Explanation**: Same functional group → similar chemistry. A common error is saying all members have the same physical properties.
**Tags**: homologous-series, characteristics, organic
