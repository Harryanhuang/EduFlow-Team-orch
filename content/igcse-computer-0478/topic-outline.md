# IGCSE Computer Science (0478) Topic Outline

Based on Cambridge IGCSE Computer Science (0478) syllabus (latest version). Covers computational thinking, programming, data representation, computer systems, networking, and security and ethics.

## Assessment Overview
- **Component 1**: Computer Science (Paper 1 — Written), 1 hour 45 minutes, 75 marks
- **Component 2**: Practical Problem-solving and Programming (Paper 2 — Written), 1 hour 45 minutes, 75 marks
- **Total**: 150 marks

## Topic List

### 1 — Information Representation
1.1 Number systems (binary, hexadecimal); 1.2 Text, sound, and images in binary (ASCII, Unicode, bitmap, digitisation, compression)

### 2 — Data Representation
2.1 Binary and hexadecimal conversion; 2.2 Two's complement (signed binary); 2.3 Floating-point numbers (normalised form, IEEE 754); 2.4 Hexadecimal representation and use

### 3 — Hardware and Software
3.1 Logic gates (AND, OR, NOT, NAND, NOR) and truth tables; 3.2 Boolean algebra (De Morgan's laws, simplification, XOR)

### 4 — Algorithms and Programming
4.1 Algorithm design (flowcharts, pseudocode); 4.2 Searching algorithms (linear, binary); 4.3 Sorting algorithms (bubble sort, insertion sort, comparison); 4.4 Detailed sorting algorithm analysis (trace, complexity, optimisation)

### 5 — Programming Concepts
5.1 Variables and data types; 5.2 Sequence, selection, and iteration (IF, CASE, FOR, WHILE, REPEAT); 5.3 Procedures and functions (parameters, return values, recursion); 5.4 File handling (sequential, OPEN/READ/WRITE/APPEND); 5.5 File handling extended (random access, CSV, error handling, binary files)

### 6 — Databases
6.1 Database concepts (fields, records, primary key, flat-file vs relational); 6.2 SQL queries (SELECT, WHERE, ORDER BY, DISTINCT, aggregate functions, GROUP BY); 6.3 Data validation and verification

### 7 — Computer Networks
7.1 Network types (LAN, WAN), topologies, client-server, P2P; 7.2 Network protocols (TCP/IP, HTTP/HTTPS, FTP, DNS, DHCP); 7.3 Network security (firewalls, encryption, VPN, authentication); 7.4 Cloud computing (IaaS, PaaS, SaaS, public/private cloud)

### 8 — Security and Ethics
8.1 Cyber security threats (malware, social engineering, phishing, SQL injection); 8.2 Data protection (GDPR, copyright, privacy, digital footprint); 8.3 Ethical issues in computing (AI, automation, digital divide, responsible use); 8.4 Environmental impact of computing (e-waste, energy, green computing)

**Total: 8 topic groups, 28 subtopics**
