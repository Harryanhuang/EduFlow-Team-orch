# Topic 1.1 — Number Systems and Binary Representation
## QA Question Level File

**1. [F]** Convert the binary number `11010110` to decimal.

**2. [F]** Convert the decimal number `215` to binary.

**3. [F]** State two advantages of using binary to represent data in a computer.

**4. [S]** Perform binary addition: `11011` + `10110`. Show all working.

**5. [S]** Convert the hexadecimal number `7F` to decimal, then to binary.

**6. [S]** A computer has 4 GB of RAM. How many bytes is this? Show your working using the binary prefix (where 1 GB = 1024 MB).

**7. [C]** Calculate how many different values can be stored using 12 bits. Then, using 12 bits to represent pixel colours in an image, if each pixel requires 3 bits for red and 4 bits for green, how many bits remain for blue?

**8. [C]** Perform binary subtraction: `110010` - `100101`. Use binary complement or other method, and show all working steps.

**9. [C]** A sensor records temperature as an unsigned 10-bit binary value. The range is 0 to 100 degrees Celsius.
(a) What is the maximum number of distinct temperature values that can be represented?
(b) What is the resolution of the sensor in degrees per unit?
(c) Explain why using binary prefixes (KiB, MiB) rather than decimal prefixes (KB, MB) matters when calculating storage capacity.
