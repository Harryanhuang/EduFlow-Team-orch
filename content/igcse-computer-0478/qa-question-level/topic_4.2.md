# Topic 4.2 — Searching and Sorting
## QA Question Level File

**1. [F]** Describe how a linear search works on an unsorted list of 10 numbers.

**2. [F]** State one advantage and one disadvantage of binary search over linear search.

**3. [F]** What condition must be true before a binary search can be performed?

**4. [S]** Trace the bubble sort algorithm on the list [5, 2, 8, 1, 4]. Show the state after each pass.

**5. [S]** Describe how insertion sort works when sorting the list [7, 3, 5, 2].

**6. [S]** Explain the time complexity difference between O(n) and O(log n) in the context of searching.

**7. [C]** A sorted list has 1024 elements. Calculate the maximum number of comparisons needed for binary search and explain your answer.

**8. [C]** Compare bubble sort and insertion sort in terms of best-case, average-case, and worst-case time complexity. Which would you choose for an almost-sorted list and why?

**9. [C]** Show step-by-step how binary search finds the value 23 in the list [4, 8, 12, 15, 19, 23, 28, 32].
