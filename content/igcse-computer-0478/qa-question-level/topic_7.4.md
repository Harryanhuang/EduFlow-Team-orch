# Topic 7.4 — Cloud Computing
## QA Question Level File

**1. [F]** Define what is meant by cloud computing and give one example of a cloud service used by individuals.

**2. [F]** State the meaning of SaaS and give one example of a SaaS application.

**3. [F]** Explain the difference between a public cloud and a private cloud.

**4. [S]** Describe two advantages of using cloud storage over local storage for a business.

**5. [S]** Explain what is meant by data sovereignty and why it is a concern for organisations using cloud services across different countries.

**6. [S]** Compare IaaS and PaaS, giving an example of when each would be appropriate for an organisation.

**7. [C]** A school is considering moving its student records system to the cloud. Evaluate the data privacy issues the school must consider before making this decision.

**8. [C]** Discuss two environmental advantages and two disadvantages of cloud computing compared to organisations running their own data centres.

**9. [C]** Explain what hybrid cloud is and evaluate why an enterprise might choose a hybrid cloud strategy over using only public or only private cloud services.
