# Topic 6.2 — SQL Queries
## QA Question Level File

**1. [F]** Write an SQL query to select all fields from a table called Students.

**2. [F]** Write an SQL query to select distinct values in the Subject column from the Enrollments table.

**3. [F]** What does the WHERE clause do in an SQL query?

**4. [S]** Write an SQL query to select all students whose age is between 16 and 18 from the Students table.

**5. [S]** Write an SQL query to select Name and Score from Results where Subject = 'Maths', ordered by Score descending.

**6. [S]** Explain the purpose of the LIKE operator and give an example.

**7. [C]** Write an SQL query to count the number of students in each class using GROUP BY.

**8. [C]** Write an SQL query using aggregate functions to show the average, maximum, and minimum score for each subject.

**9. [C]** Explain the difference between WHERE and HAVING in SQL. Write a query that uses both.
