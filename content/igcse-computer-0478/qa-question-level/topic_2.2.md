# Topic 2.2 — Two's Complement
## QA Question Level File

**1. [F]** Convert the decimal number −42 to 8-bit two's complement binary.

**2. [F]** State the range of values that can be represented using 8-bit two's complement.

**3. [F]** What is the decimal value of the 8-bit two's complement number 10011011₂?

**4. [S]** Perform two's complement subtraction: 45 − 73 using 8-bit binary. Show all working.

**5. [S]** Explain what happens when two positive numbers in two's complement produce a negative result.

**6. [S]** Convert the decimal number −128 to 8-bit two's complement binary.

**7. [C]** A computer uses 4-bit two's complement. Show how −3 and −5 are represented, then calculate −3 + (−5) in binary and explain the result.

**8. [C]** Explain why two's complement is preferred over sign-and-magnitude for representing negative numbers in computers.

**9. [C]** Using 6-bit two's complement, determine whether adding 27 + 17 produces overflow. Show your working and state how you identified overflow.
