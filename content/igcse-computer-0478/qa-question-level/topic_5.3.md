# Topic 5.3 — Procedures and Functions
## QA Question Level File

**1. [F]** State the difference between a procedure and a function.

**2. [F]** What is a parameter in the context of procedures and functions?

**3. [F]** What is a local variable and where can it be accessed?

**4. [S]** Write pseudocode for a procedure called CalculateArea with parameters length and width that returns the area.

**5. [S]** Explain the difference between passing parameters by value and passing parameters by reference.

**6. [S]** What is a global variable and state one problem with overusing global variables?

**7. [C]** Write pseudocode for a recursive function Factorial(n) that returns n! and trace it for n = 4.

**8. [C]** Write pseudocode for a recursive function that calculates the nth Fibonacci number. Trace it for n = 5.

**9. [C]** A program has a global variable counter. A procedure Increment() increments counter by 1. Explain the problem this could cause in large programs and suggest how to solve it.
