# Topic 1.2 — Text, Sound and Images in Binary
## QA Question Level File

**1. [F]** What is the difference between ASCII and Unicode character encoding systems?

**2. [F]** A bitmap image has 800 pixels wide and 600 pixels high. How many pixels does it contain in total?

**3. [F]** State two pieces of data needed to represent a bitmap image in binary.

**4. [S]** A colour image uses 24-bit colour depth. How many different colours can be represented? Show your working.

**5. [S]** Explain why increasing the sample rate of a sound file increases its file size.

**6. [S]** Describe one example of lossy compression and one example of lossless compression.

**7. [C]** A recording at 44.1 kHz sample rate and 16-bit depth is 5 minutes long. Calculate the minimum uncompressed file size in megabytes, showing your working.

**8. [C]** Compare the advantages and disadvantages of using lossy compression for photographic images versus text files.

**9. [C]** A company needs to store 1000 high-resolution photographs. Analyse which factors they should consider when choosing between BMP (uncompressed) and JPEG (lossy) formats.
