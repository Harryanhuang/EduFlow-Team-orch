# Topic 5.5 — File Handling (Extended)
## QA Question Level File

**1. [F]** What is meant by random access to a file?

**2. [F]** State two differences between opening a file in text mode versus binary mode.

**3. [F]** Describe what is meant by a fixed-length record in file handling.

**4. [S]** A file uses fixed-length records of 100 bytes. Calculate the record number needed to access byte position 2500. Show your working.

**5. [S]** Explain why CSV files are commonly used for transferring data between different applications.

**6. [S]** Describe two types of errors that should be handled when performing file operations.

**7. [C]** A database stores customer records with fixed-length fields: Name (30 chars), Address (50 chars), Phone (15 chars). Calculate the file size needed for 1000 customers and explain how random access would locate customer number 500.

**8. [C]** Compare the advantages and disadvantages of using binary files versus text files for storing numeric data.

**9. [C]** Design a file handling solution for a library system that needs to store book records with random access capability. Include consideration of record structure, file open modes, and error handling.
