# Topic 7.3 — Network Security
## QA Question Level File

**1. [F]** What is the purpose of a firewall in network security?

**2. [F]** State two examples of authentication methods.

**3. [F]** What is the difference between symmetric and asymmetric encryption?

**4. [S]** Explain how a VPN provides security for data transmitted over the internet.

**5. [S]** Describe how two-factor authentication (2FA) improves security compared to password-only authentication.

**6. [S]** Compare symmetric encryption with asymmetric encryption in terms of speed and key distribution.

**7. [C]** A company needs to secure communications between its headquarters and a remote office. Evaluate which encryption approach (symmetric, asymmetric, or hybrid) would be most suitable, with justification.

**8. [C]** Analyse three types of cyber attacks (phishing, DDoS, man-in-the-middle) and explain the specific security measures that can defend against each.

**9. [C]** A small business is implementing network security. Design a security framework that includes firewalls, encryption, authentication, and VPN, explaining how each component addresses specific threats.
