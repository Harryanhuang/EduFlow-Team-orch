# Topic 5.1 — Variables and Data Types
## QA Question Level File

**1. [F]** Name and describe the data type stored in each of the following variables:
(a) `isActive = TRUE`
(b) `grade = "A"`
(c) `height = 1.75`

**2. [F]** State the difference between a variable and a constant. Give one example of a constant in a program.

**3. [F]** Which of the following are valid variable names? Give a reason for any that are invalid.
(a) `totalScore`
(b) `2ndPlace`
(c) `course-name`
(d) `_counter`

**4. [S]** Explain the difference between DIV and MOD operations. Show the result of `17 DIV 5` and `17 MOD 5`.

**5. [S]** Describe the difference between a one-dimensional array and a two-dimensional array. Give an example of when each would be used.

**6. [S]** Explain the difference between global and local variables, including their scope and when you would use each.

**7. [C]** A program uses a record to store student data with fields: `StudentID`, `Name`, `ExamMark`, `Pass`. Write pseudocode to define this record and then show how to assign values and display the student's name and pass status.

**8. [C]** Explain the difference between assignment and comparison using the `=` operator in pseudocode. Write one example of each.

**9. [C]** An array `Scores[1:10, 1:5]` stores test scores for 10 students across 5 subjects.
(a) How many elements does this array contain?
(b) What is the data type of each element?
(c) Write pseudocode to calculate and output the average score for student 7 across all 5 subjects.
(d) If `Scores[3, 4]` represents the score for student 3 in subject 4, explain why confusion between row and column indices is a common source of bugs.
