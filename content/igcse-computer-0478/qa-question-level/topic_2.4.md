# Topic 2.4 — Hexadecimal Representation
## QA Question Level File

**1. [F]** Convert the hexadecimal number 3A to decimal.

**2. [F]** Convert the binary number 10110110 to hexadecimal.

**3. [F]** Why is hexadecimal preferred over binary for representing memory addresses?

**4. [S]** Add the hexadecimal values 7F and 42. Show your working and give your answer in hexadecimal.

**5. [S]** In HTML, a colour is specified as #FF8800. Explain what this hexadecimal representation tells you about the colour.

**6. [S]** Convert the decimal number 200 to hexadecimal. Show your working.

**7. [C]** A memory address is stored as a 32-bit binary number. How many hexadecimal digits are needed to represent this address? Explain your answer.

**8. [C]** Two hexadecimal values A5 and 3C are added together. Calculate the result in both decimal and hexadecimal, showing all working steps.

**9. [C]** Analyse why IPv6 addresses use hexadecimal notation and explain the advantages this provides for human readability compared to decimal notation.
