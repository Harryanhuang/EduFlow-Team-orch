# Topic 5.4 — File Handling
## QA Question Level File

**1. [F]** Describe what is meant by sequential access in file handling.

**2. [F]** State the purpose of the EOF marker when reading a text file.

**3. [F]** Name the file modes OPEN FOR READ, OPEN FOR WRITE, and OPEN FOR APPEND.

**4. [S]** Write pseudocode to open a text file "data.txt", read all records until EOF, and output each record.

**5. [S]** Explain the difference between reading from and writing to a text file.

**6. [S]** Describe one advantage and one disadvantage of sequential access compared to random access.

**7. [C]** Write pseudocode to create a simple contact database using sequential file access with records containing name and phone number.

**8. [C]** Explain the process of updating a specific record in a sequential file. Why is copying the file usually necessary?

**9. [C]** Compare text files and binary files in terms of structure, human readability, and typical use cases.
