# Topic 4.4 — Bubble Sort and Insertion Sort
## QA Question Level File

**1. [F]** What is the time complexity of bubble sort in the worst case?

**2. [F]** Trace through one pass of bubble sort on the list [5, 3, 8, 1]. Show the list after each comparison.

**3. [F]** State one advantage of insertion sort over bubble sort.

**4. [S]** Trace insertion sort on the list [7, 2, 5] showing the list after each insertion.

**5. [S]** Write the best case time complexity for both bubble sort and insertion sort. Explain why they differ.

**6. [S]** In bubble sort, a flag variable is used to detect when no swaps occur in a pass. Explain why this improves efficiency.

**7. [C]** Given the partially sorted list [1, 3, 5, 7, 2], trace both bubble sort and insertion sort completely. State which algorithm finishes faster and explain why.

**8. [C]** An array contains 1000 names that are already in alphabetical order. Compare the performance of bubble sort versus insertion sort on this data, including time complexity analysis.

**9. [C]** Write pseudocode for insertion sort. Then analyse why insertion sort is preferred over bubble sort for small or nearly-sorted datasets.
