# Topic 3.2 — Boolean Algebra
## QA Question Level File

**1. [F]** State De Morgan's First Law in words and as an equation.

**2. [F]** State De Morgan's Second Law in words and as an equation.

**3. [F]** What is the complement of the Boolean expression A + B?

**4. [S]** Simplify the Boolean expression A + A·B using Boolean algebra rules.

**5. [S]** Simplify the Boolean expression A·B + A·B̅.

**6. [S]** Draw the truth table for the Boolean expression Q = A ⊕ B (XOR).

**7. [C]** Prove De Morgan's First Law using a truth table.

**8. [C]** Simplify the Boolean expression (A + B)·(A + B̅) and show the circuit using the minimum number of gates.

**9. [C]** A logic circuit has output Q = A·B + A̅·C + B·C. Simplify using Boolean algebra and draw the simplified circuit.
