# Topic 4.1 — Algorithm Design and Pseudocode
## QA Question Level File

**1. [F]** State three properties that an algorithm must have.

**2. [F]** Trace the following pseudocode and state the final value of X:
```
X ← 0
FOR I FROM 1 TO 4
  X ← X + I
ENDFOR
OUTPUT X
```

**3. [S]** Write pseudocode to input 10 numbers and output the largest number.

**4. [S]** State the difference between a WHILE loop and a REPEAT-UNTIL loop. Give one use case for each.

**5. [S]** Draw a flowchart for: input exam mark, output "Pass" if mark ≥ 50, otherwise "Fail".

**6. [S]** Trace bubble sort on the list [5, 3, 8, 1] showing the state after each pass.

**7. [S]** Write pseudocode to check if a positive integer N is prime.

**8. [S]** State two advantages of pseudocode over flowcharts.

**9. [S]** Trace binary search for target 23 in [2, 5, 8, 12, 15, 19, 23, 27, 31, 35]. Show each step.

**10. [F]** What is a subroutine and why is it used in programming?

**11. [F]** What is the difference between a procedure and a function in pseudocode?

**12. [F]** What is dry running an algorithm and why is it useful?

**13. [S]** Write pseudocode to calculate the sum and average of 10 numbers.

**14. [S]** Explain why comments are important in pseudocode.

**15. [S]** Write pseudocode to swap two values A and B without using a temporary variable.

**16. [C]** Write pseudocode for linear search that returns the position of target T in ARR[1..N] or reports "Not found".

**17. [C]** Write pseudocode to calculate the factorial of a non-negative integer N.

**18. [C]** Compare linear search and binary search in terms of precondition, speed, and complexity.
