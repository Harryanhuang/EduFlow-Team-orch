# Topic 5.2 — Sequence, Selection, and Iteration
## QA Question Level File

**1. [F]** Write pseudocode using a FOR loop to output numbers 1 to 10.

**2. [F]** State the difference between a WHILE loop and a REPEAT loop in terms of when the condition is checked.

**3. [F]** Write pseudocode for an IF statement that outputs "Pass" if a student's mark is 50 or above.

**4. [S]** Write pseudocode using a CASE statement to output the grade (A/B/C/D/F) based on a mark between 0 and 100.

**5. [S]** Write pseudocode using a WHILE loop to find the first multiple of 7 greater than 100.

**6. [S]** Write pseudocode using a REPEAT loop to input numbers until a negative number is entered, then output the count of positive numbers.

**7. [C]** Write pseudocode using nested loops to output a pattern of 5 rows where row i contains i asterisks.

**8. [C]** Explain why a FOR loop is preferred over a WHILE loop when the number of iterations is known. Include an example where using a WHILE loop incorrectly could cause issues.

**9. [C]** Write pseudocode that uses both selection (IF/CASE) and iteration to calculate the average of all passing marks (50+) from a list of 20 students.
