# Topic 3.1 — Logic Gates and Boolean Algebra
## QA Question Level File

**1. [F]** Draw the standard logic gate symbols for AND, OR, NOT, NAND, and NOR.

**2. [F]** Complete the truth table for a NOT gate with input A and output Q.

**3. [S]** Draw the truth table for Q = A AND NOT B.

**4. [S]** For the circuit: input A → NOT gate → AND gate (second input B) → output Q. Write the Boolean expression and simplify if possible.

**5. [S]** Using De Morgan's Laws, write equivalent expressions for: (a) NOT(A OR B) (b) NOT(A AND B).

**6. [S]** Draw the truth table for Q = (A AND B) OR (NOT A). State when Q = 1.

**7. [S]** What is the purpose of a buffer in a logic circuit?

**8. [S]** State the truth table for an XOR gate.

**9. [S]** A logic circuit has inputs A and B. Output Q = 1 when exactly one input is 1. What gate is this? Draw the truth table.

**10. [F]** Complete the truth table for a NAND gate.

**11. [F]** State two advantages of NAND gates over other individual logic gates.

**12. [F]** Draw the circuit for Q = A XOR B using NAND gates only.

**13. [S]** Using Boolean algebra, simplify: (A + A̅ · B).

**14. [S]** What is a flip-flop and why is it important in computing?

**15. [S]** Explain the purpose of a clock signal in a digital circuit.

**16. [C]** A logic circuit has output Q = (A + B̅) · B. Simplify using Boolean algebra and state the result.

**17. [C]** Draw the circuit for Q = A XOR B using AND, OR, and NOT gates. Derive the Boolean expression.

**18. [C]** Design a half adder circuit: inputs A and B, outputs Sum and Carry. Derive the Boolean expressions.
