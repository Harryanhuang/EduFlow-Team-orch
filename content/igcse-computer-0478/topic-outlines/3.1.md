# Topic 3.1 — Logic Gates and Boolean Algebra

## Basic Info
- **Topic ID:** 3.1
- **Domain:** Hardware and Software
- **Core/Supplement:** Core
- **Weight in exam:** Medium (Paper 1 circuit and logic questions)

## Knowledge Points
1. Logic gates: AND, OR, NOT, NAND, NOR, XOR
2. Truth tables for all logic gates
3. Boolean expressions from logic circuits
4. Logic circuits from Boolean expressions
5. Laws of Boolean algebra: Commutative, Associative, Distributive, Identity, Null, Complement
6. Simplifying Boolean expressions using laws
7. De Morgan's Laws: NOT(A AND B) = NOT A OR NOT B; NOT(A OR B) = NOT A AND NOT B
8. Combining gates to form circuits

## Key Formulas
- AND: A·B or A ∧ B; output = 1 only when both inputs = 1
- OR: A + B or A ∨ B; output = 1 when at least one input = 1
- NOT: Ā or ¬A; output = opposite of input
- NAND: NOT(A AND B); universal gate
- NOR: NOT(A OR B)
- XOR: A ⊕ B; output = 1 when inputs are different

## Common Errors
- Confusing AND with OR (especially in truth tables)
- Forgetting brackets in Boolean expressions
- Incorrectly applying De Morgan's Laws
- Misidentifying NAND/NOR gate symbols

## Difficulty Band: F:2|S:4|C:3
- F: identify gate symbols, complete simple truth tables
- S: derive Boolean expressions from circuits, construct circuits from expressions, De Morgan's Laws
- C: simplify Boolean expressions, combine multiple gates, identify gates from truth tables
