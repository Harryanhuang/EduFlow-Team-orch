## 重编号后 manifest 影响

重编号后需同步更新 `content/qbank-unified-manifest.csv`:
- 被重编号的题目保留在原 topic_id 下（如 Q-1.1-01r2 仍属 topic 1.1）
- manifest 的 question_count 不变（题目数量未变，只是 ID 变了）
- 新增的 r2/r3 后缀题目需加入 manifest

## 执行建议

确认后，用 Python 脚本批量执行：
1. 对每个待改动文件，用正则定位 `### Question Q-X.Y-0Z`
2. 替换为 `### Question Q-X.Y-0Zr2`（或 r3）
3. 同时移除 4 个 Chemistry 真实重复 block
4. 更新 unified manifest CSV
5. 重跑 `qbank_verify.py` 验证冲突数 → 0

## 风险评估

- **数据丢失风险**: 零。所有题目均保留
