### Question Q-1.1-05
**Difficulty**: Standard
**Question**: A manufacturer buys raw materials for $50 and sells the finished product for $180. Calculate value added and explain why it is not the same as profit.
**Answer**: Value added = $180 - $50 = $130. Profit = value added - other costs (labour, rent, utilities, etc.). Value added only subtracts bought-in materials, not all costs.
**Explanation**: Value added ≠ profit. Profit subtracts all costs. A common error is treating them as the same.
