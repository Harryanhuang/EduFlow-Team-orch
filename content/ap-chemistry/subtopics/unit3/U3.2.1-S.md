Two solids X and Y are tested. Solid X conducts electricity in the solid state and can be hammered into a thin sheet without cracking. Solid Y has a high melting point, shatters when struck, and conducts electricity only when dissolved in water. Which pair correctly identifies the solid types?

## Options
A) X = ionic, Y = metallic
B) X = covalent network, Y = ionic
C) X = metallic, Y = ionic
D) X = metallic, Y = covalent network

## Answer
C

## Explanation
Solid X conducts in the solid state and is malleable, which are hallmarks of a metallic solid — the "electron sea" carries charge and allows atomic layers to slide past one another. Solid Y shatters when struck (brittle, because striking shifts ion layers so like charges align and repel) and conducts only when dissolved (mobile ions in solution), identifying it as an ionic solid. Covalent network solids do not conduct under any condition.
