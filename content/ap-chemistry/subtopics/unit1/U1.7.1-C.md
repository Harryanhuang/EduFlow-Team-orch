The first five successive ionization energies (in kJ/mol) for a Period 3 element X are:

$$IE_1 = 738, \quad IE_2 = 1451, \quad IE_3 = 7733, \quad IE_4 = 10540, \quad IE_5 = 13630$$

Based on this data, which element is X most likely to be?

## Options
A. $\mathrm{Na}$
B. $\mathrm{Mg}$
C. $\mathrm{Al}$
D. $\mathrm{Si}$

## Answer
B

## Explanation
The key is locating the largest jump between successive ionization energies. This large jump occurs when an electron is removed from a stable noble-gas core (an inner shell), which requires dramatically more energy.

Examine the ratios between consecutive values:
- $IE_1$ to $IE_2$: 738 to 1451 (about 2×, modest increase)
- $IE_2$ to $IE_3$: 1451 to 7733 (about 5.3×, very large jump)

The largest jump occurs between $IE_2$ and $IE_3$. This means the first two electrons are valence electrons (removed relatively easily), and the third electron comes from the stable inner core.

An element with exactly 2 valence electrons is in Group 2. In Period 3, that element is magnesium ($\mathrm{Mg}$, $1s^2 2s^2 2p^6 3s^2$). After removing both $3s$ electrons, the next removal breaks into the neon core ($2p^6$), causing the steep rise.

Option A (Na) has 1 valence electron, so the big jump would be between $IE_1$ and $IE_2$.
Option C (Al) has 3 valence electrons, so the jump would be between $IE_3$ and $IE_4$.
Option D (Si) has 4 valence electrons, so the jump would be between $IE_4$ and $IE_5$.
