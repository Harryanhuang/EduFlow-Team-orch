For the reaction $\mathrm{H_2(g) + I_2(g) \rightleftharpoons 2\,HI(g)}$, $K_c = 64.0$. A flask initially contains $[\mathrm{H_2}]_0 = 1.00\ \mathrm{M}$ and $[\mathrm{I_2}]_0 = 1.00\ \mathrm{M}$ with no HI. What is the equilibrium concentration of HI?

## Options
A. 0.80 M
B. 1.60 M
C. 0.20 M
D. 2.00 M

## Answer
B

## Explanation
Set up an ICE table (let $x$ be the amount of $\mathrm{H_2}$ that reacts):

| | $\mathrm{H_2}$ | $\mathrm{I_2}$ | $\mathrm{HI}$ |
|---|---|---|---|
| I | 1.00 | 1.00 | 0 |
| C | $-x$ | $-x$ | $+2x$ |
| E | $1.00-x$ | $1.00-x$ | $2x$ |

Write the equilibrium expression:

$$K_c = \frac{(2x)^2}{(1.00-x)(1.00-x)} = \frac{4x^2}{(1.00-x)^2} = 64.0$$

Because $K$ is large, the small-$x$ approximation fails. Take the square root of both sides (a clean shortcut for a perfect-square ratio):

$$\frac{2x}{1.00-x} = \sqrt{64.0} = 8.00$$

$$2x = 8.00(1.00 - x) = 8.00 - 8.00x$$

$$10.00x = 8.00 \quad\Rightarrow\quad x = 0.800$$

Therefore:

$$[\mathrm{HI}] = 2x = 2(0.800) = 1.60 \text{ M}$$

Option A reports $x$ instead of $2x$.
Option C uses the small-x approximation incorrectly.
Option D would require complete conversion, which is impossible at equilibrium.
