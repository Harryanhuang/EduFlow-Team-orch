A galvanic cell is built from the following standard reduction half-reactions:

$$\mathrm{Ag^+} + e^- \rightarrow \mathrm{Ag} \qquad E° = +0.80 \ \mathrm{V}$$
$$\mathrm{Ni^{2+}} + 2e^- \rightarrow \mathrm{Ni} \qquad E° = -0.25 \ \mathrm{V}$$

Calculate $\Delta G°$ for the spontaneous cell reaction. (Use $F = 96{,}485 \ \mathrm{C/mol}$.)

## Options
A. $-203 \ \mathrm{kJ/mol}$
B. $-101 \ \mathrm{kJ/mol}$
C. $+203 \ \mathrm{kJ/mol}$
D. $-405 \ \mathrm{kJ/mol}$

## Answer
A

## Explanation
**Step 1 — Identify cathode and anode and find $E°_{cell}$.** The more positive reduction potential ($\mathrm{Ag^+}/\mathrm{Ag}$) is the cathode:

$$E°_{cell} = E°_{cathode} - E°_{anode} = (+0.80) - (-0.25) = +1.05 \ \mathrm{V}$$

**Step 2 — Balance electrons to find $n$.** Silver reduction involves 1 electron; nickel oxidation involves 2 electrons. To balance, double the silver half-reaction:

$$2\,\mathrm{Ag^+} + 2e^- \rightarrow 2\,\mathrm{Ag} \qquad \mathrm{Ni} \rightarrow \mathrm{Ni^{2+}} + 2e^-$$

So $n = 2$ moles of electrons. (Note: $E°_{cell}$ is an intensive property and does **not** change when the half-reaction is scaled.)

**Step 3 — Calculate $\Delta G°$:**

$$\Delta G° = -nFE°_{cell} = -(2)(96{,}485 \ \mathrm{C/mol})(1.05 \ \mathrm{V})$$

$$\Delta G° = -202{,}619 \ \mathrm{J/mol} \approx -203 \ \mathrm{kJ/mol}$$

Option A is correct. Option B uses $n = 1$ instead of 2. Option C omits the negative sign. Option D uses $n = 4$.
