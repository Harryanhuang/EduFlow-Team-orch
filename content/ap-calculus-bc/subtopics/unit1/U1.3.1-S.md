The function $f(x) = \frac{x^2 - 4}{x - 2}$ has a discontinuity at $x = 2$. What type is it?

## Options
A) Removable discontinuity (可去间断)
B) Jump discontinuity (跳跃间断)
C) Infinite discontinuity (无穷间断)
D) The function is continuous at $x = 2$.

## Answer
A

## Explanation
At $x = 2$: $f(2)$ is undefined (division by zero), so condition 1 fails. However, $\lim_{x\to2} \frac{x^2-4}{x-2} = \lim_{x\to2} \frac{(x-2)(x+2)}{x-2} = \lim_{x\to2} (x+2) = 4$. The limit exists and is finite. Since the limit exists but $f(2)$ is undefined, this is a removable discontinuity — we could "remove" it by defining $f(2) = 4$. Option B is wrong — the left and right limits are equal (both 4). Option C is wrong — the limit is finite, not infinite. Option D is wrong — $f(2)$ is undefined.
