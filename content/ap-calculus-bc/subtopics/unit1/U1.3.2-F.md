A continuous function $f$ has $f(1) = 3$ and $f(5) = -1$. Which statement is guaranteed by the Intermediate Value Theorem?

## Options
A) $f(x) = 0$ for some $x$ in $(1, 5)$.
B) $f(x) = 4$ for some $x$ in $(1, 5)$.
C) $f(x) = -2$ for some $x$ in $(1, 5)$.
D) $f$ has a maximum on $(1, 5)$.

## Answer
A

## Explanation
The IVT states that if $f$ is continuous on $[a, b]$, then for any value $k$ between $f(a)$ and $f(b)$, there exists $c \in (a, b)$ such that $f(c) = k$. Here $f(1) = 3$ and $f(5) = -1$, so $f$ takes all values between $-1$ and $3$. Since $0$ is between $-1$ and $3$, there exists $c \in (1, 5)$ with $f(c) = 0$. Option B (4) is outside the range $[-1, 3]$. Option C (-2) is also outside. Option D is about EVT (Extreme Value Theorem), not IVT.
