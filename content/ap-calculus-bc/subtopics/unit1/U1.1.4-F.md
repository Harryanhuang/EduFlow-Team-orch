Which of the following describes a situation where a limit does NOT exist?

## Options
A) $\lim_{x \to 0} \frac{1}{x^2} = +\infty$
B) $\lim_{x \to 0} \frac{1}{x^2}$ does not exist (the function grows without bound).
C) $\lim_{x \to 0} \frac{1}{x^2} = 0$
D) $\lim_{x \to 0} \frac{1}{x^2} = \frac{1}{0}$

## Answer
B

## Explanation
As $x \to 0$, $\frac{1}{x^2} \to +\infty$. In AP Calculus, saying a limit "equals infinity" is shorthand — technically, the limit does not exist because the function grows without bound (it doesn't approach a finite number). Option A writes the shorthand notation correctly but calls it "=" which can be misleading. Option C (0) is wrong — 1/x² grows, not shrinks. Option D (1/0) is not valid mathematics. The precise statement: the limit does not exist because the function is unbounded.
