A function $f$ satisfies $\lim_{x \to 1} f(x) = 4$ and $f(1) = 6$. Which statement is correct?

## Options
A) $f$ must be continuous at $x = 1$.
B) $f$ has a removable discontinuity at $x = 1$.
C) $\lim_{x \to 1} f(x)$ does not exist because $f(1) \neq 4$.
D) The function $f$ is not defined near $x = 1$.

## Answer
B

## Explanation
The limit exists ($\lim_{x \to 1} f(x) = 4$) but $f(1) = 6 \neq 4$. This is a removable discontinuity — the function has a "hole" that could be filled by redefining $f(1) = 4$. Option A is wrong — for continuity, we need $f(1) = \lim f(x) = 4$, but $f(1) = 6$. Option C is wrong — the limit exists regardless of $f(1)$. Option D is wrong — the limit existing implies $f$ is defined near 1 (though possibly not AT 1).
