A student argues: "$f(0) = -1$ and $f(3) = 2$, so by IVT, $f(x) = 0$ for exactly one $x$ in $(0, 3)$." What is the error?

## Options
A) IVT cannot be applied because the values -1 and 2 have different signs.
B) IVT only guarantees existence, not uniqueness — there could be multiple zeros.
C) IVT requires $f(0)$ and $f(3)$ to have the same sign.
D) There is no error — the student's argument is correct.

## Answer
B

## Explanation
The IVT guarantees that $f(x) = 0$ for AT LEAST one $x \in (0, 3)$ (assuming continuity), but it does NOT guarantee exactly one. The function could cross zero multiple times. For example, a continuous function could go from -1, cross zero, go up to 5, come back down to cross zero again, then go back up to 2 — giving two zeros. The student's error is claiming uniqueness ("exactly one") when IVT only provides existence. Option A is wrong — different signs is exactly when IVT applies (0 is between -1 and 2). Option C is wrong — IVT needs different signs to guarantee a zero crossing. Option D is wrong — the uniqueness claim is an error. Note: the argument also implicitly assumes continuity, which wasn't stated.
