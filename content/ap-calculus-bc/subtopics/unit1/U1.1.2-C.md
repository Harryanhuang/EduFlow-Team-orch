The graph of $f$ shows: as $x$ approaches $-1$ from the left, the curve goes to $+\infty$; as $x$ approaches $-1$ from the right, the curve approaches $3$. Which statement is correct?

## Options
A) $\lim_{x \to -1} f(x) = 3$
B) $\lim_{x \to -1} f(x) = +\infty$
C) $\lim_{x \to -1^-} f(x) = +\infty$ and $\lim_{x \to -1^+} f(x) = 3$, so $\lim_{x \to -1} f(x)$ does not exist.
D) $f(-1) = 3$

## Answer
C

## Explanation
The left-hand limit is $+\infty$ (diverges) and the right-hand limit is 3. Since these are not equal, the two-sided limit does not exist. Option A is wrong — that's only the right-hand limit. Option B is wrong — that's only the left-hand behavior, and $+\infty$ means the limit doesn't exist in that direction. Option D is wrong — the graph behavior near $x=-1$ doesn't determine $f(-1)$. Note: even if both sides approached finite but different values, the two-sided limit still wouldn't exist.
