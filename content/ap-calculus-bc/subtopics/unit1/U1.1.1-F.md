Which of the following best describes the meaning of $\lim_{x \to 2} f(x) = 7$?

## Options
A) $f(2) = 7$
B) As $x$ approaches 2, $f(x)$ approaches 7.
C) The function $f$ is continuous at $x = 2$.
D) The function $f$ is defined only near $x = 2$.

## Answer
B

## Explanation
$\lim_{x \to 2} f(x) = 7$ means that as $x$ gets arbitrarily close to 2 (from either side), $f(x)$ gets arbitrarily close to 7. This describes the behavior of $f$ near $x = 2$, not at $x = 2$. Option A is wrong — the limit existing does not guarantee $f(2) = 7$ or even that $f(2)$ is defined. Option C is wrong — continuity requires $f(2) = \lim f(x)$. Option D is wrong — the function can be defined everywhere.
