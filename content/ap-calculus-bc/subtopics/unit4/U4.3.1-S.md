The side length of a square is measured as $10$ cm with a possible error of $\pm0.1$ cm. Use differentials to estimate the maximum possible error in the computed area.

## Options
A) $0.01$ cm²

B) $0.1$ cm²

C) $2$ cm²

D) $20$ cm²

## Answer
C

## Explanation
$A=s^2$, so $dA=2s\,ds$. With $s=10$ and $ds=0.1$: $dA=2(10)(0.1)=2$ cm². Option A uses $ds^2$. Option B uses only $ds$. Option D uses $ds=1$ cm.
