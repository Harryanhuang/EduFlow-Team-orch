A particle moves with velocity $v(t)=t^2-5t+6$. At which time(s) does the particle change direction?

## Options
A) $t=0$ only

B) $t=2$ and $t=3$

C) $t=\frac{5}{2}$ only

D) The particle never changes direction

## Answer
B

## Explanation
A particle changes direction when velocity changes sign, which occurs at zeros of velocity where the sign actually changes. $v(t)=(t-2)(t-3)$, so zeros at $t=2$ and $t=3$. For $t<2$: $v>0$; for $2<t<3$: $v<0$; for $t>3$: $v>0$. The velocity changes sign at both $t=2$ and $t=3$, so the particle changes direction at both times. Option A is the initial time. Option C is where acceleration is zero, not velocity. Option D is false.
