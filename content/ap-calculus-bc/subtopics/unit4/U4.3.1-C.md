Let $f$ be a function with $f(2)=5$ and $f'(2)=-3$. If $f$ is concave down on an interval containing $x=2$, which statement about the linearization $L(x)$ at $x=2$ is true for $x>2$ in that interval?

## Options
A) $L(x)>f(x)$

B) $L(x)<f(x)$

C) $L(x)=f(x)$

D) The relationship cannot be determined

## Answer
A

## Explanation
If $f$ is concave down, the graph of $f$ lies below its tangent lines. Therefore, for $x$ near $2$ (with $x\neq2$), $f(x)< L(x)$, which means $L(x)>f(x)$. Option B would be true if $f$ were concave up. Option C is only true at the point of tangency. Option D is incorrect because concavity determines the relationship.
