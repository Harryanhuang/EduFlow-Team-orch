On which interval does the Extreme Value Theorem guarantee that $f(x)=\frac{1}{x}$ has both a maximum and minimum value?

## Options
A) $(0,2)$

B) $[-1,1]$

C) $[1,4]$

D) $(-\infty,\infty)$

## Answer
C

## Explanation
EVT requires: continuous function on a closed, bounded interval. $f(x)=\frac{1}{x}$ is discontinuous at $x=0$. Option A: open interval, not closed. Option B: interval contains $x=0$ where $f$ is undefined. Option D: unbounded interval. Option C: $f$ is continuous on the closed interval $[1,4]$, so EVT applies.
