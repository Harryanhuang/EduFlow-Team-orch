Let $f(x)=x^2$ on $[0,3]$. What value of $c$ guaranteed by the Mean Value Theorem satisfies $f'(c)=\frac{f(3)-f(0)}{3-0}$?

## Options
A) $c=\frac{3}{2}$

B) $c=3$

C) $c=1$

D) $c=0$

## Answer
A

## Explanation
$f'(c)=2c$. $\frac{f(3)-f(0)}{3}=\frac{9}{3}=3$. Set $2c=3$, so $c=\frac{3}{2}$. This is in $(0,3)$, so MVT is satisfied. Option B is an endpoint (not in open interval). Option C gives $f'(1)=2\neq3$. Option D is an endpoint.
