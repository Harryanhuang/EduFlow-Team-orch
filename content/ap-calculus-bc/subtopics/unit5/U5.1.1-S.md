Suppose $f$ is differentiable on $[1,5]$ and $|f'(x)|\leq 2$ for all $x\in[1,5]$. If $f(1)=3$, which of the following is a possible value for $f(5)$?

## Options
A) $11$

B) $15$

C) $8$

D) $20$

## Answer
C

## Explanation
By MVT: $|f(5)-f(1)|=|f'(c)|\cdot|5-1|\leq 2\cdot4=8$. So $-8\leq f(5)-3\leq 8$, meaning $-5\leq f(5)\leq 11$. Possible values of $f(5)$ lie in $[-5,11]$. Option A ($11$) is the boundary, possible but extreme. Option B ($15$) and D ($20$) exceed the bound. Option C ($8$) is well within $[-5,11]$, so it is a valid possible value.
