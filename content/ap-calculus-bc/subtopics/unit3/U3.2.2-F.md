If $x^2+y^2=1$, it is known that $\dfrac{dy}{dx}=-\dfrac{x}{y}$. What is $\dfrac{d^2y}{dx^2}$?

## Options
A) $\frac{y^2+x^2}{y^3}$

B) $-\frac{y^2+x^2}{y^3}$

C) $\frac{1}{y^2}$

D) $-\frac{x}{y^2}$

## Answer
B

## Explanation
Differentiate $\frac{dy}{dx}=-\frac{x}{y}$ with respect to $x$ using the quotient rule: $\frac{d^2y}{dx^2}=-\frac{y\cdot1-x\cdot\frac{dy}{dx}}{y^2}=-\frac{y-x(-\frac{x}{y})}{y^2}=-\frac{y+\frac{x^2}{y}}{y^2}=-\frac{y^2+x^2}{y^3}$. Since $x^2+y^2=1$, this also equals $-\frac{1}{y^3}$. Option A has the wrong sign. Option C forgets the quotient rule structure. Option D is an incorrect first derivative.
