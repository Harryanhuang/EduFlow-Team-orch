The curve is defined by $x^2+y^2=25$. At the point $(3,4)$, which statement about concavity is correct?

## Options
A) Concave up because $\frac{d^2y}{dx^2}>0$

B) Concave down because $\frac{d^2y}{dx^2}<0$

C) Neither because $\frac{d^2y}{dx^2}=0$

D) Concavity cannot be determined from the second derivative

## Answer
B

## Explanation
From the first derivative $\frac{dy}{dx}=-\frac{x}{y}$, the second derivative is $\frac{d^2y}{dx^2}=-\frac{y^2+x^2}{y^3}$. At $(3,4)$: $\frac{d^2y}{dx^2}=-\frac{16+9}{64}=-\frac{25}{64}<0$. The curve is concave down at this point. Option A has the wrong sign. Option C is incorrect. Option D is false — the sign of the second derivative determines concavity.
