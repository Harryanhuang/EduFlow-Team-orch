Let $g(x)=\ln x$ for $x>0$. What is $\left(g^{-1}\right)'(1)$?

## Options
A) $1$

B) $e$

C) $\frac{1}{e}$

D) $0$

## Answer
B

## Explanation
$g^{-1}(x)=e^x$, so $(g^{-1})'(x)=e^x$. Thus $(g^{-1})'(1)=e$. Alternatively, using the inverse function theorem: $g^{-1}(1)=e$ and $g'(e)=\frac{1}{e}$, so $(g^{-1})'(1)=\frac{1}{1/e}=e$. Option A is the value of $g^{-1}(0)=1$, not the derivative. Option C is $\frac{1}{g'(1)}$ but $g^{-1}(1)\neq1$. Option D is $g^{-1}(1)=\ln e=1$? No, $g^{-1}(1)=e^1=e$, and $\ln 1=0$ is unrelated. Option D is the value of $\ln 1$, not the derivative.
