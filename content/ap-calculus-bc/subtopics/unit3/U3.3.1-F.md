Let $f(x)=x^3+1$. If $f^{-1}$ is the inverse of $f$, what is $\left(f^{-1}\right)'(9)$?

## Options
A) $\frac{1}{12}$

B) $\frac{1}{27}$

C) $12$

D) $3$

## Answer
A

## Explanation
First find $f^{-1}(9)$: solve $x^3+1=9$, so $x^3=8$, $x=2$. Then $f'(x)=3x^2$, so $f'(2)=12$. By the inverse function theorem: $(f^{-1})'(9)=\frac{1}{f'(2)}=\frac{1}{12}$. Option B computes $\frac{1}{f'(3)}=\frac{1}{27}$, incorrectly using the input point. Option C inverts the formula. Option D confuses $f^{-1}(9)$ with the derivative.
