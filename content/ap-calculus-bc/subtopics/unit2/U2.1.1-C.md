Let $f(x)=\dfrac{1}{x+2}$. Using the limit definition, what is $f'(1)$?

## Options
A) $-\dfrac{1}{3}$

B) $-\dfrac{1}{9}$

C) $\dfrac{1}{9}$

D) $-\dfrac{1}{4}$

## Answer
B

## Explanation
$f'(x)=\lim_{h\to0}\frac{\frac{1}{x+h+2}-\frac{1}{x+2}}{h}=\lim_{h\to0}\frac{(x+2)-(x+h+2)}{h(x+h+2)(x+2)}=\lim_{h\to0}\frac{-h}{h(x+h+2)(x+2)}=\frac{-1}{(x+2)^2}$. Evaluate at $x=1$: $f'(1)=\frac{-1}{(1+2)^2}=\frac{-1}{9}$. Option A omits squaring the denominator. Option C has the wrong sign. Option D uses $(x+2)^1$ instead of $(x+2)^2$.
