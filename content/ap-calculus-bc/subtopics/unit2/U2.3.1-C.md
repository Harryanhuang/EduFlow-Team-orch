Let $f(x)=\dfrac{x^2 e^x}{x+1}$. Using the quotient rule with $u=x^2e^x$ and $g=x+1$, which expression equals $f'(x)$?

## Options
A) $\dfrac{(2xe^x+x^2e^x)(x+1)-x^2e^x}{(x+1)^2}$

B) $\dfrac{(2xe^x+x^2e^x)(x+1)+x^2e^x}{(x+1)^2}$

C) $\dfrac{x^2e^x(2+x)}{x+1}$

D) $\dfrac{(2x+x^2)e^x-(x^2e^x)}{(x+1)^2}$

## Answer
A

## Explanation
Let $u=x^2e^x$ and $g=x+1$, so $f=u/g$. By quotient rule: $f'=\frac{u'g-ug'}{g^2}$. We need $u'=(x^2)'e^x+x^2(e^x)'=2xe^x+x^2e^x=e^x(2x+x^2)$. And $g'=1$. So $f'=\frac{(2xe^x+x^2e^x)(x+1)-x^2e^x\cdot1}{(x+1)^2}$. Option B uses addition instead of subtraction in the numerator. Option C is an incorrect simplification that omits the quotient rule denominator. Option D is missing the $(x+1)$ multiplier on the first term.
