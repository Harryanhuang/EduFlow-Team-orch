# Items: 1.2 — Classification: kingdoms, binomial system

### Question Q-1.2-01
**Difficulty**: Foundation
**Question**: What is the binomial system of naming organisms?
**Answer**: The binomial system gives each species a two-part scientific name: the genus name (capitalised) followed by the species name (lowercase). For example, Homo sapiens. Names are written in italics or underlined.
**Explanation**: Genus + species. A common error is capitalising both words or reversing the order.
**Tags**: binomial, nomenclature, genus, species

### Question Q-1.2-02
**Difficulty**: Foundation
**Question**: Name the five kingdoms of classification.
**Answer**: Animalia, Plantae, Fungi, Protoctista, Prokaryota. Organisms are grouped into these five kingdoms based on their cell structure, nutrition, and reproduction.
**Explanation**: Five kingdoms list. A common error is including "Monera" (old name for Prokaryota) or forgetting Protoctista.
**Tags**: five-kingdoms, Animalia, Plantae, Fungi, Protoctista, Prokaryota

### Question Q-1.2-03
**Difficulty**: Standard
**Question**: Classify a named vertebrate animal into its full taxonomic hierarchy.
**Answer**: Human: Kingdom = Animalia, Phylum = Chordata, Class = Mammalia, Order = Primates, Family = Hominidae, Genus = Homo, Species = sapiens. The full classification gets more specific from kingdom to species.
**Explanation**: K-P-C-O-F-G-S. A common error is missing a level or getting the order wrong.
**Tags**: taxonomy, classification, vertebrate, human

### Question Q-1.2-04
**Difficulty**: Standard
**Question**: What features are used to classify organisms into different kingdoms?
**Answer**: Cell type (prokaryotic vs eukaryotic), cell wall presence, mode of nutrition (autotrophic vs heterotrophic), number of cells (unicellular vs multicellular), and type of reproduction. These features distinguish the five kingdoms.
**Explanation**: Cell type, nutrition, structure. A common error is using habitat as a classification feature.
**Tags**: classification, features, cell-type, nutrition

### Question Q-1.2-05
**Difficulty**: Standard
**Question**: Explain why the binomial system is important for scientists worldwide.
**Answer**: The binomial system provides a universal naming system so scientists can communicate clearly regardless of language. Common names vary between countries, but scientific names are the same worldwide, avoiding confusion between different species.
**Explanation**: Universal communication. A common error is saying it "organises species" — the key point is avoiding confusion.
**Tags**: binomial, importance, communication, universal

### Question Q-1.2-06
**Difficulty**: Standard
**Question**: A student finds an organism that is unicellular, has no nucleus, and has a cell wall made of peptidoglycan. Which kingdom does it belong to and why?
**Answer**: Prokaryota (bacteria). Key features: unicellular (single-celled), no nucleus (prokaryotic — DNA is not enclosed in a membrane), and cell wall containing peptidoglycan. These features distinguish it from all other kingdoms. Protoctista are eukaryotic; Fungi have chitin cell walls; Plantae are multicellular and autotrophic.
**Explanation**: Prokaryotic + peptidoglycan cell wall. A common error is placing it in Protoctista because it is unicellular.
**Tags**: Prokaryota, bacteria, peptidoglycan, identification

### Question Q-1.2-07
**Difficulty**: Challenge
**Question**: Some scientists now propose a three-domain system (Archaea, Bacteria, Eukarya) instead of the five-kingdom system. Explain one advantage and one disadvantage of the domain system compared to the kingdom system.
**Answer**: Advantage: The domain system is based on molecular evidence (rRNA sequences) which shows Archaea are more closely related to Eukarya than to Bacteria — this reflects evolutionary relationships better. Disadvantage: The kingdom system is simpler and more practical for identification purposes — the domain system requires molecular data that is not available in a basic lab.
**Explanation**: Molecular evidence vs practical identification. A common error is saying one system is "correct" and the other "wrong."
**Tags**: three-domains, Archaea, molecular-evidence, comparison

### Question Q-1.2-08
**Difficulty**: Challenge
**Question**: Fungi were once classified as plants. Explain why they are now placed in their own kingdom, referring to three key differences between fungi and plants.
**Answer**: (1) Fungi are heterotrophic (saprophytic/parasitic) — they absorb nutrients from dead or living organic matter. Plants are autotrophic (photosynthesis). (2) Fungi have cell walls made of chitin. Plants have cell walls made of cellulose. (3) Fungi store carbohydrate as glycogen. Plants store carbohydrate as starch. These fundamental biochemical and nutritional differences justify a separate kingdom.
**Explanation**: Chitin/heterotrophic/glycogen. A common error is saying fungi are "just plants that don't photosynthesise."
**Tags**: Fungi, Plantae, chitin, heterotrophic, glycogen, classification

### Question Q-1.2-09
**Difficulty**: Challenge
**Question**: Two organisms belong to the same genus. Explain what this tells you about their evolutionary relationship and their physical similarities. Can organisms in the same genus look very different?
**Answer**: Same genus means they share a recent common ancestor and have many structural and genetic similarities. However, organisms in the same genus CAN look quite different due to adaptive radiation — different species may occupy different niches and evolve different physical features. For example, dogs and wolves are both Canis but look quite different due to domestication selection.
**Explanation**: Recent common ancestor but possible divergence. A common error is assuming same genus = looks identical.
**Tags**: genus, evolution, adaptive-radiation, Canis
