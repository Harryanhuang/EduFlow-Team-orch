### Question Q-3.1-08
**Difficulty**: Standard
**Question**: How does the thickness of a membrane affect the rate of diffusion?
**Answer**: A thinner membrane provides a shorter diffusion distance, increasing the rate of diffusion. This is why exchange surfaces (alveoli walls, capillary walls, villi walls) are only one cell thick. Doubling the thickness approximately halves the rate of diffusion.
**Explanation**: Thinner = shorter distance = faster. A common error is saying thickness has "no effect."
**Tags**: diffusion, membrane-thickness, distance, rate

### Question Q-4.2-06
**Difficulty**: Standard
**Question**: Describe the test for vitamin C in a food sample.
**Answer**: Add DCPIP solution (blue) drop by drop to the food sample (or sample juice). If vitamin C is present, the blue DCPIP is decolorised (turns colourless). The more vitamin C present, the fewer drops of DCPIP are needed to be decolorised. A negative result remains blue.
**Explanation**: DCPIP → decolorised. A common error is using iodine (that's for starch).
**Tags**: vitamin-C-test, DCPIP, decolorised

### Question Q-5.2-06
**Difficulty**: Standard
**Question**: What happens to the rate of an enzyme-controlled reaction if enzyme concentration is increased?
**Answer**: If enzyme concentration is increased (with excess substrate), the rate increases because there are more active sites available for substrate molecules to bind to. The rate is directly proportional to enzyme concentration, provided substrate is not limiting. Eventually, if substrate becomes limiting, the rate levels off.
**Explanation**: More enzymes = more active sites = faster rate. A common error is saying adding more enzyme has "no effect."
**Tags**: enzyme-concentration, rate, substrate, active-sites
