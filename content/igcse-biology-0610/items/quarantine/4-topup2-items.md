### Question Q-4.1-09
**Difficulty**: Standard
**Question**: What is the test for lipids and how does it differ from the test for proteins?
**Answer**: Lipid test (emulsion test): add ethanol, shake, add water — cloudy white emulsion = positive. Protein test (Biuret test): add Biuret solution — purple/violet = positive. The tests use different reagents and produce different colour changes. Both tests detect different types of biological molecules.
**Explanation**: Emulsion vs Biuret, different reagents, different colours. A common error is using the same test for both.
**Tags**: lipid-test, protein-test, emulsion, Biuret

### Question Q-6.2-07
**Difficulty**: Standard
**Question**: Why do greenhouse growers monitor both temperature and CO₂ levels?
**Answer**: Both temperature and CO₂ affect the rate of photosynthesis. If temperature is optimum but CO₂ is low, CO₂ becomes the limiting factor and the rate plateaus. If CO₂ is high but temperature is too low, the rate is also limited. Growers must optimise both (and light) to maximise photosynthesis and crop yield. Increasing one alone has limited effect.
**Explanation**: Both are limiting factors, must optimise together. A common error is focusing on only one factor.
**Tags**: greenhouse, temperature, CO2, limiting-factors, photosynthesis
