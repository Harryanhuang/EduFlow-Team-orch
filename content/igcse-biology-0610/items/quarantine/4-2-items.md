# Items: 4.2 — Food tests and DNA structure

### Question Q-4.2-01
**Difficulty**: Foundation
**Question**: What is the test for starch in a food sample?
**Answer**: Add a few drops of iodine solution (iodine dissolved in potassium iodide). If starch is present, the colour changes from brown/orange to blue-black. A negative result remains brown/orange.
**Explanation**: Iodine → blue-black. A common error is saying iodine turns purple.
**Tags**: starch-test, iodine, food-test

### Question Q-4.2-02
**Difficulty**: Foundation
**Question**: What are genes and where are they found?
**Answer**: Genes are sections of DNA that code for specific proteins. They are found on chromosomes, which are located in the nucleus of eukaryotic cells. Each gene contains the instructions for making a specific protein.
**Explanation**: Gene = DNA section → protein. A common error is saying genes "are" chromosomes.
**Tags**: genes, DNA, chromosomes, nucleus

### Question Q-4.2-03
**Difficulty**: Standard
**Question**: Describe how to test for reducing sugars (e.g., glucose) in a food sample.
**Answer**: Add Benedict's solution to the sample and heat in a water bath at 80-90°C for 5 minutes. A positive result shows a colour change from blue → green → yellow → orange → brick-red precipitate. Blue = no reducing sugar. The more reducing sugar present, the more dramatic the colour change.
**Explanation**: Benedict's + heat → brick-red. A common error is forgetting to heat the solution.
**Tags**: sugar-test, Benedict's, reducing-sugar

### Question Q-4.2-04
**Difficulty**: Standard
**Question**: Describe how to test for protein in a food sample.
**Answer**: Add Biuret solution (sodium hydroxide + copper sulfate). If protein is present, the colour changes from blue to purple/violet. A negative result remains blue. The Biuret test detects peptide bonds between amino acids.
**Explanation**: Biuret → purple. A common error is confusing Biuret with Benedict's test.
**Tags**: protein-test, Biuret, colour-change

### Question Q-4.2-05
**Difficulty**: Standard
**Question**: Describe the structure of DNA.
**Answer**: DNA is a double helix made of two polynucleotide strands twisted around each other. Each strand has a sugar-phosphate backbone with nitrogenous bases attached. The two strands are held together by hydrogen bonds between complementary base pairs: Adenine (A) pairs with Thymine (T); Cytosine (C) pairs with Guanine (G).
**Explanation**: Double helix, sugar-phosphate backbone, A-T, C-G. A common error is saying A pairs with C.
**Tags**: DNA, double-helix, base-pairs, structure

### Question Q-4.2-06
**Difficulty**: Standard
**Question**: Describe how to test for lipids (fats) in a food sample.
**Answer**: Emulsion test: crush the food sample, add ethanol, shake vigorously. Then add water and shake again. If lipids are present, a cloudy white emulsion forms. A negative result remains clear. This works because lipids are soluble in ethanol but insoluble in water — adding water causes lipids to come out of ethanol solution as tiny droplets that scatter light.
**Explanation**: Ethanol + water → cloudy emulsion. A common error is using water only.
**Tags**: lipid-test, emulsion-test, ethanol

### Question Q-4.2-07
**Difficulty**: Challenge
**Question**: A student finds that two different food samples both give negative results with Benedict's test, but one of them is known to contain sucrose. Explain why the result is negative and describe how the student could modify the test to detect the sucrose.
**Answer**: Sucrose is a non-reducing sugar — it does not react directly with Benedict's reagent. The student should (1) first boil a separate sample of the food with dilute hydrochloric acid (HCl) to hydrolyse the sucrose into glucose + fructose, (2) then neutralise with sodium hydrogen carbonate, (3) then perform the Benedict's test with heating. A positive result (brick-red) confirms the presence of non-reducing sugar. The other sample (still negative after hydrolysis) contains no sugar at all.
**Explanation**: Non-reducing sugar → acid hydrolysis → reducing sugars → positive Benedict's. A common error is skipping the neutralisation step.
**Tags**: sucrose, non-reducing-sugar, hydrolysis, Benedict's

### Question Q-4.2-08
**Difficulty**: Challenge
**Question**: A student is given four test tubes containing unknown solutions: one with starch, one with glucose, one with protein, and one with lipid. Describe a systematic sequence of tests to identify each solution, using only the standard food test reagents.
**Answer**: Step 1 — Iodine test on each: blue-black = starch. Step 2 — Benedict's test on remaining 3 (heat): brick-red = glucose. Step 3 — Emulsion test on remaining 2 (ethanol + water): cloudy white = lipid. Step 4 — Biuret test on last (NaOH + CuSO₄): purple = protein. This sequence avoids interference between tests and uses each reagent for its specific target molecule.
**Explanation**: Sequential testing, each test specific. A common error is performing the tests in a random order and getting false positives.
**Tags**: food-tests, systematic, identification, practical

### Question Q-4.2-09
**Difficulty**: Challenge
**Question**: Explain how DNA's structure relates to its function of storing genetic information. Why is the double helix structure important?
**Answer**: (1) The sequence of the four bases (A, T, C, G) along the DNA strand is the genetic code — this stores information (like letters in a book). (2) The sugar-phosphate backbone provides structural stability and protects the bases inside. (3) The double helix with complementary base pairing (A-T, C-G) means each strand can serve as a template for copying — when DNA replicates, the strands separate and each builds a new complementary strand, ensuring faithful information copying. (4) Hydrogen bonds between bases allow the strands to separate for replication without breaking covalent bonds, making the structure both stable and accessible.
**Explanation**: Base sequence = code; complementarity = copying; H-bonds = access. A common error is saying DNA "directly makes proteins" without explaining the genetic code.
**Tags**: DNA, double-helix, genetic-code, replication, complementary-base-pairing
