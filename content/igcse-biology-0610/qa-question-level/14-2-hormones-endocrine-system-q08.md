# Question: Question Q-14.2-08
**Difficulty**: Challenge
**Question**: Explain how the pancreas acts as both an endocrine and an exocrine organ.
**Answer**: Exocrine (~99%): pancreatic acinar cells secrete digestive enzymes (amylase, protease, lipase) + NaHCO₃ into duct system → pancreatic duct → duodenum. Aids digestion. Endocrine (~1%): islets of Langerhans; α-cells → glucagon (↑glucose); β-cells → insulin (↓glucose). Hormones enter blood directly. Dual function coordinates digestion AND glucose homeostasis after a meal. Failure: exocrine → malnutrition; endocrine → diabetes. "Gland" is not binary — some organs serve both functions.
**Explanation**: Exocrine = enzymes via duct; Endocrine = insulin/glucagon via blood. A common error is treating pancreas as only endocrine.
