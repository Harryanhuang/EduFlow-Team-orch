# Question: Question Q-1.2-06
**Difficulty**: Standard
**Question**: A student finds an organism that is unicellular, has no nucleus, and has a cell wall made of peptidoglycan. Which kingdom does it belong to and why?
**Answer**: Prokaryota (bacteria). Key features: unicellular (single-celled), no nucleus (prokaryotic — DNA is not enclosed in a membrane), and cell wall containing peptidoglycan. These features distinguish it from all other kingdoms. Protoctista are eukaryotic; Fungi have chitin cell walls; Plantae are multicellular and autotrophic.
**Explanation**: Prokaryotic + peptidoglycan cell wall. A common error is placing it in Protoctista because it is unicellular.
