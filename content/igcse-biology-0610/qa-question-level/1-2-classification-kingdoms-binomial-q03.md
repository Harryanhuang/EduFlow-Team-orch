# Question: Question Q-1.2-03
**Difficulty**: Standard
**Question**: Classify a named vertebrate animal into its full taxonomic hierarchy.
**Answer**: Human: Kingdom = Animalia, Phylum = Chordata, Class = Mammalia, Order = Primates, Family = Hominidae, Genus = Homo, Species = sapiens. The full classification gets more specific from kingdom to species.
**Explanation**: K-P-C-O-F-G-S. A common error is missing a level or getting the order wrong.
