# Question: Question Q-18.2-02
**Difficulty**: Standard
**Question**: What is genetic engineering? Describe the basic steps.
**Answer**: Genetic engineering is the transfer of a gene from one organism to another to give the recipient a new characteristic. Steps: (1) Cut the required gene from DNA using restriction enzymes. (2) Cut a vector (plasmid or virus) with the same enzyme. (3) Join the gene and vector using DNA ligase. (4) Insert the vector into the host cell. (5) The host cell expresses the new gene.
**Explanation**: Cut → join → insert → express. A common error is saying the gene replaces the host's own genes.
