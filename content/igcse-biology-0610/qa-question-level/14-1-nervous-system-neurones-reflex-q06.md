# Question: Question Q-14.1-06
**Difficulty**: Standard
**Question**: What is a synapse and how does it ensure one-directional impulse transmission?
**Answer**: Synapse = junction between two neurones (or neurone + effector). Presynaptic membrane → synaptic cleft (~20 nm) → postsynaptic membrane. Mechanism: (1) impulse → voltage-gated Ca²⁺ channels open → Ca²⁺ enters presynaptic terminal. (2) Ca²⁺ triggers neurotransmitter vesicles to fuse and release transmitter (e.g., acetylcholine) by exocytosis. (3) Transmitter diffuses across cleft → binds to specific receptors on postsynaptic membrane. (4) Ligand-gated Na⁺ channels open → Na⁺ enters → depolarisation → new impulse. One-direction only because: vesicles + transmitter are ONLY in presynaptic terminal; receptors are ONLY on postsynaptic membrane. Asymmetric distribution ensures impulse cannot travel backwards.
**Explanation**: Ca²⁺ → vesicle → transmitter → receptor → Na⁺ → impulse. One-direction due to asymmetric vesicle/receptor distribution. A common error is saying impulses "jump."
