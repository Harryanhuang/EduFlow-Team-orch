# Question: Question Q-1.2-04
**Difficulty**: Standard
**Question**: What features are used to classify organisms into different kingdoms?
**Answer**: Cell type (prokaryotic vs eukaryotic), cell wall presence, mode of nutrition (autotrophic vs heterotrophic), number of cells (unicellular vs multicellular), and type of reproduction. These features distinguish the five kingdoms.
**Explanation**: Cell type, nutrition, structure. A common error is using habitat as a classification feature.
**Tags**: classification, features, cell-type, nutrition
