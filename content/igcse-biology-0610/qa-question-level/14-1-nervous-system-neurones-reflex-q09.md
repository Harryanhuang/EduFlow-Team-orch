# Question: Question Q-14.1-09
**Difficulty**: Challenge
**Question**: A person steps on a sharp pin barefoot. Describe the full nervous system response distinguishing reflex action and conscious sensation.
**Answer**: Reflex (~0.1s, spinal, involuntary): Pin → pain receptors (nociceptors) in sole → sensory neurone (sciatic nerve → dorsal root → spinal cord) → relay neurone (grey matter) → motor neurone (ventral root) → leg muscles (hamstring contracts, quadriceps relaxes) → foot lifts off pin. Conscious sensation (~0.5s, cerebral): same sensory neurone sends branches UP spinal cord (ascending tracts) → thalamus → somatosensory cortex → brain registers "sharp pain in foot." Voluntary response (~1s): motor cortex → corticospinal tract → spinal cord → motor neurones → balance adjustment, look down, remove pin. Spinal reflex is fast and unconscious; conscious pain + voluntary response are slower. Dual pathway separates speed and awareness.
**Explanation**: Spinal reflex ~0.1s vs cortical pain ~0.5s + voluntary response. A common error is treating reflex and sensation as simultaneous.
