# Question: Question Q-13.1-03
**Difficulty**: Standard
**Question**: Name the main excretory organs in humans and state what each removes.
**Answer**: (1) Lungs — remove CO₂ (exhaled). (2) Kidneys — remove urea (from liver deamination), excess water, excess mineral ions (urine). (3) Skin — removes water, salts, trace urea via sweat (also thermoregulation). (4) Liver — deaminates excess amino acids → urea (sent to kidneys); breaks down Hb from old RBCs → bile pigments. The liver is NOT directly an excretory organ — it processes; kidneys excrete.
**Explanation**: Four organs with specific waste products. A common error is saying liver "excretes" directly.
