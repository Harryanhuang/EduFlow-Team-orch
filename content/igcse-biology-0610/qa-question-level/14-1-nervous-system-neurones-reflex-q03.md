# Question: Question Q-14.1-03
**Difficulty**: Standard
**Question**: Describe the pathway of a simple reflex arc using the withdrawal reflex as an example.
**Answer**: Hot object → thermoreceptors/nociceptors in skin → sensory neurone (via dorsal root into spinal cord) → relay neurone (grey matter) → motor neurone (via ventral root out of spinal cord) → biceps muscle contracts → hand pulled away. Action occurs BEFORE brain registers pain (~0.1s vs ~0.5s conscious awareness). Brain informed after action via ascending sensory tracts.
**Explanation**: Full arc with anatomical details. A common error is including brain in the reflex pathway.
