# Question: Question Q-10.2-02
**Difficulty**: Foundation
**Question**: What is the difference between active and passive immunity?
**Answer**: Active immunity: the body produces its own antibodies after exposure to an antigen (through infection or vaccination). It takes days to develop but provides long-lasting protection because memory lymphocytes are produced. Passive immunity: antibodies are acquired ready-made from another source (mother to baby via placenta/breast milk, or antibody injection). It provides immediate protection but is short-lived (weeks-months) because no memory cells are produced.
**Explanation**: Body makes own antibodies + memory (long) vs receives antibodies (short). A common error is saying passive immunity involves memory cells.
