# Question: Question Q-14.1-08
**Difficulty**: Challenge
**Question**: Explain the significance of the myelin sheath. What happens when myelin is damaged (e.g., multiple sclerosis)?
**Answer**: Myelin: (1) Insulation — prevents ion leakage. (2) Saltatory conduction — action potentials only at nodes → impulse "jumps" between nodes → conduction velocity 1→100 m/s. (3) Energy efficiency — fewer Na⁺/K⁺ pumps needed. MS: immune system attacks myelin → scar tissue (sclerosis) → demyelination → current leaks between nodes → conduction slowed/blocked → axons may degenerate → muscle weakness, numbness, vision loss, fatigue. No cure — treatments suppress immune attack. Myelin is not just insulation — it enables normal nervous function.
**Explanation**: Myelin → saltatory conduction → speed + efficiency. MS → demyelinated → blocked. A common error is describing myelin without saltatory mechanism.
