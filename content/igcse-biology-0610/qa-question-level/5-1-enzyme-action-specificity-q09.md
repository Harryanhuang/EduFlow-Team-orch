# Question: Question Q-5.1-09
**Difficulty**: Challenge
**Question**: Some drugs work by inhibiting enzymes. Design an explanation for how an inhibitor with a shape similar to the substrate could reduce the rate of an enzyme-controlled reaction, using the "lock and key" model.
**Answer**: A competitive inhibitor has a shape similar enough to the substrate to fit into the enzyme's active site. When the inhibitor binds, it blocks the active site — the real substrate cannot enter. The inhibitor does not react (no product is formed), so the active site is occupied uselessly. This reduces the reaction rate because fewer active sites are available for the real substrate. The effect depends on the relative concentrations of inhibitor and substrate — if enough substrate is present, it can outcompete the inhibitor. This is "competitive inhibition."
**Explanation**: Similar shape → binds active site → blocks substrate → less product. A common error is saying the inhibitor "destroys" the enzyme.
