# Question: Question Q-14.2-02
**Difficulty**: Foundation
**Question**: What is negative feedback? Give an example.
**Answer**: Negative feedback is a control mechanism that reverses a deviation from the set point to maintain steady internal state (homeostasis). When a level deviates, the body detects this and triggers a response that returns it to normal. Example — blood glucose: high glucose → pancreas releases insulin → liver converts glucose → glycogen → glucose falls → insulin stops. Low glucose → glucagon → liver converts glycogen → glucose → glucose rises → glucagon stops. Response is always opposite to stimulus.
**Explanation**: Detects deviation → triggers opposing response → returns to set point. A common error is saying negative feedback "amplifies" change.
