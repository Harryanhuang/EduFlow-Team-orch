# Question: Question Q-1.2-01
**Difficulty**: Foundation
**Question**: What is the binomial system of naming organisms?
**Answer**: The binomial system gives each species a two-part scientific name: the genus name (capitalised) followed by the species name (lowercase). For example, Homo sapiens. Names are written in italics or underlined.
**Explanation**: Genus + species. A common error is capitalising both words or reversing the order.
