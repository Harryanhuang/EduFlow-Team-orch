# Question: Question Q-7.1-03
**Difficulty**: Standard
**Question**: Explain the effects of scurvy and rickets, and state which nutrient deficiency causes each.
**Answer**: Scurvy: caused by vitamin C deficiency. Symptoms include bleeding gums, slow wound healing, weakened blood vessel walls (easy bruising). Collagen synthesis requires vitamin C. Rickets: caused by vitamin D deficiency. Symptoms include soft, weak bones that bend (bowed legs) because calcium absorption is impaired without vitamin D. Both are deficiency diseases preventable by a balanced diet.
**Explanation**: Vit C → scurvy; Vit D → rickets. A common error is confusing which vitamin causes which disease.
