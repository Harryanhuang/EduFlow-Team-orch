# Question: Question Q-18.2-03
**Difficulty**: Standard
**Question**: How is genetic engineering used to produce human insulin?
**Answer**: The human insulin gene is cut from human DNA using restriction enzymes. It is inserted into a bacterial plasmid (cut with the same enzyme) using DNA ligase. The plasmid is inserted into bacteria. The bacteria are grown in a fermenter and produce human insulin, which is harvested and purified for use by diabetic patients.
**Explanation**: Human gene → plasmid → bacteria → insulin. A common error is saying insulin is extracted from human pancreases.
