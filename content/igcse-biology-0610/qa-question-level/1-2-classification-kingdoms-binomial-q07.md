# Question: Question Q-1.2-07
**Difficulty**: Challenge
**Question**: Some scientists now propose a three-domain system (Archaea, Bacteria, Eukarya) instead of the five-kingdom system. Explain one advantage and one disadvantage of the domain system compared to the kingdom system.
**Answer**: Advantage: The domain system is based on molecular evidence (rRNA sequences) which shows Archaea are more closely related to Eukarya than to Bacteria — this reflects evolutionary relationships better. Disadvantage: The kingdom system is simpler and more practical for identification purposes — the domain system requires molecular data that is not available in a basic lab.
**Explanation**: Molecular evidence vs practical identification. A common error is saying one system is "correct" and the other "wrong."
