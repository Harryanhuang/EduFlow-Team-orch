# Question: Question Q-4.2-03
**Difficulty**: Foundation
**Question**: Describe how to test for the presence of starch in a food sample.
**Answer**: Add a few drops of iodine solution (iodine dissolved in potassium iodide) to the food sample. If starch is present, the colour changes from brown/orange to blue-black. A negative result remains brown/orange.
**Explanation**: Iodine → blue-black. A common error is saying iodine turns purple.
