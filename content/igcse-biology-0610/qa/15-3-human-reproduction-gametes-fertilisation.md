# QA: 15.3 — Human reproduction: gametes, fertilisation and development

## Topic 名称
Human reproduction: gametes, fertilisation and development（人类生殖：配子、受精与发育）

## Topic 定义
掌握人类有性生殖的核心过程：配子（gametes）形成（精子 spermatogenesis 与卵子 oogenesis）、受精（fertilisation）、胚胎与胎儿发育、以及分娩（birth）。理解减数分裂（meiosis）如何产生遗传上不同的配子（交叉互换 + 自由组合），以及受精如何恢复二倍体并产生基因独特的后代。掌握受精后受精卵（zygote）到胚泡（blastocyst）的分裂过程、植入（implantation）子宫内膜、胎盘（placenta）与羊膜囊（amniotic sac）的结构功能区别。理解月经周期（menstrual cycle）中的激素调控（FSH/LH/oestrogen/progesterone 的反馈关系）、受精成功后的激素维持（hCG 的作用）、以及分娩的正反馈机制（oxytocin）。掌握避孕（contraception）方法及其作用原理。

## 关键知识点
- Gamete adaptations：
  - Sperm: small, motile (flagellum), many mitochondria (energy for swimming), acrosome (digestive enzymes to penetrate zona pellucida)
  - Egg: large, non-motile, cytoplasm packed with yolk nutrients, zona pellucida (species-specific sperm receptor + block to polyspermy), jelly coat
- Spermatogenesis：seminiferous tubules → primary spermatocytes → meiosis I → secondary spermatocytes → meiosis II → spermatids → mature sperm; takes ~64 days; continuous from puberty
- Oogenesis：primary oocytes begin meiosis I in foetus → arrested at prophase I → resume at ovulation → complete meiosis II only if fertilised → ovum; one mature egg per cycle
- Fertilisation (human)：copulation → sperm in vagina → through cervix/uterus to oviduct → acrosome reaction → one sperm penetrates zona pellucida → membrane fusion → zygote (diploid 2n=46); zona pellucida changes (block to polyspermy); cleavage divisions as zygote travels to uterus
- Cleavage and implantation：zygote → mitosis → morula (solid ball ~day 3) → blastocyst (hollow ball ~day 5) → hatches → implants in endometrium (~day 6–7)
- Placenta：chorionic villi in maternal blood spaces; separate maternal/foetal circulations; exchange (O₂, glucose, amino acids → foetus; CO₂, urea → mother); hormones (hCG, progesterone, oestrogen); partial barrier (blocks some pathogens, alcohol/nicotine pass through)
- Amniotic sac：membrane containing amniotic fluid; shock absorption, temperature regulation, infection barrier, allows movement for muscle development
- Birth (parturition)：oxytocin from posterior pituitary → rhythmic uterine contractions → cervical stretching → positive feedback for more oxytocin → full dilation → baby expelled → placenta expelled (afterbirth); progesterone fall triggers these events
- Contraception：abstinence / fertility awareness (keep gametes apart); condom (physical barrier); contraceptive pill (oestrogen/progesterone → inhibit FSH → no ovulation); IUD (prevents implantation)
- Teratogens：substances that cross placenta and damage foetus (e.g., alcohol → foetal alcohol syndrome; nicotine → low birth weight; rubella virus → organ defects); most critical during organogenesis (weeks 3–8)

## 常见错误
- 说受精发生在子宫（实际在 oviduct/Fallopian tube）
- 说卵子是可运动的（精子的尾是运动器官，卵子是被动接受）
- 说母体血液与胎儿血液在胎盘中混合（实际 separate circulations，跨膜交换）
- 混淆胎盘与羊膜囊的功能：胎盘 = 物质交换 + 激素，羊膜囊 = 物理缓冲
- 在描述 meiosis 时忘记交叉互换（crossing over）是遗传变异的主要来源
- 说受精后胎盘立即形成（胎盘在植入后逐渐发育）
- 把避孕药的机制说成"杀死精子"（实际是抑制 FSH 从而阻止排卵）
- 忘记 hCG 的作用是维持黄体以分泌孕酮（早孕测试检测的就是 hCG）
- 在描述 birth 时混淆 oxytocin 的作用：刺激子宫收缩，不是让胎盘脱落
