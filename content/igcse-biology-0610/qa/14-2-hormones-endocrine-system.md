# QA: 14.2 — Hormones and endocrine system

## Topic 名称
Hormones and endocrine system（激素与内分泌系统）

## Topic 定义
掌握 hormone definition (chemical messenger via bloodstream)。理解 negative feedback 的普遍闭环控制原理。掌握 insulin/glucagon 双激素拮抗调控血液 glucose 的完整机制（β-cells→insulin→glycogenesis; α-cells→glucagon→glycogenolysis/gluconeogenesis）。掌握 Nervous vs Hormonal 的比较（speed/duration/pathway/signal）。能区分 Type 1 (autoimmune β-cell destruction) 与 Type 2 (insulin resistance) diabetes。理解 pancreas dual function (exocrine—digestive enzymes via duct; endocrine—insulin/glucagon via blood)。能描述餐后 3 小时血糖 hormone timeline。

## 关键知识点
- Hormone = chemical messenger: endocrine gland → blood → target organ → specific effect
- Negative feedback: deviation → opposing response → return to set point (insulin/glucagon, ADH)
- Insulin (β-cells): high glucose → glycogenesis, glucose uptake → ↓glucose
- Glucagon (α-cells): low glucose → glycogenolysis + gluconeogenesis → ↑glucose
- Nervous vs Hormonal: ms/wired/localised/short vs s-hr/broadcast/widespread/long
- Type 1 diabetes: autoimmune β-cell loss → no insulin → injections
- Type 2 diabetes: insulin resistance → diet/exercise/drugs (metformin)
- Pancreas dual: exocrine (enzymes via duct) + endocrine (islets←hormones via blood)
- Fight-or-flight: adrenal→adrenaline→heart↑/breathing↑/glucose↑/blood to muscles

## 常见错误
- 说 hormones travel in nerves
- 说 negative feedback "amplifies"
- 颠倒 insulin 和 glucagon 的作用

- 混淆 nervous 和 hormonal speed
