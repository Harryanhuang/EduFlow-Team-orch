# QA: 10.2 — Defence mechanisms and immunity

## Topic 名称
Defence mechanisms and immunity（防御机制与免疫）

## Topic 定义
掌握人体三层防御体系——一线非特异性屏障（skin/mucus/acid/lysozyme/cilia）、二线非特异性吞噬（phagocytes）、三线特异性免疫（B/T lymphocytes + memory）。理解 active vs passive immunity 的本质区别、vaccination 的免疫学原理、herd immunity 的群体保护机制。能通过 measles vs influenza 对比阐述 immunological memory 与 pathogen mutation 的军备竞赛，以及 immunocompromised 人群的疫苗安全考量。

## 关键知识点
- 第一线：Skin (keratin + sebum), Mucus membranes + cilia, Stomach acid (pH 1-2), Lysozyme (tears/saliva)
- 第二线：Phagocytes (neutrophils, macrophages) — non-specific phagocytosis + antigen presentation
- 第三线：B-lymphocytes (antibodies) + T-lymphocytes (cell-mediated) + Memory lymphocytes
- Active immunity (own antibodies + memory, long) vs Passive immunity (received antibodies, short, no memory)
- Vaccination: dead/attenuated pathogen → lymphocyte response → antibody + memory → rapid response on re-exposure
- Herd immunity: 高接种率→传播链断裂→保护未接种者（阈值 measles ~95%, polio ~80%）
- Antibiotics vs viruses: 抗生素靶向细菌特有结构 (cell wall/70S ribosome)，病毒无此类结构
- Antigenic drift (point mutation) vs shift (segment re-assortment) → influenza 疫苗需每年更新
- 器官移植免疫排斥与免疫抑制剂：foreign HLA antigens→T-cell attack→immunosuppressants; trade-off: infection risk

## 常见错误
- 把非特异性屏障叫 "immune system"（immune = specific）
- 说 passive immunity 产生 memory cells
- 说 vaccines "contain antibodies"（含病原体/抗原，不含抗体）
- "antibiotics kill viruses"
- 说 "所有人都必须打疫苗"（忽略了 herd immunity 保护禁忌接种者）
- 混淆 antigenic drift（点突变→部分逃逸）和 shift（重配→全新株）
