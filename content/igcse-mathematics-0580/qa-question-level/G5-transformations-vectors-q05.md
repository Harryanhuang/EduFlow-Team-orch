### Question Q-G5-04
**Difficulty**: Standard
**Question**: Triangle A(1,2), B(3,2), C(2,4) is enlarged by scale factor 2, centre (0,0). Find the image coordinates.
**Answer**: A'(2,4), B'(6,4), C'(4,8). Each coordinate is multiplied by 2.
**Explanation**: Enlargement sf 2, centre (0,0): multiply each coordinate by 2. A common error is adding 2 to each coordinate.
