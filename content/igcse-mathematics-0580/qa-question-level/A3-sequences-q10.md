### Question Q-A3-09
**Difficulty**: Challenge
**Question**: The nth term of a sequence is an² + bn + c. The first three terms are 2, 6, 12. Find a, b, and c.
**Answer**: a = 1, b = 1, c = 0, so nth term = n² + n.
**Explanation**: n=1: a+b+c=2. n=2: 4a+2b+c=6. n=3: 9a+3b+c=12. Second difference = 2, so a=1. Then b=1, c=0. A common error is not using simultaneous equations.
