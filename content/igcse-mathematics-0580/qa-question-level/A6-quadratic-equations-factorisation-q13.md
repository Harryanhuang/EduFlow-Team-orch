### Question Q-A6-12
**Difficulty**: Challenge
**Question**: Find the values of k for which x² + kx + 9 = 0 has exactly one solution.
**Answer**: Discriminant = k² - 36 = 0, k² = 36, k = ±6
**Explanation**: One solution means discriminant = 0. b²-4ac = k²-36 = 0, k = ±6. A common error is k = 6 only.
