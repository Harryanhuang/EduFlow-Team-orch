### Question Q-G1-07
**Difficulty**: Standard
**Question**: A quadrilateral has angles x°, 2x°, 3x°, and 4x°. Find the value of x.
**Answer**: x + 2x + 3x + 4x = 360°, 10x = 360°, x = 36°.
**Explanation**: Sum of angles in a quadrilateral = 360°. 10x = 360, x = 36. A common error is using 180° (triangle sum).
