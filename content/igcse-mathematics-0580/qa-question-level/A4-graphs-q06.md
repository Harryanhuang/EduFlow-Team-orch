### Question Q-A4-05
**Difficulty**: Standard
**Question**: Complete the table for y = x² - 2x for x = -1, 0, 1, 2, 3.
**Answer**: x=-1: y=3, x=0: y=0, x=1: y=-1, x=2: y=0, x=3: y=3
**Explanation**: Substitute each x value: (-1)²-2(-1)=1+2=3, 0-0=0, 1-2=-1, 4-4=0, 9-6=3. A common error is (-1)² = -1.
