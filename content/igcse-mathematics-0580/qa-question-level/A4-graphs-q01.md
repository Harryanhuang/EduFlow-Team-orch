### Question Q-A4-13
**Difficulty**: Standard
**Question**: Find the equation of the line perpendicular to y = 2x + 1 passing through (4, 3).
**Answer**: Gradient = -1/2. y = (-1/2)x + c: 3 = -2 + c, c = 5. y = (-1/2)x + 5.
**Explanation**: Perpendicular gradient = -1/m. Substitute point to find c. A common error is gradient = 2.
