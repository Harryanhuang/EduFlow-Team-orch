# Question: Question Q-A7-07
**Difficulty**: Challenge
**Question**: Solve f(x) = 5 given f(x) = (2x − 1)/(x + 3). Find all values of x.
**Answer**: (2x − 1)/(x + 3) = 5 → 2x − 1 = 5(x + 3) → 2x − 1 = 5x + 15 → −3x = 16 → x = −16/3. Check: denominator ≠ 0, x ≠ −3. −16/3 ≠ −3, so valid.
**Explanation**: Cross-multiply, solve linear equation, check domain. A common error is forgetting to check the denominator ≠ 0.
