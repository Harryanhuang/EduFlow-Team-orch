# Question: Question Q-S6-01
**Difficulty**: Foundation
**Question**: What is cumulative frequency?
**Answer**: Cumulative frequency is the running total of frequencies. To find it, add each frequency to the sum of all previous frequencies.
**Explanation**: CF accumulates as you move down a grouped frequency table. A common error is restarting the sum at each class interval.
**Tags**: cumulative-frequency, statistics, data

# Question: Question Q-S6-02
**Difficulty**: Foundation
**Question**: What is a box plot and what does it show?
**Answer**: A box plot shows the minimum, lower quartile (Q1), median (Q2), upper quartile (Q3), and maximum of a data set. The box spans Q1 to Q3, with a line at the median.
**Explanation**: Five-number summary. A common error is confusing Q1 with the minimum.
**Tags**: box-plot, statistics, quartile

# Question: Question Q-S6-03
**Difficulty**: Standard
**Question**: Draw a scatter graph and describe the types of correlation.
**Answer**: Types: positive correlation (both increase), negative correlation (one increases, other decreases), no correlation. If points lie close to a line, correlation is strong; if spread out, weak.
**Explanation**: Direction and strength. A common error is confusing direction with strength.
**Tags**: scatter-graph, correlation, statistics

# Question: Question Q-S6-04
**Difficulty**: Standard
**Question**: What is the equation of the line of best fit? Give an example.
**Answer**: y = mx + c where m is the gradient and c is the y-intercept. Example: y = 0.5x + 2. The gradient shows the rate of change.
**Explanation**: Linear regression line. A common error is guessing m and c without using the data points.
**Tags**: line-of-best-fit, regression, statistics

# Question: Question Q-S6-05
**Difficulty**: Standard
**Question**: From a cumulative frequency curve, find the median and interquartile range.
**Answer**: Median = CF = n/2. Q1 = CF = n/4. Q3 = CF = 3n/4. IQR = Q3 − Q1.
**Explanation**: Read from CF curve: find n, locate CF = n/2 for median. A common error is finding the median value instead of the position.
**Tags**: cumulative-frequency, quartile, median, IQR

# Question: Question Q-S6-06
**Difficulty**: Standard
**Question**: What is a standard deviation and what does it measure?
**Answer**: Standard deviation (σ) measures the spread of data around the mean. σ = √(Σ(x − x̄)² / n for the population. A small σ means data is close to the mean.
**Explanation**: Spread measure. A common error is confusing variance with standard deviation (σ² = variance).
**Tags**: standard-deviation, statistics, spread

# Question: Question Q-S6-07
**Difficulty**: Challenge
**Question**: Two data sets have the same mean but different standard deviations. What does this tell you?
**Answer**: The data with the larger standard deviation is more spread out from the mean. Both have the same average value but the second is more variable.
**Explanation**: SD measures dispersion around the mean. A common error is saying the means are different.
**Tags**: standard-deviation, statistics, comparison

# Question: Question Q-S6-08
**Difficulty**: Challenge
**Question**: Compare a box plot and a histogram for presenting data.
**Answer**: Histogram shows the distribution shape and frequency density for grouped data. Box plot shows the five-number summary. Histogram is better for shape; box plot is better for comparing multiple data sets.
**Explanation**: Different purposes. A common error is using the wrong graph for the purpose.
**Tags**: box-plot, histogram, statistics, comparison

# Question: Question Q-S6-09
**Difficulty**: Challenge
**Question**: Find the mean and standard deviation of the data set: 4, 6, 6, 7, 8, 10, 12.
**Answer**: Mean = (4+6+6+7+8+10+12)/7 = 53/7 = 7.57. σ = √(Σ(x−x̄)²/n) = √(53.43/7) = √7.63 = 2.76.
**Explanation**: Mean = Σx/n. SD = √(Σ(x−x̄)²/n). A common error is forgetting to square the deviations.
**Tags**: standard-deviation, mean, statistics
