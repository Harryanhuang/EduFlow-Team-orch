### Question Q-A6-07
**Difficulty**: Standard
**Question**: Use the quadratic formula to solve: 3x² - 5x - 2 = 0
**Answer**: x = (5±√(25+24))/6 = (5±7)/6. x = 2 or x = -1/3.
**Explanation**: a=3, b=-5, c=-2. Discriminant = 25+24 = 49. x = (5±7)/6. A common error is forgetting the ± or using -b instead of +5.
