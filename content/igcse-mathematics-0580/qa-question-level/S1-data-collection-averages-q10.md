### Question Q-S1-08
**Difficulty**: Standard
**Question**: Classify the data: the number of pets owned by each student in a class. Is it discrete or continuous?
**Answer**: Discrete. The number of pets is countable (0, 1, 2, 3, ...). It cannot take any value within a range.
**Explanation**: Discrete = countable, whole numbers. Continuous = measurable, any value. A common error is saying continuous.
