### Question Q-A2-11
**Difficulty**: Challenge
**Question**: Solve: (2x+1)/(x-3) = 5
**Answer**: 2x + 1 = 5(x-3), 2x + 1 = 5x - 15, 16 = 3x, x = 16/3
**Explanation**: Multiply both sides by (x-3): 2x+1 = 5x-15, 16 = 3x, x = 16/3. Note: x ≠ 3 (denominator). A common error is cross-multiplying incorrectly.
