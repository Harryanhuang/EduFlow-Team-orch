# Question: Question Q-T2-01
**Difficulty**: Foundation
**Question**: State the sine rule for a non-right-angled triangle.
**Answer**: a/sin A = b/sin B = c/sin C. Also written as sin A/a = sin B/b = sin C/c.
**Explanation**: Sine rule relates sides and opposite angles. A common error is putting the wrong angle in the denominator.
