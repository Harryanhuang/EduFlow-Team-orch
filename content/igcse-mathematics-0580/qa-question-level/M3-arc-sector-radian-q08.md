# Question: Question Q-M3-08
**Difficulty**: Challenge
**Question**: A cone has base radius 5 cm and slant height 13 cm. Find the total surface area.
**Answer**: CSA = πrl = 65π. Base = πr² = 25π. Total = 90π ≈ 282.7 cm².
**Explanation**: CSA + base. A common error is forgetting the base.
