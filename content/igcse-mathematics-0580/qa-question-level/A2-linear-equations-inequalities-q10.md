### Question Q-A2-09
**Difficulty**: Standard
**Question**: Solve the inequality: 5 - 2x ≥ 1. List the integer values of x.
**Answer**: -2x ≥ -4, x ≤ 2. Integers: ..., -2, -1, 0, 1, 2.
**Explanation**: 5-2x ≥ 1, -2x ≥ -4, x ≤ 2 (divide by negative, flip inequality). A common error is x ≥ 2 (not flipping the sign).
**Tags**: inequality, integer-values, negative-division
