### Question Q-A4-12
**Difficulty**: Challenge
**Question**: A line passes through (1, 2) and (4, 14). Find its equation in the form y = mx + c, then find where it crosses the x-axis.
**Answer**: m = (14-2)/(4-1) = 12/3 = 4. y = 4x + c: 2 = 4(1) + c, c = -2. y = 4x - 2. x-axis: y=0, 0=4x-2, x=1/2.
**Explanation**: Gradient = 4, c = -2. x-intercept: 0 = 4x-2, x = 0.5. A common error is c = 2 (sign error).
