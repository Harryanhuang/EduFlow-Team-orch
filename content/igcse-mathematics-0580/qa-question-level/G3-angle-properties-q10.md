### Question Q-G3-09
**Difficulty**: Challenge
**Question**: In triangle ABC, angle A = 2x, angle B = 3x, angle C = 4x. Find each angle.
**Answer**: 2x + 3x + 4x = 180°, 9x = 180°, x = 20°. A = 40°, B = 60°, C = 80°.
**Explanation**: Angles in triangle = 180°. 9x = 180, x = 20. A common error is not setting up the equation correctly.
