# Question: Question Q-A7-04
**Difficulty**: Standard
**Question**: Find f(3) if f(x) = 2x² − 5x + 1.
**Answer**: f(3) = 2(3)² − 5(3) + 1 = 2(9) − 15 + 1 = 18 − 15 + 1 = 4.
**Explanation**: Substitute x = 3. A common error is forgetting to square 3 first.
