### Question Q-G5-12
**Difficulty**: Standard
**Question**: If a = (1, 5) and b = (3, -2), find the magnitude of a + b.
**Answer**: a + b = (4, 3). Magnitude = √(16+9) = √25 = 5.
**Explanation**: Add vectors component-wise, then find magnitude. A common error is adding magnitudes.
