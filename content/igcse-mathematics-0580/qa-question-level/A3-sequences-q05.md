### Question Q-A3-04
**Difficulty**: Standard
**Question**: Is 50 a term in the sequence 3n + 2? Explain.
**Answer**: 3n + 2 = 50, 3n = 48, n = 16. Yes, 50 is the 16th term.
**Explanation**: Set 3n+2 = 50, solve for n. n = 16 (whole number = yes). A common error is 3n = 50, n = 16.67 (not subtracting 2 first).
