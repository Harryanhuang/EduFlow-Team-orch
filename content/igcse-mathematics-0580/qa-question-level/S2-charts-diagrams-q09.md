### Question Q-S2-08
**Difficulty**: Challenge
**Question**: A histogram has unequal class widths. The frequency density for the class 10-20 is 5, and for 20-40 is 3. Which class has the higher frequency?
**Answer**: Frequency = frequency density × class width. Class 10-20: 5 × 10 = 50. Class 20-40: 3 × 20 = 60. The second class has higher frequency.
**Explanation**: In a histogram, area (not height) represents frequency. A common error is comparing heights directly (5 > 3, but 50 < 60).
