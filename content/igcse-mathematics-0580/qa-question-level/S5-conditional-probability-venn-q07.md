# Question: Question Q-S5-07
**Difficulty**: Challenge
**Question**: A test for a disease affects 1 in 100 people. The test is 95% accurate (P(positive | has disease) = 0.95, P(negative | no disease) = 0.95). Find P(disease | positive).
**Answer**: P(disease | positive) = [P(positive | disease) × P(disease)] / P(positive). P(positive) = (0.95 × 0.01) + (0.05 × 0.99) = 0.0095 + 0.0495 = 0.059. P(disease | positive) = 0.0095/0.059 = 0.161 = 16.1%.
**Explanation**: Bayes' theorem. A common error is forgetting the false positive rate in P(positive).
