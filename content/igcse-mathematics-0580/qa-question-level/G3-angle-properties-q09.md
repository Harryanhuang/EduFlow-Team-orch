### Question Q-G3-08
**Difficulty**: Standard
**Question**: An exterior angle of a regular polygon is 36°. Find the interior angle and the number of sides.
**Answer**: Interior = 180° - 36° = 144°. Number of sides = 360°/36° = 10 (decagon).
**Explanation**: Interior + exterior = 180°. n = 360°/exterior. A common error is 360°/144° = 2.5.
