# Question: Question Q-A7-02
**Difficulty**: Foundation
**Question**: What is the domain and range of a function?
**Answer**: The domain is the set of all possible input values (x). The range is the set of all possible output values (f(x)). Example: f(x) = x² with domain x ≥ 0 has range f(x) ≥ 0.
**Explanation**: Domain = inputs, range = outputs. A common error is saying the range is the same as the domain.
