### Question Q-G3-07
**Difficulty**: Standard
**Question**: In the diagram, ABCD is a parallelogram. Angle A = 65°. Find angles B, C, and D.
**Answer**: B = 115°, C = 65°, D = 115°. Adjacent angles in a parallelogram are supplementary (sum to 180°). Opposite angles are equal.
**Explanation**: Opposite angles equal: C = A = 65°. Adjacent: B = 180-65 = 115°, D = B = 115°. A common error is all angles = 65°.
