### Question Q-N3-08
**Difficulty**: Standard
**Question**: Write 0.072 recurring (0.0727272...) as a fraction.
**Answer**: 4/55
**Explanation**: Let x = 0.0727272... Then 100x = 7.272727... and x = 0.072727... Subtract: 99x = 7.2, so x = 7.2/99 = 72/990 = 4/55. A common error is treating it as 0.072 (terminating) = 72/1000.
