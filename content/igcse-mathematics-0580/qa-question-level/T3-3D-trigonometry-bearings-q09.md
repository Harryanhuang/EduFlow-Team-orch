### Question Q-T3-09
**Difficulty**: Challenge
**Question**: Point P is at (3, 4, 0) and point Q is at (6, 7, 5) in a 3D coordinate system. Find the angle that line PQ makes with the horizontal plane.
**Answer**: PQ = (3, 3, 5). Length of PQ = √(3²+3²+5²) = √43 = 6.56. Horizontal component = √(3²+3²) = √18 = 4.24. tan θ = 5/4.24 → θ = 49.8°.
**Explanation**: In 3D, the angle with the horizontal uses the vertical component and the horizontal projection (x-y plane). A common error is using the full length as the denominator instead of the horizontal component.
