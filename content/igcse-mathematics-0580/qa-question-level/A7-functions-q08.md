# Question: Question Q-A7-08
**Difficulty**: Challenge
**Question**: Find the composite function fg(x) if f(x) = x + 2 and g(x) = x². Then find (fg)(3).
**Answer**: fg(x) = f(g(x)) = f(x²) = x² + 2. (fg)(3) = 3² + 2 = 11. Note: gf(x) = g(f(x)) = (x + 2)² = x² + 4x + 4 (different from fg).
**Explanation**: Apply g first, then f. A common error is applying them in the wrong order.
