# Question: Question Q-A7-05
**Difficulty**: Standard
**Question**: What is the inverse function f⁻¹(x)? If f(x) = 3x − 2, find f⁻¹(7).
**Answer**: Let y = 3x − 2. Swap x and y: x = 3y − 2 → 3y = x + 2 → y = (x + 2)/3. So f⁻¹(x) = (x + 2)/3. f⁻¹(7) = (7 + 2)/3 = 9/3 = 3.
**Explanation**: Swap x and y, then solve for y. A common error is forgetting to swap before solving.
