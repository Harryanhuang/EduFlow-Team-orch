### Question Q-S1-12
**Difficulty**: Standard
**Question**: A grouped frequency table shows: 0-10 (5), 10-20 (8), 20-30 (12), 30-40 (5). Estimate the mean using midpoints.
**Answer**: Midpoints: 5, 15, 25, 35. Sum of (midpoint × frequency): 5×5+15×8+25×12+35×5 = 25+120+300+175 = 620. Total frequency = 30. Mean ≈ 620/30 ≈ 20.7.
**Explanation**: Use midpoints as estimates. A common error is using the class boundaries instead of midpoints.
