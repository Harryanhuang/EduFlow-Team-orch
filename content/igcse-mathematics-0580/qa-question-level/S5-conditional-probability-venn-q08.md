# Question: Question Q-S5-08
**Difficulty**: Challenge
**Question**: Using a Venn diagram with P(A) = 0.4, P(B) = 0.5, P(A ∩ B) = 0.2, find P(A ∩ B').
**Answer**: P(A ∩ B') = P(A) − P(A ∩ B) = 0.4 − 0.2 = 0.2.
**Explanation**: Intersection with complement = in A but not in B. A common error is confusing with union.
