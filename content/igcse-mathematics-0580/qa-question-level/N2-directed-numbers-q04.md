### Question Q-N2-02
**Difficulty**: Foundation
**Question**: Work out: -8 - (-3) + (-5)
**Answer**: -8 + 3 + (-5) = -10
**Explanation**: Subtracting a negative = adding: -8 - (-3) = -8 + 3 = -5. Then -5 + (-5) = -10. A common error is: -8 - (-3) = -8 - 3 = -11.
