### Question Q-G2-06
**Difficulty**: Standard
**Question**: Construct a rectangle 8 cm by 5 cm using a ruler and set square.
**Answer**: 1) Draw base 8 cm. 2) At each end, construct perpendiculars. 3) Measure 5 cm up each perpendicular. 4) Join the top points.
**Explanation**: A rectangle has four right angles. Use set square or compass for perpendiculars. A common error is not ensuring the opposite sides are parallel.
