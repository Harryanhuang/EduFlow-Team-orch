### Question Q-A3-02
**Difficulty**: Foundation
**Question**: Find the nth term of: 10, 7, 4, 1, -2, ...
**Answer**: -3n + 13
**Explanation**: Common difference = -3, so -3n. When n=1: -3(1) = -3, but first term is 10, so add 13: -3n+13. A common error is 3n+7.
