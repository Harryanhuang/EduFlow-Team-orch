### Question Q-A5-01
**Difficulty**: Standard
**Question**: Solve: x + y = 8, 2x - y = 4.
**Answer**: Add: 3x = 12, x = 4. y = 8 - 4 = 4. Check: 2(4)-4 = 4 ✓.
**Explanation**: Adding eliminates y. x=4, y=4. A common error is subtracting: 3x = 4, which is wrong.
