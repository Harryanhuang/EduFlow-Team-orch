# Question: Question Q-A7-09
**Difficulty**: Challenge
**Question**: The function f(x) = kx + 3 passes through the point (2, 11). Find k and find the value of a such that f⁻¹(a) = 5.
**Answer**: f(2) = 2k + 3 = 11 → 2k = 8 → k = 4. So f(x) = 4x + 3. For f⁻¹(a) = 5, we need f(5) = a. f(5) = 4(5) + 3 = 23. So a = 23.
**Explanation**: Use the point to find k, then use inverse definition. A common error is mixing up f(a) and f⁻¹(a).
