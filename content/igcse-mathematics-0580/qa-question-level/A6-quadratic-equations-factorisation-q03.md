### Question Q-A6-02
**Difficulty**: Foundation
**Question**: Solve: x² + 5x = 0
**Answer**: x(x + 5) = 0, so x = 0 or x = -5
**Explanation**: Factor out x: x(x+5) = 0. Set each factor to zero. A common error is dividing by x: x+5=0, x=-5 (losing the x=0 solution).
