# QA: A2 — Linear equations, inequalities

## Topic 名称
Linear equations, inequalities（线性方程、不等式）

## Topic 定义
学习解一元一次方程、含有括号的方程、含有分数的方程，以及在数轴上表示不等式的解。

## 关键知识点
- 解基本一元一次方程：ax + b = c
- 解含括号的方程
- 解含分数的方程（去分母）
- 解未知数在两边的方程：ax + b = cx + d
- 列方程解应用题
- 解一元一次不等式并在数轴上表示
- 不等式符号：>, <, ≥, ≤ 的含义和使用
- 不等式两边同时乘除负数时不等号方向改变

## 前置知识
- A1（代数运算、公式）

## 常见错误
- 移项时忘记变号
- 解含分数方程时忘记等号两边都乘分母
- 不等式两边同时乘除负数时忘记翻转不等号
- 数轴表示时混淆实心点（≥, ≤）和空心点（>, <）
- 列方程应用题中设错未知数或列错等量关系
- 检验时不代入原方程验证

## 可出题方向
- 解一元一次方程
- 解含括号或分数的方程
- 解不等式并在数轴上表示
- 列方程解应用题（年龄、距离、价格等）
- 求不等式的整数解个数

## 难度提示
Core：基础。要求学生能解各类一元一次方程和不等式。

## 适合题型
- 计算题
- 数轴作图题
- 应用题（列方程）
- 填空题（不等式整数解）
