# QA: A6 — Quadratic equations, factorisation

## Topic 名称
Quadratic equations, factorisation（二次方程、因式分解）

## Topic 定义
学习解一元二次方程的方法：因式分解法、公式法（求根公式），以及从图像理解二次方程的根。

## 关键知识点
- 标准形式：ax² + bx + c = 0
- 因式分解法解二次方程（十字相乘法）
- 完全平方式：(x + p)² = x² + 2px + p²
- 配方法（completing the square）
- 求根公式：x = (-b ± √(b²-4ac)) / 2a
- 判别式 b²-4ac 的含义：>0 两实根, =0 一实根, <0 无实根
- 二次方程两个解的含义
- 简单分式方程的求解

## 前置知识
- A1（代数运算、公式）
- A2（线性方程、不等式）

## 常见错误
- 十字相乘时符号搞反
- 配方时忘记处理一次项系数的一半
- 用求根公式时代入 a, b, c 值出错
- 判别式计算错误（忘记 b² 可能为负数的平方）
- 只写出一个解而漏掉另一个
- 分式方程忘记检验分母不为零

## 可出题方向
- 因式分解法解二次方程
- 配方法解二次方程
- 求根公式法解二次方程
- 判断方程根的情况（判别式）
- 简单的分式方程

## 难度提示
Core：基础因式分解和求根公式。Extended 增加更复杂的系数和分式方程。

## 适合题型
- 计算题
- 判别式分析题
- 选择题
- 结构化解答题（展示配方过程）
