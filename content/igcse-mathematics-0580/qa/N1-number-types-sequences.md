# QA: N1 — Number types, ordering, sequences

## Topic 名称
Number types, ordering, sequences

## Topic 定义
识别和分类不同数集，理解数的排序关系，识别并延续数字序列模式。

## 关键知识点
- 自然数 N = {1, 2, 3, ...}，整数 Z = {..., -2, -1, 0, 1, 2, ...}
- 质数：仅能被 1 和自身整除的大于 1 的自然数
- 合数：有超过两个因数的自然数
- 平方数、立方数、三角数
- 有理数 vs 无理数（π、√2 等）
- 奇数和偶数的性质
- 序列中的模式识别：递推关系、逐项差
- HCF（最大公因数）和 LCM（最小公倍数）
- 质因数分解（质因数树/连续除法）

## 前置知识
- 基础算术运算
- 乘法表

## 常见错误
- 把 1 当作质数（1 不是质数，因为它只有一个因数）
- 混淆 HCF 和 LCM 的求法
- 质因数分解时遗漏因数
- 将负数排序时出错（-5 < -3，不是 -5 > -3）
- 认为所有小数都是有理数（循环小数是有理数，但无限不循环小数不是）
- 混淆"项"和"项数"（第 n 项 vs 第 n 个位置）

## 可出题方向
- 给定一组数，分类哪些属于哪些数集
- 求两个或多个数的 HCF 和 LCM
- 质因数分解并写成 2^a × 3^b × ... 形式
- 识别序列规律并写出下一项或第 n 项
- 判断一个数是否为质数
- 给定 Venn 图分类数集

## 难度提示
Core：基础。Extended 在此基础上增加更复杂的序列规律识别。

## 适合题型
- 简答题（分类、计算 HCF/LCM）
- 填空题（序列下一项）
- 选择题（数的分类）
- 结构化解答题（质因数分解展示过程）
