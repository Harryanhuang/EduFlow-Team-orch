# QA: A3 — Sequences

## Topic 名称
Sequences（数列、序列）

## Topic 定义
学习识别数列的规律，求数列的下一项和通项公式（nth term），包括等差数列和简单递推数列。

## 关键知识点
- 等差数列：逐项差相同，通项公式 an = a + (n-1)d
- 等差数列的通项求法：先求 d，再求 a
- 识别递增/递减/摆动数列
- 通过逐项差判断数列类型
- 递推数列：给出前几项的规律求后续项
- 平方数列、三角数列等特殊数列
- 从实际情境中识别数列规律
- 判断某个数是否在给定数列中

## 前置知识
- N1（数的分类、排序、序列）

## 常见错误
- 把首项 a 和公差 d 混淆（通项写成 n + d 而非 a + (n-1)d）
- 递推数列中忽略初始条件
- 判断数列类型时只看前两项不看更多项
- 求"第几个数"时把项数 n 和项的值 an 混淆
- 验证时不代入通项公式检查

## 可出题方向
- 求等差数列的通项公式
- 求数列的下几项
- 判断某数是否在数列中
- 根据前几项写出递推关系
- 应用题中的数列建模（如存款增长、台阶数）

## 难度提示
Core：基础。要求学生能识别常见数列规律并求通项。Extended 增加更复杂的二阶等差数列。

## 适合题型
- 计算题（求通项）
- 填空题（求下一项）
- 判断题（某数是否在数列中）
- 应用题
