# Mathematics 0580 — A7 Functions Items

## Item 1
**Difficulty**: Foundation
**Question**: What is a function? Give an example.
**Answer**: A function is a relationship where each input value (x) gives exactly one output value (f(x)). Example: f(x) = 2x + 3. If x = 5, f(5) = 13.
**Explanation**: One input → one output. A common error is confusing a function with a general relation.
**Tags**: functions, definition, notation

## Item 2
**Difficulty**: Foundation
**Question**: What is the domain and range of a function?
**Answer**: The domain is all possible inputs (x). The range is all possible outputs (f(x)). Example: f(x) = x² with domain x ≥ 0 has range f(x) ≥ 0.
**Explanation**: Domain = inputs, range = outputs. A common error is confusing the two.
**Tags**: functions, domain, range

## Item 3
**Difficulty**: Standard
**Question**: Sketch the graph of f(x) = x² and describe its key features.
**Answer**: Parabola opening upwards, vertex at (0, 0), y ≥ 0, symmetrical about the y-axis.
**Explanation**: Basic quadratic shape. A common error is forgetting the axis of symmetry.
**Tags**: quadratic, graph, parabola, functions

## Item 4
**Difficulty**: Standard
**Question**: Find f(3) if f(x) = 2x² − 5x + 1.
**Answer**: f(3) = 2(9) − 15 + 1 = 4.
**Explanation**: Substitute x = 3. A common error is forgetting to square first.
**Tags**: functions, substitution, quadratic

## Item 5
**Difficulty**: Standard
**Question**: Find the inverse function f⁻¹(x) if f(x) = 3x − 2.
**Answer**: Let y = 3x − 2. Swap x and y: x = 3y − 2 → 3y = x + 2 → y = (x + 2)/3. So f⁻¹(x) = (x + 2)/3.
**Explanation**: Swap x and y, solve for y. A common error is forgetting to swap before solving.
**Tags**: inverse-function, algebra, functions

## Item 6
**Difficulty**: Standard
**Question**: Describe the transformation mapping f(x) = x² to g(x) = (x − 3)² + 2.
**Answer**: Translation: shift 3 units right and 2 units up. Vertex moves from (0, 0) to (3, 2).
**Explanation**: Inside bracket shifts x, outside shifts y. A common error is reading the direction incorrectly.
**Tags**: transformation, translation, functions, quadratic

## Item 7
**Difficulty**: Challenge
**Question**: Solve f(x) = 5 for f(x) = (2x − 1)/(x + 3). Find x.
**Answer**: (2x − 1)/(x + 3) = 5 → 2x − 1 = 5x + 15 → −3x = 16 → x = −16/3. Check: denominator ≠ 0, −16/3 ≠ −3, so valid.
**Explanation**: Cross-multiply, solve, check domain. A common error is forgetting the domain check.
**Tags**: functions, rational-function, equation-solving

## Item 8
**Difficulty**: Challenge
**Question**: Find the composite function fg(x) given f(x) = x + 2 and g(x) = x². Then find (fg)(3).
**Answer**: fg(x) = f(g(x)) = x² + 2. (fg)(3) = 3² + 2 = 11. Note: gf(x) = (x + 2)² = x² + 4x + 4 (different from fg).
**Explanation**: Apply g first, then f. A common error is applying in the wrong order.
**Tags**: composite-function, functions, notation

## Item 9
**Difficulty**: Challenge
**Question**: f(x) = kx + 3 passes through (2, 11). Find k and find a such that f⁻¹(a) = 5.
**Answer**: f(2) = 2k + 3 = 11 → k = 4. f⁻¹(a) = 5 means f(5) = a. f(5) = 4(5) + 3 = 23. So k = 4, a = 23.
**Explanation**: Use the point to find k, then use the definition of inverse. A common error is mixing f(a) with f⁻¹(a).
**Tags**: functions, inverse-function, finding-k
