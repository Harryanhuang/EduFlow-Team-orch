# Mathematics 0580 — T2 Sine Rule, Cosine Rule, Area Items

## Item 1
**Difficulty**: Foundation
**Question**: State the sine rule for a non-right-angled triangle.
**Answer**: a/sin A = b/sin B = c/sin C.
**Explanation**: Relates sides and opposite angles. A common error is putting the wrong angle in the denominator.
**Tags**: sine-rule, trigonometry, non-right-triangle

## Item 2
**Difficulty**: Foundation
**Question**: State the cosine rule.
**Answer**: a² = b² + c² − 2bc cos A.
**Explanation**: Relates all three sides and one angle. A common error is forgetting the minus sign.
**Tags**: cosine-rule, trigonometry, non-right-triangle

## Item 3
**Difficulty**: Standard
**Question**: In triangle ABC, A = 40°, B = 65°, a = 7 cm. Find b using the sine rule.
**Answer**: b = 7 × sin 65°/sin 40° = 9.86 cm.
**Explanation**: b/sin B = a/sin A. A common error is swapping sin 65° and sin 40°.
**Tags**: sine-rule, trigonometry

## Item 4
**Difficulty**: Standard
**Question**: In triangle ABC, a = 8 cm, b = 5 cm, A = 70°. Find the possible value of c.
**Answer**: Using b² = a² + c² − 2ac cos B. c² − 5.49c + 39 = 0 → c = 6.77 or 5.76 cm.
**Explanation**: Cosine rule needs the angle opposite the side being solved. A common error is using the wrong side.
**Tags**: cosine-rule, trigonometry

## Item 5
**Difficulty**: Standard
**Question**: Find the area of triangle ABC given a = 10 cm, b = 8 cm, C = 50°.
**Answer**: Area = (1/2)ab sin C = 0.5 × 10 × 8 × sin 50° = 30.64 cm².
**Explanation**: Two sides and included angle. A common error is forgetting (1/2).
**Tags**: area, trigonometry, triangle

## Item 6
**Difficulty**: Standard
**Question**: A triangle has sides 9 cm, 7 cm, 12 cm. Find the largest angle.
**Answer**: cos A = (49 + 81 − 144)/(2 × 7 × 9) = −14/126 = −0.111. A = 96.4°.
**Explanation**: Largest side → largest angle. A common error is using the wrong side as a².
**Tags**: cosine-rule, trigonometry, triangle

## Item 7
**Difficulty**: Challenge
**Question**: Two triangles have a = 12 cm, b = 8 cm, A = 50°. Find both possible values of angle B.
**Answer**: sin B = 8 × sin 50°/12 = 0.511. B = 30.7° or 149.3°. Both valid since 50° + 149.3° < 180°.
**Explanation**: Ambiguous case: if h < b < a, two solutions. A common error is only giving the acute solution.
**Tags**: ambiguous-case, sine-rule, trigonometry

## Item 8
**Difficulty**: Challenge
**Question**: A triangle has sides 11 cm, 9 cm, 7 cm. Find its area using Heron's formula.
**Answer**: s = 13.5. Area = √(13.5 × 2.5 × 4.5 × 6.5) = 31.4 cm².
**Explanation**: Heron's formula. A common error is forgetting to halve the perimeter.
**Tags**: area, Heron, trigonometry

## Item 9
**Difficulty**: Challenge
**Question**: Triangle ABC has A = 55°, a = 15 cm, b = 20 cm. Can this be solved? Explain.
**Answer**: sin B = 20 × sin 55°/15 = 1.092 > 1. No solution exists. Side b is too short to reach the base.
**Explanation**: Check b/a > sin A before solving. A common error is forcing a solution.
**Tags**: sine-rule, trigonometry, triangle-solve
