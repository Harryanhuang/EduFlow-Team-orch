### Question Q-N6-01
**Difficulty**: Foundation
**Question**: Calculate 5³ × 5², giving your answer as a single power of 5.
**Answer**: 5⁵
**Explanation**: When multiplying with the same base, add the exponents: 5³ × 5² = 5⁽³⁺²⁾ = 5⁵. A common error is multiplying the exponents: 5⁶.
**Tags**: indices, index-laws, multiplication

### Question Q-N6-02
**Difficulty**: Foundation
**Question**: Simplify (7⁴) ÷ (7²).
**Answer**: 7² = 49
**Explanation**: When dividing with the same base, subtract the exponents: 7⁽⁴⁻²⁾ = 7² = 49. A common error is dividing the exponents: 7² = 7⁽⁴÷²⁾ = 7² (correct answer but wrong method).
**Tags**: indices, index-laws, division

### Question Q-N6-03
**Difficulty**: Standard
**Question**: A bank account has $5000 earning 3% compound interest per year. How much is in the account after 2 years?
**Answer**: $5000 × 1.03² = $5000 × 1.0609 = $5304.50
**Explanation**: Year 1: $5000 × 1.03 = $5150. Year 2: $5150 × 1.03 = $5304.50. Or $5000 × 1.03². A common error is simple interest: $5000 + 2×$150 = $5300.
**Tags**: compound-interest, money, percentage

### Question Q-N6-04
**Difficulty**: Standard
**Question**: Simplify √12 + √27.
**Answer**: √12 = 2√3, √27 = 3√3. So 2√3 + 3√3 = 5√3.
**Explanation**: Simplify each surd first: √12 = √(4×3) = 2√3, √27 = √(9×3) = 3√3. Then add: 5√3. A common error is √12 + √27 = √39.
**Tags**: surds, simplification, addition

### Question Q-N6-05
**Difficulty**: Standard
**Question**: Expand and simplify (2 + √3)(2 - √3).
**Answer**: 4 - 2√3 + 2√3 - 3 = 4 - 3 = 1
**Explanation**: This is the difference of two squares: (a+b)(a-b) = a² - b². So 4 - 3 = 1. A common error is 4 - 2√3 - 2√3 + 3 = 7 - 4√3 (wrong signs).
**Tags**: surds, expansion, difference-of-squares

### Question Q-N6-06
**Difficulty**: Standard
**Question**: A car depreciates by 15% per year. Its value is $12,000. What will it be worth after 3 years?
**Answer**: $12,000 × 0.85³ = $12,000 × 0.614125 = $7,369.50
**Explanation**: Each year the value is multiplied by 0.85 (100% - 15%). After 3 years: $12,000 × 0.85³. A common error is $12,000 - 3×15% = $12,000 - $5,400 = $6,600 (simple depreciation).
**Tags**: depreciation, compound-decrease, money

### Question Q-N6-07
**Difficulty**: Standard
**Question**: Simplify (2a³)² × (3a⁻¹).
**Answer**: 4a⁶ × 3a⁻¹ = 12a⁵
**Explanation**: (2a³)² = 4a⁶ (square the coefficient and double the exponent). Then 4a⁶ × 3a⁻¹ = 12a⁽⁶⁻¹⁾ = 12a⁵. A common error is (2a³)² = 2a⁶ (forgetting to square the 2).
**Tags**: indices, expansion, negative-powers

### Question Q-N6-09
**Difficulty**: Challenge
**Question**: Solve: 2^x = 32 × 4. Find x.
**Answer**: 32 × 4 = 2⁵ × 2² = 2⁷. So 2^x = 2⁷, x = 7.
**Explanation**: Express both sides as powers of 2: 32 = 2⁵, 4 = 2². 2⁵ × 2² = 2⁷. So x = 7. A common error is 32 + 4 = 36, then trying to find 2^x = 36.
**Tags**: indices, exponential-equation, powers

### Question Q-N6-10
**Difficulty**: Challenge
**Question**: Simplify (√50 - √8) / √2.
**Answer**: √50 = 5√2, √8 = 2√2. (5√2 - 2√2)/√2 = 3√2/√2 = 3.
**Explanation**: Simplify surds: √50 = 5√2, √8 = 2√2. Numerator = 3√2. 3√2/√2 = 3. A common error is (√50-√8)/√2 = √(50-8)/√2 = √42/√2.
**Tags**: surds, simplification, division
