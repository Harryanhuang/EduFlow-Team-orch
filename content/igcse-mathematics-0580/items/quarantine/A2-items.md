### Question Q-A2-01
**Difficulty**: Foundation
**Question**: Solve: 3x + 7 = 22
**Answer**: x = 5
**Explanation**: 3x = 22 - 7 = 15, x = 15/3 = 5. A common error is 3x = 22 + 7 = 29, x = 29/3.
**Tags**: linear-equation, one-step

### Question Q-A2-02
**Difficulty**: Foundation
**Question**: Solve: 5(x - 2) = 20
**Answer**: x = 6
**Explanation**: Method 1: x-2 = 4, x = 6. Method 2: 5x-10 = 20, 5x = 30, x = 6. A common error is 5x-2 = 20, 5x = 22 (not distributing the 5).
**Tags**: linear-equation, bracket

### Question Q-A2-12
**Difficulty**: Challenge
**Question**: Solve the simultaneous equations by elimination: 2x + y = 10, x - y = 2.
**Answer**: Add: 3x = 12, x = 4. Substitute: 4 - y = 2, y = 2. Check: 2(4)+2 = 10 ✓.
**Explanation**: Adding eliminates y: 3x = 12, x = 4. Then y = 10 - 2(4) = 2. A common error is subtracting instead of adding.
**Tags**: simultaneous-equations, elimination
