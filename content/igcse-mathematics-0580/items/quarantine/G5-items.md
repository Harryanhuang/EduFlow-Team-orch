### Question Q-G5-01
**Difficulty**: Foundation
**Question**: Describe the single transformation that maps triangle A(1,1), B(3,1), C(2,3) to A'(4,2), B'(6,2), C'(5,4).
**Answer**: Translation by vector (3, 1). Each point moves 3 right and 1 up.
**Explanation**: A'(4,2) - A(1,1) = (3,1). Same for all points. Translation vector is (3,1). A common error is reflection or rotation.
**Tags**: transformations, translation

### Question Q-G5-02
**Difficulty**: Foundation
**Question**: Triangle A(2,3) is reflected in the x-axis. Write down the coordinates of the image.
**Answer**: A'(2, -3). The y-coordinate changes sign.
**Explanation**: Reflection in x-axis: (x,y) → (x,-y). A common error is (-2,3) (confusing with y-axis reflection).
**Tags**: transformations, reflection, x-axis

### Question Q-G5-03
**Difficulty**: Standard
**Question**: Point A(3, 4) is rotated 90° clockwise about the origin. Find the coordinates of the image.
**Answer**: A'(4, -3). Rule: (x,y) → (y,-x) for 90° clockwise.
**Explanation**: 90° clockwise about origin: (x,y) → (y,-x). So (3,4) → (4,-3). A common error is (-4,3) (90° anticlockwise).
**Tags**: transformations, rotation

### Question Q-G5-04
**Difficulty**: Standard
**Question**: Triangle A(1,2), B(3,2), C(2,4) is enlarged by scale factor 2, centre (0,0). Find the image coordinates.
**Answer**: A'(2,4), B'(6,4), C'(4,8). Each coordinate is multiplied by 2.
**Explanation**: Enlargement sf 2, centre (0,0): multiply each coordinate by 2. A common error is adding 2 to each coordinate.
**Tags**: transformations, enlargement

### Question Q-G5-05
**Difficulty**: Standard
**Question**: Calculate the magnitude of vector (3, 4).
**Answer**: √(3² + 4²) = √(9+16) = √25 = 5
**Explanation**: Magnitude = √(x² + y²). This is a Pythagorean triple. A common error is 3 + 4 = 7.
**Tags**: vectors, magnitude

### Question Q-G5-06
**Difficulty**: Standard
**Question**: Given a = (2, 3) and b = (1, -2), find 2a - b.
**Answer**: 2a = (4, 6). 2a - b = (4-1, 6-(-2)) = (3, 8).
**Explanation**: Scalar multiplication first, then component-wise subtraction. 6-(-2) = 6+2 = 8. A common error is (3, 4) (subtracting 2 instead of -2).
**Tags**: vectors, scalar-multiplication, subtraction

### Question Q-G5-07
**Difficulty**: Standard
**Question**: A is at (1, 2) and B is at (5, 8). Find vector AB.
**Answer**: AB = (5-1, 8-2) = (4, 6)
**Explanation**: Vector AB = B - A = (5-1, 8-2). A common error is (1-5, 2-8) = (-4,-6) (direction reversed).
**Tags**: vectors, position-vector

### Question Q-G5-08
**Difficulty**: Standard
**Question**: Describe the enlargement with scale factor -1, centre (0,0), applied to point (3, 5).
**Answer**: Image is (-3, -5). A negative scale factor means the image is on the opposite side of the centre.
**Explanation**: Multiply each coordinate by -1. Negative scale factor = 180° rotation about centre. A common error is (-3, 5).
**Tags**: transformations, enlargement, negative-scale-factor

### Question Q-G5-09
**Difficulty**: Challenge
**Question**: Point A(2, 1) is reflected in the line y = x. Find the image coordinates.
**Answer**: A'(1, 2). Reflection in y=x swaps the x and y coordinates.
**Explanation**: Reflection in y = x: (x,y) → (y,x). A common error is (-2,-1) (confusing with origin reflection).
**Tags**: transformations, reflection, y=x

### Question Q-G5-10
**Difficulty**: Challenge
**Question**: A shape has area 12 cm². It is enlarged by scale factor 3. What is the area of the image?
**Answer**: 12 × 3² = 12 × 9 = 108 cm²
**Explanation**: Area scales by the square of the scale factor. A common error is 12 × 3 = 36 (not squaring the scale factor).
**Tags**: transformations, enlargement, area-scaling
