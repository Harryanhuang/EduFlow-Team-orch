### Question Q-A6-01
**Difficulty**: Foundation
**Question**: Solve: x² - 9 = 0
**Answer**: x² = 9, x = ±3
**Explanation**: Take the square root: x = ±3. Both positive and negative. A common error is x = 3 only (missing the negative root).

### Question Q-A6-02
**Difficulty**: Foundation
**Question**: Solve: x² + 5x = 0
**Answer**: x(x + 5) = 0, so x = 0 or x = -5
**Explanation**: Factor out x: x(x+5) = 0. Set each factor to zero. A common error is dividing by x: x+5=0, x=-5 (losing the x=0 solution).
**Tags**: quadratic, common-factor

**Answer**: (x+2)(x+4) = 0, x = -2 or x = -4

### Question Q-A6-05
**Difficulty**: Standard
**Question**: Solve: 2x² + 7x + 3 = 0
**Answer**: (2x+1)(x+3) = 0, x = -1/2 or x = -3
**Explanation**: Need (2x+a)(x+b): a×b=3, 2b+a=7. Try a=1, b=3: 2(3)+1=7 ✓. x=-1/2 or x=-3. A common error is (2x+3)(x+1).
**Tags**: quadratic, non-mono-coefficient

### Question Q-A6-06
**Difficulty**: Standard
**Question**: Solve by completing the square: x² + 4x - 5 = 0
**Answer**: (x+2)² - 4 - 5 = 0, (x+2)² = 9, x+2 = ±3, x = 1 or x = -5
**Explanation**: (x+2)² = x²+4x+4. So x²+4x-5 = (x+2)²-9 = 0. (x+2)²=9, x+2=±3. A common error is (x+4)² (halving wrong).
**Tags**: quadratic, completing-the-square

### Question Q-A6-07

**Question**: Use the quadratic formula to solve: 3x² - 5x - 2 = 0

**Explanation**: a=3, b=-5, c=-2. Discriminant = 25+24 = 49. x = (5±7)/6. A common error is forgetting the ± or using -b instead of +5.
