### Question Q-G1-01
**Difficulty**: Foundation
**Question**: Name the type of triangle with angles 60°, 60°, 60°.
**Answer**: Equilateral triangle (all sides and angles equal).
**Explanation**: All angles are 60°, so all sides are equal. This is an equilateral triangle. A common error is isosceles (which has only two equal sides).
**Tags**: triangles, equilateral

### Question Q-G1-02
**Difficulty**: Foundation
**Question**: How many lines of symmetry does a rectangle have?
**Answer**: 2 (one horizontal, one vertical through the midpoints of opposite sides).
**Explanation**: A rectangle has 2 lines of symmetry. A common error is 4 (confusing with a square).
**Tags**: symmetry, rectangle

### Question Q-G1-03
**Difficulty**: Foundation
**Question**: What is the sum of the interior angles of a hexagon?
**Answer**: (6-2) × 180° = 720°
**Explanation**: Formula: (n-2) × 180° where n is the number of sides. For a hexagon, n=6: 4×180 = 720°. A common error is 6×180 = 1080°.
**Tags**: polygons, interior-angles, hexagon

### Question Q-G1-04
**Difficulty**: Standard
**Question**: A regular polygon has exterior angles of 45°. How many sides does it have?
**Answer**: 360° ÷ 45° = 8 sides (octagon).
**Explanation**: Sum of exterior angles = 360°. Each exterior angle = 360°/n. So n = 360°/45° = 8. A common error is 180°/45° = 4.
**Tags**: exterior-angles, regular-polygon

### Question Q-G1-05
**Difficulty**: Standard
**Question**: Calculate the size of each interior angle of a regular octagon.
**Answer**: Sum = (8-2)×180 = 1080°. Each angle = 1080°/8 = 135°. Or: exterior = 360°/8 = 45°, interior = 180°-45° = 135°.
**Explanation**: Two methods: sum/number or 180-exterior. Both give 135°. A common error is 1080°/6 = 180°.
**Tags**: interior-angles, regular-polygon, octagon

### Question Q-G1-06
**Difficulty**: Standard
**Question**: Describe the rotational symmetry of a parallelogram.
**Answer**: Order 2 (rotates 180° onto itself).
**Explanation**: A parallelogram maps onto itself when rotated 180° about its centre. A common error is order 4 (confusing with a square).
**Tags**: rotational-symmetry, parallelogram

### Question Q-G1-07
**Difficulty**: Standard
**Question**: A quadrilateral has angles x°, 2x°, 3x°, and 4x°. Find the value of x.
**Answer**: x + 2x + 3x + 4x = 360°, 10x = 360°, x = 36°.
**Explanation**: Sum of angles in a quadrilateral = 360°. 10x = 360, x = 36. A common error is using 180° (triangle sum).
**Tags**: quadrilateral, angles, algebra

### Question Q-G1-08
**Difficulty**: Standard
**Question**: What is the order of rotational symmetry of an equilateral triangle?
**Answer**: Order 3 (maps onto itself at 120°, 240°, and 360°).
**Explanation**: Rotates 360°/3 = 120° at a time. A common error is order 6 (confusing with a regular hexagon).
**Tags**: rotational-symmetry, equilateral-triangle

### Question Q-G1-09
**Difficulty**: Challenge
**Question**: The interior angle of a regular polygon is 150°. How many sides does it have?
**Answer**: Exterior angle = 180° - 150° = 30°. n = 360°/30° = 12 sides.
**Explanation**: Interior + exterior = 180°. n = 360°/exterior. A common error is 360°/150° = 2.4.
**Tags**: interior-angles, exterior-angles, regular-polygon

### Question Q-G1-10
**Difficulty**: Challenge
**Question**: Prove that the sum of the exterior angles of any polygon is 360°.
**Answer**: Each exterior angle = 180° - interior angle. Sum of all exterior = n×180° - sum of interiors = n×180° - (n-2)×180° = 2×180° = 360°.
**Explanation**: At each vertex: exterior + interior = 180°. Total: n×180° - (n-2)×180° = 360°. A common error is not using the interior angle sum formula.
**Tags**: exterior-angles, proof, polygon
