### Question Q-N2-01
**Difficulty**: Foundation
**Question**: Calculate: (-4) × (-3) + (-2) × 5
**Answer**: 12 + (-10) = 2
**Explanation**: (-4)×(-3) = 12 (negative × negative = positive). (-2)×5 = -10 (negative × positive = negative). 12 + (-10) = 2. A common error is getting the sign wrong: (-4)×(-3) = -12.
**Tags**: directed-numbers, multiplication, order-of-operations

### Question Q-N2-02
**Difficulty**: Foundation
**Question**: Work out: -8 - (-3) + (-5)
**Answer**: -8 + 3 + (-5) = -10
**Explanation**: Subtracting a negative = adding: -8 - (-3) = -8 + 3 = -5. Then -5 + (-5) = -10. A common error is: -8 - (-3) = -8 - 3 = -11.
**Tags**: directed-numbers, subtraction, negative-numbers

### Question Q-N2-03
**Difficulty**: Standard
**Question**: Calculate: (-2)³ + (-3)²
**Answer**: -8 + 9 = 1
**Explanation**: (-2)³ = (-2)×(-2)×(-2) = -8 (odd power of negative is negative). (-3)² = (-3)×(-3) = 9 (even power of negative is positive). -8 + 9 = 1. A common error is writing (-2)³ = 8.
**Tags**: directed-numbers, powers, negative-numbers

### Question Q-N2-04
**Difficulty**: Standard
**Question**: The temperature at midnight was -7°C. By noon it had risen by 15°C. By midnight it had fallen by 8°C from the noon temperature. What was the temperature at the second midnight?
**Answer**: Noon: -7 + 15 = 8°C. Second midnight: 8 - 8 = 0°C.
**Explanation**: Track changes step by step: -7 + 15 = 8, then 8 - 8 = 0. A common error is: -7 + 15 - 8 = 0 but calculating -7 - 8 + 15 instead and getting it wrong.
**Tags**: directed-numbers, temperature, word-problem

### Question Q-N2-05
**Difficulty**: Standard
**Question**: Fill in the missing numbers: (-3) × ☐ = 21, ☐ ÷ (-4) = -6
**Answer**: (-3) × (-7) = 21, 24 ÷ (-4) = -6
**Explanation**: For (-3) × ☐ = 21: since 21 is positive and -3 is negative, ☐ must be negative. 21 ÷ 3 = 7, so ☐ = -7. For ☐ ÷ (-4) = -6: ☐ = (-6) × (-4) = 24. A common error is getting signs wrong.
**Tags**: directed-numbers, missing-number, inverse-operations

### Question Q-N2-06
**Difficulty**: Standard
**Question**: Calculate: √144 + √(-8)³. If the expression is not possible, explain why.
**Answer**: √144 = 12. (-8)³ = -512. √(-512) is not a real number (cannot take the square root of a negative number). So the expression is not possible in real numbers.
**Explanation**: √144 = 12. (-8)³ = -512. √(-512) does not exist in real numbers. A common error is trying to calculate √(-512) as a real number.
**Tags**: directed-numbers, square-roots, domain

### Question Q-N2-07
**Difficulty**: Standard
**Question**: Arrange in order: -5, -|-3|, 0, |-7|, -2²
**Answer**: -5, -4, -3, 0, 7 (i.e., -5, -2², -|-3|, 0, |-7|)
**Explanation**: Evaluate each: -|-3| = -3, |-7| = 7, -2² = -4 (not (-2)² = 4). Order from smallest: -5, -4, -3, 0, 7. A common error is thinking -2² = 4.
**Tags**: directed-numbers, ordering, absolute-value

### Question Q-N2-08
**Difficulty**: Challenge
**Question**: Find all pairs of integers (a, b) such that a × b = -24 and a + b = -5.
**Answer**: (-8, 3): (-8)×3 = -24, (-8)+3 = -5. Also (3, -8).
**Explanation**: Factors of -24: list pairs with opposite signs. (1,-24), (-1,24), (2,-12), (-2,12), (3,-8), (-3,8), (4,-6), (-4,6). Check which pair sums to -5: 3+(-8) = -5. A common error is not listing all factor pairs.
**Tags**: directed-numbers, factors, simultaneous

### Question Q-N2-09
**Difficulty**: Challenge
**Question**: A submarine is at -120 m below sea level. It rises 45 m, then descends 80 m, then rises 30 m. What is its final position relative to sea level?
**Answer**: -120 + 45 = -75. -75 - 80 = -155. -155 + 30 = -125 m (125 m below sea level).
**Explanation**: Track position step by step: -120+45-80+30 = -125. A common error is treating 'descends 80' as adding +80 instead of subtracting 80 (or adding -80).
**Tags**: directed-numbers, word-problem, sea-level
