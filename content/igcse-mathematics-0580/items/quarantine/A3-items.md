### Question Q-A3-02

### Question Q-A3-09
**Difficulty**: Challenge
**Question**: The nth term of a sequence is an² + bn + c. The first three terms are 2, 6, 12. Find a, b, and c.
**Answer**: a = 1, b = 1, c = 0, so nth term = n² + n.
**Explanation**: n=1: a+b+c=2. n=2: 4a+2b+c=6. n=3: 9a+3b+c=12. Second difference = 2, so a=1. Then b=1, c=0. A common error is not using simultaneous equations.
**Tags**: sequences, quadratic, coefficients

### Question Q-A3-10
**Difficulty**: Challenge
**Question**: A sequence has nth term n(n+1)/2. Write down the first 5 terms. What is this sequence called?
**Answer**: 1, 3, 6, 10, 15. These are triangular numbers.
**Explanation**: n=1: 1×2/2=1, n=2: 2×3/2=3, n=3: 3×4/2=6, n=4: 4×5/2=10, n=5: 5×6/2=15. Triangular numbers. A common error is not recognising the formula.
**Tags**: sequences, triangular-numbers, special-sequences
