### Question Q-A5-01
**Difficulty**: Standard
**Question**: Solve: x + y = 8, 2x - y = 4.
**Answer**: Add: 3x = 12, x = 4. y = 8 - 4 = 4. Check: 2(4)-4 = 4 ✓.
**Explanation**: Adding eliminates y. x=4, y=4. A common error is subtracting: 3x = 4, which is wrong.
**Tags**: simultaneous-equations, elimination

### Question Q-A5-04
**Difficulty**: Standard
**Question**: Solve: 2x + 3y = 13, 4x - y = 5.
**Answer**: From second: y = 4x-5. Substitute: 2x + 3(4x-5) = 13, 2x+12x-15=13, 14x=28, x=2. y=4(2)-5=3.
**Explanation**: Express y from the easier equation, substitute. x=2, y=3. Check: 2(2)+3(3)=13 ✓.
**Tags**: simultaneous-equations, substitution

### Question Q-A5-10
**Difficulty**: Challenge
**Question**: The sum of two numbers is 15 and their difference is 7. Find the numbers.
**Answer**: x+y=15, x-y=7. Add: 2x=22, x=11. y=15-11=4. Numbers: 11 and 4.
**Explanation**: Set up simultaneous equations. Add to find x. A common error is x+y=15, x-y=7, then x=15-7=8 (treating as one equation).
