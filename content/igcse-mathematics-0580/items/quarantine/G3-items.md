### Question Q-G3-01
**Difficulty**: Foundation
**Question**: Two angles on a straight line are 65° and x°. Find x.
**Answer**: x = 180° - 65° = 115°
**Explanation**: Angles on a straight line sum to 180°. A common error is 65° + x° = 90° (confusing with complementary).
**Tags**: angles, straight-line

### Question Q-G3-02
**Difficulty**: Foundation
**Question**: Two vertically opposite angles are formed by intersecting lines. One is 73°. What is the other?
**Answer**: 73° (vertically opposite angles are equal).
**Explanation**: Vertically opposite angles are always equal. A common error is 180° - 73° = 107°.
**Tags**: angles, vertically-opposite

### Question Q-G3-03
**Difficulty**: Standard
**Question**: In a triangle, two angles are 55° and 72°. Find the third angle.
**Answer**: 180° - 55° - 72° = 53°
**Explanation**: Angles in a triangle sum to 180°. A common error is 360° - 55° - 72°.
**Tags**: angles, triangle

### Question Q-G3-04
**Difficulty**: Standard
**Question**: Two parallel lines are cut by a transversal. If one angle is 110°, find the corresponding angle.
**Answer**: 110° (corresponding angles are equal when lines are parallel).
**Explanation**: Corresponding angles (F-angles) are equal. A common error is 180° - 110° = 70° (that would be the co-interior angle).
**Tags**: angles, parallel-lines, corresponding

### Question Q-G3-05
**Difficulty**: Standard
**Question**: Two parallel lines are cut by a transversal. One angle is 110°. Find the co-interior (consecutive interior) angle.
**Answer**: 180° - 110° = 70°
**Explanation**: Co-interior angles (C-angles) sum to 180° when lines are parallel. A common error is saying they are equal.
**Tags**: angles, parallel-lines, co-interior

### Question Q-G3-06
**Difficulty**: Standard
**Question**: An isosceles triangle has one angle of 100°. Find the other two angles.
**Answer**: The 100° angle must be the vertex (base angles are equal and acute). Base angles = (180°-100°)/2 = 40° each.
**Explanation**: In an isosceles triangle, the two base angles are equal. 100° cannot be a base angle (two × 100° > 180°). A common error is (180-100)/1 = 80°.
**Tags**: angles, isosceles-triangle

### Question Q-G3-07
**Difficulty**: Standard
**Question**: In the diagram, ABCD is a parallelogram. Angle A = 65°. Find angles B, C, and D.
**Answer**: B = 115°, C = 65°, D = 115°. Adjacent angles in a parallelogram are supplementary (sum to 180°). Opposite angles are equal.
**Explanation**: Opposite angles equal: C = A = 65°. Adjacent: B = 180-65 = 115°, D = B = 115°. A common error is all angles = 65°.
**Tags**: angles, parallelogram

### Question Q-G3-08
**Difficulty**: Standard
**Question**: An exterior angle of a regular polygon is 36°. Find the interior angle and the number of sides.
**Answer**: Interior = 180° - 36° = 144°. Number of sides = 360°/36° = 10 (decagon).
**Explanation**: Interior + exterior = 180°. n = 360°/exterior. A common error is 360°/144° = 2.5.
**Tags**: angles, exterior, regular-polygon

### Question Q-G3-09
**Difficulty**: Challenge
**Question**: In triangle ABC, angle A = 2x, angle B = 3x, angle C = 4x. Find each angle.
**Answer**: 2x + 3x + 4x = 180°, 9x = 180°, x = 20°. A = 40°, B = 60°, C = 80°.
**Explanation**: Angles in triangle = 180°. 9x = 180, x = 20. A common error is not setting up the equation correctly.
**Tags**: angles, algebra, triangle

### Question Q-G3-10
**Difficulty**: Challenge
**Question**: Lines AB and CD are parallel. A transversal cuts them at E and F. If angle AEF = 3x + 10 and angle EFD = 5x - 30, find x and both angles. (These are alternate angles.)
**Answer**: Alternate angles are equal: 3x+10 = 5x-30, 40 = 2x, x = 20. Both angles = 3(20)+10 = 70°. Check: 5(20)-30 = 70° ✓.
**Explanation**: Alternate (Z-angles) are equal. Set up and solve. A common error is 3x+10 + 5x-30 = 180 (treating as co-interior).
**Tags**: angles, parallel-lines, alternate, algebra
