### Question Q-A1-01
**Difficulty**: Foundation
**Question**: Simplify: 3a + 2b - a + 4b
**Answer**: 2a + 6b
**Explanation**: Collect like terms: 3a - a = 2a, 2b + 4b = 6b. Answer: 2a + 6b. A common error is 3a + 2b - a + 4b = 2a + 2b (forgetting to combine the b terms).
**Tags**: simplification, like-terms

### Question Q-A1-02
**Difficulty**: Foundation
**Question**: Expand: 3(2x - 5)
**Answer**: 6x - 15
**Explanation**: Multiply each term inside the bracket by 3: 3×2x = 6x, 3×(-5) = -15. Answer: 6x - 15. A common error is 6x - 5 (only multiplying the first term).
**Tags**: expansion, single-bracket

### Question Q-A1-03
**Difficulty**: Foundation
**Question**: Expand: x(x + 3)
**Answer**: x² + 3x
**Explanation**: x × x = x², x × 3 = 3x. Answer: x² + 3x. A common error is x² + 3 (forgetting the x on the second term).
**Tags**: expansion, single-bracket

### Question Q-A1-04
**Difficulty**: Standard
**Question**: Expand and simplify: (x + 3)(x - 5)
**Answer**: x² - 5x + 3x - 15 = x² - 2x - 15
**Explanation**: FOIL: x×x = x², x×(-5) = -5x, 3×x = 3x, 3×(-5) = -15. Combine: x² - 2x - 15. A common error is x² - 15 (only multiplying first and last).
**Tags**: expansion, double-bracket, quadratic

### Question Q-A1-05
**Difficulty**: Standard
**Question**: Factorise: 6x + 15
**Answer**: 3(2x + 5)
**Explanation**: HCF of 6 and 15 is 3. Factor out 3: 3(2x + 5). A common error is 3(2x + 15) (only dividing the first term).
**Tags**: factorisation, hcf, single-bracket

### Question Q-A1-06
**Difficulty**: Standard
**Question**: Factorise: x² + 7x + 12
**Answer**: (x + 3)(x + 4)
**Explanation**: Find two numbers that multiply to 12 and add to 7: 3 and 4. So (x+3)(x+4). Check: (x+3)(x+4) = x²+7x+12. A common error is (x+6)(x+2) = x²+8x+12.
**Tags**: factorisation, quadratic, trinomial

### Question Q-A1-07
**Difficulty**: Standard
**Question**: Factorise: x² - 9
**Answer**: (x + 3)(x - 3)
**Explanation**: This is the difference of two squares: x² - 3² = (x+3)(x-3). A common error is (x-3)² = x²-6x+9 (which has a middle term).
**Tags**: factorisation, difference-of-squares

### Question Q-A1-08
**Difficulty**: Standard
**Question**: Substitute a = -2, b = 3 into the formula P = 2a + 3b.
**Answer**: P = 2(-2) + 3(3) = -4 + 9 = 5
**Explanation**: 2×(-2) = -4, 3×3 = 9, -4+9 = 5. A common error is 2×(-2) = -2 (forgetting to multiply by 2).
**Tags**: substitution, formulae, negative-numbers

### Question Q-A1-09
**Difficulty**: Standard
**Question**: Simplify: (4x²y³)²
**Answer**: 16x⁴y⁶
**Explanation**: Square the coefficient: 4² = 16. Double each exponent: (x²)² = x⁴, (y³)² = y⁶. Answer: 16x⁴y⁶. A common error is 4x⁴y⁶ (not squaring the 4).
**Tags**: indices, algebra, simplification

### Question Q-A1-10
**Difficulty**: Standard
**Question**: Expand: (2x + 1)(3x - 4)
**Answer**: 6x² - 8x + 3x - 4 = 6x² - 5x - 4
**Explanation**: FOIL: 2x×3x = 6x², 2x×(-4) = -8x, 1×3x = 3x, 1×(-4) = -4. Combine: 6x² - 5x - 4. A common error is forgetting the middle terms.
**Tags**: expansion, double-bracket

### Question Q-A1-11
**Difficulty**: Standard
**Question**: Factorise: 2x² + 8x
**Answer**: 2x(x + 4)
**Explanation**: Common factor: 2x. 2x²÷2x = x, 8x÷2x = 4. Answer: 2x(x+4). A common error is 2(x²+4x) (not taking out the x).
**Tags**: factorisation, common-factor

### Question Q-A1-12
**Difficulty**: Challenge
**Question**: Factorise: 3x² - 11x - 4
**Answer**: (3x + 1)(x - 4)
**Explanation**: Need (3x+a)(x+b) where a×b = -4 and 3b+a = -11. Try a=1, b=-4: 3(-4)+1 = -11 ✓. So (3x+1)(x-4). Check: 3x²-12x+x-4 = 3x²-11x-4. A common error is (3x-1)(x+4) = 3x²+11x-4 (wrong sign on middle term).
