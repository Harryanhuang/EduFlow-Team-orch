### Question Q-A4-01
**Difficulty**: Foundation
**Question**: Find the gradient of the line passing through (1, 3) and (5, 11).
**Answer**: (11-3)/(5-1) = 8/4 = 2
**Explanation**: Gradient = (y₂-y₁)/(x₂-x₁) = (11-3)/(5-1) = 8/4 = 2. A common error is (5-1)/(11-3) = 4/8 = 0.5 (reversed).
**Tags**: gradient, coordinates

### Question Q-A4-04
**Difficulty**: Standard
**Question**: Find the equation of the line with gradient 3 passing through (2, 7).
**Answer**: y = 3x + 1
**Explanation**: y = mx + c, m = 3. 7 = 3(2) + c, c = 1. So y = 3x + 1. A common error is y = 3x + 7 (using the y-value as c).
**Tags**: equation-of-line, gradient, point

### Question Q-A4-05

**Question**: Complete the table for y = x² - 2x for x = -1, 0, 1, 2, 3.
**Answer**: x=-1: y=3, x=0: y=0, x=1: y=-1, x=2: y=0, x=3: y=3
**Explanation**: Substitute each x value: (-1)²-2(-1)=1+2=3, 0-0=0, 1-2=-1, 4-4=0, 9-6=3. A common error is (-1)² = -1.

### Question Q-A4-08
**Difficulty**: Standard
**Question**: Find the gradient of a line perpendicular to y = 2x + 3.
**Answer**: -1/2
**Explanation**: Perpendicular lines: m₁ × m₂ = -1. So m₂ = -1/m₁ = -1/2. A common error is -2 (negating but not taking the reciprocal).
**Tags**: perpendicular-lines, gradient

### Question Q-A4-09
**Difficulty**: Standard
**Question**: Describe the shape of the graph of y = 1/x. What happens as x approaches 0?
**Answer**: Hyperbola with two branches. As x → 0⁺, y → +∞. As x → 0⁻, y → -∞. The axes are asymptotes.
**Explanation**: y = 1/x has asymptotes at x=0 and y=0. Never touches either axis. A common error is saying it crosses the axes.
**Tags**: reciprocal-graph, asymptotes, hyperbola

### Question Q-A4-10
**Difficulty**: Standard
**Question**: Sketch y = 2ˣ, stating the y-intercept and the behaviour as x → -∞.
**Answer**: Exponential growth curve. y-intercept: (0, 1). As x → -∞, y → 0 (horizontal asymptote at y = 0).
**Explanation**: 2⁰ = 1, so y-intercept at (0,1). 2^(-large) approaches 0. A common error is thinking 2ˣ goes through the origin.
**Tags**: exponential-graph, asymptote

### Question Q-A4-11
**Difficulty**: Challenge
**Question**: The line y = x + 2 intersects the curve y = x² - 4. Find the coordinates of the points of intersection.
**Answer**: x+2 = x²-4, x²-x-6 = 0, (x-3)(x+2) = 0. x=3: y=5. x=-2: y=0. Points: (3, 5) and (-2, 0).
**Explanation**: Set equations equal: x+2 = x²-4. Rearrange: x²-x-6=0. Factor: (x-3)(x+2)=0. A common error is not checking both points.
**Tags**: intersection, line, curve, quadratic

### Question Q-A4-12
**Difficulty**: Challenge
**Question**: A line passes through (1, 2) and (4, 14). Find its equation in the form y = mx + c, then find where it crosses the x-axis.
**Answer**: m = (14-2)/(4-1) = 12/3 = 4. y = 4x + c: 2 = 4(1) + c, c = -2. y = 4x - 2. x-axis: y=0, 0=4x-2, x=1/2.
**Explanation**: Gradient = 4, c = -2. x-intercept: 0 = 4x-2, x = 0.5. A common error is c = 2 (sign error).
