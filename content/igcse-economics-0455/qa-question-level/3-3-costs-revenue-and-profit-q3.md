### Question Q-3.3-03
**Difficulty**: Standard
**Question**: Calculate total cost if fixed costs are $5000, variable cost per unit is $10, and output is 300 units.
**Answer**: Total cost = Fixed costs + (Variable cost per unit x output) = 5000 + (10 x 300) = 5000 + 3000 = $8000.
**Explanation**: TC = FC + (VC per unit x output). A common error is adding FC and VC per unit without multiplying by output: 5000 + 10 = 5010.
