## 一、设计原则

1. **横向一致**：4 所学校都按同一骨架展开，便于家长横向对比。
2. **纵向独立**：每所学校的特色板块（差异化项）允许独立子节，避免被通用骨架磨平个性。
3. **信息真实**：每个一级章节均能在 4 所样本学校的官网信息板块中找到对应（已逐校验证）。
4. **决策导向**：板块围绕择校核心问题——课程、升学、费用、适配度——组织，而非按行政维度（学校概况→历史→...）堆砌。
5. **可扩展**：方案 A 是「一校一文件夹」，新增学校时复制 `_template/学校介绍模板.md` 即可。

## 二、通用骨架（12 节）

### 1. 学校速览（TL;DR）
> 用途：给家长 30 秒内抓取关键决策因子。
- 学校名称（中/英/常用缩写）
- 办学类型（公立 / 私立 / 国际）
- 课程体系（A-Level / IBDP / AP / DSE / 双语 等）
- 所在城市 / 区 / 校区
- 招生学段 / 入学年级
- 学费区间（人民币 / 港币 / 美元 / 年）
- 建校年份 / 办学年限
- 一句话定位（如"美式教育 + 基督教背景 + 45 国籍"）

### 2. 学校概况（About）
- 使命 / 愿景 / 核心价值
- 校训 / 文化标识
- 隶属机构（如香港圣公会、CIS、WASC、EARCOS 等认证）
- 学生多元构成（国籍 / 地区分布）

### 3. 办学历史（History）
- 创办年份 / 关键里程碑
- 重要资质与认证清单（剑桥、Edexcel、College Board 等）

### 4. 课程体系（Curriculum）
#### 4.1 主修课程
- 主流国际课程（A-Level / IBDP / AP / DSE 等）
- 双轨或单轨
#### 4.2 学科部门
- 文 / 理 / 工 / 商 / 艺术 / 体育 / 宗教 等
#### 4.3 特色项目
- 双语沉浸、IB 预备班、多语种、研学项目等
#### 4.4 选修课 / 课外学术

### 5. 招生信息（Admissions）
#### 5.1 招生年级
#### 5.2 申请流程（本地 / 国际 / 寄宿）
#### 5.3 入学考试 / 面试
#### 5.4 重要日期（信息会 / 报名截止 / 考试日 / 放榜日）
#### 5.5 学费、奖学金 / 学费减免

### 6. 升学成果（Academic Excellence）
#### 6.1 公开考试成绩（DSE / A-Level / IBDP / AP）
#### 6.2 大学录取（按国别 / 排名 / 人数分布）
#### 6.3 校友导师计划
#### 6.4 职业辅导

### 7. 校园设施（Campus & Facilities）
- 教学楼 / 实验室 / 图书馆
- 体育设施（体育馆 / 游泳池 / 操场）
- 艺术设施（剧场 / 琴房 / 画室 / 黑箱剧场）
- 寄宿设施（如适用）

### 8. 课外活动与学生生活（Student Life）
- 学生组织 / 社团（学术 / 兴趣 / 公益 / 服务）
- 体育校队
- 艺术与表演
- 学生活动周 / 主题项目
- 交换 / 研学 / 国际项目（如 Round Square、姊妹学校）
- 夏令营 / 冬令营

### 9. 师资力量（Faculty）
- 师生比
- 教师资质（硕士 / 博士 / 外籍占比）
- 顾问与辅导体系（升学、心理、生活）
- 教师发展 / 培训

### 10. 联系方式（Contact）
- 校区地址 / 总机 / 招生热线
- 邮箱 / 官网 / 微信公众号
- 访校预约方式（线上 / 线下）

### 11. 特色亮点（差异化板块，按校填写）
> 仅记录该校独有的特色项目或历史亮点。
- 深国交：绿洲 SOIS、TEDxSCIE Youth、纵横杯辩论、HackMIT China、牛剑录取列表（2023-2026）、21 枚藤校 Top10（2026 届美本）、香港本科录取。
- 广外 AP：1962 年首批 7 所外语学校之一、2009 年南沙重建、多语种特色（俄 / 法 / 德 / 西 / 日）、AP + 本土高中双轨、U.S. College Board 代码学校、歌德学院合作。
- 圣保罗男女（SPCC）：香港圣公会成员校、DSE + IBDP 双轨、F4 Preparatory Year、Round Square 国际组织成员、110 周年校庆、IBDP & F4 Prep 招生独立线、设有 Primary School（spccps.edu.hk）。
- 汉基 HKIS：美式教育 + 基督教背景、双语沉浸项目（2025-26 R1 启动）、45+ 国籍、Dragon Center for Activities & Athletics（2025-11-01 开幕）、18 位顾问、Great Place to Work 认证。

### 12. 家长决策参考（FAQ / 适配画像）
- 适合什么样的学生？（画像）
- 优势 / 短板
- 申请建议时间线
- 常见问题（入学考难度、转学政策、走读 / 寄宿、签证）

## 四、与现有知识库的对接

- 现有 `content/ap-calculus-ab/`、`content/igcse-*/`、`content/ap-physics*/` 等学科题库保持不变。
- 新建 `content/school/` 作为**平行分类（学校维度）**，与学科题库并列。

## 五、与其它方案的关系

- 方案 A（本方案）：**一校一文件夹** + 12 节通用骨架 + 差异化子节。优点：横向易对比、纵向保留个性、信息密度高。适合"决策导向"的家长。
- 与方案 B（待议）：若老板后续倾向"先按课程体系分类（如 A-Level 区 / IBDP 区）再装学校"，需要再起一份方案 B 文档对比。
- 与方案 C（待议）：若老板倾向"轻量卡片（每校只 1 页 Markdown，无子文件）"，也需独立方案。

## 六、验收口径

- ✅ 每个一级章节在 4 所样本校官网均有真实对应板块（已逐校验证）。
- ✅ 12 节结构能覆盖家长择校时最高频的 7 类问题（课程 / 升学 / 费用 / 学制 / 地理位置 / 适配度 / 申请难度）。
- ✅ 差异化板块（第 11 节）允许 4 校各自扩展，不被通用骨架压平。
