A block of mass $m$ is attached to a spring with force constant $k$ and oscillates without friction. Newton's second law gives:

$$\frac{d^2x}{dt^2} = -\frac{k}{m}\,x$$

Which of the following is the general solution to this differential equation?

## Options
A) $x(t) = A\sin\!\left(\sqrt{\frac{k}{m}}\,t\right)$
B) $x(t) = A\cos\!\left(\sqrt{\frac{k}{m}}\,t + \phi\right)$
C) $x(t) = A\,e^{-\gamma t}\cos\!\left(\sqrt{\frac{k}{m}}\,t\right)$
D) $x(t) = \frac{k}{m}\,t^2 + C_1\,t + C_2$

## Answer
B

## Explanation
We identify $\omega^2 = k/m$, so $\omega = \sqrt{k/m}$. The equation becomes $\frac{d^2x}{dt^2} + \omega^2 x = 0$, the defining ODE of simple harmonic motion. Its general solution is $x(t) = A\cos(\omega t + \phi)$, containing two arbitrary constants.

Option A is only one particular solution (equivalent to choosing $\phi = -\pi/2$); it lacks the arbitrary phase needed to represent all solutions. Option C describes damped harmonic motion, which requires a velocity-dependent resistive force absent here. Option D is a polynomial; substituting it into the left side yields $2k/m$, which cannot equal $-(k/m) \cdot (k/m)t^2$ for all $t$, so it does not satisfy the ODE.
