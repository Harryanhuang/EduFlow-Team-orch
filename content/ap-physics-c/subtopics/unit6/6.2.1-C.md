A $0.500\,\text{kg}$ block on a spring ($k = 32.0\,\text{N/m}$) oscillates with amplitude $A = 0.400\,\text{m}$. Using energy conservation, find the speed of the block when it passes through $x = 0.200\,\text{m}$.

## Options
A) $2.77\,\text{m/s}$
B) $0.800\,\text{m/s}$
C) $1.60\,\text{m/s}$
D) $4.80\,\text{m/s}$

## Answer
A

## Explanation
**Method 1 -- Energy conservation:**

$$E = \frac{1}{2}kA^2 = \frac{1}{2}(32.0)(0.400)^2 = \frac{1}{2}(32.0)(0.160) = 2.56\,\text{J}$$

At $x = 0.200$ m:

$$U = \frac{1}{2}kx^2 = \frac{1}{2}(32.0)(0.200)^2 = \frac{1}{2}(32.0)(0.0400) = 0.640\,\text{J}$$

$$K = E - U = 2.56 - 0.640 = 1.92\,\text{J}$$

$$v = \sqrt{\frac{2K}{m}} = \sqrt{\frac{2 \times 1.92}{0.500}} = \sqrt{7.68} = 2.77\,\text{m/s}$$

**Method 2 -- Using v(x) formula:**

$$\omega = \sqrt{\frac{k}{m}} = \sqrt{\frac{32.0}{0.500}} = \sqrt{64.0} = 8.00\,\text{rad/s}$$

$$v = \omega\sqrt{A^2 - x^2} = 8.00\sqrt{(0.400)^2 - (0.200)^2} = 8.00\sqrt{0.160 - 0.0400} = 8.00\sqrt{0.120} = 8.00 \times 0.346 = 2.77\,\text{m/s}$$

Option B (0.800) results from $v = \omega x$, which is incorrect. Option C (1.60) results from $v = \omega A$, the maximum speed. Option D (4.80) results from $v = \sqrt{2kA^2/m}$, an arithmetic error.

The answer is **A**.
