common_mistake: Using A = √(x₀² + v₀²) without dividing v₀ by ω.
explanation_context: Squaring the two initial condition equations and adding:
A² cos²φ + A² sin²φ = x₀² + v₀²/ω² → A² = x₀² + v₀²/ω².

A block-spring system has $\omega = 10.0\,\text{rad/s}$. At $t = 0$, the block is at $x_0 = 0.300\,\text{m}$ and has velocity $v_0 = -4.00\,\text{m/s}$. What is the amplitude?

## Options
A) $0.300\,\text{m}$
B) $0.400\,\text{m}$
C) $0.500\,\text{m}$
D) $0.700\,\text{m}$

## Answer
C

## Explanation
Using the amplitude formula derived from initial conditions:

$$A = \sqrt{x_0^2 + \frac{v_0^2}{\omega^2}} = \sqrt{(0.300)^2 + \frac{(-4.00)^2}{(10.0)^2}}$$

$$A = \sqrt{0.0900 + \frac{16.0}{100}} = \sqrt{0.0900 + 0.160} = \sqrt{0.250} = 0.500\,\text{m}$$

This can also be understood energetically. Conservation of energy gives $\frac{1}{2}kA^2 = \frac{1}{2}kx_0^2 + \frac{1}{2}mv_0^2$. Substituting $k = m\omega^2$ and dividing by $\frac{1}{2}m\omega^2$ yields exactly $A^2 = x_0^2 + v_0^2/\omega^2$.

Option A (0.300) would be correct only if $v_0 = 0$. Option B (0.400) results from an arithmetic error. Option D (0.700) is the sum $x_0 + |v_0|/\omega = 0.300 + 0.400$, which is not the correct formula.

The answer is **C**.
