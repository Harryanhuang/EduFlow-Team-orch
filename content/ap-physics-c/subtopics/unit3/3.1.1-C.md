A particle moves along a path from $(0,0)$ to $(2,4)$ along the parabola $y = x^2$, subject to a force $\vec{F}(x,y) = (2xy)\hat{i} + (x^2)\hat{j}$ (N). What is the work done by the force?

## Options
A) $W = 0$ J
B) $W = 8$ J
C) $W = 16$ J
D) $W = 32$ J

## Answer
C

## Explanation
Work is the line integral of the force along the path:

$$W = \int_C \vec{F} \cdot d\vec{r} = \int_C (F_x\, dx + F_y\, dy)$$

Parameterize the path using $x$ as the parameter: $y = x^2$, so $dy = 2x\, dx$, with $x$ going from $0$ to $2$.

Substitute into the integral:

$$W = \int_{0}^{2} \left[(2xy)\, dx + (x^2)(2x)\, dx\right]$$

With $y = x^2$: $F_x = 2x \cdot x^2 = 2x^3$, $F_y = x^2$.

$$W = \int_0^2 (2x^3 + 2x^3)\, dx = \int_0^2 4x^3\, dx = \left[x^4\right]_0^2 = 16 \text{ J}$$

The work done is $16$ J.

**Note on conservative force:** Check $\partial F_y/\partial x = 2x$ and $\partial F_x/\partial y = 2x$. Since these are equal, the force is conservative, and the work should be path-independent. Indeed, integrating $F_x = -\partial U/\partial x = 2xy$ gives $U = -x^2 y + g(y)$. Then $-\partial U/\partial y = x^2 - g'(y) = F_y = x^2 \implies g'(y) = 0 \implies g(y) = \text{const}$. So $U(x,y) = -x^2 y + C$, and $W = -\Delta U = -[(-4 \cdot 4) - 0] = 16$ J. The path-independence is verified. The correct answer is **C**.
