A force $F(t) = 4t$ N acts on a $3$ kg particle starting from rest at $t = 0$. Find the work done by the force from $t = 0$ to $t = 3$ s.

## Options
A) $W = 18$ J
B) $W = 27$ J
C) $W = 36$ J
D) $W = 54$ J

## Answer
D

## Explanation
First, find the acceleration and velocity by integrating Newton's second law:

$$a(t) = \frac{F(t)}{m} = \frac{4t}{3} \text{ m/s}^2$$
$$v(t) = \int_0^t a(t')\, dt' = \int_0^t \frac{4t'}{3}\, dt' = \frac{2t^2}{3} \text{ m/s}$$

Work is defined as $W = \int F\, dx$. Use the substitution $dx = v(t)\, dt$ to convert the integration variable to time:

$$W = \int_{t_i}^{t_f} F(t)\, v(t)\, dt = \int_0^3 (4t)\left(\frac{2t^2}{3}\right) dt = \int_0^3 \frac{8}{3}t^3\, dt$$
$$W = \frac{8}{3}\left[\frac{t^4}{4}\right]_0^3 = \frac{8}{3} \cdot \frac{81}{4} = \frac{8}{3} \cdot 20.25 = 54 \text{ J}$$

The work done is 54 J. The correct answer is **D**.

We can verify using the work-energy theorem: $v(3) = 2(9)/3 = 6$ m/s, so $\Delta KE = \frac{1}{2}(3)(6)^2 = 54$ J. This matches.
