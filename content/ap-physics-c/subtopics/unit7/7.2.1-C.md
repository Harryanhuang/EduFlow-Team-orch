Two satellites, X and Y, are in circular orbits around the same planet. Satellite X orbits at radius $r_X = R$ and satellite Y orbits at radius $r_Y = 4R$, where $R$ is some reference distance. What is the ratio of their orbital periods $T_Y / T_X$?

## Options
A) $4$
B) $2$
C) $8$
D) $16$

## Answer
C

## Explanation
Kepler's third law derived from centripetal force balance:

$$T = 2\pi\sqrt{\frac{r^3}{GM}}$$

Since both satellites orbit the same central mass $M$, the constant $2\pi/\sqrt{GM}$ is identical for both. Therefore:

$$\frac{T_Y}{T_X} = \frac{2\pi\sqrt{r_Y^3/(GM)}}{2\pi\sqrt{r_X^3/(GM)}} = \sqrt{\frac{r_Y^3}{r_X^3}} = \left(\frac{r_Y}{r_X}\right)^{3/2}$$

Substituting $r_Y/r_X = 4$:

$$\frac{T_Y}{T_X} = (4)^{3/2} = (\sqrt{4})^3 = 2^3 = 8$$

Option A ($T \propto r$) is a common misconception (confusing period with radius). Option B ($\sqrt{4} = 2$) would result from $T \propto r^{1/2}$, which undercorrects. Option D ($4^2 = 16$) would result from $T \propto r^2$.

The answer is **C**.
