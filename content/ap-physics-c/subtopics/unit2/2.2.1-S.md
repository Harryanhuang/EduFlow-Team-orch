A $3$ kg object falls from rest through a resistive medium with linear drag $F_{drag} = -bv$ where $b = 2$ N·s/m. What is the terminal velocity, and what is the speed after one time constant $\tau = m/b$?

## Options
A) $v_T = 14.7$ m/s; $v(\tau) = 9.29$ m/s
B) $v_T = 4.90$ m/s; $v(\tau) = 3.10$ m/s
C) $v_T = 14.7$ m/s; $v(\tau) = 12.7$ m/s
D) $v_T = 1.63$ m/s; $v(\tau) = 1.03$ m/s

## Answer
A

## Explanation
Newton's second law for vertical motion with linear drag (downward positive):

$$m\frac{dv}{dt} = mg - bv$$

Terminal velocity when $dv/dt = 0$: $mg - bv_T = 0 \implies v_T = \frac{mg}{b} = \frac{3(9.8)}{2} = 14.7$ m/s.

The time constant: $\tau = m/b = 3/2 = 1.5$ s.

Solve the ODE by separation of variables:

$$\frac{dv}{g - \frac{b}{m}v} = dt$$

Integrate: $-\frac{m}{b}\ln\left(g - \frac{b}{m}v\right) = t + C$

Using $v(0) = 0$: $-\frac{m}{b}\ln(g) = C$

$$-\frac{m}{b}\ln\left(g - \frac{b}{m}v\right) = t - \frac{m}{b}\ln(g)$$
$$\ln\left(\frac{g - \frac{b}{m}v}{g}\right) = -\frac{b}{m}t$$
$$g - \frac{b}{m}v = ge^{-bt/m}$$
$$v(t) = \frac{mg}{b}\left(1 - e^{-bt/m}\right) = v_T(1 - e^{-t/\tau})$$

After one time constant $t = \tau = 1.5$ s:
$$v(\tau) = 14.7(1 - e^{-1}) = 14.7(1 - 0.368) = 14.7(0.632) \approx 9.29 \text{ m/s}$$

The object reaches about 63.2% of terminal velocity after one time constant. The correct answer is **A**.
