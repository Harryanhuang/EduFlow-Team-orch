A $4$ kg mass is subject to a force $F(t) = 24t - 12$ N. At $t = 0$, the mass is at $x = 2$ m with velocity $v = 5$ m/s. What is the position $x(t)$ as a function of time?

## Options
A) $x(t) = t^3 - 1.5t^2 + 5t + 2$
B) $x(t) = 3t^3 - 3t^2 + 5t + 2$
C) $x(t) = t^3 - 1.5t^2 + 2.5t + 5$
D) $x(t) = 3t^3 - 1.5t^2 + 2$

## Answer
A

## Explanation
Newton's second law gives the acceleration:

$$a(t) = \frac{d^2x}{dt^2} = \frac{F(t)}{m} = \frac{24t - 12}{4} = 6t - 3$$

Integrate once to find velocity, using $v(0) = 5$:

$$v(t) = \int (6t - 3)\, dt = 3t^2 - 3t + C_1$$

Using $v(0) = 5$: $3(0)^2 - 3(0) + C_1 = 5 \implies C_1 = 5$.

$$v(t) = 3t^2 - 3t + 5$$

Integrate again to find position, using $x(0) = 2$:

$$x(t) = \int (3t^2 - 3t + 5)\, dt = t^3 - \frac{3}{2}t^2 + 5t + C_2$$

Using $x(0) = 2$: $0 - 0 + 0 + C_2 = 2 \implies C_2 = 2$.

$$x(t) = t^3 - 1.5t^2 + 5t + 2$$

Let's verify at $t = 1$: $x(1) = 1 - 1.5 + 5 + 2 = 6.5$ m.
$v(1) = 3 - 3 + 5 = 5$ m/s.
$a(1) = 6 - 3 = 3$ m/s².
Check: $F(1) = 24(1) - 12 = 12$ N. $ma = 4(3) = 12 = F(1)$. Verified.

The correct answer is **A**.
