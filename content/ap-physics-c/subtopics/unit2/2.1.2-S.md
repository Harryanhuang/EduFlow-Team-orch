Two blocks $m_1 = 2$ kg and $m_2 = 4$ kg are connected by a massless string on a horizontal frictionless surface. Block $m_1$ is pulled by a force $F(t) = 12t$ N. What is the tension in the string at $t = 2$ s?

## Options
A) $T = 8$ N
B) $T = 12$ N
C) $T = 16$ N
D) $T = 24$ N

## Answer
C

## Explanation
Since the string is massless and inextensible, both blocks have the same acceleration $a(t)$ at any instant.

Apply Newton's second law to each block:

For $m_1$ (being pulled rightward by $F(t)$ and leftward by $T$):
$$F(t) - T = m_1 a \implies 12t - T = 2a$$

For $m_2$ (pulled rightward only by $T$):
$$T = m_2 a \implies T = 4a$$

Solve simultaneously: from $T = 4a$, we have $a = T/4$. Substitute:

$$12t - T = 2\left(\frac{T}{4}\right) = \frac{T}{2}$$

$$\implies 12t = T + \frac{T}{2} = \frac{3T}{2} \implies T(t) = 8t$$

At $t = 2$ s: $T(2) = 8(2) = 16$ N.

We can verify by computing the acceleration: $a = T/4 = 4$ m/s², so $F_{net, total} = (m_1 + m_2)a = 6(4) = 24$ N, which equals $F(2) = 12(2) = 24$ N. Correct. The tension at $t = 2$ s is **C) 16 N**.
