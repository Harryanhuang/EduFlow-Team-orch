An object's position in frame $A$ is $\vec{r}_A(t) = (3t^2 + 2)\hat{i} + (4t - 5)\hat{j}$ (meters). Frame $B$ moves with constant velocity $\vec{v}_{B/A} = (-5\hat{i} + 3\hat{j})$ m/s relative to frame $A$, and at $t = 0$ the origins coincide. What are the velocity and acceleration of the object in frame $B$?

## Options
A) $\vec{v}_B = (6t + 5)\hat{i} + (1)\hat{j}$ m/s; $\vec{a}_B = 6\hat{i}$ m/s²
B) $\vec{v}_B = (6t + 2)\hat{i} + (4)\hat{j}$ m/s; $\vec{a}_B = 6\hat{i}$ m/s²
C) $\vec{v}_B = (6t - 2)\hat{i} + (7)\hat{j}$ m/s; $\vec{a}_B = 6\hat{i}$ m/s²
D) $\vec{v}_B = (6t)\hat{i} + (4)\hat{j}$ m/s; $\vec{a}_B = 6\hat{i}$ m/s²

## Answer
A

## Explanation
The Galilean transformation for position (frame $B$ moves at $\vec{v}_{B/A}$ relative to $A$):

$$\vec{r}_B = \vec{r}_A - \vec{v}_{B/A}t$$

Substituting: $\vec{v}_{B/A} = -5\hat{i} + 3\hat{j}$.

$$\vec{r}_B = [(3t^2 + 2) - (-5t)]\hat{i} + [(4t - 5) - (3t)]\hat{j} = (3t^2 + 5t + 2)\hat{i} + (t - 5)\hat{j}$$

Differentiate to find velocity in frame $B$:

$$\vec{v}_B = \frac{d\vec{r}_B}{dt} = (6t + 5)\hat{i} + (1)\hat{j} \text{ m/s}$$

The acceleration is:

$$\vec{a}_B = \frac{d\vec{v}_B}{dt} = 6\hat{i} \text{ m/s}^2$$

In frame $A$: $\vec{v}_A = (6t)\hat{i} + 4\hat{j}$ m/s, $\vec{a}_A = 6\hat{i}$ m/s².

Since $\vec{v}_{B/A}$ is constant (zero derivative), $\vec{a}_B = \vec{a}_A$. Acceleration is invariant under Galilean transformations. The velocities differ by the constant relative frame velocity: $\vec{v}_B = \vec{v}_A - \vec{v}_{B/A} = (6t\hat{i} + 4\hat{j}) - (-5\hat{i} + 3\hat{j}) = (6t + 5)\hat{i} + \hat{j}$. The correct answer is **A**.
