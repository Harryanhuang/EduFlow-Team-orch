Particle A moves along the x-axis with $x_A(t) = 3t^2 + 2t$ (m). Particle B moves along the x-axis with $x_B(t) = 5t^2 - t$ (m). What is the relative acceleration of B with respect to A?

## Options
A) $a_{B/A} = 4.00$ m/s²
B) $a_{B/A} = -4.00$ m/s²
C) $a_{B/A} = 2.00$ m/s²
D) $a_{B/A} = -2.00$ m/s²

## Answer
A

## Explanation
Relative position: $x_{B/A} = x_B - x_A = (5t^2 - t) - (3t^2 + 2t) = 2t^2 - 3t$.

Relative velocity: $v_{B/A} = \frac{d}{dt}(x_{B/A}) = 4t - 3$.

Relative acceleration: $a_{B/A} = \frac{d}{dt}(v_{B/A}) = 4.00$ m/s².

Alternatively, compute directly: $a_A = dx_A/dt^2 = 6.00$ m/s², $a_B = dx_B/dt^2 = 10.0$ m/s², and $a_{B/A} = a_B - a_A = 10.0 - 6.00 = 4.00$ m/s².

The relative acceleration is constant and equals the difference of the individual accelerations. This is a key result: in any inertial frame, accelerations are identical (no fictitious forces), so $a_{B/A} = a_B - a_A$ regardless of the observer's reference frame.
