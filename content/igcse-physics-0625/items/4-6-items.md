# Items: 4.6 — Thin converging lens

### Question Q-4.6-01
**Difficulty**: Foundation
**Question**: What is the principal focus of a converging lens?
**Answer**: The point on the principal axis where rays parallel to the axis converge after passing through the lens.
**Explanation**: The key is "parallel rays converge." A common error is saying "all rays converge at the focus."
**Tags**: converging-lens, focal-point, definition

### Question Q-4.6-02
**Difficulty**: Standard
**Question**: An object is placed 15 cm from a converging lens of focal length 10 cm. Draw a ray diagram to locate the image and describe its nature.
**Answer**: The image is real, inverted, and magnified (larger than the object). It forms beyond 2f on the other side. Two rays: one parallel to axis through focus, one through centre undeviated.
**Explanation**: Object between f and 2f -> real, inverted, magnified image beyond 2f. A common error is drawing the image on the same side as the object.
**Tags**: converging-lens, ray-diagram, image-nature

### Question Q-4.6-03
**Difficulty**: Foundation
**Question**: When a converging lens is used as a magnifying glass, where must the object be placed relative to the focal point?
**Answer**: The object must be placed between the lens and the focal point (closer to the lens than the focal length). This produces a virtual, upright, and magnified image.
**Explanation**: Object inside f -> virtual magnified image. A common error is placing the object at or beyond f.
**Tags**: magnifying-glass, converging-lens, object-position

### Question Q-4.6-04
**Difficulty**: Standard
**Question**: Calculate the magnification of a lens if the image height is 6 cm and the object height is 2 cm. Is the image magnified or diminished?
**Answer**: Magnification = 6/2 = 3. The image is magnified (3 times larger than the object).
**Explanation**: Magnification = image height / object height. A common error is inverting the ratio.
**Tags**: magnification, lens, calculation

### Question Q-4.6-05
**Difficulty**: Standard
**Question**: How can you measure the focal length of a converging lens experimentally?
**Answer**: Focus a distant object (e.g., a window or tree) onto a screen using the lens. The distance from the lens to the sharp image on the screen is approximately the focal length (since the object is effectively at infinity).
**Explanation**: Distant object -> image at focus. A common error is using a nearby object without accounting for the object distance.
**Tags**: focal-length, experiment, converging-lens

### Question Q-4.6-06
**Difficulty**: Standard
**Question**: An object is placed 30 cm from a converging lens with focal length 10 cm. Calculate the image distance using the lens formula: 1/f = 1/u + 1/v.
**Answer**: 15 cm
**Explanation**: 1/f = 1/u + 1/v. 1/10 = 1/30 + 1/v. 1/v = 1/10 - 1/30 = 3/30 - 1/30 = 2/30 = 1/15. v = 15 cm. A common error is sign convention mistakes or arithmetic errors.
**Tags**: lens-formula, calculation, image-distance

### Question Q-4.6-07
**Difficulty**: Challenge
**Question**: A converging lens has a focal length of 8 cm. An object is placed 4 cm from the lens. Describe the nature and position of the image formed.
**Answer**: The object is inside the focal length (u < f), so the image is virtual, upright, and magnified. Using the lens formula: 1/8 = 1/4 + 1/v, so 1/v = 1/8 - 1/4 = -1/8, giving v = -8 cm. The negative sign indicates a virtual image on the same side as the object, 8 cm from the lens.
**Explanation**: Object inside f produces a virtual image. A common error is not recognizing that the image distance should be negative for virtual images.
**Tags**: converging-lens, virtual-image, lens-formula, calculation

### Question Q-4.6-08
**Difficulty**: Challenge
**Question**: Compare the images formed by a converging lens when the object is placed: (a) beyond 2f, (b) between f and 2f, (c) inside f. For each case, state whether the image is real/virtual, upright/inverted, and magnified/diminished.
**Answer**: (a) Beyond 2f: real, inverted, diminished (image between f and 2f). (b) Between f and 2f: real, inverted, magnified (image beyond 2f). (c) Inside f: virtual, upright, magnified (image on same side as object). Cases (a) and (b) produce real images on the opposite side; case (c) produces a virtual image.
**Explanation**: The object position relative to f and 2f determines image properties. A common error is confusing which positions produce real vs virtual images.
**Tags**: converging-lens, image-properties, comparison, extended-response

### Question Q-4.6-09
**Difficulty**: Challenge
**Question**: In a camera, a converging lens forms an image of a distant object on film. Where should the film be placed relative to the lens? If the object moves closer, how should the lens position be adjusted?
**Answer**: For a distant object, the film should be placed at the focal point (one focal length from the lens) since parallel rays converge there. If the object moves closer, the image forms further from the lens, so the lens must be moved away from the film (or the film moved away from the lens) to maintain focus.
**Explanation**: As object distance decreases, image distance increases. A common error is thinking the film position stays fixed regardless of object distance.
**Tags**: camera, converging-lens, focusing, application
