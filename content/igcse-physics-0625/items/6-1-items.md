# Items: 6.1 — Atomic model and nuclide notation

### Question Q-6.1-01
**Difficulty**: Foundation
**Question**: An atom has 11 protons, 12 neutrons, and 11 electrons. What is its atomic number, mass number, and element symbol?
**Answer**: Atomic number = 11, mass number = 23, element symbol = Na (sodium). Nuclide notation: ²³₁₁Na.
**Explanation**: The atomic number equals the number of protons (11). The mass number equals protons + neutrons (11 + 12 = 23). Element 11 is sodium. A common error is including electrons in the mass number calculation.
**Tags**: atomic-number, mass-number, nuclide-notation, sodium

### Question Q-6.1-02
**Difficulty**: Foundation
**Question**: How many protons, neutrons, and electrons are in a neutral atom of carbon-14 (¹⁴₆C)?
**Answer**: Protons = 6, neutrons = 8, electrons = 6.
**Explanation**: The atomic number (subscript 6) gives the number of protons = 6. In a neutral atom, electrons = protons = 6. Neutrons = mass number - atomic number = 14 - 6 = 8. A common error is using 14 as the atomic number instead of the mass number.
**Tags**: carbon-14, protons, neutrons, electrons, isotope

### Question Q-6.1-03
**Difficulty**: Standard
**Question**: Define an isotope.
**Answer**: Isotopes are atoms of the same element that have the same number of protons (same atomic number) but different numbers of neutrons (different mass numbers).
**Explanation**: Isotopes share the same chemical properties because they have identical electron configurations, but differ in mass due to different neutron counts. A common error is reversing the definition to state "same neutrons, different protons."
**Tags**: isotope, definition, atomic-structure

### Question Q-6.1-04
**Difficulty**: Standard
**Question**: A magnesium ion Mg²⁺ has 12 protons and 12 neutrons. How many electrons does it have?
**Answer**: 10 electrons.
**Explanation**: A neutral magnesium atom has 12 electrons (equal to its 12 protons). The 2+ charge indicates the atom has lost 2 electrons, so 12 - 2 = 10 electrons remain. A common error is adding 2 electrons to give 14 instead of subtracting.
**Tags**: ion, electrons, charge, magnesium, atomic-structure

### Question Q-6.1-05
**Difficulty**: Standard
**Question**: Why do isotopes of the same element have identical chemical properties?
**Answer**: Chemical properties are determined by the arrangement of electrons in an atom. Since isotopes have the same number of protons, they also have the same number of electrons in a neutral atom, giving them identical chemical behaviour.
**Explanation**: The number and arrangement of electrons govern how an atom bonds and reacts chemically. Neutrons affect only the mass, not chemical reactivity. A common error is stating that isotopes "look the same" rather than focusing on electron configuration.
**Tags**: isotope, chemical-properties, electrons, atomic-structure

### Question Q-6.1-06
**Difficulty**: Challenge
**Question**: State the relative charge and approximate relative mass of a proton, neutron, and electron.
**Answer**: Proton: relative charge +1, relative mass 1. Neutron: relative charge 0, relative mass 1. Electron: relative charge -1, relative mass approximately 1/1836 (negligible compared to nucleons).
**Explanation**: Protons and neutrons each have a relative mass of 1 and make up nearly all the atom's mass. Electrons carry negligible mass but equal and opposite charge to protons. A common error is stating that electrons have zero mass rather than negligible mass.
**Tags**: subatomic-particles, charge, mass, proton, neutron, electron

### Question Q-6.1-07
**Difficulty**: Challenge
**Question**: Write the nuclide notation for an atom with 17 protons and 18 neutrons. Identify the element.
**Answer**: ³⁵₁₇Cl (chlorine-35). The atomic number is 17 (protons), mass number is 17 + 18 = 35, and element 17 is chlorine.
**Explanation**: Nuclide notation follows the format ᴬzX where A = mass number (protons + neutrons), Z = atomic number (protons), and X = element symbol. A common error is swapping the mass and atomic numbers or misidentifying the element from the periodic table.
**Tags**: nuclide-notation, chlorine, atomic-number, mass-number

### Question Q-6.1-08
**Difficulty**: Standard
**Question**: An atom contains 15 protons and 16 neutrons. State the number of electrons if the atom is neutral, and write its nuclide notation.
**Answer**: Electrons = 15. Nuclide notation: ³¹₁₅P (phosphorus-31).
**Explanation**: In a neutral atom, the number of electrons equals the number of protons (15). The mass number is 15 + 16 = 31, and atomic number 15 corresponds to phosphorus. A common error is assuming the atom is charged when not stated, or miscounting the mass number.
**Tags**: neutral-atom, nuclide-notation, phosphorus, electrons

### Question Q-6.1-09
**Difficulty**: Challenge
**Question**: Two atoms are represented as ²³⁵₉₂U and ²³⁸₉₂U. Explain why these are isotopes of the same element, and calculate the difference in the number of neutrons between them.
**Answer**: Both atoms have 92 protons (same atomic number), so they are both uranium — this makes them isotopes. For ²³⁵₉₂U: neutrons = 235 - 92 = 143. For ²³⁸₉₂U: neutrons = 238 - 92 = 146. Difference = 146 - 143 = 3 neutrons.
**Explanation**: Isotopes share the same atomic number but differ in mass number due to different neutron counts. Both are uranium because Z = 92. A common error is calculating the neutron difference as 238 - 235 = 3 without showing the full working, which risks errors when the atomic numbers differ.
**Tags**: isotope, uranium, neutrons, nuclide-notation, comparison
