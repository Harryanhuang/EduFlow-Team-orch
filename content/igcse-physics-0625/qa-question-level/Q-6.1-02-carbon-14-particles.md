# Question Q-6.1-02

**Difficulty**: Foundation

**Question**: How many protons, neutrons, and electrons are in a neutral atom of carbon-14 (¹⁴₆C)?

**Answer**: Protons = 6, neutrons = 8, electrons = 6.

**Explanation**: The atomic number (subscript 6) gives the number of protons = 6. In a neutral atom, electrons = protons = 6. Neutrons = mass number - atomic number = 14 - 6 = 8. A common error is using 14 as the atomic number instead of the mass number.
