# Question Q-4.5-02

**Difficulty**: Standard

**Question**: Calculate the critical angle for glass with refractive index 1.5.

**Answer**: 41.8°

**Explanation**: sin c = 1/n = 1/1.5 = 0.6667. c = sin⁻¹(0.6667) = 41.8°. A common error is using n = sin c instead of sin c = 1/n.
