# Question Q-5.10-07

**Difficulty**: Challenge

**Question**: Use Fleming's left-hand rule to explain what happens if the current in a motor coil is reversed.

**Answer**: Reversing the current reverses the force on each side of the coil, so the direction of rotation reverses.

**Explanation**: In Fleming's rule, changing the second finger direction changes the thumb direction.
