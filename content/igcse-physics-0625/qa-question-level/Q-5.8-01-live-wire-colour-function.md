# Question Q-5.8-01

**Difficulty**: Foundation

**Question**: State the colour and function of the live wire in a mains plug.

**Answer**: The live wire is brown. It carries the alternating potential difference from the supply to the appliance.

**Explanation**: The live wire can be dangerous even when an appliance is switched off if the switch is wired incorrectly.
