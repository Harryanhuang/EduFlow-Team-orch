# Question Q-5.4-06

**Difficulty**: Standard

**Question**: The I-V graph for a filament lamp is a curve that gets less steep as voltage increases. Explain why.

**Answer**: As voltage increases, current heats the filament. Higher temperature increases resistance, so current rises less for each extra volt.

**Explanation**: A filament lamp is non-ohmic because its temperature changes.
