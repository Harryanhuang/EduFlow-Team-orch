# Question Q-3.2-02

**Difficulty**: Foundation

**Question**: A thermometer is calibrated using the ice point and steam point. The distance between these marks is 20 cm. If the mercury rises 5 cm above the ice point, what is the temperature?

**Answer**: 25°C

**Explanation**: Temperature = (5/20) × 100 = 25°C. The scale is linear between the two fixed points. A common error is not using the proportional method.
