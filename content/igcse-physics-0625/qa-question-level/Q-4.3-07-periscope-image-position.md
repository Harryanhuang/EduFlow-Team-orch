# Question Q-4.3-07

**Difficulty**: Challenge

**Question**: A periscope uses two plane mirrors placed at 45° to the horizontal. If the object is 2 m above the top mirror and the bottom mirror is 1.5 m below the top mirror, how far above the observer's eye does the image appear?

**Answer**: 3.5 m

**Explanation**: In a plane mirror, the image distance equals the object distance. The total vertical distance from object to eye = 2 m + 1.5 m = 3.5 m. Each mirror produces a virtual image at the same distance behind it. A common error is adding distances incorrectly.
