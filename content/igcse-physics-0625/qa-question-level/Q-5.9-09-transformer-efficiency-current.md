# Question Q-5.9-09

**Difficulty**: Challenge

**Question**: A transformer is 90 percent efficient. It supplies 45 W to the secondary circuit from a 12 V primary supply. Calculate the input power and primary current.

**Answer**: Efficiency = output/input, so input power = 45/0.90 = 50 W. Primary current = P/V = 50/12 = 4.17 A.

**Explanation**: Use efficiency before calculating input current.
