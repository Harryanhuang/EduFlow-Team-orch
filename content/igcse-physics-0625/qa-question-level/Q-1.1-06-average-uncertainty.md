# Question Q-1.1-06

**Difficulty**: Standard

**Question**: A student records five measurements of a wire's diameter using a micrometer: 0.52 mm, 0.54 mm, 0.51 mm, 0.53 mm, 0.55 mm. Calculate the average diameter and estimate the uncertainty.

**Answer**: 0.53 mm, uncertainty ± 0.02 mm

**Explanation**: Average = (0.52 + 0.54 + 0.51 + 0.53 + 0.55) / 5 = 2.65 / 5 = 0.53 mm. Uncertainty = (max - min) / 2 = (0.55 - 0.51) / 2 = ± 0.02 mm. A common error is using only the range without dividing by 2.
