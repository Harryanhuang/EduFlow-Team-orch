# Question Q-6.1-07

**Difficulty**: Challenge

**Question**: Write the nuclide notation for an atom with 17 protons and 18 neutrons. Identify the element.

**Answer**: ³⁵₁₇Cl (chlorine-35). The atomic number is 17 (protons), mass number is 17 + 18 = 35, and element 17 is chlorine.

**Explanation**: Nuclide notation follows the format ᴬzX where A = mass number (protons + neutrons), Z = atomic number (protons), and X = element symbol. A common error is swapping the mass and atomic numbers or misidentifying the element from the periodic table.
