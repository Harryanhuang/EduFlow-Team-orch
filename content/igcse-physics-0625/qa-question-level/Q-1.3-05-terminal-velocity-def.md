# Question Q-1.3-05

**Difficulty**: Standard

**Question**: What is meant by terminal velocity?

**Answer**: Terminal velocity is the constant maximum speed reached by a falling object when the upward air resistance (drag) equals the downward weight, so the resultant force and acceleration are both zero.

**Explanation**: The key point is the force balance. A common error is saying terminal velocity is when "the object stops falling" — it keeps falling, just at constant speed.
