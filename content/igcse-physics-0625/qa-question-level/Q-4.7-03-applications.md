# Question Q-4.7-03

**Difficulty**: Standard

**Question**: Give one use each for: (a) infrared, (b) ultraviolet, (c) X-rays.

**Answer**: (a) Infrared: remote controls, thermal imaging, heaters. (b) Ultraviolet: sterilisation, fluorescent lamps, security marking. (c) X-rays: medical imaging, security scanning.

**Explanation**: Each use must match the correct region. A common error is swapping infrared and ultraviolet uses.
