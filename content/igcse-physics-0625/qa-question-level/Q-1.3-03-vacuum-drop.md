# Question Q-1.3-03

**Difficulty**: Standard

**Question**: Two objects, one heavy and one light, are dropped from the same height in a vacuum. Which hits the ground first? Explain.

**Answer**: Both hit the ground at the same time. In a vacuum, there is no air resistance, so both accelerate at g = 10 m/s² regardless of mass.

**Explanation**: This is Galileo's principle. A common error is saying the heavier object falls faster, which is only true in air due to different terminal velocities.
