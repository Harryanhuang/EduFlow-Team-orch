# Question Q-1.4-03

**Difficulty**: Standard

**Question**: Describe how you would find the centre of gravity of an irregularly shaped thin card.

**Answer**: Suspend the card from a point near one edge using a pin. Hang a plumb line from the same point and draw the line on the card. Repeat from a different suspension point. The centre of gravity is where the two lines intersect.

**Explanation**: The plumb line method works because the centre of gravity always lies vertically below the suspension point. A common error is only using one suspension point.
