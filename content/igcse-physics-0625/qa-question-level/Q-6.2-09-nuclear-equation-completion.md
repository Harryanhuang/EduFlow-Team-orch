# Question Q-6.2-09

**Difficulty**: Challenge

**Question**: Complete the following nuclear equation: ²²⁶₈₈Ra → ? + ⁴₂α. Identify the daughter element and determine how many protons and neutrons it contains.

**Answer**: ²²⁶₈₈Ra → ²²²₈₆Rn + ⁴₂α. The daughter element is radon-222 (Rn). It contains 86 protons and 222 - 86 = 136 neutrons.

**Explanation**: Conservation of mass number: 226 = A + 4, so A = 222. Conservation of atomic number: 88 = Z + 2, so Z = 86. Element 86 is radon. A common error is miscalculating the neutron number by subtracting incorrectly.
