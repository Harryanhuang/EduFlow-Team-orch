# Question Q-5.3-09

**Difficulty**: Challenge

**Question**: Design a circuit with two lamps that can be switched on and off independently from the same battery. Explain where the switches should be placed.

**Answer**: Put the lamps in parallel, with one switch in series with each lamp branch. Each switch controls only its own branch.

**Explanation**: Independent control requires separate branches. A common error is placing one switch before the junction.
