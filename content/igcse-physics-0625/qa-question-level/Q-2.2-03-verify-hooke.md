# Question Q-2.2-03

**Difficulty**: Standard

**Question**: Describe how you could verify that a spring obeys Hooke's law using standard lab equipment.

**Answer**: Hang the spring vertically. Add known masses one at a time and measure the extension each time. Plot force against extension. If the graph is a straight line through the origin, the spring obeys Hooke's law.

**Explanation**: The key is the linear relationship. A common error is only measuring one or two points and claiming Hooke's law.
