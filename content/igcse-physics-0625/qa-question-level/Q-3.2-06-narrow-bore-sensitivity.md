# Question Q-3.2-06

**Difficulty**: Standard

**Question**: A liquid-in-glass thermometer has a very narrow bore. How does this affect its sensitivity?

**Answer**: A narrower bore means a greater change in the length of the liquid column for the same temperature change, making the thermometer more sensitive.

**Explanation**: The same volume expansion produces a longer column change in a narrower tube. A common error is saying it makes it "more accurate" — sensitivity and accuracy are different.
