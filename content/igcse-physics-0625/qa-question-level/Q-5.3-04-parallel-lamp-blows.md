# Question Q-5.3-04

**Difficulty**: Standard

**Question**: In a parallel circuit with two lamps, one lamp blows. What happens to the other lamp? Explain.

**Answer**: The other lamp stays on because each parallel branch has its own path for current.

**Explanation**: Parallel branches can work independently. A common error is treating the circuit as one series path.
