# Question Q-4.6-05

**Difficulty**: Standard

**Question**: How can you measure the focal length of a converging lens experimentally?

**Answer**: Focus a distant object (e.g., a window or tree) onto a screen using the lens. The distance from the lens to the sharp image on the screen is approximately the focal length (since the object is effectively at infinity).

**Explanation**: Distant object → image at focus. A common error is using a nearby object without accounting for the object distance.
