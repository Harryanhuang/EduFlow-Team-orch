# Question Q-1.3-06

**Difficulty**: Standard

**Question**: Sketch a speed-time graph for a skydiver from jumping to reaching terminal velocity. Label key features.

**Answer**: The graph starts steeply (high gradient = large acceleration due to gravity), then curves and gradually flattens as air resistance increases, finally becoming horizontal (zero gradient = zero acceleration) at terminal velocity.

**Explanation**: The changing gradient is key — it shows decreasing acceleration. A common error is drawing a straight line (constant acceleration) for the entire fall.
