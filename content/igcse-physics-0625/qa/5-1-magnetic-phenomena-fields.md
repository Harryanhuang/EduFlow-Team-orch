# QA: 5.1 — Simple magnetic phenomena, magnetic fields

## Topic 名称
Simple magnetic phenomena, magnetic fields（简单磁现象、磁场）

## Topic 定义
学习磁体的基本性质、磁场线的特征以及磁化的方法。

## 关键知识点
- 磁体有两极：北极（N）和南极（S）
- 同名磁极相斥，异名磁极相吸
- 磁场线从 N 极出发回到 S 极
- 磁场线密集处磁场强，稀疏处磁场弱
- 磁场线不相交
- 地球是一个大磁体：地磁 N 极在地理南极附近
- 磁化方法：摩擦法、感应法、电流入法
- 去磁方法：加热、敲打、通反向交流电
- 铁磁性材料（铁、钴、镍）能被磁化
- 磁场方向的定义：小磁针 N 极所指方向

## 前置知识
- 无

## 常见错误
- 认为磁体可以只有单极（切断后每段仍有两极）
- 磁场线方向画反（应从 N 到 S）
- 认为磁场线在磁体内部也从 N 到 S（内部是从 S 到 N）
- 磁化和去磁方法混淆
- 地磁 N 极和地理北极混淆
- 认为铜/铝能被磁铁吸引

## 可出题方向
- 磁极间作用力判断
- 磁场线绘制
- 磁化/去磁方法
- 磁体性质的概念判断
- 指南针在磁场中的指向

## 难度提示
Core：基础。要求学生能描述磁现象和绘制磁场线。
