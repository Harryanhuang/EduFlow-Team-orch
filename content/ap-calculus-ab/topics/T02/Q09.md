Evaluate $\displaystyle \lim_{x \to 0} \frac{\sqrt{x + 1} - 1}{\sqrt{x + 4} - 2}$.

Rationalize by multiplying by both conjugates:

$\dfrac{\sqrt{x + 1} - 1}{\sqrt{x + 4} - 2} \cdot \dfrac{\sqrt{x + 1} + 1}{\sqrt{x + 1} + 1} \cdot \dfrac{\sqrt{x + 4} + 2}{\sqrt{x + 4} + 2}$

$= \dfrac{[(x + 1) - 1](\sqrt{x + 4} + 2)}{[(x + 4) - 4](\sqrt{x + 1} + 1)} = \dfrac{x(\sqrt{x + 4} + 2)}{x(\sqrt{x + 1} + 1)}$

$= \dfrac{\sqrt{x + 4} + 2}{\sqrt{x + 1} + 1}$ for $x \neq 0$.

$\lim\limits_{x \to 0} \dfrac{\sqrt{x + 4} + 2}{\sqrt{x + 1} + 1} = \dfrac{2 + 2}{1 + 1} = \dfrac{4}{2} = 2$.
