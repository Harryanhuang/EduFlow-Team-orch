The table gives values of a differentiable function $g(x)$. Estimate $g'(3)$ using a symmetric difference quotient.

| $x$ | 2.8 | 2.9 | 3.0 | 3.1 | 3.2 |
|-----|-----|-----|-----|-----|-----|
| $g(x)$ | 7.84 | 8.41 | 9.00 | 9.61 | 10.24 |

<question>
The table gives values of a differentiable function $g(x)$. Estimate $g'(3)$ using a symmetric difference quotient.

| $x$ | 2.8 | 2.9 | 3.0 | 3.1 | 3.2 |
|-----|-----|-----|-----|-----|-----|
| $g(x)$ | 7.84 | 8.41 | 9.00 | 9.61 | 10.24 |
</question>
<answer>
Using the symmetric difference quotient with $h = 0.1$ (using $x = 2.9$ and $x = 3.1$):

$$g'(3) \approx \frac{g(3.1) - g(2.9)}{3.1 - 2.9} = \frac{9.61 - 8.41}{0.2} = \frac{1.20}{0.2} = 6.00$$

(Using $h = 0.2$ with $x = 2.8$ and $x = 3.2$ gives the same result: $\frac{10.24 - 7.84}{0.4} = \frac{2.40}{0.4} = 6.00$. The underlying function is $g(x) = x^2$, whose derivative at $x = 3$ is exactly 6.)
</answer>
