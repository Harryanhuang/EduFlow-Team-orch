Determine whether the function $f(x) = |x + 1|$ is differentiable at $x = -1$. Justify using the limit definition.

<question>
Determine whether the function $f(x) = |x + 1|$ is differentiable at $x = -1$. Justify using the limit definition.
</question>
<answer>
Check the one-sided limits of the difference quotient at $x = -1$:

$$\lim_{h \to 0} \frac{f(-1 + h) - f(-1)}{h} = \lim_{h \to 0} \frac{|(-1 + h) + 1| - 0}{h} = \lim_{h \to 0} \frac{|h|}{h}$$

Left-hand limit ($h \to 0^-$): $\dfrac{|h|}{h} = \dfrac{-h}{h} = -1$

Right-hand limit ($h \to 0^+$): $\dfrac{|h|}{h} = \dfrac{h}{h} = 1$

Since the left-hand and right-hand limits differ ($-1 \neq 1$), the limit does not exist. Therefore $f'(-1)$ does not exist, and $f$ is NOT differentiable at $x = -1$.

Note: $f$ IS continuous at $x = -1$ since $\lim_{x \to -1} |x + 1| = 0 = f(-1)$. This shows continuity does not imply differentiability.
</answer>
