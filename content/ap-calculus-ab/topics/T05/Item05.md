Let $f(x) = 3x + 2$.

(a) Write the difference quotient for $f$ at $x = 1$.

(b) Use the limit definition of the derivative to find $f'(1)$.

## Answer
(a) The difference quotient is $\dfrac{f(1 + h) - f(1)}{h} = \dfrac{[3(1 + h) + 2] - [3(1) + 2]}{h} = \dfrac{3h + 5 - 5}{h} = \dfrac{3h}{h}$

(b) $f'(1) = \displaystyle\lim_{h \to 0} \frac{3h}{h} = \lim_{h \to 0} 3 = 3$

## Explanation
Since $f(x) = 3x + 2$ is a linear function, its derivative is constant and equals the slope of the line, which is 3. The difference quotient simplifies to $\frac{3h}{h} = 3$ for all $h \neq 0$, so the limit as $h \to 0$ is simply 3. This confirms that the derivative of a linear function equals its slope.
