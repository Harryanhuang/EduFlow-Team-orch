Two particles A and B move along the $x$-axis. Particle A has velocity $v_A(t) = \sin(2t)$ and particle B has acceleration $a_B(t) = \cos(2t)$, both for $0 \leq t \leq \pi$. At $t = 0$, particle A is at $x = 0$ with $v_A(0) = 0$, and particle B is at $x = 1$ with $v_B(0) = 0$.

(a) Find the position function for particle A.
(b) Find the velocity and position functions for particle B.
(c) Do the particles ever collide? If so, at what time?

## Answer
(a) $s_A(t) = \int \sin(2t)\, dt = -\frac{1}{2}\cos(2t) + C$. Using $s_A(0) = 0$: $0 = -\frac{1}{2}(1) + C \Rightarrow C = \frac{1}{2}$. So $s_A(t) = \frac{1 - \cos(2t)}{2} = \sin^2(t)$.

(b) $v_B(t) = \int \cos(2t)\, dt = \frac{1}{2}\sin(2t) + D$. Using $v_B(0) = 0$: $D = 0$. So $v_B(t) = \frac{1}{2}\sin(2t)$.

$s_B(t) = \int \frac{1}{2}\sin(2t)\, dt = -\frac{1}{4}\cos(2t) + E$. Using $s_B(0) = 1$: $1 = -\frac{1}{4}(1) + E \Rightarrow E = \frac{5}{4}$. So $s_B(t) = \frac{5}{4} - \frac{1}{4}\cos(2t)$.

(c) Collision when $s_A(t) = s_B(t)$: $\frac{1 - \cos(2t)}{2} = \frac{5}{4} - \frac{1}{4}\cos(2t)$. Multiply by 4: $2 - 2\cos(2t) = 5 - \cos(2t) \Rightarrow -\cos(2t) = 3 \Rightarrow \cos(2t) = -3$. Since $|\cos(2t)| \leq 1$, this has no solution. The particles never collide.
