The function $f$ is differentiable on the interval $[-3, 5]$. It is known that $f'(x) > 0$ for $-3 < x < 1$ and $f'(x) < 0$ for $1 < x < 5$. On which interval is $f$ increasing?

## Options
A) $(-3, 1)$ only
B) $(1, 5)$ only
C) $(-3, 5)$
D) $(0, 3)$ only

## Answer
A

## Explanation
A function $f$ is increasing on intervals where $f'(x) > 0$. Since $f'(x) > 0$ for $-3 < x < 1$, the function $f$ is increasing on $(-3, 1)$. The function is decreasing on $(1, 5)$ because $f'(x) < 0$ there.
