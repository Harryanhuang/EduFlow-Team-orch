Consider the function \(f(x) = \dfrac{x^2 - 4}{x - 2}\). Which of the following best describes the value of \(\displaystyle\lim_{x \to 2} f(x)\)?

## Options
A) The limit does not exist because \(f(2)\) is undefined.
B) The limit is 0.
C) The limit is 4.
D) The limit is 2.

## Answer
C

## Explanation
The value of a limit depends on the behavior of \(f(x)\) as \(x\) approaches 2, not on the value at \(x = 2\). For \(x \neq 2\), we can simplify: \(f(x) = \dfrac{(x-2)(x+2)}{x-2} = x + 2\). As \(x \to 2\), the expression \(x + 2 \to 4\). Therefore \(\displaystyle\lim_{x \to 2} f(x) = 4\). This illustrates that the limit can exist even when the function is not defined at the point in question.
