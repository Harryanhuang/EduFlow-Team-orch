**Q:** Show that the midpoint Riemann sum M_n for any continuous f on [a,b] approximates ∫_a^b f(x)dx with error ≤ (b-a)²·M/(2n) where M ≥ |f'(x)| on [a,b].

**A:** By the error bound for midpoint rule: |∫_a^b f(x)dx − M_n| ≤ (b-a)³·K/(24n²) where K ≥ |f''(x)|.
Alternatively, using linear approximation: M_n uses f at interval midpoints, each approximates the average value of f on that subinterval more accurately than endpoints. The midpoint rule's error for continuous f on [a,b] with f'' continuous is bounded by (b-a)³/(24n²)·max|f''(x)|.
For the specific bound with f': error ≤ (b-a)²·M/(2n) follows from Taylor's theorem centered at midpoint, but this is not the standard bound (standard uses f'').
