Evaluate $\displaystyle \lim_{x \to -\infty} \frac{\sqrt{9x^2 + 4x}}{2x + 1}$. Be careful with the sign when simplifying $\sqrt{x^2}$ for negative $x$.

## Answer
For $x < 0$, we have $\sqrt{x^2} = |x| = -x$. So:
$$\sqrt{9x^2 + 4x} = \sqrt{x^2\left(9 + \frac{4}{x}\right)} = |x|\sqrt{9 + \frac{4}{x}} = -x\sqrt{9 + \frac{4}{x}}.$$
Therefore:
$$\lim_{x \to -\infty} \frac{-x\sqrt{9 + 4/x}}{2x + 1} = \lim_{x \to -\infty} \frac{-x\sqrt{9 + 4/x}}{x(2 + 1/x)} = \lim_{x \to -\infty} \frac{-\sqrt{9 + 4/x}}{2 + 1/x} = \frac{-\sqrt{9 + 0}}{2 + 0} = -\frac{3}{2}.$$
