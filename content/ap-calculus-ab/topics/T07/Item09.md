If $f(x) = \dfrac{e^x}{\sin x}$, what is $f'(x)$?

## Options
A) $\dfrac{e^x \sin x - e^x \cos x}{\sin^2 x}$
B) $\dfrac{e^x \sin x + e^x \cos x}{\sin^2 x}$
C) $\dfrac{e^x \cos x - e^x \sin x}{\sin^2 x}$
D) $\dfrac{e^x}{\cos x}$

## Answer
A

## Explanation
Using the quotient rule: $u = e^x$, $v = \sin x$, $u' = e^x$, $v' = \cos x$. So $f'(x) = \frac{e^x \sin x - e^x \cos x}{\sin^2 x}$.
