Find the derivative of $f(x) = e^x \cos x$.

## Answer
$f'(x) = e^x \cos x - e^x \sin x = e^x(\cos x - \sin x)$

## Explanation
Using the product rule: $u = e^x$, $v = \cos x$. Then $u' = e^x$ and $v' = -\sin x$. So $f'(x) = e^x \cos x + e^x(-\sin x) = e^x(\cos x - \sin x)$.
