If $f(x) = 3x \tan x$, what is $f'(x)$?

## Options
A) $3 \tan x + 3x \sec^2 x$
B) $3 \tan x + 3x \sec x$
C) $3 \sec^2 x$
D) $3x \sec^2 x$

## Answer
A

## Explanation
Using the product rule: $u = 3x$, $v = \tan x$. Then $u' = 3$ and $v' = \sec^2 x$. So $f'(x) = 3 \tan x + 3x \sec^2 x$.
