State the Second Derivative Test for local extrema. If f'(c) = 0 and f''(c) < 0, what can you conclude?
If f'(c) = 0 and f''(c) < 0, then f has a local maximum at c. If f'(c) = 0 and f''(c) > 0, then f has a local minimum at c. If f''(c) = 0, the test is inconclusive.
