For g(x) = e^(-x^2), find all local extrema and inflection points. Justify using both the First/Second Derivative Tests.
g'(x) = -2xe^(-x^2), so critical points at x = 0. g''(x) = 2e^(-x^2)(2x^2 - 1). At x = 0, g''(0) = -2 < 0, so by Second Derivative Test, x = 0 is a local maximum. g''(x) = 0 when 2x^2 - 1 = 0, so x = ±1/√2. Testing concavity: g'' < 0 for |x| < 1/√2 (concave down), g'' > 0 for |x| > 1/√2 (concave up). Points of inflection at (-1/√2, e^(-1/2)) and (1/√2, e^(-1/2)).
