If $f(x) = e^x + x$, what is the value of $(f^{-1})'(1)$?

## Options
A) $1$  B) $\frac{1}{2}$  C) $\frac{1}{e}$  D) $e$

## Answer
B) $\frac{1}{2}$

## Explanation
Since $f(0) = e^0 + 0 = 1$, we have $(f^{-1})'(1) = \frac{1}{f'(0)}$. Since $f'(x) = e^x + 1$, we get $f'(0) = e^0 + 1 = 2$. Therefore $(f^{-1})'(1) = \frac{1}{2}$.
