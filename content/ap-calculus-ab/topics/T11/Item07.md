Let $f(x) = x^4 - 4x^3 + 2$. On which interval is the graph of $f$ concave up?

## Options
A) $(-\infty, 0)$ only  B) $(0, 2)$  C) $(-\infty, 0) \cup (2, \infty)$  D) $(2, \infty)$ only

## Answer
C) $(-\infty, 0) \cup (2, \infty)$

## Explanation
$f'(x) = 4x^3 - 12x^2$, so $f''(x) = 12x^2 - 24x = 12x(x - 2)$. The graph is concave up when $f''(x) > 0$, which occurs when $x < 0$ or $x > 2$, i.e., $(-\infty, 0) \cup (2, \infty)$.
