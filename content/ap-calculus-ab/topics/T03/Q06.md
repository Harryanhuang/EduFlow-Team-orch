## Question
Let \( f(x) = \dfrac{\cos x - 1}{x \sin x} \).

(a) Find all discontinuities of \( f \) in the interval \( (-\pi, \pi) \).
(b) For each, determine whether the discontinuity is removable or non-removable.
(c) For each removable discontinuity, give the value that would make \( f \) continuous there.

## Answer
(a) The denominator is zero when \( x \sin x = 0 \), i.e., \( x = 0 \) or \( \sin x = 0 \). In \( (-\pi, \pi) \), \( \sin x = 0 \) only at \( x = 0 \). So the only discontinuity is at \( x = 0 \).

(b) Evaluate the limit as \( x \to 0 \):
\[
\lim_{x \to 0} \frac{\cos x - 1}{x \sin x}.
\]
Using \( \cos x - 1 = -2\sin^2(x/2) \) and \( \sin x = 2\sin(x/2)\cos(x/2) \):
\[
\frac{\cos x - 1}{x \sin x} = \frac{-2\sin^2(x/2)}{x \cdot 2\sin(x/2)\cos(x/2)} = \frac{-\sin(x/2)}{x \cos(x/2)}.
\]
Now write \( x = 2 \cdot (x/2) \):
\[
= \frac{-\sin(x/2)}{2(x/2)\cos(x/2)} = -\frac{1}{2} \cdot \frac{\sin(x/2)}{x/2} \cdot \frac{1}{\cos(x/2)}.
\]
As \( x \to 0 \): \( \frac{\sin(x/2)}{x/2} \to 1 \) and \( \frac{1}{\cos(x/2)} \to 1 \). So the limit is \( -\frac{1}{2} \).

The limit exists, so the discontinuity is **removable**.

(c) Define \( f(0) = -\dfrac{1}{2} \) to make \( f \) continuous at \( x = 0 \).
