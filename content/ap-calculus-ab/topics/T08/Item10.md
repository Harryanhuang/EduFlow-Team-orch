If g(x) = arctan(e^x), what is g'(x)?

## Options
A) e^x/(1 + e^(2x))
B) 1/(1 + e^(2x))
C) e^x/(1 + e^x)
D) 1/(1 + e^x)

## Answer
A

## Explanation
Outer: arctan(u), inner: u = e^x. d/dx[arctan(u)] = 1/(1 + u^2) * du/dx = 1/(1 + e^(2x)) * e^x = e^x/(1 + e^(2x)).
