If f(x) = (3x^2 + 1)^4, what is f'(x)?

## Options
A) 4(3x^2 + 1)^3
B) 12x(3x^2 + 1)^3
C) 24x(3x^2 + 1)^3
D) 4(6x)(3x^2 + 1)^4

## Answer
B

## Explanation
Apply the chain rule: d/dx[g(h(x))] = g'(h(x)) * h'(x). Here the outer function is u^4 and the inner function is u = 3x^2 + 1. The derivative of the outer function (with respect to u) is 4u^3 = 4(3x^2 + 1)^3. The derivative of the inner function is d/dx(3x^2 + 1) = 6x. Multiply: f'(x) = 4(3x^2 + 1)^3 * 6x = 12x(3x^2 + 1)^3.
