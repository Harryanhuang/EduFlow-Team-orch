Find dy/dx at the point (π, 0) on the curve sin(xy) + x = y.

<answer>
dy/dx = -1/(π - 1) = 1/(1 - π)
</answer>

<explanation>
Differentiate implicitly: cos(xy) · (y + x·dy/dx) + 1 = dy/dx. At (π, 0): cos(0)(0 + π·dy/dx) + 1 = dy/dx, so π·dy/dx + 1 = dy/dx. Rearrange: π·dy/dx - dy/dx = -1, so dy/dx(π - 1) = -1. Therefore dy/dx = -1/(π - 1) = 1/(1 - π).
</explanation>
