At which points on the curve x² + y² = 2x is the tangent line horizontal?

## Options
A) (0, 0) and (2, 0)
B) (1, 1) and (1, -1)
C) (1, 0) only
D) (0, 1) and (0, -1)

## Answer
B) (1, 1) and (1, -1)

## Explanation
Differentiate implicitly: 2x + 2y(dy/dx) = 2, so dy/dx = (2 - 2x)/(2y) = (1 - x)/y. Horizontal tangent requires dy/dx = 0, so 1 - x = 0, i.e., x = 1. Substitute into the curve: 1 + y² = 2, giving y² = 1, so y = ±1. The points are (1, 1) and (1, -1).
