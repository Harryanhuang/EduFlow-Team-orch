Find d²y/dx² at the point (3/2, 3/2) on the curve x³ + y³ = 3xy.

<answer>
d²y/dx² = -32/3
</answer>

<explanation>
First find dy/dx: 3x² + 3y²(dy/dx) = 3y + 3x(dy/dx), so dy/dx = (y - x²)/(y² - x). At (3/2, 3/2): dy/dx = -1.

Apply the quotient rule for d²y/dx²:
d²y/dx² = [(y² - x)(dy/dx - 2x) - (y - x²)(2y·dy/dx - 1)] / (y² - x)²

At (3/2, 3/2) with dy/dx = -1:
- Numerator = (3/4)(-4) - (-3/4)(-4) = -3 - 3 = -6
- Denominator = (3/4)² = 9/16

d²y/dx² = -6 ÷ (9/16) = -96/9 = -32/3
</explanation>
