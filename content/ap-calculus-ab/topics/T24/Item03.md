The area between the curves y = x squared and y = x + 2 from x = negative 1 to x = 2 is

## Options (if mcq)
A) 21/2
B) 15/2
C) 9/2
D) 27/2

## Answer
C

## Explanation
Upper minus lower: (x + 2) minus x squared.
Integral from -1 to 2 of [(x + 2) minus x squared] dx = [x squared / 2 + 2x minus x cubed / 3] from -1 to 2.
At x = 2: 4/2 + 4 minus 8/3 = 2 + 4 minus 8/3 = 10/3.
At x = -1: 1/2 minus 2 + 1/3 = negative 7/6.
Difference: 10/3 minus (negative 7/6) = 20/6 + 7/6 = 27/6 = 9/2.
