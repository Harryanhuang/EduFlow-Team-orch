Let $f$ be the function with $f(0) = 2$ and $f'(x) = 3x^2 - 4x + 1$ for all $x$.

(a) Find the average value of $f$ on the interval $[0, 4]$.
(b) Find the $x$-coordinates of all local extrema of $f$ on $(0, 4)$ and classify each as a local maximum or local minimum.
(c) Find the volume of the solid generated by revolving the region bounded by $y = f(x)$, the $x$-axis, $x = 0$, and $x = 4$ about the $x$-axis.

## Answer
(a) $\dfrac{28}{3}$
(b) Local maximum at $x = \dfrac{1}{3}$; local minimum at $x = 1$.
(c) $V = \dfrac{25808\pi}{35} \approx 2316.5$ cubic units.

## Explanation
(a) Recover $f$ by antidifferentiating: $f(x) = \int (3x^2 - 4x + 1)\,dx = x^3 - 2x^2 + x + C$. Since $f(0) = 2$, $C = 2$, so $f(x) = x^3 - 2x^2 + x + 2$.

The average value is $\frac{1}{4}\int_0^4 f(x)\,dx$. Compute $\int_0^4 (x^3 - 2x^2 + x + 2)\,dx = \left[\frac{x^4}{4} - \frac{2x^3}{3} + \frac{x^2}{2} + 2x\right]_0^4 = 64 - \frac{128}{3} + 8 + 8 = \frac{112}{3}$. Thus the average value is $\frac{1}{4}\cdot\frac{112}{3} = \frac{28}{3}$.

(b) Set $f'(x) = 3x^2 - 4x + 1 = 0$. By the quadratic formula, $x = \frac{4 \pm \sqrt{16 - 12}}{6} = \frac{4 \pm 2}{6}$, so $x = \frac{1}{3}$ or $x = 1$. Sign of $f'$: positive on $(0, \frac{1}{3})$, negative on $(\frac{1}{3}, 1)$, positive on $(1, 4)$. Therefore $x = \frac{1}{3}$ is a local maximum and $x = 1$ is a local minimum.

(c) By the disk method, $V = \pi \int_0^4 [f(x)]^2\,dx$. Expanding, $[f(x)]^2 = x^6 - 4x^5 + 6x^4 - 7x^2 + 4x + 4$. Then $\int_0^4 [f(x)]^2\,dx = \left[\frac{x^7}{7} - \frac{2x^6}{3} + \frac{6x^5}{5} - \frac{7x^3}{3} + 2x^2 + 4x\right]_0^4 = \frac{77424}{105} = \frac{25808}{35}$. Therefore $V = \frac{25808\pi}{35} \approx 2316.5$ cubic units.
