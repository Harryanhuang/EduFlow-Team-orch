The volume of the solid generated by revolving the region in the first quadrant bounded by y = x squared, y = 4, and the y-axis about the x-axis is

## Options (if mcq)
A) (64/5)pi
B) (128/5)pi
C) (256/5)pi
D) (512/5)pi

## Answer
B

## Explanation
Using the washer method: outer radius R = 4 (distance from x-axis to y = 4), inner radius r = x squared (distance from x-axis to y = x squared).
Intersection: x squared = 4 implies x = 2. In first quadrant, x from 0 to 2.
V = pi times integral from 0 to 2 of (R squared minus r squared) dx = pi times integral from 0 to 2 of (16 minus x to the fourth) dx = pi [16x minus x to the fifth / 5] from 0 to 2 = pi [32 minus 32/5] = (160 minus 32)/5 pi = (128/5) pi.
