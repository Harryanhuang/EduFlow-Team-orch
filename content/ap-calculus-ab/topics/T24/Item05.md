The arc length of the curve y = square root of (1 minus x squared) from x = 0 to x = 1 is

## Options (if mcq)
A) pi/2
B) pi
C) 2
D) 1

## Answer
A

## Explanation
y = square root of (1 minus x squared). Then dy/dx = negative x / square root of (1 minus x squared).
1 + (dy/dx) squared = 1 + x squared / (1 minus x squared) = (1 minus x squared + x squared) / (1 minus x squared) = 1 / (1 minus x squared).
Square root of this = 1 / square root of (1 minus x squared).
Arc length = integral from 0 to 1 of 1 / square root of (1 minus x squared) dx = [arcsin(x)] from 0 to 1 = pi/2 minus 0 = pi/2.
