Find the area between the curves \(y = x^2\) and \(y = x + 2\).

Intersection: \(x^2 = x + 2\) gives \(x^2 - x - 2 = 0\), so \((x-2)(x+1) = 0\), giving \(x = -1\) and \(x = 2\).

For \(-1 < x < 2\), the line \(y = x + 2\) is above the parabola \(y = x^2\).

\[
\text{Area} = \int_{-1}^{2} \left[(x + 2) - x^2\right]dx = \int_{-1}^{2} (-x^2 + x + 2)dx = \left[-\frac{x^3}{3} + \frac{x^2}{2} + 2x\right]_{-1}^{2}
\]

\[
= \left(-\frac{8}{3} + 2 + 4\right) - \left(\frac{1}{3} + \frac{1}{2} - 2\right) = \left(-\frac{8}{3} + 6\right) - \left(\frac{5}{6} - 2\right) = \frac{10}{3} - \left(-\frac{7}{6}\right) = \frac{10}{3} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2}
\]
