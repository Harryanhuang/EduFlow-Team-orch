Evaluate \(\int_0^{\pi} \sin^2(x)\,dx\) using the power-reduction identity.

Using \(\sin^2(x) = \frac{1 - \cos(2x)}{2}\):

\[
\int_0^{\pi} \sin^2(x)\,dx = \int_0^{\pi} \frac{1 - \cos(2x)}{2}\,dx = \frac{1}{2}\left[x - \frac{\sin(2x)}{2}\right]_0^{\pi}
\]

\[
= \frac{1}{2}\left(\pi - \frac{\sin(2\pi)}{2} - 0\right) = \frac{1}{2}\left(\pi - 0\right) = \frac{\pi}{2}
\]
