The graph of f'(x) is shown below. At which x-value(s) does f(x) have a horizontal tangent?

f'(x) = 3x^2 - 12

## Options
A) x = 2 only
B) x = -2 only
C) x = -2 and x = 2
D) x = 0

## Answer
C) x = -2 and x = 2

## Explanation
f has a horizontal tangent where f'(x) = 0: 3x^2 - 12 = 0, so x^2 = 4, giving x = 2 or x = -2.
