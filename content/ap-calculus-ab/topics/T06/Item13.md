Find the equation of the tangent line to the curve y = sin(x) at x = pi/4.

## Answer
y = (sqrt(2)/2)x + (sqrt(2)/2)(1 - pi/4)

## Explanation
The point: when x = pi/4, y = sin(pi/4) = sqrt(2)/2. So the point is (pi/4, sqrt(2)/2).
The slope: dy/dx = cos(x), so at x = pi/4 the slope is cos(pi/4) = sqrt(2)/2.
Tangent line: y - sqrt(2)/2 = (sqrt(2)/2)(x - pi/4).
Simplified: y = (sqrt(2)/2)x + sqrt(2)/2 - (sqrt(2)/2)(pi/4) = (sqrt(2)/2)x + (sqrt(2)/2)(1 - pi/4).
