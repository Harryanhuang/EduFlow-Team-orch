What is the particular solution of \(\frac{dy}{dx} = 4x\) with the initial condition \(y(0) = 3\)?

A) \(y = 2x^{2} + 3\)
B) \(y = 4x^{2} + 3\)
C) \(y = 2x^{2}\)
D) \(y = 4x + 3\)

## Answer
A

## Explanation
Integrating: \(y = \int 4x\,dx = 2x^{2} + C\).
Apply the initial condition \(y(0) = 3\):
\[3 = 2(0)^{2} + C \implies C = 3\]
Thus the particular solution is \(y = 2x^{2} + 3\).
