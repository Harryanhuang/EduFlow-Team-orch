Find the inflection point of f(x)=x^3-3x^2+2.

## Options
A) (1,0)
B) (1,2)
C) (0,2)
D) (0,0)

## Answer
A

## Explanation
f'(x)=3x^2-6x, f''(x)=6x-6. Setting f''(x)=0 gives x=1. For x<1, f''(0)=-6<0 (concave down). For x>1, f''(2)=6>0 (concave up). So concavity changes at x=1, making it an inflection point. f(1)=1-3+2=0, so the inflection point is (1,0). Option A is correct. Option B has the wrong y-coordinate. Option C has the wrong x-coordinate. Option D is not an inflection point because f''(x)=6≠0 there.
