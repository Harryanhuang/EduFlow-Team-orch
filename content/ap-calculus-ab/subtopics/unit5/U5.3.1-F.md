A function f is concave up on an interval if which condition holds?

## Options
A) f''(x)<0 on the interval
B) f''(x)>0 on the interval
C) f'(x)>0 on the interval
D) f'(x)<0 on the interval

## Answer
B

## Explanation
A function is concave up on an interval when f''(x)>0 on that interval. Option B states this correctly. Option A is wrong because f''(x)>0 means concave up, not concave down. Option C is wrong because f'(x)>0 means increasing, not concave up. Option D is wrong because f'(x)<0 means decreasing, not concave up.
