Consider the function f(x) = { 2x for 0 ≤ x < 1; 1 for x = 1; x-1 for 1 < x ≤ 2 } on the interval [0,2].
I. f attains an absolute maximum on [0,2].
II. f attains an absolute minimum on [0,2].
III. f attains both an absolute maximum and an absolute minimum on [0,2].
Which statements are guaranteed by the Extreme Value Theorem?

## Options
A) I only
B) II only
C) I and II only
D) None of I, II, or III are guaranteed

## Answer
D

## Explanation
The function is continuous on [0,2] except at x=1 where it has a jump discontinuity (left limit 2, right limit 0, value 1). Since f is not continuous on the closed interval [0,2], the Extreme Value Theorem does not guarantee absolute extrema. Indeed, f approaches 2 from the left at x=1 but never attains a value greater than 2 on [0,1), and on (1,2] the values are in (0,1]. The function has no absolute maximum (supremum 2 but not attained) and no absolute minimum (infimum 0 but not attained). Thus none of the statements I, II, or III are guaranteed, making D the correct answer.
