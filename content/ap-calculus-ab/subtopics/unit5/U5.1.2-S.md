Find the absolute maximum value of f(x)=x^3-3x^2+1 on the interval [-1,3].

## Options
A) 1
B) -3
C) 3
D) 0

## Answer
A

## Explanation
The function is continuous on [-1,3] (polynomial), so EVT applies. Critical points: f'(x)=3x^2-6x=3x(x-2), so x=0 and x=2 are in (-1,3). Evaluate: f(-1)=-1-3+1=-3, f(0)=1, f(2)=8-12+1=-3, f(3)=27-27+1=1. The absolute maximum is 1, attained at both x=0 and x=3. Option A is correct. Option B gives the minimum value. Option C gives the maximum x-coordinate but not the value. Option D is incorrect because the maximum value is 1, not 3.
