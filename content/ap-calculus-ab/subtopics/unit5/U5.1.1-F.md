Which statement correctly gives the hypotheses and conclusion of the Mean Value Theorem for a function f on a closed interval [a,b]?

## Options
A) f is continuous on [a,b], differentiable on (a,b), and c must be an endpoint.
B) f is continuous on [a,b], differentiable on (a,b), and there exists c in (a,b) with f'(c)=(f(b)-f(a))/(b-a).
C) f is continuous on [a,b], differentiable on (a,b), and there is exactly one c in (a,b) with f'(c)=0.
D) f is continuous on [a,b], differentiable on (a,b), and f(a)=f(b).

## Answer
B

## Explanation
The Mean Value Theorem requires f to be continuous on [a,b] and differentiable on (a,b). The conclusion guarantees at least one c in (a,b) where f'(c)=(f(b)-f(a))/(b-a). This is exactly option B. Option A is false because c must be in the open interval, not endpoints. Option C is false because the theorem only guarantees existence, not uniqueness. Option D is false because the theorem does not require f(a)=f(b); that is Rolle's Theorem.
