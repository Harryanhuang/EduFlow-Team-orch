Find the local maximum value of f(x)=x^3-6x^2+9x+2 using the Second Derivative Test.

## Options
A) 2
B) 6
C) 1
D) 10

## Answer
B

## Explanation
f'(x)=3x^2-12x+9=3(x^2-4x+3)=3(x-1)(x-3). Critical points at x=1 and x=3. f''(x)=6x-12. f''(1)=6-12=-6<0, so x=1 is a local maximum. f''(3)=18-12=6>0, so x=3 is a local minimum. The local maximum value is f(1)=1-6+9+2=6. Option B is correct. Option A gives the local minimum value. Option C gives the x-coordinate. Option D is incorrect.
