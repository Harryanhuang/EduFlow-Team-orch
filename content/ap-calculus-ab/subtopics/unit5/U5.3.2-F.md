Which statement correctly describes the Second Derivative Test?

## Options
A) If f'(c)=0 and f''(c)<0, then f has a local maximum at x=c.
B) If f'(c)=0 and f''(c)<0, then f has a local minimum at x=c.
C) If f'(c)=0 and f''(c)>0, then f has a local maximum at x=c.
D) If f'(c)=0 and f''(c)=0, then f has a local maximum at x=c.

## Answer
A

## Explanation
The Second Derivative Test states that if f'(c)=0 and f''(c)<0, then f has a local maximum at c. Option A states this correctly. Option B is wrong because f''(c)<0 gives a maximum, not a minimum. Option C is wrong because f''(c)>0 gives a minimum, not a maximum. Option D is wrong because f''(c)=0 means the test is inconclusive.
