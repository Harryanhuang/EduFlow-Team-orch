For dy/dx = 2x with y(0) = 3, what is the particular solution?

## Options
A) y = x² + 3
B) y = 2x + 3
C) y = x²
D) y = 2x² + 3

## Answer
A

## Explanation
Separating variables (or simply integrating): y = x² + C. Applying y(0) = 3: 3 = 0 + C, so C = 3. The particular solution is y = x² + 3. Option A is correct.
