---For the differential equation dy/dx = x - y, which statement about the slope field is true?

## Options
A. At (1,1) the slope is 1
B. At (1,2) the slope is 0
C. At (2,2) the slope is 0
D. At (1,1) the slope is -1

## Answer
C

## Explanation
At (1,1): dy/dx = 1 - 1 = 0 (horizontal). At (1,2): dy/dx = 1 - 2 = -1 (downward). At (2,2): dy/dx = 2 - 2 = 0 (horizontal). So the description in C matches.
