A substance decays according to dN/dt = −0.03N. What is the half-life?

## Options
A) ln(2)/0.03 ≈ 23.1 time units
B) 0.03/ln(2) ≈ 0.043 time units
C) 1/0.03 ≈ 33.3 time units
D) −ln(2)/0.03 ≈ −23.1 time units

## Answer
A

## Explanation
dN/dt = −0.03N has solution N = N₀e^(−0.03t). For half-life, set N = N₀/2: 1/2 = e^(−0.03t), so ln(1/2) = −0.03t, giving t = −ln(2)/(−0.03) = ln(2)/0.03 ≈ 23.1 time units. The negative k confirms decay, and the half-life formula t₁/₂ = ln(2)/|k| gives answer A.
