---Solve dy/dx = y^2 with y(0) = 1. What is y(0.5)?

## Options
A. 0.5
B. 1
C. 1.5
D. 2

## Answer
D

## Explanation
Separate: dy/(y^2) = dx. Integrate: -1/y = x + C, so y = -1/(x+C). Using y(0)=1: 1 = -1/C, so C = -1. Thus y = -1/(x-1) = 1/(1-x). At x=0.5: y = 1/(1-0.5) = 2.
