The slope field for which differential equation has all zero-slope segments along the y-axis (where x = 0)?

## Options
A) dy/dx = x + y
B) dy/dx = x − y
C) dy/dx = y − x
D) dy/dx = x²

## Answer
D

## Explanation
Along x = 0: Option A gives dy/dx = y (varies with y, not zero). Option B gives dy/dx = −y (varies with y). Option C gives dy/dx = y (varies with y). Option D gives dy/dx = 0 for all y when x = 0, producing horizontal segments everywhere on the y-axis. Only D has slope = 0 independent of y along x = 0. Answer is D.
