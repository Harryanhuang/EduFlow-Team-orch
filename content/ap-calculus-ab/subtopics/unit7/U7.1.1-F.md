---Consider the slope field for dy/dx = x. Which of the following is true?

## Options
A. All points with the same x-coordinate have the same slope segment
B. All points with the same y-coordinate have the same slope segment
C. The slope is always positive
D. The slope is always zero along y = x

## Answer
A

## Explanation
If dy/dx depends only on x (here dy/dx = x), all points with the same x-coordinate have the same slope. That means vertical columns of the slope field have identical segments.
