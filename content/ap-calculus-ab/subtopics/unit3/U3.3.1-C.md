设 $f(x) = x^3 + 2x + 1$，$g = f^{-1}$。求反函数 $g$ 在 $y = 1$ 处的切线方程。

## Options
A) $y = \frac{1}{2}(x - 1)$
B) $y = x - 1$
C) $y = 2(x - 1)$
D) $y = \frac{1}{4}(x - 1) + 1$

## Answer
A

## Explanation
反函数 $g$ 在 $y = 1$ 处的切线，即 $g$ 在点 $(1, g(1))$ 处的切线。

第一步：求 $g(1)$。由 $f(x) = 1$：
$$x^3 + 2x + 1 = 1 \Rightarrow x^3 + 2x = 0 \Rightarrow x(x^2 + 2) = 0 \Rightarrow x = 0$$
所以 $g(1) = 0$，切点为 $(1, 0)$。

第二步：求 $g'(1)$。$f'(x) = 3x^2 + 2$，故 $f'(0) = 2$，于是
$$g'(1) = \frac{1}{f'(g(1))} = \frac{1}{f'(0)} = \frac{1}{2}.$$

第三步：切线方程为
$$y - 0 = \frac{1}{2}(x - 1) \Rightarrow y = \frac{1}{2}(x - 1).$$

选项 B 忘记取倒数；选项 C 直接用 $f'(0)=2$ 作为斜率；选项 D 混淆了切点坐标。
