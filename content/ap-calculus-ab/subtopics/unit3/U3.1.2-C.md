若 $y = \ln\left(\frac{e^{2x}}{x^2 + 1}\right)$，则 $\frac{dy}{dx} =$

## Options
A) $\frac{2e^{2x}}{e^{2x}} - \frac{2x}{x^2 + 1}$
B) $\frac{2}{x^2 + 1} - \frac{2x}{e^{2x}}$
C) $\frac{2(x^2 + 1) - 2xe^{2x}}{e^{2x}(x^2 + 1)}$
D) $\frac{2(x^2 + 1) - 2x}{x^2 + 1}$

## Answer
D

## Explanation
方法一：先化简再求导。
$$y = \ln(e^{2x}) - \ln(x^2 + 1) = 2x - \ln(x^2 + 1)$$

求导：
$$\frac{dy}{dx} = 2 - \frac{2x}{x^2 + 1} = \frac{2(x^2 + 1) - 2x}{x^2 + 1} = \frac{2x^2 + 2 - 2x}{x^2 + 1}$$

方法二：直接用链式法则。
令 $u = \frac{e^{2x}}{x^2 + 1}$，则 $y = \ln u$，$\frac{dy}{dx} = \frac{1}{u} \cdot u'$。

$u' = \frac{2e^{2x}(x^2 + 1) - e^{2x} \cdot 2x}{(x^2 + 1)^2} = \frac{e^{2x}[2(x^2 + 1) - 2x]}{(x^2 + 1)^2}$

$\frac{dy}{dx} = \frac{x^2 + 1}{e^{2x}} \cdot \frac{e^{2x}[2(x^2 + 1) - 2x]}{(x^2 + 1)^2} = \frac{2(x^2 + 1) - 2x}{x^2 + 1}$

选项 A 是分子分别求导的错误形式；选项 B 混淆了求导对象和分母；选项 C 是未化简的中间步骤且分子结构错误。
