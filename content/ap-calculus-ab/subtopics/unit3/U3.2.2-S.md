曲线 $x^2 + xy + y^2 = 7$ 在点 $(1, 2)$ 处的切线斜率为

## Options
A) $-\frac{4}{5}$
B) $-\frac{1}{2}$
C) $-1$
D) $0$

## Answer
A

## Explanation
对方程两边同时对 $x$ 求导：
$$2x + y + x\frac{dy}{dx} + 2y\frac{dy}{dx} = 0$$

解出 $\frac{dy}{dx}$：
$$(x + 2y)\frac{dy}{dx} = -(2x + y)$$
$$\frac{dy}{dx} = -\frac{2x + y}{x + 2y}$$

在点 $(1, 2)$ 处：
$$\frac{dy}{dx} = -\frac{2(1) + 2}{1 + 2(2)} = -\frac{4}{5}$$

选项 B 错误地计算为 $-\frac{2+2}{1+2} = -\frac{4}{3}$ 后约简错误；选项 C 是未正确代入分子；选项 D 是误以为 $x + 2y = 0$。
