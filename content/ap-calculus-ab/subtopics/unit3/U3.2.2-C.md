曲线 $x^2 + y^2 = 25$ 上，在点 $(3, 4)$ 处的 $\frac{d^2y}{dx^2} =$

## Options
A) $-\frac{25}{64}$
B) $-\frac{3}{4}$
C) $-\frac{1}{4}$
D) $\frac{25}{64}$

## Answer
A

## Explanation
第一步：求一阶导。
$$2x + 2y\frac{dy}{dx} = 0 \Rightarrow \frac{dy}{dx} = -\frac{x}{y}$$

第二步：对一阶导再次求导（用商法则）：
$$\frac{d^2y}{dx^2} = -\frac{y \cdot 1 - x \cdot \frac{dy}{dx}}{y^2} = -\frac{y - x(-\frac{x}{y})}{y^2} = -\frac{y + \frac{x^2}{y}}{y^2}$$

化简分子：
$$y + \frac{x^2}{y} = \frac{y^2 + x^2}{y} = \frac{25}{y}$$（利用 $x^2 + y^2 = 25$）

因此：
$$\frac{d^2y}{dx^2} = -\frac{\frac{25}{y}}{y^2} = -\frac{25}{y^3}$$

在点 $(3, 4)$ 处：
$$\frac{d^2y}{dx^2} = -\frac{25}{4^3} = -\frac{25}{64}$$

选项 B 是一阶导在 $(3, 4)$ 的值；选项 C 是计算过程中遗漏了 $25$ 的错误结果；选项 D 符号错误。
