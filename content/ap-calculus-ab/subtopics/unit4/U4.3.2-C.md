函数 $f(x) = x^3 - 6x^2 + 9x + 2$ 在闭区间 $[0, 4]$ 上的绝对最大值与绝对最小值之差是多少？

## Options
A) $4$
B) $6$
C) $8$
D) $10$

## Answer
A

## Explanation
求导：$f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3) = 3(x-1)(x-3)$。

临界点：$x = 1$ 和 $x = 3$（均在区间 $[0, 4]$ 内）。

比较函数值：
- $f(0) = 0 - 0 + 0 + 2 = 2$
- $f(1) = 1 - 6 + 9 + 2 = 6$
- $f(3) = 27 - 54 + 27 + 2 = 2$
- $f(4) = 64 - 96 + 36 + 2 = 6$

绝对最大值为 $6$（在 $x = 1$ 和 $x = 4$ 处取得）。
绝对最小值为 $2$（在 $x = 0$ 和 $x = 3$ 处取得）。

差值：$6 - 2 = 4$。

因此答案为 A。
