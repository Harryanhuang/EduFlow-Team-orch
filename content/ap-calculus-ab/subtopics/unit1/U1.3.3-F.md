Which of the following functions is continuous for all real numbers?

## Options
A) $f(x) = \frac{1}{x}$
B) $f(x) = x^3 - 2x + 1$
C) $f(x) = \frac{x^2 - 1}{x - 1}$
D) $f(x) = \sqrt{x}$

## Answer
B

## Explanation
A) $f(x) = \frac{1}{x}$ is undefined at $x = 0$, so it is not continuous there.
B) $f(x) = x^3 - 2x + 1$ is a polynomial, and all polynomials are continuous for all real numbers.
C) $f(x) = \frac{x^2 - 1}{x - 1}$ is undefined at $x = 1$ (before simplifying), so it is not continuous there.
D) $f(x) = \sqrt{x}$ is only defined for $x \geq 0$, so it is not continuous for all real numbers (specifically, it is not defined for $x < 0$).
