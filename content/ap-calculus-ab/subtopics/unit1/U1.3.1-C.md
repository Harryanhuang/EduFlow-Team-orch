Let $f$ be the function defined by
$$f(x) = \begin{cases} \frac{x^2 - 9}{x - 3} & \text{if } x < 3 \\ k & \text{if } x = 3 \\ x + 3 & \text{if } x > 3 \end{cases}$$
For which value of $k$ is $f$ continuous at $x = 3$?

## Options
A) 3
B) 6
C) 9
D) No such value exists.

## Answer
B

## Explanation
For continuity at $x = 3$, we need $\lim_{x \to 3} f(x) = f(3) = k$. Left-hand limit: for $x < 3$, $\frac{x^2 - 9}{x - 3} = \frac{(x-3)(x+3)}{x-3} = x + 3$, so $\lim_{x \to 3^-} f(x) = 6$. Right-hand limit: $\lim_{x \to 3^+} (x + 3) = 6$. Since both one-sided limits equal 6, $\lim_{x \to 3} f(x) = 6$. Therefore, $k = 6$ makes $f$ continuous at $x = 3$.
