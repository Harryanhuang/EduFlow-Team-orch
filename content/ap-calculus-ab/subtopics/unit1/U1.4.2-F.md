Which of the following statements about the Intermediate Value Theorem (IVT) is true?

## Options
A) IVT guarantees exactly one solution to $f(x) = k$ in $(a, b)$.
B) IVT applies only if $f$ is differentiable on $(a, b)$.
C) IVT requires $f$ to be continuous on $[a, b]$.
D) IVT can be used to find the exact value of $c$ such that $f(c) = k$.

## Answer
C

## Explanation
The Intermediate Value Theorem states that if $f$ is continuous on the closed interval $[a, b]$ and $k$ is any number between $f(a)$ and $f(b)$, then there exists at least one $c$ in $(a, b)$ such that $f(c) = k$.

A is false: IVT guarantees at least one solution, not exactly one.
B is false: IVT only requires continuity, not differentiability.
C is true: continuity on $[a, b]$ is a necessary condition for IVT.
D is false: IVT is an existence theorem; it does not provide the exact value of $c$.
