The function $f$ is continuous on the closed interval $[1, 5]$. If $f(1) = -2$ and $f(5) = 4$, which of the following statements is guaranteed by the Intermediate Value Theorem?

## Options
A) $f$ has a maximum value of 4 on $[1, 5]$.
B) There exists at least one $c$ in $(1, 5)$ such that $f(c) = 0$.
C) $f$ is increasing on $[1, 5]$.
D) $f(3) = 1$.

## Answer
B

## Explanation
The Intermediate Value Theorem states that if $f$ is continuous on $[a, b]$ and $k$ is any number between $f(a)$ and $f(b)$, then there exists at least one $c$ in $(a, b)$ such that $f(c) = k$. Since $f(1) = -2 < 0 < 4 = f(5)$, the value $k = 0$ is between $f(1)$ and $f(5)$. Therefore, there must exist at least one $c$ in $(1, 5)$ with $f(c) = 0$.

A is about the Extreme Value Theorem, not IVT. C and D are not guaranteed by any theorem.
