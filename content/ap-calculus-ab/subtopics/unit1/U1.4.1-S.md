Let $f$ be the function defined by $f(x) = x^3 - 2x + 1$. On which of the following intervals does the equation $f(x) = 2$ have a solution?

## Options
A) $(-2, -1)$
B) $(-1, 0)$
C) $(0, 1)$
D) $(1, 2)$

## Answer
D

## Explanation
Since $f(x) = x^3 - 2x + 1$ is a polynomial, it is continuous everywhere. We evaluate $f$ at the endpoints of each interval to find where $f(x) = 2$ is bracketed:
- $f(1) = 1 - 2 + 1 = 0 < 2$
- $f(2) = 8 - 4 + 1 = 5 > 2$

Since $f(1) = 0 < 2 < 5 = f(2)$ and $f$ is continuous on $[1, 2]$, by the Intermediate Value Theorem there exists at least one $c$ in $(1, 2)$ such that $f(c) = 2$.

Checking the other intervals:
- $f(-2) = -8 + 4 + 1 = -3$, $f(-1) = -1 + 2 + 1 = 2$. Since $f(-1) = 2$, the equation has a solution at the endpoint, not guaranteed in the interior by IVT.
- $f(-1) = 2$, $f(0) = 1$. Since $2 > 1$, and we want $f(x) = 2$, the value 2 is at the endpoint $x = -1$, not in $(-1, 0)$.
- $f(0) = 1 < 2$, $f(1) = 0 < 2$. Both are less than 2, so IVT does not guarantee a solution.
