Let $f$ be the function defined by
$$f(x) = \begin{cases} x + 3 & \text{if } x < 2 \\ ax + b & \text{if } 2 \leq x < 3 \\ 2x & \text{if } x \geq 3 \end{cases}$$
If $\lim_{x \to 2} f(x)$ exists and $\lim_{x \to 3} f(x) = 6$, what is the value of $a + b$?

## Options
A) 2
B) 4
C) 6
D) 8

## Answer
B

## Explanation
For $\lim_{x \to 2} f(x)$ to exist, the left-hand and right-hand limits must be equal. Left-hand limit: $\lim_{x \to 2^-} (x + 3) = 5$. Right-hand limit: $\lim_{x \to 2^+} (ax + b) = 2a + b$. So $2a + b = 5$.

For $\lim_{x \to 3} f(x) = 6$, the left-hand and right-hand limits must equal 6. Left-hand limit: $\lim_{x \to 3^-} (ax + b) = 3a + b$. Right-hand limit: $\lim_{x \to 3^+} (2x) = 6$. So $3a + b = 6$.

Subtracting the first equation from the second: $(3a + b) - (2a + b) = 6 - 5$, giving $a = 1$. Substituting into $2a + b = 5$: $2(1) + b = 5$, so $b = 3$. Therefore, $a + b = 1 + 3 = 4$.
