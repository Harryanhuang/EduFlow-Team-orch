Let $f$ be the function defined by $f(x) = \frac{x^2 - 5x + 6}{x^2 - 4}$. At which of the following $x$-values does $f$ have a removable discontinuity?

## Options
A) $x = -2$ only
B) $x = 2$ only
C) $x = -2$ and $x = 2$
D) $x = 2$ and $x = 3$

## Answer
B

## Explanation
Factor the numerator and denominator:
$$f(x) = \frac{(x-2)(x-3)}{(x-2)(x+2)}$$
At $x = 2$: both numerator and denominator are zero. After canceling $(x-2)$ for $x \neq 2$, we get $\frac{x-3}{x+2}$, which has limit $-\frac{1}{4}$ at $x = 2$. Since the limit exists but $f(2)$ is undefined, this is a removable discontinuity.
At $x = -2$: the denominator is zero but the numerator is $(-2-2)(-2-3) = (-4)(-5) = 20 \neq 0$. So $\lim_{x \to -2} f(x)$ is infinite, giving a vertical asymptote (infinite discontinuity).
At $x = 3$: after canceling, $f(3) = \frac{3-3}{3+2} = 0$, so the function is continuous (or at least the limit equals the function value if we define the simplified form). The original function has $f(3) = 0$, so $x = 3$ is not a discontinuity.
Therefore, only $x = 2$ is a removable discontinuity.
