The function $f$ has a discontinuity at $x = 2$, where $\lim_{x \to 2} f(x) = 3$ and $f(2) = 5$. What type of discontinuity does $f$ have at $x = 2$?

## Options
A) Jump discontinuity
B) Removable discontinuity
C) Infinite discontinuity
D) The function is continuous at $x = 2$.

## Answer
B

## Explanation
A removable discontinuity occurs when the limit exists but does not equal the function value (or the function is undefined at that point). Here, $\lim_{x \to 2} f(x) = 3$ exists, but $f(2) = 5 \neq 3$, so this is a removable discontinuity. A jump discontinuity would have unequal left and right limits. An infinite discontinuity involves an infinite limit.
