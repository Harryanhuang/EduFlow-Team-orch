Let $f$ be the function defined by
$$f(x) = \begin{cases} x + 3 & \text{if } x < 1 \\ 5 & \text{if } x = 1 \\ 2x + 1 & \text{if } x > 1 \end{cases}$$
What is $\lim_{x \to 1^-} f(x)$?

## Options
A) 3
B) 4
C) 5
D) The limit does not exist.

## Answer
B

## Explanation
To find the left-hand limit as $x$ approaches 1, we use the piece of the function defined for $x < 1$, which is $f(x) = x + 3$. Substituting $x = 1$ gives $\lim_{x \to 1^-} f(x) = 1 + 3 = 4$. The value $f(1) = 5$ is irrelevant for the left-hand limit.
