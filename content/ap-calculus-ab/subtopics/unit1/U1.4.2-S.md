Let $f$ be the function defined by
$$f(x) = \begin{cases} x & \text{if } x < 2 \\ x + 2 & \text{if } x \geq 2 \end{cases}$$
Which of the following statements is true?

## Options
A) $f(1) = 1$ and $f(3) = 5$, so by IVT there exists $c$ in $(1, 3)$ with $f(c) = 2.5$.
B) $f$ is not continuous on $[1, 3]$, so IVT does not apply.
C) $f(1) = 1$ and $f(3) = 5$, so by IVT there exists $c$ in $(1, 3)$ with $f(c) = 3$.
D) $f$ is continuous on $[1, 3]$ and $f(c) = 2.5$ for some $c$ in $(1, 3)$.

## Answer
B

## Explanation
At $x = 2$: left-hand limit is $\lim_{x \to 2^-} x = 2$, right-hand limit is $\lim_{x \to 2^+} (x + 2) = 4$, and $f(2) = 4$. Since the left and right limits are not equal, $f$ is discontinuous at $x = 2$. Therefore, $f$ is not continuous on the entire interval $[1, 3]$, and the Intermediate Value Theorem does not apply on $[1, 3]$.

To verify that IVT would fail if blindly applied: $f(c) = 2.5$ requires $c = 2.5$ (first piece, but $2.5 \notin [1, 2)$) or $c + 2 = 2.5$ giving $c = 0.5$ (second piece, but $0.5 \notin [2, 3]$). So no such $c$ exists in $(1, 3)$, confirming that A and C are false. D is false because $f$ is not continuous on $[1, 3]$.
