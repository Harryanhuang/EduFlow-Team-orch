If $f$ and $g$ are continuous functions for all real numbers, which of the following must also be continuous for all real numbers?

## Options
A) $f(x) + g(x)$
B) $\frac{f(x)}{g(x)}$
C) $f(g(x))$
D) Both A and C

## Answer
D

## Explanation
If $f$ and $g$ are continuous everywhere, then:
- $f(x) + g(x)$ is continuous everywhere (sum of continuous functions is continuous).
- $f(g(x))$ is continuous everywhere (composition of continuous functions is continuous).
- $\frac{f(x)}{g(x)}$ is continuous only where $g(x) \neq 0$. If $g(x) = 0$ at any point, the quotient is not continuous there. So B is not necessarily continuous everywhere.
Therefore, both A and C must be continuous for all real numbers.
