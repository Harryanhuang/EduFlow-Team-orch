Let $f$ be the function defined by
$$f(x) = \begin{cases} \frac{x^2 - 4}{x - 2} & \text{if } x < 2 \\ 5 & \text{if } x = 2 \\ x + 2 & \text{if } x > 2 \end{cases}$$
Which of the following correctly describes the behavior of $f$ at $x = 2$?

## Options
A) $\lim_{x \to 2} f(x)$ does not exist because $f(2) = 5$.
B) $\lim_{x \to 2} f(x)$ does not exist because the left and right limits are not equal.
C) $\lim_{x \to 2} f(x) = 4$, but $f(2) \neq 4$.
D) $\lim_{x \to 2} f(x) = 5$.

## Answer
C

## Explanation
Left-hand limit: for $x < 2$, $\frac{x^2 - 4}{x - 2} = \frac{(x-2)(x+2)}{x-2} = x + 2$, so $\lim_{x \to 2^-} f(x) = 2 + 2 = 4$.
Right-hand limit: $\lim_{x \to 2^+} f(x) = 2 + 2 = 4$.
Since both one-sided limits equal 4, $\lim_{x \to 2} f(x) = 4$. However, $f(2) = 5$, which is not equal to the limit. This is a removable discontinuity. The limit exists and equals 4, but the function value is 5.
