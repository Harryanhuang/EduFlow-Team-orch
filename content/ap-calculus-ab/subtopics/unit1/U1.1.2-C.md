Let $f$ be the function defined by
$$f(x) = \begin{cases} \frac{x^2 - 1}{x + 1} & \text{if } x < -1 \\ kx + 1 & \text{if } x \geq -1 \end{cases}$$
For what value of $k$ does $\lim_{x \to -1} f(x)$ exist?

## Options
A) 1
B) 2
C) 3
D) 4

## Answer
C

## Explanation
For the limit to exist, the left-hand and right-hand limits must be equal. Left-hand limit: for $x < -1$, $\frac{x^2 - 1}{x + 1} = \frac{(x-1)(x+1)}{x+1} = x - 1$ (for $x \neq -1$). So $\lim_{x \to -1^-} f(x) = -1 - 1 = -2$. Right-hand limit: $\lim_{x \to -1^+} f(x) = k(-1) + 1 = -k + 1$. Setting equal: $-2 = -k + 1$, so $k = 3$.
