Let $g(x) = \dfrac{\sec x}{1 + \tan x}$. Find $g'(x)$ and simplify.

$g'(x) = \dfrac{\sec x \tan x(1 + \tan x) - \sec x \cdot \sec^2 x}{(1 + \tan x)^2} = \dfrac{\sec x[\tan x + \tan^2 x - \sec^2 x]}{(1 + \tan x)^2}$.

Since $\sec^2 x - \tan^2 x = 1$: $\tan^2 x - \sec^2 x = -1$.

$g'(x) = \dfrac{\sec x(\tan x - 1)}{(1 + \tan x)^2}$.
