Let $f(x) = \dfrac{\sec x}{x^2 + 1}$.

(a) Find $f'(x)$.
(b) Evaluate $f'(\pi)$.

## Answer
(a) Using the quotient rule: $u = \sec x$, $v = x^2 + 1$, $u' = \sec x \tan x$, $v' = 2x$.

$f'(x) = \dfrac{\sec x \tan x(x^2 + 1) - \sec x \cdot 2x}{(x^2 + 1)^2}$

(b) At $x = \pi$: $\sec \pi = -1$, $\tan \pi = 0$.

$f'(\pi) = \dfrac{(-1)(0)(\pi^2 + 1) - (-1)(2\pi)}{(\pi^2 + 1)^2} = \dfrac{2\pi}{(\pi^2 + 1)^2}$

## Explanation
This problem requires applying the quotient rule with a trig function (sec x) whose derivative is itself nontrivial, then evaluating at a specific point where the trig values simplify.
