If $f(x) = \dfrac{\sin x}{x}$, what is $f'(x)$?

## Options
A) $\dfrac{x \cos x - \sin x}{x^2}$
B) $\dfrac{\cos x}{1}$
C) $\dfrac{\sin x - x \cos x}{x^2}$
D) $\dfrac{x \cos x + \sin x}{x^2}$

## Answer
A

## Explanation
Using the quotient rule: $\frac{d}{dx}\left[\frac{u}{v}\right] = \frac{u'v - uv'}{v^2}$. Here $u = \sin x$, $v = x$, so $u' = \cos x$ and $v' = 1$. Thus $f'(x) = \frac{\cos x \cdot x - \sin x \cdot 1}{x^2} = \frac{x \cos x - \sin x}{x^2}$.
