Let f be the function with f(0) = 0 and f prime(x) = x squared minus 4x + 3 for all x.

(a) Find the average value of f on the interval [0, 4].
(b) Find the x-coordinate of all local extrema of f on the open interval (0, 4).
(c) Find the volume of the solid generated by revolving the region under y = f(x) from x = 0 to x = 4 about the x-axis.

## Answer
(a) First find f(x). f(x) = integral of (x squared minus 4x + 3) dx = x cubed/3 minus 2x squared + 3x + C. Using f(0) = 0, we get C = 0. So f(x) = x cubed/3 minus 2x squared + 3x.
Average value = (1/4) times the integral from 0 to 4 of f(x) dx.
Integral of f(x) = integral of (x cubed/3 minus 2x squared + 3x) dx = x to the fourth/12 minus (2/3)x cubed + (3/2)x squared.
Evaluate from 0 to 4: 4^4/12 minus (2/3)(64) + (3/2)(16) = 256/12 minus 128/3 + 24 = 64/3 minus 128/3 + 24 = negative 64/3 + 24 = negative 64/3 + 72/3 = 8/3.
Average = (1/4)(8/3) = 2/3.

(b) f prime(x) = x squared minus 4x + 3 = (x minus 1)(x minus 3). Set equal to 0: x = 1, x = 3.
Test intervals: (0,1): f prime > 0. (1,3): f prime < 0. (3,4): f prime > 0.
Therefore, x = 1 is a local maximum. x = 3 is a local minimum.

(c) V = pi times the integral from 0 to 4 of [f(x)] squared dx.
f(x) = x cubed/3 minus 2x squared + 3x = x(x squared/3 minus 2x + 3).
[f(x)] squared = (x cubed/3 minus 2x squared + 3x) squared = x squared (x squared/3 minus 2x + 3) squared.
This is messy to expand, but we can write the integral setup and evaluate.
Expand: (x/3)(x^2 - 6x + 9)? No. f(x) = (1/3)(x^3 - 6x^2 + 9x) = (x/3)(x^2 - 6x + 9) = (x/3)(x-3)^2.
So f(x) = (x)(x-3)^2 / 3.
[f(x)]^2 = x^2 (x-3)^4 / 9.
V = (pi/9) times the integral from 0 to 4 of x^2 (x-3)^4 dx.
This can be expanded or evaluated with substitution. Let u = x-3, then x = u+3, dx = du. When x=0, u=negative 3. x=4, u=1.
Integral of (u+3)^2 u^4 du from u=negative 3 to u=1 = integral of (u^2 + 6u + 9) u^4 du = integral of (u^6 + 6u^5 + 9u^4) du = u^7/7 + 6u^6/6 + 9u^5/5 = u^7/7 + u^6 + (9/5)u^5.
Evaluate from -3 to 1: at u=1: 1/7 + 1 + 9/5 = (5/35 + 35/35 + 63/35) = 103/35.
at u=negative 3: (negative 3)^7/7 + (negative 3)^6 + (9/5)(negative 3)^5 = (negative 2187)/7 + 729 + (9/5)(negative 243) = negative 2187/7 + 729 minus (2187/5).
729 = (729)(35)/35 = 25515/35. 2187/7 = (2187)(5)/35 = 10935/35. 2187/5 = (2187)(7)/35 = 15309/35.
So at u=negative 3: negative 10935/35 + 25515/35 minus 15309/35 = (25515 - 10935 - 15309)/35 = (negative 729)/35.
Difference: 103/35 minus (negative 729/35) = 832/35.
V = (pi/9) times (832/35) = (832 pi) / 315.
Simplify: 832/315 = divide by... 832 and 315 share no factors. So V = 832pi/315.
This is approximately 2.64pi.
