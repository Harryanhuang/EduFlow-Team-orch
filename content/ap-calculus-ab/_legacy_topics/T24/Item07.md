The area of the region enclosed by the curves y = x squared and y = 2x minus x squared is

## Options (if mcq)
A) 1/3
B) 2/3
C) 4/3
D) 2

## Answer
A

## Explanation
Find intersection: x squared = 2x minus x squared implies 2x squared minus 2x = 0 implies 2x(x minus 1) = 0, so x = 0, 1.
Top curve: 2x minus x squared. Bottom: x squared.
Area = integral from 0 to 1 of [(2x minus x squared) minus x squared] dx = integral of (2x minus 2x squared) dx = [x squared minus (2/3)x cubed] from 0 to 1 = 1 minus 2/3 = 1/3.
