The function f(x) = x^3 - 6x^2 + 9x has a horizontal tangent at which of the following x-values?

## Options
A) x = 1 only
B) x = 3 only
C) x = 1 and x = 3
D) x = 0 and x = 6

## Answer
C) x = 1 and x = 3

## Explanation
A horizontal tangent occurs when f'(x) = 0. f'(x) = 3x^2 - 12x + 9 = 3(x^2 - 4x + 3) = 3(x - 1)(x - 3). Setting f'(x) = 0 gives x = 1 or x = 3.
