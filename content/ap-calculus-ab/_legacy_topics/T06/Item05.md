Find the derivative of f(x) = x^(-3) + 2*sqrt(x). Write your answer using only positive exponents and no radicals.

## Answer
f'(x) = -3/x^4 + 1/sqrt(x) = -3/x^4 + 1/x^(1/2)

## Explanation
Rewrite: f(x) = x^(-3) + 2x^(1/2). Apply the power rule to each term:
- d/dx[x^(-3)] = -3 * x^(-4) = -3/x^4
- d/dx[2x^(1/2)] = 2 * (1/2) * x^(-1/2) = x^(-1/2) = 1/sqrt(x)
So f'(x) = -3/x^4 + 1/sqrt(x).
