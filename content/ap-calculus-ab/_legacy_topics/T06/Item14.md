Let f(x) = x^4 - 4x^3. On which interval is f(x) both decreasing and concave up?

## Options
A) (0, 2)
B) (2, 3)
C) (3, infinity)
D) (-infinity, 0)

## Answer
B) (2, 3)

## Explanation
f'(x) = 4x^3 - 12x^2 = 4x^2(x - 3). f is decreasing when f'(x) < 0: x^2 > 0 always (except x=0), so f'(x) < 0 when x < 3. That gives (-infinity, 0) union (0, 3).
f''(x) = 12x^2 - 24x = 12x(x - 2). f is concave up when f''(x) > 0: x > 2 or x < 0.
Both conditions: f is decreasing AND concave up. Decreasing on (0, 3); concave up on (-infinity, 0) union (2, infinity). Intersection: (2, 3).
