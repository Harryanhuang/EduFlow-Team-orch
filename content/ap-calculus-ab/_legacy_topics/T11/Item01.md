Let $f(x) = x^3 + 2x$. If $f(1) = 3$, what is the value of $\left(f^{-1}\right)'(3)$?

## Options
A) $\frac{1}{5}$  B) $\frac{1}{3}$  C) $3$  D) $5$

## Answer
A) $\frac{1}{5}$

## Explanation
Using the inverse function derivative formula: $(f^{-1})'(b) = \frac{1}{f'(a)}$ where $f(a) = b$. Here $a = 1$ since $f(1) = 3$. We have $f'(x) = 3x^2 + 2$, so $f'(1) = 3(1)^2 + 2 = 5$. Therefore $(f^{-1})'(3) = \frac{1}{5}$.
