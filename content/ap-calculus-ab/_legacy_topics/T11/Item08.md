Let $y = x \sin x$. Find $\frac{d^2 y}{dx^2}$.

## Answer
$\frac{d^2y}{dx^2} = 2\cos x - x \sin x$

## Explanation
First derivative: $\frac{dy}{dx} = \sin x + x \cos x$ (product rule). Second derivative: $\frac{d^2y}{dx^2} = \cos x + \cos x + x(-\sin x) = 2\cos x - x \sin x$.
