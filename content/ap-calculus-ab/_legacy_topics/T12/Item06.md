A particle moves along the $x$-axis with velocity $v(t) = t^2 - 4t + 3$ for $t \geq 0$. Which of the following is true about the particle's motion on the interval $0 \leq t \leq 4$?

## Options
A) The particle is always moving to the right.
B) The particle is always moving to the left.
C) The particle changes direction exactly once.
D) The particle changes direction exactly twice.

## Answer
D

## Explanation
Factor: $v(t) = (t - 1)(t - 3)$. The velocity is zero at $t = 1$ and $t = 3$. On $(0, 1)$: test $t = 0.5$, $v(0.5) = 0.25 - 2 + 3 = 1.25 > 0$ (moving right). On $(1, 3)$: test $t = 2$, $v(2) = 4 - 8 + 3 = -1 < 0$ (moving left). On $(3, 4)$: test $t = 3.5$, $v(3.5) = 12.25 - 14 + 3 = 1.25 > 0$ (moving right). The velocity changes sign at both $t = 1$ and $t = 3$, so the particle changes direction exactly twice.
