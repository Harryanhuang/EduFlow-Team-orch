Evaluate the limit:

$\displaystyle \lim_{x \to 0} \frac{\dfrac{1}{x + 3} - \dfrac{1}{3}}{x}$

## Answer
$-\dfrac{1}{9}$

## Explanation
Combine the fractions in the numerator using a common denominator:

$$\frac{1}{x + 3} - \frac{1}{3} = \frac{3 - (x + 3)}{3(x + 3)} = \frac{-x}{3(x + 3)}$$

So the limit becomes:
$$\lim_{x \to 0} \frac{\dfrac{-x}{3(x + 3)}}{x} = \lim_{x \to 0} \frac{-x}{3x(x + 3)} = \lim_{x \to 0} \frac{-1}{3(x + 3)}$$

$$= \frac{-1}{3(0 + 3)} = -\frac{1}{9}$$
