Use integration by parts to find \(\int x e^{2x}\,dx\).

Let \(u = x\) and \(dv = e^{2x}dx\), then \(du = dx\) and \(v = \frac{1}{2}e^{2x}\).

\[
\int x e^{2x}\,dx = \frac{x e^{2x}}{2} - \int \frac{1}{2}e^{2x}\,dx = \frac{x e^{2x}}{2} - \frac{1}{4}e^{2x} + C = e^{2x}\left(\frac{x}{2} - \frac{1}{4}\right) + C
\]
