Let $f(x) = x^3 - x$ on $[0, 2]$. Find the value $c$ in $(0, 2)$ guaranteed by the Mean Value Theorem.

$f'(x) = 3x^2 - 1$. Average rate of change = $(f(2)-f(0))/2 = (8-2-0)/2 = 6/2 = 3$. Set $3c^2 - 1 = 3$, so $3c^2 = 4$, $c = 2/\sqrt{3} = 2\sqrt{3}/3 \approx 1.155$.
