A function $g$ is continuous and differentiable on $[0, 4]$ with $g(0) = 1$, $g(2) = 9$, and $g(4) = 5$. Using the Mean Value Theorem on $[0, 2]$ and $[2, 4]$, show that there is a point $c$ in $(2, 4)$ where $g'(c) = -2$, and explain why this implies $g$ has a local minimum at some point in $(0, 4)$.

On $[2, 4]$, MVT gives $g'(c) = (g(4)-g(2))/(4-2) = (5-9)/2 = -2$ for some $c \in (2, 4)$. By IVT on $g'$ (which is continuous if $g$ is twice differentiable), $g'$ goes from a positive value (from MVT on $[0, 2]$, $g'(c_1) = 4 > 0$) to a negative value ($-2$), so $g'$ must be zero at some point between — indicating a local minimum.
