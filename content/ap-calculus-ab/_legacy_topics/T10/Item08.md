Given x² + y² = 25, what is d²y/dx² in terms of x and y?

## Options
A) -1/y
B) -25/y³
C) -x/y²
D) 25/y³

## Answer
B) -25/y³

## Explanation
From dy/dx = -x/y, differentiate using the quotient rule: d²y/dx² = [(-1)(y) - (-x)(dy/dx)] / y² = [-y + x(dy/dx)]/y². Substitute dy/dx = -x/y: d²y/dx² = [-y + x(-x/y)]/y² = [-y - x²/y]/y² = [-(y² + x²)/y]/y² = -(x² + y²)/y³. Since x² + y² = 25: d²y/dx² = -25/y³.
