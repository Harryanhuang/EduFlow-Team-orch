Find all points on the curve x² + y²/4 = 5 where the tangent line is horizontal.

## Answer
(0, 2√5) and (0, -2√5)

## Explanation
Differentiate implicitly: 2x + (y/2)(dy/dx) = 0, so dy/dx = -4x/y. Horizontal tangent when dy/dx = 0, which requires x = 0 (and y ≠ 0). Substitute x = 0 into the curve: y²/4 = 5, giving y² = 20, so y = ±2√5. The points are (0, 2√5) and (0, -2√5).
