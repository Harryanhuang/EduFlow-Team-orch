Find dy/dx for the curve e^(xy) + y = x².

<answer>
dy/dx = (2x - y·e^(xy)) / (x·e^(xy) + 1)
</answer>

<explanation>
Differentiate implicitly using the chain and product rules: e^(xy) · (y + x·dy/dx) + dy/dx = 2x. Distribute: y·e^(xy) + x·e^(xy)·dy/dx + dy/dx = 2x. Factor dy/dx: dy/dx(x·e^(xy) + 1) = 2x - y·e^(xy). Therefore dy/dx = (2x - y·e^(xy)) / (x·e^(xy) + 1).
</explanation>
