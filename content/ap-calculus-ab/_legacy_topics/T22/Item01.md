Which of the following is a solution to the differential equation \(\frac{dy}{dx} = 3y\)?

A) \(y = e^{3x}\)
B) \(y = 3e^{x}\)
C) \(y = x^{3}\)
D) \(y = 5 + 3x\)

## Answer
B

## Explanation
If \(y = e^{3x}\), then \(\frac{dy}{dx} = 3e^{3x} = 3y\), so \(y = e^{3x}\) is a solution.
If \(y = 3e^{x}\), then \(\frac{dy}{dx} = 3e^{x} = y\), not \(3y\), so it is not a solution.
If \(y = x^{3}\), then \(\frac{dy}{dx} = 3x^{2}\), which is not proportional to \(y\).
If \(y = 5 + 3x\), then \(\frac{dy}{dx} = 3\), which is not proportional to \(y\).
Only option A satisfies the differential equation.
