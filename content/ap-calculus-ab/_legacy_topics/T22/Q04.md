Solve the initial value problem \(\dfrac{dy}{dx} + 2y = 6\) with \(y(0) = 1\) using an integrating factor.
The integrating factor is \(\mu(x) = e^{\int 2\,dx} = e^{2x}\). Multiplying through and integrating gives \(e^{2x}y = 3e^{2x} + C\), so \(y = 3 + Ce^{-2x}\). Using \(y(0) = 1\): \(1 = 3 + C \implies C = -2\). Thus \(y = 3 - 2e^{-2x}\).
