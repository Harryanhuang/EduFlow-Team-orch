Describe the key feature that distinguishes the slope field of \(\dfrac{dy}{dx} = x\) from the slope field of \(\dfrac{dy}{dx} = y\).
In \(\frac{dy}{dx} = x\), slopes depend only on the x-coordinate, so line segments in the same vertical column have identical slopes (forming vertical "stripes"). In \(\frac{dy}{dx} = y\), slopes depend only on the y-coordinate, so segments in the same horizontal row have identical slopes (forming horizontal "stripes").
