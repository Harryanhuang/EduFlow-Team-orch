The function $f$ has domain all real numbers. Selected values of $f$ and $f'$ are shown in the table below. If $f''$ changes sign only at $x = 0$, which of the following could be the sign of $f''(0)$?

| $x$ | $-2$ | $-1$ | $0$ | $1$ | $2$ |
|-----|------|------|-----|-----|-----|
| $f$ | $3$ | $2$ | $0$ | $2$ | $3$ |
| $f'$ | $1$ | $0$ | $0$ | $0$ | $-1$ |

A) $f''(0) > 0$
B) $f''(0) < 0$
C) $f''(0) = 0$
D) Any of the above could be true

## Answer
D

## Explanation
The table shows $f$ has values $3, 2, 0, 2, 3$ with $f'$ values $1, 0, 0, 0, -1$. This is consistent with $f$ having a local maximum at $x = 0$ (since $f'$ goes from positive to negative). For a point of inflection to occur at $x = 0$ (where $f''$ changes sign), we need concavity to change. If $f''(0) > 0$, concavity changes from down to up at $x = 0$. If $f''(0) < 0$, concavity changes from up to down. If $f''(0) = 0$, we still need to check if the sign changes on either side. All three cases could potentially work, so any sign is possible.
