For f(x) = x^3 - 3x, find all points where f''(x) = 0 and determine if they are inflection points.
f'(x) = 3x^2 - 3, f''(x) = 6x. Setting f''(x) = 0 gives x = 0. Since f'' changes from negative to positive at x = 0, the point (0, 0) is an inflection point.
