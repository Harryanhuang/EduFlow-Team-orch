Explain why the Second Derivative Test fails when f''(c) = 0 at a critical point c where f'(c) = 0.
When f''(c) = 0, the Second Derivative Test is inconclusive because the test requires f''(c) to be either positive (indicating concave up, hence minimum) or negative (indicating concave down, hence maximum). When f''(c) = 0, the graph could have a minimum, maximum, or inflection point. The First Derivative Test should be used instead.
