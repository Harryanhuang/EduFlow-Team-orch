Find the equation of the tangent line to $f(x) = x^3 - 2x$ at $x = 1$ using the limit definition of the derivative.

<question>
Find the equation of the tangent line to $f(x) = x^3 - 2x$ at $x = 1$ using the limit definition of the derivative.
</question>
<answer>
Step 1: Find $f'(1)$ using the limit definition.

$$f'(1) = \lim_{h \to 0} \frac{f(1 + h) - f(1)}{h}$$

$f(1 + h) = (1 + h)^3 - 2(1 + h) = 1 + 3h + 3h^2 + h^3 - 2 - 2h = h^3 + 3h^2 + h - 1$

$f(1) = 1 - 2 = -1$

$$f'(1) = \lim_{h \to 0} \frac{h^3 + 3h^2 + h - 1 - (-1)}{h} = \lim_{h \to 0} \frac{h^3 + 3h^2 + h}{h}$$
$$= \lim_{h \to 0}(h^2 + 3h + 1) = 1$$

Step 2: The point of tangency is $(1, f(1)) = (1, -1)$ and the slope is $m = 1$.

Step 3: Tangent line equation:
$$y - (-1) = 1(x - 1) \implies y = x - 2$$
</answer>
