The derivative of a function $f$ at $x = a$ is defined as

$$f'(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

If $f(x) = x^2$, which of the following expressions represents the difference quotient used to find $f'(3)$?

## Options
A) $\dfrac{(3 + h)^2 - 9}{h}$
B) $\dfrac{(3 + h)^2 - 3}{h}$
C) $\dfrac{9 + h - 9}{h}$
D) $\dfrac{(3 + h)^2 - h^2}{h}$

## Answer
A

## Explanation
Substitute $a = 3$ and $f(x) = x^2$ into the difference quotient formula:
$$\frac{f(3 + h) - f(3)}{h} = \frac{(3 + h)^2 - 3^2}{h} = \frac{(3 + h)^2 - 9}{h}$$
Option A correctly represents this expression.
