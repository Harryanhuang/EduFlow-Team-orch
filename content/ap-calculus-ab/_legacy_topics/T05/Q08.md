Let
$$h(x) = \begin{cases} x^2 + cx + d & \text{if } x \leq 0 \\ 2x + 1 & \text{if } x > 0 \end{cases}$$

Find the values of $c$ and $d$ such that $h$ is differentiable at $x = 0$.

<question>
Let
$$h(x) = \begin{cases} x^2 + cx + d & \text{if } x \leq 0 \\ 2x + 1 & \text{if } x > 0 \end{cases}$$

Find the values of $c$ and $d$ such that $h$ is differentiable at $x = 0$.
</question>
<answer>
For differentiability at $x = 0$, $h$ must be continuous there and the one-sided derivatives must agree.

**Continuity:** $\lim_{x \to 0^-} h(x) = 0 + 0 + d = d$. $\lim_{x \to 0^+} h(x) = 2(0) + 1 = 1$. So $d = 1$.

**Equal derivatives:** Left derivative at $x = 0$:
$$\lim_{h \to 0^-} \frac{h(0 + h) - h(0)}{h} = \lim_{h \to 0^-} \frac{h^2 + ch + d - d}{h} = \lim_{h \to 0^-} (h + c) = c$$

Right derivative at $x = 0$:
$$\lim_{h \to 0^+} \frac{h(0 + h) - h(0)}{h} = \lim_{h \to 0^+} \frac{2h + 1 - 1}{h} = \lim_{h \to 0^+} \frac{2h}{h} = 2$$

So $c = 2$ and $d = 1$.

With these values: $h(x) = x^2 + 2x + 1$ for $x \leq 0$ and $h(x) = 2x + 1$ for $x > 0$. Note that $x^2 + 2x + 1 = (x+1)^2$, which equals 1 at $x = 0$ and has derivative $2x + 2 = 2$ at $x = 0$, matching the right piece.
</answer>
