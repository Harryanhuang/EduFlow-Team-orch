Compute $f'(2)$ using the limit definition of the derivative for $f(x) = x^2 + 3x$.

<question>
Compute $f'(2)$ using the limit definition of the derivative for $f(x) = x^2 + 3x$.
</question>
<answer>
$f'(2) = \displaystyle\lim_{h \to 0} \frac{f(2 + h) - f(2)}{h}$

$f(2 + h) = (2 + h)^2 + 3(2 + h) = 4 + 4h + h^2 + 6 + 3h = h^2 + 7h + 10$

$f(2) = 4 + 6 = 10$

$f'(2) = \displaystyle\lim_{h \to 0} \frac{h^2 + 7h + 10 - 10}{h} = \lim_{h \to 0} \frac{h^2 + 7h}{h} = \lim_{h \to 0}(h + 7) = 7$
</answer>
