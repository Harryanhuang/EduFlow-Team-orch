Let $f(x) = \dfrac{x^2 - x - 6}{x - 3}$.

(a) Evaluate $\displaystyle \lim_{x \to 3} f(x)$. Does the graph have a vertical asymptote at $x = 3$? Explain.
(b) Determine whether $f$ has a horizontal asymptote. If not, describe the end behavior and identify any slant asymptote.

## Answer
(a) Factor: $f(x) = \dfrac{(x-3)(x+2)}{x-3} = x + 2$ for $x \neq 3$. So $\displaystyle \lim_{x \to 3} f(x) = 3 + 2 = 5$. Since the limit is finite, there is **no vertical asymptote** at $x = 3$. The function has a removable discontinuity (hole) at the point $(3, 5)$.

(b) Since $f(x) = x + 2$ for all $x \neq 3$, we have $\displaystyle \lim_{x \to \infty} f(x) = \infty$ and $\displaystyle \lim_{x \to -\infty} f(x) = -\infty$. There is **no horizontal asymptote**. However, since $f(x) = x + 2$ (for $x \neq 3$), the graph approaches the line $y = x + 2$ as $x \to \pm\infty$. In fact, $f$ coincides with $y = x + 2$ everywhere except at $x = 3$. The line $y = x + 2$ is a slant asymptote (though trivially so, since the function equals this line for all $x \neq 3$).
