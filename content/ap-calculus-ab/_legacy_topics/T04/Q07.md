Find all horizontal and vertical asymptotes of $f(x) = \dfrac{5x}{\sqrt{x^2 + 9}}$. Evaluate the appropriate limits to justify each asymptote.

## Answer
Vertical asymptotes: The denominator $\sqrt{x^2 + 9} \geq 3 > 0$ for all real $x$. The function is defined and continuous everywhere. **No vertical asymptotes.**

Horizontal asymptotes:
- As $x \to \infty$: $\sqrt{x^2 + 9} = x\sqrt{1 + 9/x^2}$ (since $x > 0$), so
$$\lim_{x \to \infty} \frac{5x}{x\sqrt{1 + 9/x^2}} = \lim_{x \to \infty} \frac{5}{\sqrt{1 + 9/x^2}} = \frac{5}{1} = 5.$$
- As $x \to -\infty$: $\sqrt{x^2 + 9} = |x|\sqrt{1 + 9/x^2} = -x\sqrt{1 + 9/x^2}$ (since $x < 0$), so
$$\lim_{x \to -\infty} \frac{5x}{-x\sqrt{1 + 9/x^2}} = \lim_{x \to -\infty} \frac{-5}{\sqrt{1 + 9/x^2}} = \frac{-5}{1} = -5.$$

Horizontal asymptotes: $y = 5$ and $y = -5$.
