Let $f(x) = \dfrac{3x^2 + 2x - 8}{x^2 - 4}$.

(a) Identify all vertical asymptotes and all removable discontinuities. For each vertical asymptote, compute both one-sided limits.
(b) Find the horizontal asymptote(s).
(c) Determine whether the graph of $f$ intersects its horizontal asymptote. If so, at what point(s)?

## Answer
(a) Factor: $f(x) = \dfrac{(3x - 4)(x + 2)}{(x - 2)(x + 2)}$. Denominator zeros: $x = 2, x = -2$. At $x = -2$, the factor $(x+2)$ cancels — removable discontinuity (hole). At $x = 2$, the numerator is $(6 - 4)(2 + 2) = 8 \neq 0$ — vertical asymptote.

One-sided limits at $x = 2$:
- $\displaystyle \lim_{x \to 2^{+}} f(x) = \lim_{x \to 2^{+}} \frac{3x - 4}{x - 2} = \frac{2}{0^{+}} = +\infty$
- $\displaystyle \lim_{x \to 2^{-}} f(x) = \lim_{x \to 2^{-}} \frac{3x - 4}{x - 2} = \frac{2}{0^{-}} = -\infty$

(b) $\displaystyle \lim_{x \to \pm\infty} f(x) = \lim_{x \to \pm\infty} \frac{3x - 4}{x - 2} = \frac{3}{1} = 3$. Horizontal asymptote: $y = 3$.

(c) Set $f(x) = 3$: $\dfrac{3x - 4}{x - 2} = 3$ for $x \neq -2$. This gives $3x - 4 = 3x - 6$, so $-4 = -6$, which is false. The graph **does not** intersect its horizontal asymptote $y = 3$.
