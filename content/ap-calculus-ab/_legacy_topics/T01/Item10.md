Using a graphing calculator, a student graphs \(f(x) = \dfrac{\sqrt{x + 4} - 2}{x}\) and uses the trace feature to examine values near \(x = 0\). Which value should the calculator display as \(x\) approaches 0?

## Options
A) 0
B) 0.25
C) 0.5
D) 1

## Answer
B

## Explanation
We can verify algebraically: multiply numerator and denominator by the conjugate:
\[
\frac{\sqrt{x + 4} - 2}{x} \cdot \frac{\sqrt{x + 4} + 2}{\sqrt{x + 4} + 2} = \frac{(x + 4) - 4}{x(\sqrt{x + 4} + 2)} = \frac{x}{x(\sqrt{x + 4} + 2)} = \frac{1}{\sqrt{x + 4} + 2}
\]
As \(x \to 0\), this becomes \(\dfrac{1}{\sqrt{4} + 2} = \dfrac{1}{4} = 0.25\). A calculator trace would show values approaching 0.25 from both sides.
