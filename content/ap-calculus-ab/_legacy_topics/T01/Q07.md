**Question:** Use the \(\varepsilon\)-\(\delta\) definition to prove that \(\displaystyle\lim_{x \to 3} (2x - 1) = 5\). Given an arbitrary \(\varepsilon > 0\), find an appropriate \(\delta\) and verify the implication.

**Answer:** Let \(\varepsilon > 0\) be given. We need to find \(\delta > 0\) such that if \(0 < |x - 3| < \delta\), then \(|(2x - 1) - 5| < \varepsilon\). Simplify: \(|(2x - 1) - 5| = |2x - 6| = 2|x - 3|\). We want \(2|x - 3| < \varepsilon\), which means \(|x - 3| < \varepsilon/2\). So choose \(\delta = \varepsilon/2\). Verification: if \(0 < |x - 3| < \varepsilon/2\), then \(|(2x - 1) - 5| = 2|x - 3| < 2 \cdot (\varepsilon/2) = \varepsilon\). Therefore \(\displaystyle\lim_{x \to 3} (2x - 1) = 5\).
