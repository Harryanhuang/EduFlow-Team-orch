What is d/dx[arctan(x)]?

## Options
A) 1/sqrt(1 - x^2)
B) -1/sqrt(1 - x^2)
C) 1/(1 + x^2)
D) -1/(1 + x^2)

## Answer
C

## Explanation
The derivative of arctan(x) is 1/(1 + x^2) for all real x. Derived via implicit differentiation: if y = arctan(x), then tan(y) = x, so sec^2(y) * dy/dx = 1, so dy/dx = 1/sec^2(y) = 1/(1 + tan^2(y)) = 1/(1 + x^2).
