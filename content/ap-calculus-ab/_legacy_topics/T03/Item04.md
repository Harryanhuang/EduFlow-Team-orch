State the three conditions required for a function \( f \) to be continuous at \( x = c \).

## Answer
1. \( f(c) \) is defined (i.e., \( c \) is in the domain of \( f \)).
2. \( \displaystyle\lim_{x \to c} f(x) \) exists (i.e., the left-hand and right-hand limits both exist and are equal).
3. \( \displaystyle\lim_{x \to c} f(x) = f(c) \).

## Explanation
These three conditions constitute the formal definition of continuity at a point. If any one of them fails, the function is discontinuous at \( x = c \). Condition 1 ensures there is a function value. Condition 2 ensures the function approaches a single value from both sides. Condition 3 ensures that the approaching value matches the actual function value.
