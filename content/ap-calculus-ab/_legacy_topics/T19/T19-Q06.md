<question>
Given f(x) = x² on the interval [0, 3], write out the left Riemann sum with n = 3 subintervals and calculate its numerical approximation.
</question>
<answer>
To form the left Riemann sum, we divide [0, 3] into n = 3 subintervals of equal width.

**Step 1: Determine Δx**
$$\Delta x = \frac{b - a}{n} = \frac{3 - 0}{3} = 1$$

**Step 2: Identify the left endpoints**
The left endpoints are:
- x₀ = 0
- x₁ = 1
- x₂ = 2

**Step 3: Write out the left Riemann sum**
$$L_3 = \sum_{i=0}^{2} f(x_i)\,\Delta x = f(0)\cdot 1 + f(1)\cdot 1 + f(2)\cdot 1$$

**Step 4: Evaluate**
$$L_3 = (0)^2(1) + (1)^2(1) + (2)^2(1) = 0 + 1 + 4 = 5$$

**Answer:** The left Riemann sum with n = 3 subintervals is 5.

**Comparison:** The actual area is $\int_0^3 x^2\,dx = \left[\frac{x^3}{3}\right]_0^3 = 9$. The left sum underestimates because f(x) = x² is increasing on [0, 3], so the left endpoints yield rectangles below the curve.
</answer>
