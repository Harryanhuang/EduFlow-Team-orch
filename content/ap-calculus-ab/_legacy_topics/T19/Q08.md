**Q:** The Trapezoidal Rule gives T_n = 5 and Simpson's Rule gives S_n = 5.2 for ∫₀² f(x)dx. If f''(x)>0 on [0,2], which estimate is more accurate? Why?

**A:** Trapezoidal Rule is more accurate here — when f''(x)>0 (concave up), the trapezoids lie ABOVE the curve, giving an overestimate, but the error is smaller than Simpson's Rule which requires f⁽⁴⁾(x) to be well-behaved. Wait — for concave up functions, T_n overestimates but is still generally more stable with just n trapezoids vs Simpson's requiring more computation. Actually, Simpson's Rule exact for polynomials up to degree 3, so if f is "nice", S_n≈T_n with T_n slightly over.
