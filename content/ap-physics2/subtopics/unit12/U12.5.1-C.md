A circuit has 4 branches and 3 junctions. What is the minimum number of independent equations (combining junction and loop rules) needed to solve for all branch currents?

## Options
A) $3$
B) $4$
C) $5$
D) $6$

## Answer
B

## Explanation
Number of unknowns = number of branches = 4. Independent junction equations = $J - 1 = 3 - 1 = 2$. Independent loop equations = $B - (J - 1) = 4 - 2 = 2$. Total independent equations = $2 + 2 = 4$, which equals the number of unknowns.
