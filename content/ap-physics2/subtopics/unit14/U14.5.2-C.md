An object is placed at the center of curvature ($C = 2f$) of a concave mirror. What are the image properties?

## Options
A) Virtual, erect, same size
B) Real, inverted, diminished
C) Real, inverted, same size
D) Virtual, inverted, magnified

## Answer
C

## Explanation
At $C$ ($d_o = 2f$): $1/d_i = 1/f - 1/(2f) = 1/(2f)$, so $d_i = 2f = C$. The image is at $C$, real, inverted, and the same size as the object ($m = -d_i/d_o = -1$).
