An ideal gas expands from volume $V_1$ to volume $V_2$ along three different paths on a $P$-$V$ diagram: Path A is a straight line (diagonal), Path B is above Path A (higher pressure throughout), and Path C is below Path A (lower pressure throughout). Which correctly ranks the work done by the gas?

## Options
A) $W_B > W_A > W_C$
B) $W_A > W_B > W_C$
C) $W_C > W_A > W_B$
D) $W_B = W_A = W_C$

## Answer
A

## Explanation
Work done by a gas equals the area under the $P$-$V$ curve between the initial and final volumes. Since Path B lies above Path A, and Path A lies above Path C, the areas satisfy $A_B > A_A > A_C$, so $W_B > W_A > W_C$. This demonstrates that work is a path function — different paths between the same states yield different amounts of work.
