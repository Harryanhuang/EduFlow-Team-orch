A 5 kg box sits on a rough surface (μ_s = 0.6). A horizontal force of 20 N is applied. The maximum static friction is 29.4 N. Does the box move?

## Options
A) Yes, because 20 N > 0.
B) No, because 20 N < 29.4 N; static friction = 20 N.
C) No, because static friction = 29.4 N.
D) Yes, because μ_s < 1.

## Answer
B

## Explanation
The applied force (20 N) is less than the maximum static friction (29.4 N), so the box does NOT move. Static friction adjusts to equal the applied force: f_s = 20 N. It does NOT automatically equal μ_s N — that is only the maximum. Option B is correct.
