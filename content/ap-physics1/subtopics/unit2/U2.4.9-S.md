Spring A has k = 100 N/m and Spring B has k = 200 N/m. They are connected in series and a force of 50 N is applied. What is the total extension of the system?

## Options
A) 0.17 m
B) 0.25 m
C) 0.50 m
D) 0.75 m

## Answer
D

## Explanation
In series, each spring experiences the same force F = 50 N. Extension of Spring A: x_A = F/k_A = 50/100 = 0.50 m. Extension of Spring B: x_B = F/k_B = 50/200 = 0.25 m. Total extension: x = x_A + x_B = 0.50 + 0.25 = 0.75 m. Alternatively: 1/k_eq = 1/100 + 1/200 = 3/200, so k_eq = 200/3 ≈ 66.7 N/m. Then x = F/k_eq = 50/66.7 ≈ 0.75 m. Option A (0.17 m) uses the parallel formula: k_eq = 100+200 = 300, x = 50/300 ≈ 0.17 m. Option B (0.25 m) only counts Spring B's extension. Option C (0.50 m) only counts Spring A's extension. The softer spring stretches more in series.
