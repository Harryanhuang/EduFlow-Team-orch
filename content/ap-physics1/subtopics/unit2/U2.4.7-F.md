A person stands on a scale in an elevator. The scale reads a value greater than the person's actual weight. What can be concluded about the elevator's motion?

## Options
A) The elevator is moving upward at constant speed.
B) The elevator is accelerating upward.
C) The elevator is accelerating downward.
D) The elevator is in free fall.

## Answer
B

## Explanation
The scale reads the normal force N, which equals the apparent weight. Newton's Second Law: N - mg = ma, so N = m(g + a). If N > mg, then a > 0 (accelerating upward). Option A (constant speed upward) gives a = 0, so N = mg (normal weight, not greater). Option C (accelerating downward) gives a < 0, so N < mg (apparent weight is less — feels lighter). Option D (free fall) gives a = -g, so N = 0 (weightlessness). The key: it's the acceleration direction that determines apparent weight, not the velocity direction. An elevator moving upward but decelerating would also read less than actual weight.
