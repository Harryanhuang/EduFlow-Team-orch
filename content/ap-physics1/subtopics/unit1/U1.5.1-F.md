The position-time graph for an object moving along a straight line is a straight line with a positive slope. What can be concluded about the object's motion?

## Options
A) The object is accelerating.
B) The object is moving with constant positive velocity.
C) The object is at rest.
D) The object is moving with constant negative velocity.

## Answer
B

## Explanation
x-t 图像的斜率表示速度。直线表示斜率恒定，即速度恒定。正斜率表示正方向运动。因此物体以恒定的正速度运动。选项 B 正确。常见错误是将 x-t 图像的纵坐标值当作速度，或认为直线表示静止。
