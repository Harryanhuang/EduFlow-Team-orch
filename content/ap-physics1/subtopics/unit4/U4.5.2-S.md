A force acts on an object along the x-axis. From x = 0 to x = 3 m, F = +10 N (constant). From x = 3 m to x = 6 m, F = -10 N (constant). What is the total work done from x = 0 to x = 6 m?

## Options
A) 60 J
B) 30 J
C) 0 J
D) -60 J

## Answer
C

## Explanation
From x = 0 to 3 m: W₁ = F₁d₁ = (+10)(3) = +30 J (rectangle above axis). From x = 3 to 6 m: W₂ = F₂d₂ = (-10)(3) = -30 J (rectangle below axis). Total work: W = W₁ + W₂ = +30 + (-30) = 0 J. The positive work in the first half is completely cancelled by the negative work in the second half. The object's speed first increases (0-3 m) then decreases by the same amount (3-6 m), returning to its original speed. Option A (60 J) adds the magnitudes without signs. Option B only counts one section. Option D only counts the negative work.
