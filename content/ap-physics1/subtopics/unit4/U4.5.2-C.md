A spring (k = 100 N/m) is compressed from equilibrium to x = -0.10 m, then released. The spring pushes a block from x = -0.10 m to x = +0.10 m. What is the work done by the SPRING force on the block during this entire process?

## Options
A) 0 J
B) 0.5 J
C) 1.0 J
D) -1.0 J

## Answer
A

## Explanation
The spring force is F = -kx. From x = -0.10 to x = 0: F points in +x direction (pushing), block moves in +x direction → positive work. W₁ = ½k(0.10)² = ½(100)(0.01) = 0.5 J. From x = 0 to x = +0.10: F points in -x direction (pulling back), block moves in +x direction → negative work. W₂ = -½k(0.10)² = -0.5 J. Total: W = W₁ + W₂ = 0.5 + (-0.5) = 0 J. This makes sense: the F-x graph is symmetric about x = 0, so the area above and below the axis cancel. The spring gives KE to the block (first half) then takes it back (second half). At x = +0.10, the block has the same speed as at x = -0.10 (both zero at the turning points), so net work = ΔKE = 0.
