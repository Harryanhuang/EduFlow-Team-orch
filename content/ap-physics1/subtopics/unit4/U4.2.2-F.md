Two objects have the same momentum p. Object A has mass m and Object B has mass 4m. What is the ratio KE_A/KE_B?

## Options
A) 1/4
B) 1/2
C) 1
D) 4

## Answer
D

## Explanation
KE = p²/(2m). For Object A: KE_A = p²/(2m). For Object B: KE_B = p²/(2·4m) = p²/(8m). Ratio: KE_A/KE_B = [p²/(2m)]/[p²/(8m)] = 8m/(2m) = 4. The lighter object has more kinetic energy for the same momentum. This makes sense: same momentum means lighter object has higher velocity (v_A = p/m, v_B = p/4m), and KE depends on v², so the velocity advantage more than compensates for the mass disadvantage.
