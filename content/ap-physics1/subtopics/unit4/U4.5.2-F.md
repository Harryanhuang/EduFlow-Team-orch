A force varies with position as shown: F increases linearly from 0 to 20 N over 4.0 m. What is the work done by this force?

## Options
A) 20 J
B) 40 J
C) 80 J
D) 10 J

## Answer
B

## Explanation
The F-x graph is a triangle with base 4.0 m and height 20 N. Work = area under the curve = ½ × base × height = ½ × 4.0 × 20 = 40 J. This is equivalent to a spring being stretched (F = kx with k = 20/4 = 5 N/m, W = ½kx² = ½(5)(16) = 40 J). Option A (20 J) uses base × height without the ½ factor. Option C (80 J) uses base × height. Option D (10 J) uses an incorrect formula.
