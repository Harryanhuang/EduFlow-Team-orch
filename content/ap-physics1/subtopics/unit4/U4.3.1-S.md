A ball is dropped from height H to the ground. If the reference level is changed from the ground to a table at height H/2, how does the change in gravitational potential energy ΔPE compare?

## Options
A) ΔPE doubles because the reference is higher.
B) ΔPE halves because the ball starts from a lower height relative to the new reference.
C) ΔPE stays the same — it depends only on the height difference, not the reference.
D) ΔPE changes sign.

## Answer
C

## Explanation
ΔPE = mgΔh = mg(H - 0) = mgH with ground reference. With table reference (at H/2): initial height = H - H/2 = H/2 above table. Final height = 0 - H/2 = -H/2 (below table). ΔPE = mg(-H/2) - mg(H/2) = -mgH/2 - mgH/2 = -mgH. Same magnitude! The absolute PE values change with reference, but the CHANGE in PE depends only on the actual height difference H, which is independent of reference. This is a fundamental property: only ΔPE has physical meaning (it equals the work done by gravity).
