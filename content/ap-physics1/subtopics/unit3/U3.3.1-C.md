A curve of radius R is banked at angle θ for design speed v₀. With static friction coefficient μ_s, what is the maximum speed the car can travel without sliding up the bank?

## Options
A) v_max = √(Rg·tanθ)
B) v_max = √(Rg·(sinθ + μ_s cosθ)/(cosθ − μ_s sinθ))
C) v_max = √(Rg·(tanθ + μ_s)/(1 − μ_s tanθ))
D) v_max = v₀ + μ_s·Rg

## Answer
C

## Explanation
At maximum speed, friction acts downward along the bank (preventing the car from sliding up). Resolving forces parallel and perpendicular to the bank: perpendicular: N - mg cosθ - mv²sinθ/R = 0 → N = mg cosθ + mv²sinθ/R. Parallel (toward center): N sinθ + μ_s N = mv²/R → N(sinθ + μ_s) = mv²/R. Substituting N: (mg cosθ + mv²sinθ/R)(sinθ + μ_s) = mv²/R. After algebra: v_max² = Rg(sinθ + μ_s cosθ)/(cosθ - μ_s sinθ). Dividing numerator and denominator by cosθ: v_max = √(Rg(tanθ + μ_s)/(1 - μ_s tanθ)). Option A is the frictionless design speed. Option D is dimensionally incorrect.
