A satellite in circular orbit at radius r has period T. If the orbit radius is increased to 4r, what is the new period?

## Options
A) 2T
B) 4T
C) 8T
D) 16T

## Answer
C

## Explanation
Kepler's Third Law: T² ∝ r³, so T ∝ r^(3/2). If r → 4r: T_new/T = (4r/r)^(3/2) = 4^(3/2) = (√4)³ = 2³ = 8. So T_new = 8T. Option A (2T) uses T ∝ √r. Option B (4T) uses T ∝ r. Option D (16T) uses T ∝ r². The key is the 3/2 power: quadrupling the distance gives 4^(3/2) = 8 times the period. This can also be seen from T² = (4)³ = 64, so T = √64 = 8 times the original.
