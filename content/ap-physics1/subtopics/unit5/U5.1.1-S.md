Two objects have the same kinetic energy. Object A has mass m and Object B has mass 9m. What is the ratio p_B/p_A of their momenta?

## Options
A) 1/9
B) 1/3
C) 3
D) 9

## Answer
C

## Explanation
KE = p²/(2m), so p = √(2mKE). For the same KE: p ∝ √m. So p_B/p_A = √(m_B/m_A) = √(9m/m) = √9 = 3. The heavier object has more momentum for the same kinetic energy. This makes sense: Object B has 9× the mass but 1/9 the velocity squared (to have the same KE), meaning 1/3 the velocity. So p_B = 9m × v/3 = 3mv = 3p_A. Option A (1/9) confuses the mass ratio with the momentum ratio. Option B (1/3) uses v ratio instead of p ratio. Option D (9) uses the mass ratio directly.
