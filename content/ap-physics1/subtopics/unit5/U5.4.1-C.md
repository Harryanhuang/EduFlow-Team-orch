A rocket in deep space (no gravity, no air resistance) burns fuel and ejects exhaust gases backward. Consider the system to be the rocket only (not the fuel). Which statement correctly describes this system?

## Options
A) The rocket is a closed system; its momentum is conserved.
B) The rocket is an open system; the ejected gases carry momentum away, so the rocket's momentum changes even without external forces on the rocket.
C) The rocket is a closed system; the thrust is an external force.
D) The rocket is an open system, but its momentum is still conserved because there are no external forces.

## Answer
B

## Explanation
The rocket alone is an open system — fuel/exhaust matter leaves the system boundary, carrying momentum with it. As gases are ejected backward (negative momentum), the rocket gains forward momentum. This is NOT momentum conservation for the rocket alone — the rocket's momentum changes because mass (carrying momentum) leaves the system. If we chose rocket + all exhaust gases as the system, THAT would be a closed system with conserved total momentum. Option A is wrong — matter leaves the rocket, so it's not closed. Option C is wrong — thrust is not an "external force" in the traditional sense; it arises from mass ejection. Option D is wrong — open system momentum is not conserved when matter carries momentum in/out.
