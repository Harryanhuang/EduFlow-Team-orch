A uniform beam of weight W and length L is supported at both ends by vertical forces F₁ (left) and F₂ (right). A load of weight w hangs at distance d from the left end. If you calculate torques about the left end, you get F₂ = (w·d + W·L/2)/L. If you calculate about the right end, which equation do you get?

## Options
A) F₁ = (w(L-d) + W·L/2)/L — different forces are found.
B) F₁ = (w(L-d) + W·L/2)/L — the same physical result, just solving for F₁ instead of F₂.
C) F₂ = (w·d + W·L/2)/L — you get the same equation regardless of pivot.
D) The problem cannot be solved by choosing the right end as pivot.

## Answer
B

## Explanation
When you take torques about the right end, F₂ has zero moment arm (it acts AT the right end), so it drops out. You're left solving for F₁:
- w acts at distance (L-d) from right end → τ_w = -w(L-d) (CW)
- W acts at L/2 from right end → τ_W = -W(L/2) (CW)
- F₁ acts at distance L from right end → τ₁ = +F₁·L (CCW)

Setting net torque = 0: F₁·L = w(L-d) + W·L/2, so F₁ = (w(L-d) + W·L/2)/L.

This is consistent with the left-pivot calculation: ΣF_y = 0 gives F₁ + F₂ = W + w. Both pivot choices give correct, consistent results. Option A is correct in substance but wrong in saying "different forces" — you're just solving for a different unknown, which is normal. Option C is wrong — different pivots give different equations (solving for different unknowns). Option D is wrong — any pivot choice is valid.
