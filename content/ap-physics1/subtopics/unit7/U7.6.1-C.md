Object A has I_A and ω_A. Object B has I_B = 2I_A and ω_B = ω_A/2. What is the ratio KE_B/KE_A?

## Options
A) 1/4
B) 1/2
C) 1
D) 2

## Answer
B

## Explanation
KE_A = ½I_A ω_A². KE_B = ½I_B ω_B² = ½(2I_A)(ω_A/2)² = ½(2I_A)(ω_A²/4) = ½I_A ω_A² × (2/4) = KE_A × ½. So KE_B/KE_A = 1/2. Object B has twice the moment of inertia (×2 KE) but half the angular velocity (×¼ KE since ω is squared). Net effect: 2 × ¼ = ½. Option A (1/4) only accounts for the ω change. Option C (1) thinks the effects cancel completely. Option D (2) only accounts for the I change.
