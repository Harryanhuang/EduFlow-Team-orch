A mass-spring system has a period of 1.0 s. If the spring constant is quadrupled (k → 4k) while the mass stays the same, what is the new period?

## Options
A) 0.25 s
B) 0.5 s
C) 2.0 s
D) 4.0 s

## Answer
B

## Explanation
T = 2π√(m/k). If k → 4k: T_new = 2π√(m/(4k)) = 2π√(m/k) × √(1/4) = T × (1/2) = 1.0/2 = 0.5 s. Quadrupling k makes the spring 4× stiffer, so oscillations are faster → period halves. Option A (0.25) would be T/4 — forgetting the square root. Option C (2.0) doubles — as if k decreased. Option D (4.0) quadruples — inversely wrong. Remember: T ∝ 1/√k.
