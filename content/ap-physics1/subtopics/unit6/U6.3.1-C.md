Two identical springs (each with constant k) are connected in parallel and attached to a mass m on a frictionless surface. What is the period of oscillation compared to a single spring with the same mass?

## Options
A) T_parallel = T_single / √2 (shorter period — stiffer system)
B) T_parallel = T_single (unchanged)
C) T_parallel = √2 × T_single (longer period)
D) T_parallel = 2 × T_single

## Answer
A

## Explanation
For springs in parallel, both springs stretch by the same amount x when the mass moves. Each spring exerts force -kx, so total restoring force = -2kx. The effective spring constant is k_eq = 2k (stiffer).

Period: T_parallel = 2π√(m/k_eq) = 2π√(m/(2k)) = (1/√2) × 2π√(m/k) = T_single/√2.

The parallel system is stiffer → higher frequency → shorter period. Option B is wrong — adding springs changes k. Option C would be for springs in series (k_eq = k/2, so T = √2 × T_single). Option D uses k_eq = k/4, which is incorrect. Key: parallel → stiffer → faster oscillation; series → softer → slower oscillation.
