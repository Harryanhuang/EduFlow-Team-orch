**Item 1 [F]**
Question: Solve by elimination: 2x + 3y = 7, x − y = 4
Answer: x = 19/5, y = −1/5
Difficulty: F
Topic: 1.1
Explanation: Multiply the second equation by 3: 3x − 3y = 12. Add to the first: 5x = 19 → x = 19/5. Back-substitute: y = 19/5 − 4 = −1/5.
Tags: simultaneous equations, elimination method, linear equations, substitution
**Item 2 [F]**
Question: Solve by substitution: y = 3x − 2, 2x + y = 8
Answer: x = 2, y = 4
Difficulty: F
Topic: 1.1
Explanation: Substitute y = 3x − 2 into 2x + y = 8: 2x + 3x − 2 = 8 → 5x = 10 → x = 2. Then y = 3(2) − 2 = 4.

**Item 4 [S]**
Question: Solve: 2x/3 + y/2 = 5/6, x/4 − y/3 = 1/12
Answer: x = 23/25, y = 11/25
Difficulty: S
Topic: 1.1
Explanation: Multiply the first equation by 6 to clear denominators: 4x + 3y = 5. Multiply the second equation by 12: 3x − 4y = 1. Solve by elimination: multiply the first equation by 4 and the second by 3, then add: 16x + 12y = 20; 9x − 12y = 3 → 25x = 23 → x = 23/25. Substitute back: 4(23/25) + 3y = 5 → 92/25 + 3y = 125/25 → 3y = 33/25 → y = 11/25. Verify: (2/3)(23/25) + (1/2)(11/25) = 46/75 + 11/50 = 92/150 + 33/150 = 125/150 = 5/6 ✓; (1/4)(23/25) − (1/3)(11/25) = 23/100 − 11/75 = 69/300 − 44/300 = 25/300 = 1/12 ✓.
Tags: simultaneous equations, elimination, fractional coefficients, linear equations

**Item 6 [S]**
Question: A shop sells pens at $3 each and notebooks at $5 each. A customer buys 8 items for $30. How many of each?
Answer: 5 pens, 3 notebooks
Difficulty: S
Topic: 1.1
Explanation: Let p = pens, n = notebooks. p + n = 8 and 3p + 5n = 30. Multiply first by 3: 3p + 3n = 24. Subtract: 2n = 6 → n = 3. Then p = 5.
Tags: simultaneous equations, word problem, elimination, shopping context
**Item 7 [C]**
Question: Solve: 2x + y = 4, x² + y = 7
Answer: (3, −2) and (−1, 6)
Difficulty: C
Topic: 1.1
Explanation: From 2x + y = 4, y = 4 − 2x. Substitute into x² + y = 7: x² + 4 − 2x = 7 → x² − 2x − 3 = 0 → (x − 3)(x + 1) = 0 → x = 3 or x = −1. At x = 3, y = 4 − 6 = −2. Check: 9 + (−2) = 7 ✓. At x = −1, y = 4 − (−2) = 6. Check: 1 + 6 = 7 ✓. Solutions: (3, −2) and (−1, 6).
Tags: simultaneous equations, linear and quadratic, substitution, intersection, discriminant

**Item 9 [C]**
Question: Find k for which 3x + 2y = 7 and 6x + ky = 14 have infinitely many solutions.
Answer: k = 4
Difficulty: C
Topic: 1.1
Explanation: For infinitely many solutions, the two equations must be proportional: 6x/3x = ky/2y = 14/7 → 2 = k/2 = 2 → k = 4. When k = 4, the second equation becomes 6x + 4y = 14, which is exactly 2 times the first equation (6x + 4y = 14), so the system is dependent with infinitely many solutions.
Tags: simultaneous equations, infinitely many solutions, consistency, proportionality
