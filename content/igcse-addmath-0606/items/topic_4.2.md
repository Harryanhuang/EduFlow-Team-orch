**Item 7 [C]**
Question: Solve: |x − 2| = |x + 3|
Answer: x = −1/2
Difficulty: C
Topic: 4.2
Explanation: Square both sides: (x−2)² = (x+3)² → x²−4x+4 = x²+6x+9 → −4x−6x = 9−4 → −10x = 5 → x = −1/2. Verify: |−1/2−2| = |−2.5| = 2.5; |−1/2+3| = |2.5| = 2.5 ✓.
Tags: absolute value, solving equations, equating distances
**Item 8 [C]**
Question: Solve: |x + 1| + |x − 2| = 5
Answer: x = −2 or x = 3
Difficulty: C
Topic: 4.2
Explanation: Critical points at x = −1 and x = 2. Case 1: x < −1: −(x+1) − (x−2) = 5 → −2x+1 = 5 → x = −2 (valid, x < −1). Case 2: −1 ≤ x < 2: (x+1) − (x−2) = 5 → 3 = 5 (no solution). Case 3: x ≥ 2: (x+1) + (x−2) = 5 → 2x−1 = 5 → x = 3 (valid, x ≥ 2). So x = −2 or x = 3.
Tags: absolute value, piecewise solving, distance sum
**Item 9 [C]**
Question: Prove: |a + b| ≤ |a| + |b| for all real a, b.
Answer: By triangle inequality proof: |a+b|² = (a+b)² = a²+2ab+b² ≤ a²+2|a||b|+b² = (|a|+|b|)². Since all quantities are non-negative, |a+b| ≤ |a|+|b|. Equality holds when ab ≥ 0 (a and b have the same sign). QED.
Difficulty: C
Topic: 4.2
Explanation: Square both sides: |a+b|² = (a+b)² = a²+2ab+b². Compare with (|a|+|b|)² = a²+2|a||b|+b². Since ab ≤ |a||b|, we have a²+2ab+b² ≤ a²+2|a||b|+b². Taking positive square roots: |a+b| ≤ |a|+|b|. Equality iff ab = |a||b|, i.e., a and b have the same sign. QED.
Tags: triangle inequality, proof, absolute value properties
