**Item 1 [F]**
Question: Simplify: 2³ × 2⁴
Answer: 2⁷ = 128
Difficulty: F
Topic: 2.1
Explanation: Using the law of indices: aᵐ × aⁿ = aᵐ⁺ⁿ. So 2³ × 2⁴ = 2³⁺⁴ = 2⁷ = 128.
Tags: laws of indices, multiplication, indices, algebra

**Item 4 [S]**
Question: Simplify using logarithm laws: log₂ 8 + log₂ 4
Answer: log₂ 8 + log₂ 4 = log₂(8×4) = log₂ 32 = 5
Difficulty: S
Topic: 2.1
Explanation: Using the product law of logarithms: log(ab) = log a + log b. So log₂ 8 + log₂ 4 = log₂(8×4) = log₂ 32. Since 32 = 2⁵, log₂ 32 = 5.
Tags: logarithm laws, product law, simplification, log₂

**Item 7 [C]**
Question: Solve: 2ˣ⁺¹ = 5ˣ. Give your answer in terms of logarithms.
Answer: x = ln 2 / (ln 5 − ln 2)
Difficulty: C
Topic: 2.1
Explanation: 2ˣ⁺¹ = 5ˣ → 2 · 2ˣ = 5ˣ → 2 = (5/2)ˣ → x = log₍₅/₂₎ 2 = ln 2 / ln(5/2) = ln 2 / (ln 5 − ln 2). Numerically: ≈ 1.356.
Tags: exponential equations, logarithms, solving equations, change of base
**Item 8 [C]**
Question: Prove: log(ab) = log a + log b
Answer: Let log a = m → a = 10ᵐ and log b = n → b = 10ⁿ. Then ab = 10ᵐ · 10ⁿ = 10ᵐ⁺ⁿ → log(ab) = m + n = log a + log b. QED.
Difficulty: C
Topic: 2.1
Explanation: Let log a = m, so a = 10ᵐ (using base 10). Similarly log b = n, so b = 10ⁿ. Then ab = 10ᵐ × 10ⁿ = 10ᵐ⁺ⁿ (using law of indices: aᵐ × aⁿ = aᵐ⁺ⁿ). Taking log of both sides: log(ab) = m + n = log a + log b. QED. Note: this proof works for any base; the result holds for log base a where a > 0, a ≠ 1.
Tags: logarithm laws, proof, product law, properties
**Item 9 [C]**
Question: Solve simultaneously: log₂ x + log₂ y = 4 and x − 2y = 0
Answer: x = 4√2, y = 2√2
Difficulty: C
Topic: 2.1
Explanation: From log₂ x + log₂ y = 4, use the product law: log₂(xy) = 4 → xy = 2⁴ = 16. From x − 2y = 0, x = 2y. Substitute into xy = 16: (2y)(y) = 16 → 2y² = 16 → y² = 8 → y = 2√2 (y > 0, since log₂ y requires y > 0). Then x = 2y = 4√2. Verify: xy = (4√2)(2√2) = 16 ✓; log₂(4√2) + log₂(2√2) = log₂(16) = 4 ✓.
Tags: simultaneous equations, logarithms, algebra, system of equations
