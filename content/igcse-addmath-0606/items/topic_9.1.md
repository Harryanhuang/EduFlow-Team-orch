# Topic 9.1 — Equation of a Straight Line and Circle

**Item 1 [F]**
Question: Find the equation of the line through (1, 3) and (4, 11).
Answer: y = (8/3)x + 1/3
Difficulty: F
Topic: 9.1
Explanation: m = (11−3)/(4−1) = 8/3. Using point-slope form: y − 3 = (8/3)(x − 1) → y = (8/3)x + 1/3.
Tags: coordinate geometry, straight line, equation
**Item 2 [F]**
Question: Write the equation of the circle with centre (2, −3) and radius 5.
Answer: (x − 2)² + (y + 3)² = 25
Difficulty: F
Topic: 9.1
Explanation: Using (x−a)² + (y−b)² = r²: centre (a,b) = (2,−3), r = 5. So (x−2)² + (y+3)² = 25.
Tags: coordinate geometry, circle, equation
**Item 3 [S]**
Question: Find the equation of the line perpendicular to y = 2x + 1 passing through (3, 4).
Answer: x + 2y − 11 = 0
Difficulty: S
Topic: 9.1
Explanation: Original line has m = 2. Perpendicular slope = −1/2. Using point-slope: y − 4 = −1/2(x − 3) → 2y − 8 = −x + 3 → x + 2y − 11 = 0.

**Item 4 [S]**
Question: Find the centre and radius of x² + y² − 4x + 6y − 3 = 0.
Answer: Centre (2, −3), radius = 4
Difficulty: S
Topic: 9.1
Explanation: Complete the square: (x²−4x) + (y²+6y) = 3 → (x−2)²−4 + (y+3)²−9 = 3 → (x−2)² + (y+3)² = 16. Centre = (2, −3), radius = √16 = 4.
Tags: coordinate geometry, circle, completing the square

**Item 6 [S]**
Question: Find where y = 2x + 1 intersects (x − 1)² + (y − 2)² = 9.
Answer: x = (3 ± 2√11)/5, giving points ((3+2√11)/5, (11+4√11)/5) and ((3−2√11)/5, (11−4√11)/5)
Difficulty: S
Topic: 9.1
Explanation: Substitute y = 2x + 1 into (x − 1)² + (y − 2)² = 9: (x − 1)² + (2x + 1 − 2)² = 9 → (x − 1)² + (2x − 1)² = 9 → (x² − 2x + 1) + (4x² − 4x + 1) = 9 → 5x² − 6x + 2 = 9 → 5x² − 6x − 7 = 0. Using the quadratic formula: x = [6 ± √(36 + 140)] / 10 = [6 ± √176] / 10 = [6 ± 4√11] / 10 = (3 ± 2√11) / 5. At x = (3 + 2√11)/5: y = 2x + 1 = 2(3+2√11)/5 + 1 = (6+4√11)/5 + 5/5 = (11+4√11)/5. At x = (3 − 2√11)/5: y = 2(3−2√11)/5 + 1 = (11−4√11)/5. So the intersection points are ((3±2√11)/5, (11±4√11)/5).
Tags: coordinate geometry, line-circle intersection

**Item 8 [C]**
Question: Find the length of the chord of x² + y² = 25 cut off by y = x + 1.
Answer: 7√2
Difficulty: C
Topic: 9.1
Explanation: Distance from centre (0,0) to the line y = x + 1 (rewritten as x − y + 1 = 0): d = |0 − 0 + 1| / √(1² + (−1)²) = 1 / √2. The perpendicular from the centre meets the chord at its midpoint. Half-chord length = √(r² − d²) = √(25 − 1/2) = √(49/2) = 7/√2 = (7√2)/2. Full chord length = 2 × (7/√2) = 7√2.
Tags: coordinate geometry, chord, circle
**Item 9 [C]**
Question: Prove that the angle in a semicircle is a right angle.
Answer: Thales' theorem proved
Difficulty: C
Topic: 9.1
Explanation: By Thales' theorem: the angle subtended by a diameter at any point on the circle is a right angle. Let AB be the diameter and C be any point on the circle. Proof using the cosine rule: Let O be the centre, so OA = OB = OC = r. In triangles AOC and BOC: AC² = 2r² − 2r² cos ∠AOC and BC² = 2r² − 2r² cos ∠BOC. Since A, O, B are collinear, ∠AOC + ∠BOC = 180°, so cos ∠BOC = −cos ∠AOC. Then AC² + BC² = 4r² − 2r²(cos ∠AOC + cos ∠BOC) = 4r² − 0 = 4r². By the cosine rule in triangle ACB: cos ∠ACB = (AC² + BC² − AB²) / (2·AC·BC) = (4r² − (2r)²) / (2·AC·BC) = 0. Hence ∠ACB = 90°. QED.
Tags: coordinate geometry, Thales theorem, circle, proof
