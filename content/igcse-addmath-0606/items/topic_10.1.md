**Item 1 [F]**
Question: Find the magnitude of the vector (3, 4).
Answer: 5
Difficulty: F
Topic: 10.1
Explanation: |a| = √(a₁² + a₂²) = √(3² + 4²) = √(9+16) = √25 = 5.

**Item 3 [S]**
Question: Find the angle between a = (1, 1) and b = (1, 0).
Answer: 45°
Difficulty: S
Topic: 10.1
Explanation: a·b = 1×1 + 1×0 = 1. |a| = √2, |b| = 1. cos θ = (a·b)/(|a||b|) = 1/√2 = √2/2. So θ = 45°.
Tags: vectors, scalar product, angle between vectors

**Item 5 [S]**
Question: If a · b = 6, |a| = 3, |b| = 5, find the angle between a and b.
Answer: arccos(2/5) ≈ 66.42°
Difficulty: S
Topic: 10.1
Explanation: a·b = |a||b| cos θ → 6 = 3×5×cos θ → cos θ = 6/15 = 2/5. So θ = arccos(2/5) ≈ 66.42°.

Answer: k = 1

**Item 7 [C]**
Question: Show that the vectors (1, 2, 2) and (2, −2, 1) are perpendicular.
Answer: Dot product = 1×2 + 2×(−2) + 2×1 = 2 − 4 + 2 = 0. Since the scalar product is 0, the vectors are perpendicular.
Difficulty: C
Topic: 10.1
Explanation: For 3D vectors, a · b = a₁b₁ + a₂b₂ + a₃b₃. Computing (1,2,2) · (2,−2,1) = 2 + (−4) + 2 = 0. Since the dot product equals 0, the vectors are perpendicular. QED.
Tags: vectors, scalar product, perpendicular, 3D
**Item 8 [C]**
Question: Triangle with vertices A(1, 2), B(4, 6), C(7, 2). Find the angle at B.
Answer: ∠B ≈ 73.74°
Difficulty: C
Topic: 10.1
Explanation: BA = A − B = (−3, −4), BC = C − B = (3, −4). BA·BC = (−3)(3) + (−4)(−4) = −9 + 16 = 7. |BA| = √(9+16) = 5, |BC| = √(9+16) = 5. cos ∠ABC = 7/(5×5) = 7/25. So ∠B = arccos(7/25) ≈ 73.74°.
Tags: vectors, scalar product, angle, triangle
**Item 9 [C]**
Question: Prove that the diagonals of a parallelogram bisect each other.
Answer: Let vertices be A, B, C, D (in order). With AB = DC, position vectors satisfy c = a + (b−a) + (d−b) = b + d − a. Midpoint of AC = (a+c)/2 = (a + b + d − a)/2 = (b+d)/2. Midpoint of BD = (b+d)/2. These are equal; hence the diagonals bisect each other. QED.
Difficulty: C
Topic: 10.1
Explanation: In a parallelogram ABCD with A, B, C, D as vertices in order: AB = DC and AB ∥ DC. Let position vectors be a, b, c, d. Since AB = DC: b − a = d − c → c = b + d − a. Midpoint of diagonal AC: M = (a + c)/2 = (a + b + d − a)/2 = (b + d)/2. Midpoint of diagonal BD: N = (b + d)/2. Since M = N, the diagonals bisect each other. QED.
Tags: vectors, parallelogram, bisect, proof
