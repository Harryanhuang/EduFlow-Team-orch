**Item 1 [F]**
Question: Expand: (x + 3)(x − 2)
Answer: x² + x − 6
Difficulty: F
Topic: 1.2
Explanation: Multiply each term in the first bracket by each term in the second: x·x + x·(−2) + 3·x + 3·(−2) = x² − 2x + 3x − 6 = x² + x − 6.
Tags: expanding brackets, FOIL, algebra, quadratic expansion
**Item 2 [F]**
Question: Expand and simplify: (2x + 1)²
Answer: 4x² + 4x + 1
Difficulty: F
Topic: 1.2
Explanation: (2x+1)² = (2x+1)(2x+1) = 4x² + 2x + 2x + 1 = 4x² + 4x + 1. Using the identity (a+b)² = a²+2ab+b² with a=2x, b=1: (2x)² + 2(2x)(1) + 1² = 4x² + 4x + 1.
Tags: expanding brackets, square of a binomial, algebra identity

**Item 6 [S]**
Question: Simplify: (x² − 4) ÷ (x − 2)
Answer: x + 2, for x ≠ 2
Difficulty: S
Topic: 1.2
Explanation: Factor the numerator: x²−4 = (x+2)(x−2) (difference of squares). Then (x²−4)÷(x−2) = (x+2)(x−2)÷(x−2) = x+2, provided x ≠ 2 (since division by zero is undefined).

**Item 7 [C]**
Question: Expand (x + 2)³ using (a + b)³ = a³ + 3a²b + 3ab² + b³.
Answer: x³ + 6x² + 12x + 8
Difficulty: C
Topic: 1.2
Explanation: Let a = x, b = 2. (x+2)³ = x³ + 3(x²)(2) + 3(x)(2²) + 2³ = x³ + 6x² + 12x + 8. Verify by multiplying (x+2)(x+2)(x+2) step by step: (x+2)² = x²+4x+4, then (x²+4x+4)(x+2) = x³+2x²+4x²+8x+4x+8 = x³+6x²+12x+8 ✓.

**Item 8 [C]**
Question: Change the subject to r: V = πr²h
Answer: r = √(V/(πh))
Difficulty: C
Topic: 1.2
Explanation: V = πr²h. Divide by (πh): V/(πh) = r². Take square root: r = √(V/(πh)) or r = −√(V/(πh)). Since r represents a radius (length), we take the positive root: r = √(V/(πh)).

**Item 9 [C]**
Question: Prove: (a + b)² − (a − b)² = 4ab
Answer: LHS = (a²+2ab+b²) − (a²−2ab+b²) = a²+2ab+b²−a²+2ab−b² = 4ab = RHS ✓
Difficulty: C
Topic: 1.2
Explanation: Expand both squares using the identity (p±q)² = p²±2pq+q². For (a+b)² = a²+2ab+b². For (a−b)² = a²−2ab+b². Subtract: (a²+2ab+b²) − (a²−2ab+b²) = 4ab. The terms a² and b² cancel, leaving 4ab. QED.
Tags: algebraic proof, identity proof, expanding brackets, algebra
