**Item 4 [S]**
Question: Express 3 sin x − 4 cos x in the form R sin(x − α).
Answer: 5 sin(x − 53.13°)
Difficulty: S
Topic: 7.3
Explanation: R = √(1+3) = 2. Write sin x + √3 cos x = 2 sin(x + π/3). Set equal to 2: 2 sin(x + π/3) = 2 → sin(x + π/3) = 1. This gives x + π/3 = π/2 + 2πn, so x = π/2 − π/3 + 2πn = (3π−2π)/6 + 2πn = π/6 + 2πn. In [0, 2π]: x = π/6 and x = 13π/6. Verify: x = π/6: sin(π/6) + √3 cos(π/6) = 1/2 + √3(√3/2) = 1/2 + 3/2 = 2 ✓. x = 13π/6: sin(13π/6) + √3 cos(13π/6) = 1/2 + √3(√3/2) = 1/2 + 3/2 = 2 ✓.
Tags: R-formula, trigonometry, solving equations

**Item 8 [C]**
Question: Solve: sin x + √3 cos x = 2 for 0 ≤ x ≤ 2π.
Answer: x = π/6
Difficulty: C
Topic: 7.3
Explanation: sin x + √3 cos x = 2 → R = √(1² + (√3)²) = 2, so 2 sin(x + π/3) = 2 → sin(x + π/3) = 1. Then x + π/3 = π/2 → x = π/6. In the interval [0, 2π): x = π/6. Verification: sin(π/6) + √3 cos(π/6) = 1/2 + √3 × (√3/2) = 1/2 + 3/2 = 2 ✓.
Tags: R-formula, trigonometry, solving equations
**Item 9 [C]**
Question: Prove that the maximum value of a sin x + b cos x is √(a² + b²).
Answer: Write a sin x + b cos x = R sin(x + α) with R = √(a²+b²). Since sin(x+α) ≤ 1, max = R = √(a²+b²). QED.
Difficulty: C
Topic: 7.3
Explanation: Set R = √(a²+b²). Then a = R cos α, b = R sin α. So a sin x + b cos x = R(cos α sin x + sin α cos x) = R sin(x + α). Since −1 ≤ sin(θ) ≤ 1 for all θ, max = R. QED.
Tags: R-formula, maximum value, proof, trigonometry
