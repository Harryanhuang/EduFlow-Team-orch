# Topic 3.1 — Quadratic Equations

**Item 1 [F]**
Question: Solve by factorisation: x² − 5x + 6 = 0
Answer: x = 2 or x = 3
Difficulty: F
Topic: 3.1
Explanation: Find two numbers that multiply to +6 (c) and add to −5 (b): −2 and −3. Factor: (x − 2)(x − 3) = 0. So x = 2 or x = 3.
Tags: quadratic equations, factorisation, solving quadratics, roots

Answer: x = 2 or x = −2

**Item 3 [S]**
Question: Solve using the quadratic formula: x² − 3x − 4 = 0
Answer: x = 4 or x = −1
Difficulty: S
Topic: 3.1
Explanation: a = 1, b = −3, c = −4. Quadratic formula: x = [−(−3) ± √(9 − 4(1)(−4))] / 2 = [3 ± √25] / 2 = [3 ± 5] / 2 → x = 4 or x = −1.
Tags: quadratic equations, quadratic formula, discriminant, solving quadratics
**Item 4 [S]**
Question: Solve by completing the square: x² + 6x + 5 = 0
Answer: x = −1 or x = −5
Difficulty: S
Topic: 3.1
Explanation: x² + 6x = −5. Complete the square: x² + 6x + 9 = −5 + 9 → (x + 3)² = 4 → x + 3 = ±2 → x = −1 or x = −5.
Tags: quadratic equations, completing the square, solving quadratics
**Item 5 [S]**
Question: Solve: 2x² − 5x + 2 = 0. State the nature of the roots.
Answer: x = 1/2 or x = 2; two distinct real roots (Δ = 9 > 0)
Difficulty: S
Topic: 3.1
Explanation: a = 2, b = −5, c = 2. Discriminant: Δ = 25 − 16 = 9 > 0, so two distinct real roots. x = [5 ± 3] / 4 → x = 2 or x = 1/2.
Tags: quadratic equations, discriminant, nature of roots, quadratic formula
**Item 6 [S]**
Question: Find the sum and product of the roots of 3x² + 4x − 7 = 0 without solving.
Answer: sum = −4/3, product = −7/3
Difficulty: S
Topic: 3.1
Explanation: For ax² + bx + c = 0, sum of roots α + β = −b/a, product αβ = c/a. Here a = 3, b = 4, c = −7. So α + β = −4/3 and αβ = −7/3.
Tags: quadratic equations, Vieta's formulas, sum of roots, product of roots
**Item 7 [C]**
Question: Form the quadratic equation whose roots are 3 and −2.
Answer: x² − x − 6 = 0
Difficulty: C
Topic: 3.1
Explanation: If roots are α = 3 and β = −2, then sum α + β = 1 and product αβ = −6. For equation x² − (sum)x + (product) = 0: x² − x − 6 = 0. Check: (x − 3)(x + 2) = x² − x − 6 ✓.
Tags: quadratic equations, forming equations from roots, Vieta's formulas, reverse problem
**Item 8 [C]**
Question: Find the value of k for which x² + kx + 9 = 0 has equal roots.
Answer: k = 6 or k = −6
Difficulty: C
Topic: 3.1
Explanation: Equal roots require discriminant Δ = 0: k² − 36 = 0 → k² = 36 → k = ±6.
Tags: quadratic equations, equal roots, discriminant, parameter finding
**Item 9 [C]**
Question: The roots of x² − 3x + 2 = 0 are α and β. Given that α + 2β = 5, verify α = 2, β = 1 and state the value of p in x² + px + 2 = 0.
Answer: p = −3
Difficulty: C
Topic: 3.1
Explanation: The equation x² − 3x + 2 = 0 factors as (x−2)(x−1) = 0, so roots are α = 2, β = 1. Verify: α + 2β = 2 + 2(1) = 2 + 2 = 4 ≠ 5. Change the equation: for x² + px + 2 = 0 with roots α, β, we need αβ = 2 and α + 2β = 5. Try α = 1, β = 2: αβ = 2 ✓, α + 2β = 1 + 4 = 5 ✓. Then p = −(α+β) = −(1+2) = −3.
Tags: quadratic equations, Vieta's formulas, conditional roots, parameter solving
