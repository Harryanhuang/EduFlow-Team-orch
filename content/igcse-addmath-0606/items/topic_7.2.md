# Topic 7.2 — Trigonometric Equations and Double-Angle Formulas

**Item 1 [F]**
Question: Express sin 2A in terms of sin A and cos A.
Answer: sin 2A = 2 sin A cos A
Difficulty: F
Topic: 7.2
Explanation: Derived from the addition formula: sin 2A = sin(A+A) = sin A cos A + cos A sin A = 2 sin A cos A.
Tags: double-angle, trigonometry, identities
**Item 2 [F]**
Question: Given sin A = 3/5 (A acute), find sin 2A.
Answer: 24/25
Difficulty: F
Topic: 7.2
Explanation: cos A = √(1−sin²A) = √(1−9/25) = 4/5 (A acute, cos positive). sin 2A = 2 sin A cos A = 2 × (3/5) × (4/5) = 24/25.
Tags: double-angle, trigonometry, trigonometry values
**Item 3 [S]**
Question: Solve: sin 2x = sin x for 0° ≤ x ≤ 360°.
Answer: x = 0°, 60°, 180°, 300°, 360°
Difficulty: S
Topic: 7.2
Explanation: sin 2x = sin x → 2 sin x cos x = sin x → sin x(2 cos x − 1) = 0. Case 1: sin x = 0 → x = 0°, 180°, 360°. Case 2: 2 cos x − 1 = 0 → cos x = 1/2 → x = 60°, 300°. All solutions satisfy the original equation.
Tags: trigonometric equations, double-angle, solving equations

**Item 7 [C]**
Question: Solve: sin 2x = √3 cos x for 0° ≤ x ≤ 360°.
Answer: x = 60°, 90°, 120°, 270°
Difficulty: C
Topic: 7.2
Explanation: sin 2x = 2 sin x cos x, so 2 sin x cos x = √3 cos x → cos x(2 sin x − √3) = 0. Case 1: cos x = 0 → x = 90°, 270°. Case 2: 2 sin x = √3 → sin x = √3/2 → x = 60°, 120°. All four satisfy the equation.
Tags: trigonometric equations, double-angle, solving
**Item 8 [C]**
Question: Prove: (sin x + cos x)² = 1 + sin 2x.
Answer: (sin x + cos x)² = sin²x + 2 sin x cos x + cos²x = (sin²x+cos²x) + 2 sin x cos x = 1 + sin 2x ✓. QED.
Difficulty: C
Topic: 7.2
Explanation: Expand and apply Pythagorean and double-angle identities. QED.
Tags: double-angle, identity proof, trigonometry
**Item 9 [C]**
Question: Solve: cos 2x + 3 cos x + 2 = 0 for 0 ≤ x ≤ 2π.
Answer: x = π, 2π/3, 4π/3
Difficulty: C
Topic: 7.2
Explanation: cos 2x = 2 cos²x − 1: 2 cos²x − 1 + 3 cos x + 2 = 0 → 2 cos²x + 3 cos x + 1 = 0 → (2 cos x + 1)(cos x + 1) = 0. So cos x = −1/2 → x = 2π/3, 4π/3; or cos x = −1 → x = π. All verified.
Tags: trigonometric equations, double-angle, solving, quadratic
