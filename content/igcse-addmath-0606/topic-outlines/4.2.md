# Topic 4.2 — Absolute Value Equations and Inequalities

## Basic Info
- **Topic ID:** 4.2
- **Domain:** Inequalities and Equations
- **Core/Supplement:** Supplement
- **Weight in exam:** Medium (Paper 1 and Paper 2)

## Knowledge Points
1. Definition: |x| = distance of x from zero on the number line
2. |x| = a → x = a or x = −a (for a > 0)
3. |x| = |a| → x = a or x = −a
4. Graph of y = |x|: V-shaped, vertex at origin
5. Solving |ax + b| = c
6. Solving inequalities with absolute values: |x| < a, |x| > a
7. Geometrical interpretation of |x − a|

## Key Formulas
- |x| = x if x ≥ 0; |x| = −x if x < 0
- |x|² = x²
- |x − a| = distance between x and a on the number line

## Common Errors
- Forgetting the negative solution when solving |x| = a
- Incorrectly expanding |a + b| as |a| + |b|
- Forgetting domain restrictions when squaring both sides
- Mixing up < and > directions for inequalities

## Difficulty Band: F:2|S:4|C:3
- F: solve simple |x| = a, |ax + b| = c
- S: solve inequalities with absolute values, interpret geometrically
- C: compound absolute value equations, proving properties
