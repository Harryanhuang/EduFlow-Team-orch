# Topic 8.3 — Integration

## Basic Info
- **Topic ID:** 8.3
- **Domain:** Calculus
- **Core/Supplement:** Core
- **Weight in exam:** High (Paper 1 and Paper 2)

## Knowledge Points
1. Integration as the reverse of differentiation (antiderivative)
2. Indefinite integrals: ∫xⁿ dx = x^(n+1)/(n+1) + C for n ≠ −1
3. Integration of standard functions: polynomial, trigonometric, exponential
4. Definite integrals: evaluate between limits, area under curve
5. Fundamental theorem of calculus: ∫_a^b f(x)dx = F(b) − F(a)
6. Area under a curve (above x-axis)
7. Area between curve and x-axis (signed area)
8. Integration as the reverse of differentiation: F'(x) = f(x) → ∫f(x)dx = F(x) + C

## Key Formulas
- ∫xⁿ dx = x^(n+1)/(n+1) + C (n ≠ −1)
- ∫e^x dx = e^x + C
- ∫1/x dx = ln|x| + C
- ∫sin x dx = −cos x + C
- ∫cos x dx = sin x + C
- ∫_a^b f(x)dx = [F(x)]_a^b = F(b) − F(a)

## Common Errors
- Forgetting the constant of integration C in indefinite integrals
- Forgetting to evaluate between limits in definite integrals
- Incorrect signs when integrating trigonometric functions
- Confusing integration with differentiation

## Difficulty Band: F:2|S:4|C:3
- F: integrate polynomials, evaluate definite integrals
- S: area under curve, integration of trig/exponential functions
- C: area between curve and x-axis, volume of revolution (if applicable), integration techniques
