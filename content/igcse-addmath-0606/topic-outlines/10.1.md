# Topic 10.1 — Vector Notation, Magnitude, Direction, Addition, Scalar Product

## Basic Info
- **Topic ID:** 10.1
- **Domain:** Vectors
- **Core/Supplement:** Core
- **Weight in exam:** Medium (Paper 1 and Paper 2)

## Knowledge Points
1. Vector notation: →a, a, AB
2. Position vectors: from origin to a point
3. Magnitude: |a| = √(a₁² + a₂²) in 2D; √(a₁² + a₂² + a₃²) in 3D
4. Direction: unit vector a/|a|
5. Vector addition and subtraction: triangle law, parallelogram law
6. Scalar multiplication of vectors
7. Scalar (dot) product: a · b = |a||b| cos θ = a₁b₁ + a₂b₂
8. Angle between two vectors: cos θ = (a · b) / (|a||b|)
9. Perpendicular vectors: a · b = 0

## Key Formulas
- a · b = a₁b₁ + a₂b₂ = |a||b| cos θ
- |a| = √(a₁² + a₂²)
- cos θ = (a · b) / (|a||b|)
- a ⟂ b ⟺ a · b = 0

## Common Errors
- Confusing scalar product (dot product) with vector product
- Forgetting to use the angle between vectors in dot product
- Mixing up the order of subtraction in position vectors
- Not checking perpendicular condition before claiming

## Difficulty Band: F:2|S:4|C:3
- F: find magnitude, add/subtract vectors, scalar multiplication
- S: scalar product, angle between vectors, unit vectors
- C: proving perpendicular/parallel, complex vector problems
