# Topic 7.3 — Solving Trigonometric Equations in Real Numbers

## Basic Info
- **Topic ID:** 7.3
- **Domain:** Trigonometry
- **Core/Supplement:** Core
- **Weight in exam:** Medium-High (Paper 1 and Paper 2)

## Knowledge Points
1. General solution of sin x = a: x = (−1)^k arcsin(a) + kπ or x = nπ + (−1)^n arcsin(a)
2. General solution of cos x = a: x = 2nπ ± arccos(a)
3. General solution of tan x = a: x = nπ + arctan(a)
4. Solving equations with multiple angles
5. Using R-formula: a sin x + b cos x = R sin(x ± α) or R cos(x ± α)
6. Maximum and minimum values of a sin x + b cos x
7. Solving equations that require the R-formula

## Key Formulas
- R = √(a² + b²)
- R sin(x + α): a = R cos α, b = R sin α
- R cos(x − α): a = R cos α, b = R sin α (for a sin x + b cos x = R cos(x − α))

## Common Errors
- Forgetting the ± in cos general solution
- Incorrectly finding the R-formula parameters
- Losing solutions when transforming equations
- Not restricting to valid ranges for inverse trig

## Difficulty Band: F:2|S:4|C:3
- F: solve basic sin/cos/tan equations, recall general solution patterns
- S: solve equations using R-formula, find max/min values
- C: general solutions in radians, complex trigonometric equations
