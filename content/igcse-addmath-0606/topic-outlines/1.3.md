# Topic 1.3 — Functions: Domain, Range, Inverse, Composite

## Basic Info
- **Topic ID:** 1.3
- **Domain:** Algebra
- **Core/Supplement:** Core
- **Weight in exam:** Medium-High (both Paper 1 and Paper 2)

## Knowledge Points
1. Function notation: f(x), domain, range
2. Types of functions: linear, quadratic, cubic, reciprocal, exponential, logarithmic
3. Composite functions: fg(x) = f(g(x))
4. Inverse functions: f⁻¹(x), conditions for inverse to exist (one-to-one)
5. Finding inverse functions algebraically
6. Graphical relationship between f(x) and f⁻¹(x) (reflection in y=x)
7. Domain restrictions for inverse functions

## Key Formulas
- Composite: if f(x) = ax+b, g(x) = cx+d, then fg(x) = a(cx+d)+b
- Inverse linear: f⁻¹(x) = (x−b)/a (for f(x) = ax+b, a≠0)
- For f to have an inverse, it must be one-to-one (passes horizontal line test)

## Common Errors
- Confusing f(g(x)) with g(f(x)) — order matters
- Forgetting domain restrictions when finding inverses
- Incorrectly simplifying composite function expressions
- Not checking that f is one-to-one before claiming an inverse exists

## Difficulty Band: F:2|S:4|C:3
- F: evaluate f(x) for given x, identify domain/range for simple functions
- S: find inverse of linear function, form composite functions, find domain of composite
- C: inverse of quadratic (with domain restriction), composite of non-linear functions, prove functions are inverses
