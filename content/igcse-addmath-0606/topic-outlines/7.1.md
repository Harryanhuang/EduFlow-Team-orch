# Topic 7.1 — Trigonometric Functions and Identities

## Basic Info
- **Topic ID:** 7.1
- **Domain:** Trigonometry
- **Core/Supplement:** Core
- **Weight in exam:** Medium-High (Paper 1 & 2)

## Knowledge Points
1. Radian measure: conversion between degrees and radians
2. The six trigonometric functions: sin, cos, tan, cosec, sec, cot
3. Pythagorean identity: sin²θ + cos²θ = 1
4. Reciprocal identities: tan θ = sin θ / cos θ, etc.
5. Graphs of sin, cos, tan (amplitude, period, phase shift)
6. Values of trig functions at special angles (0°, 30°, 45°, 60°, 90°)
7. Using identities to simplify expressions

## Key Formulas
- sin²θ + cos²θ = 1
- 1 + tan²θ = sec²θ
- 1 + cot²θ = cosec²θ
- Degrees ↔ Radians: θ_rad = θ_deg × π/180

## Common Errors
- Mixing up radians and degrees in the same problem
- Forgetting the ± sign when using inverse trig
- Incorrectly applying Pythagorean identities
- Not restricting domain when finding inverse trig values

## Difficulty Band: Medium-Hard (F:2|S:4|C:3)
- F: evaluate trig at special angles, basic identity checks
- S: use identities to simplify expressions, radian conversion
- C: prove identities, graph transformations, multiple-angle reasoning
