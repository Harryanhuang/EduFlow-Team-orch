# Topic 4.1 — Linear and Quadratic Inequalities

## Basic Info
- **Topic ID:** 4.1
- **Domain:** Inequalities and Equations
- **Core/Supplement:** Core
- **Weight in exam:** Medium (Paper 1 and Paper 2)

## Knowledge Points
1. Solving linear inequalities: rules for reversing inequality sign
2. Representing solutions on number lines
3. Solving quadratic inequalities: critical values, sign analysis
4. Regions: linear inequalities in two variables (shading regions)
5. Combined inequalities: solving simultaneous inequalities
6. Finding solution sets for inequalities involving fractions

## Key Formulas / Methods
- When multiplying or dividing by a negative number, reverse the inequality sign
- For quadratic inequalities: find roots, sketch parabola, determine sign
- "Critical values" method: solve equality, test intervals
- Combined: a < x < b means x > a AND x < b

## Common Errors
- Forgetting to reverse the sign when multiplying/dividing by negative
- Confusing AND/OR when writing solution sets
- For quadratic inequalities, drawing the wrong parabola shape
- Not testing all intervals when using sign analysis
- Confusing open circles (strict inequality) with closed circles (= included)

## Difficulty Band: F:2|S:4|C:3
- F: solve linear inequalities, represent on number line
- S: solve quadratic inequalities by sign analysis, simultaneous linear inequalities
- C: inequalities with absolute values, region problems, prove inequality results
