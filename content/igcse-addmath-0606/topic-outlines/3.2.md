# Topic 3.2 — Polynomial Operations and the Remainder/Factor Theorem

## Basic Info
- **Topic ID:** 3.2
- **Domain:** Quadratics and Polynomials
- **Core/Supplement:** Core
- **Weight in exam:** Medium (Paper 1 and Paper 2)

## Knowledge Points
1. Polynomial operations: addition, subtraction, multiplication, division
2. Polynomial long division
3. The Remainder Theorem: when polynomial f(x) is divided by (x−a), remainder = f(a)
4. The Factor Theorem: (x−a) is a factor of f(x) iff f(a) = 0
5. Finding unknown coefficients using the Factor/Remainder Theorem
6. Factorising polynomials completely using the Factor Theorem
7. Cubic polynomials: finding all factors given one factor

## Key Formulas
- Remainder Theorem: remainder when f(x) ÷ (x−a) is f(a)
- Factor Theorem: (x−a) is a factor ⟺ f(a) = 0
- Polynomial division: dividend = divisor × quotient + remainder
- For cubic f(x) = (x−a)(x−b)(x−c) + remainder

## Common Errors
- Forgetting to substitute the correct value when using Remainder Theorem
- Not setting the divisor equal to zero when using Factor Theorem (e.g., saying f(2)=0 for factor (x−2))
- Sign errors in polynomial long division
- Not checking all possible factors when factorising completely

## Difficulty Band: F:2|S:4|C:3
- F: evaluate polynomial at a value, identify factors given a root
- S: use Remainder Theorem to find remainders, factorise given one factor
- C: find unknown coefficients, factorise completely, polynomial division problems
