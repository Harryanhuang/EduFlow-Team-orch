# Topic 3.1 — Quadratic Equations
## QA Question Level File

**1. [F]** Solve by factorisation: x² − 5x + 6 = 0
**2. [F]** Solve by factorisation: x² − 4 = 0
**3. [S]** Solve using the quadratic formula: x² − 3x − 4 = 0
**4. [S]** Solve by completing the square: x² + 6x + 5 = 0
**5. [S]** Solve: 2x² − 5x + 2 = 0. State the nature of the roots.
**6. [S]** Find the sum and product of the roots of 3x² + 4x − 7 = 0 without solving.
**7. [C]** Form the quadratic equation whose roots are 3 and −2.
**8. [C]** Find the value of k for which x² + kx + 9 = 0 has equal roots.
**9. [C]** The roots of x² + px + 2 = 0 are α and β. Given that α + 2β = 5, find p.
