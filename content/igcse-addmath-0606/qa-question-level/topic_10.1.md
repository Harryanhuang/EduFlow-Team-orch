# Topic 10.1 — Vector Notation, Magnitude, Direction, Addition, Scalar Product
## QA Question Level File

**1. [F]** Find the magnitude of the vector (3, 4).

**2. [F]** If a = (1, 2) and b = (3, −1), find a + b.

**3. [S]** Find the angle between a = (1, 1) and b = (1, 0).

**4. [S]** Find the unit vector in the direction of (4, 3).

**5. [S]** If a · b = 6, |a| = 3, |b| = 5, find the angle between a and b.

**6. [S]** Find the value of k such that (2, k) is perpendicular to (3, −6).

**7. [C]** Show that the vectors (1, 2, 2) and (2, 2, −1) are perpendicular.

**8. [C]** In a triangle with vertices A(1, 2), B(4, 6), C(7, 2), find the angle at B.

**9. [C]** Prove that the diagonal of a parallelogram bisect each other.
