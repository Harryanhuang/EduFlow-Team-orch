# Topic 1.3 — Functions: Domain, Range, Inverse, Composite
## QA Question Level File

**1. [F]** If f(x) = 3x − 2, find f(4).
**2. [F]** State the domain and range of f(x) = 2x + 1.
**3. [S]** If f(x) = x² − 4 and g(x) = x + 1, find (a) f(g(x)) (b) g(f(x)).
**4. [S]** Find the inverse function f⁻¹(x) for f(x) = (3x − 1)/2.
**5. [S]** The function f is defined as f(x) = x² for x ≥ 0. Find f⁻¹(x) and state its domain.
**6. [S]** If f(x) = 2x + 3 and g(x) = x², find fg(2) and gf(2).
**7. [C]** The function f is one-to-one. Given that f⁻¹(5) = 3, find the value of f(3).
**8. [C]** If f(x) = eˣ and g(x) = ln x, show that fg(x) = x and gf(x) = x. What can you conclude about f and g?
**9. [C]** A function is defined as f(x) = (2x − 3)/(x + 1) for x ≠ −1. Find the inverse function f⁻¹(x) and state the domain and range of both f and f⁻¹.
