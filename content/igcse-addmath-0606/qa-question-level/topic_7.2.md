# Topic 7.2 — Trigonometric Equations and Double-Angle Formulas
## QA Question Level File

**1. [F]** Express sin 2A in terms of sin A and cos A.
**2. [F]** Given sin A = 3/5 (A acute), find sin 2A.
**3. [S]** Solve: sin 2x = sin x for 0° ≤ x ≤ 360°.
**4. [S]** Solve: cos 2x = 1/2 for 0° ≤ x ≤ 180°.
**5. [S]** Given cos θ = 4/5, find cos 2θ.
**6. [S]** Solve: tan 2x = 1 for 0° ≤ x ≤ 180°.
**7. [C]** Solve: sin 2x = √3 cos x for 0° ≤ x ≤ 360°.
**8. [C]** Prove: (sin x + cos x)² = 1 + sin 2x.
**9. [C]** Solve: cos 2x + 3 cos x + 2 = 0 for 0 ≤ x ≤ 2π.
