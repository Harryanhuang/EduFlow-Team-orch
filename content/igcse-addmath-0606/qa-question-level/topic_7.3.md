# Topic 7.3 — Solving Trigonometric Equations in Real Numbers
## QA Question Level File

**1. [F]** Find the general solution of sin x = 1/2.
**2. [F]** Find the general solution of tan x = 1.
**3. [S]** Solve: 2 sin x = √3 for 0° ≤ x ≤ 360°.
**4. [S]** Express 3 sin x − 4 cos x in the form R sin(x − α).
**5. [S]** Solve: sin x = cos x for 0° ≤ x ≤ 360°.
**6. [S]** Find the maximum value of 5 cos x + 12 sin x.
**7. [C]** Solve: 3 sin x + 4 cos x = 3 for 0° ≤ x ≤ 360°.
**8. [C]** Solve: sin x + √3 cos x = 1 for 0 ≤ x ≤ 2π.
**9. [C]** Prove that the maximum value of a sin x + b cos x is √(a² + b²).
