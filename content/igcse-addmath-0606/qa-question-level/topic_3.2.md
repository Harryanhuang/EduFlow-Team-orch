# Topic 3.2 — Polynomial Operations and the Remainder/Factor Theorem
## QA Question Level File

**1. [F]** Given f(x) = x³ − 2x² + 3x − 1, find f(1).
**2. [F]** If (x − 2) is a factor of x² + kx − 10, find k.
**3. [S]** Using the Remainder Theorem, find the remainder when x³ + 4x² − 3x + 2 is divided by (x − 1).
**4. [S]** Factorise completely: x³ − 3x² − 4x + 12, given that x = 2 is a root.
**5. [S]** If f(x) = 2x³ − 5x² + ax + 6 and f(2) = 4, find a.
**6. [S]** Divide x³ + 3x² − x − 3 by (x + 3).
**7. [C]** Find the value of a and b if (x − 1) and (x + 2) are both factors of x³ + ax² + bx − 6.
**8. [C]** It is given that x³ + px² + qx + 6 is divisible by (x − 2) and (x + 1). Find p and q.
**9. [C]** Find all the factors of x³ + x² − 10x + 8 and hence solve x³ + x² − 10x + 8 = 0.
