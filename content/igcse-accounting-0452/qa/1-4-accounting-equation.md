# QA: 1.4 — The accounting equation

## Topic 名称
The accounting equation（会计方程式）

## Topic 定义
理解并应用会计方程式：资产 = 负债 + 资本（Assets = Liabilities + Capital），分析交易对会计方程式各项目的影响。

## 关键知识点
- 会计方程式：A = L + C
- 方程式的变形：C = A - L（资本 = 资产 - 负债）
- 每笔交易都保持方程式平衡
- 交易对方程式的影响类型：
  - 一项资产增加，另一项资产减少
  - 一项资产增加，一项负债增加
  - 一项资产增加，资本增加
  - 一项资产减少，一项负债减少
  - 一项资产减少，资本减少
- 用方程式计算未知数
- 期初和期末资本的变化分析

## 前置知识
- 1.3 资产、负债和资本

## 常见错误
- 方程式写错（A = L - C 是错的）
- 交易分析时忘记保持方程式平衡
- 利润增加资本但忘记利润本身不是方程式的直接项
- 计算资本时忘记加利润减提款
- 多项交易同时发生时遗漏某些影响

## 可出题方向
- 已知两项求第三项
- 分析交易对方程式各项目的影响
- 计算期末资本
- 多项交易的综合影响分析
- 方程式的变形应用
