# Question Q-5.6-07

**Difficulty**: Challenge

**Question**: Prepare a company's income statement (from revenue to profit after tax) and appropriation account: Revenue $500,000; cost of sales $280,000; administrative expenses $95,000 (including directors' remuneration $35,000); 5% debentures $240,000; corporation tax 25% on profit before tax. The company proposes an ordinary dividend of $15,000 and has 8% preference shares of $100,000. Show all workings.

**Answer**:

**Income Statement:**

| | $ | $ |
|---|---|---|
| Revenue | | 500,000 |
| Cost of sales | | (280,000) |
| **Gross profit** | | **220,000** |
| Administrative expenses | | (95,000) |
| Debenture interest (5% × 240,000) | | (12,000) |
| **Profit before tax** | | **113,000** |
| Corporation tax (25% × 113,000) | | (28,250) |
| **Profit after tax** | | **84,750** |

**Appropriation Account:**

| | $ |
|---|---|
| Profit after tax | 84,750 |
| Less: Preference dividend (8% × 100,000) | (8,000) |
| Less: Ordinary dividend proposed | (15,000) |
| **Retained profit for the year** | **61,750** |

**Explanation**: Directors' remuneration is already included in administrative expenses — do not deduct it again. Debenture interest is a finance cost in the income statement, NOT an appropriation. Corporation tax is calculated on profit before tax. Preference dividends are paid before ordinary dividends. A common error is placing debenture interest in the appropriation account — it is an expense.
