# Question Q-5.5-09

**Difficulty**: Challenge

**Question**: Prepare a full appropriation account from the following scenario: Alex and Ben's partnership has net profit of $85,000 for the year. Their agreement specifies: capital balances — Alex $60,000, Ben $40,000; interest on capital at 5% p.a.; Alex receives a salary of $10,000 p.a.; interest on drawings — Alex $1,200, Ben $800; residual profit shared 3:2. Show the appropriation account and calculate each partner's total appropriation.

**Answer**:

**Appropriation Account for the year ended:**

| | $ | $ |
|---|---|---|
| Net profit | | 85,000 |
| Add: Interest on drawings — Alex | 1,200 | |
| Interest on drawings — Ben | 800 | 2,000 |
| **Total available for appropriation** | | **87,000** |
| Less: Interest on capital — Alex (5% × 60,000) | 3,000 | |
| Interest on capital — Ben (5% × 40,000) | 2,000 | |
| Less: Partner's salary — Alex | 10,000 | |
| **Total appropriations** | | **(15,000)** |
| **Residual profit** | | **72,000** |
| Shared 3:2 — Alex (3/5 × 72,000) | | 43,200 |
| Shared 3:2 — Ben (2/5 × 72,000) | | 28,800 |

**Each partner's total appropriation:**

| | Interest on capital | Salary | Residual share | Total |
|---|---|---|---|---|
| Alex | $3,000 | $10,000 | $43,200 | **$56,200** |
| Ben | $2,000 | $0 | $28,800 | **$30,800** |
| **Total** | **$5,000** | **$10,000** | **$72,000** | **$87,000** |

Verification: Total appropriations ($56,200 + $30,800 = $87,000) match the total available for appropriation.

**Explanation**: Always verify that total appropriations equal net profit plus interest on drawings. A common error is deducting interest on drawings instead of adding it back, or miscalculating the residual profit. Work systematically: start with net profit, add interest on drawings, deduct interest on capital and salaries, then share the remainder in the agreed ratio.
