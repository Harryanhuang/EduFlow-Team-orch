# Question Q-7.2-07

**Difficulty**: Challenge

**Question**: A business has profit before items of $100,000. It spent $20,000 on a new machine (capital) but incorrectly expensed it. It spent $5,000 on building repairs (revenue) but incorrectly capitalized it. Calculate correct profit, incorrect profit, and the combined balance sheet effect.

**Answer**:

**Correct treatment:**
- Profit before items: $100,000
- Machine $20,000 → capital (not expensed)
- Repairs $5,000 → revenue expense (deducted)
- Correct profit = $100,000 - $5,000 = $95,000

**Incorrect treatment:**
- Machine $20,000 → incorrectly expensed
- Repairs $5,000 → incorrectly capitalized (not expensed)
- Incorrect profit = $100,000 - $20,000 = $80,000

**Effect:**
- Profit understated by $15,000 ($95,000 vs $80,000)
- Non-current assets: machine ($20,000) missing, repairs ($5,000) wrongly added. Net: assets understated by $15,000.
- Retained earnings: understated by $15,000.

The two errors reinforce each other — both reduce profit and assets. If they had been in opposite directions, the errors would partially offset.

**Explanation**: Always analyze each error separately before combining. A common error is treating the two errors as offsetting when they actually reinforce each other.
