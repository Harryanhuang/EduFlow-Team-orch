### Question Q-3.1-07
**Difficulty**: Standard
**Question**: A bookkeeper purchases a non-current asset (a machine for $5,000 cash) but records it as: Debit Purchases $5,000, Credit Cash $5,000. Will the trial balance detect this error? Explain.
**Answer**: No. The trial balance will still agree because debits ($5,000) equal credits ($5,000). This is an error of principle — the machine (a non-current asset) should have been debited to the Machinery account, not the Purchases account (which is for goods/inventory). The debit/credit sides are correct, but the wrong account type was used. The trial balance cannot detect this.
**Explanation**: The trial balance only checks that debits equal credits, not whether the right accounts were chosen. A common error is saying "yes it will be detected because Purchases is wrong" — the trial balance has no way of knowing which accounts should have been used.
