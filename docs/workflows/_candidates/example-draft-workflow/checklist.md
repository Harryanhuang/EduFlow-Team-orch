# checklist: example-draft-workflow

## Candidate Intake

- [ ] Source evidence or gap note exists.
- [ ] Candidate status is `draft`.
- [ ] Owner is `worker_builder`.
- [ ] manager closeout is explicit.
- [ ] Builder followup is explicit.
- [ ] Forbidden moves are explicit.
- [ ] Reassurance boundary is explicit.

## Block Promotion If

- [ ] Candidate duplicates an active workflow.
- [ ] Candidate lacks manager closeout.
- [ ] Candidate lacks builder followup.
- [ ] Candidate treats worker reassurance as formal manager result.
- [ ] Candidate has not passed strict validation.
