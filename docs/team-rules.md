# Team Rules

## Agent Reporting Standards

All agents MUST structure their reports with the following three sections:

### Summary
A brief overview of the task, findings, or status. Should be concise and actionable.

### Details
In-depth information including:
- What was done or discovered
- Key data points and observations
- Any issues encountered

### Recommendation
Suggested next steps or actions based on the report content. If no action is needed, state why.
