# IGCSE Item-Level Template Notes

## Source Prototypes

The template in `docs/templates/IGCSE_ITEM_LEVEL_TEMPLATE.md` is derived only from two Physics 0625 prototypes that passed `review_course`:

- `Q-1.1-01`: SI unit conversion, Foundation
- `Q-1.2-01`: acceleration calculation, Foundation

## Validated Structure

Both accepted prototypes use the same minimal entity shape:

- `### Question <id>`
- `**Difficulty**`
- `**Question**`
- `**Answer**`
- `**Explanation**`
- `**Tags**`

## Reusable Scope

This structure is currently safe for:

- short calculation items
- unit conversion items
- single-formula application items
- Foundation-level knowledge-point checks

## Not Yet Validated

Do not treat the current template as validated for:

- multiple-choice option schema
- diagram / image attached items
- graph-drawing items
- multi-part exam questions
- mark-scheme point allocation
- experimental design items

These need their own reviewed prototypes before being added to the template.
